# 第二十九章：多方势力关注

木叶村，黄昏。

李武正在训练场练习新技能。

【技能训练：念气代码·Lv.1】

金色的代码符号在李武周围浮现。

他正在分析各种术式的逻辑。

"火遁·豪火球之术..."李武喃喃自语。

"核心逻辑：将查克拉转化为火属性。"

"转化公式：C×火=火焰。"

"其中C是查克拉...火是性质变化。"

"而念气..."他手心凝聚金色念气。

"念气不需要性质变化。"

"它本身就是...纯粹的能量。"

"所以..."

金色的念气形成代码：

```
// 念气vs查克拉
念气 = 纯粹能量
查克拉 = 混合能量
念气 > 查克拉 // 优先判定
```

"原来如此..."李武若有所思。

"念气的优先级...比查克拉高。"

"这就是为什么...念气技能能压制忍术。"

...

【感知警报：多方监视】

李武的感知系统突然警报！

【感知警报：监视者增加】
【监视来源：7个】
【威胁等级分布】
- 根部（团藏）：紫色（上忍级）
- 岩隐使者：紫色（上忍级）
- 云隐使者：紫色（上忍级）
- 雾隐使者：蓝色（精英级）
- 砂隐使者：蓝色（精英级）
- 晓组织：紫色（上忍级）
- 大蛇丸：粉色偏橙（影级）

"这么多人..."李武若有所思。

系统界面弹出：

【警告：你已引起多方关注】
【建议：谨慎行事】

"我知道。"李武平静道。

他继续训练。

...

岩隐大使馆。

两名岩隐上忍在讨论。

"李武的情报..."一人道。

"收集得怎么样了？"

"不少。"另一人道。

"他的力量...叫'念气'。"

"不是查克拉。"

"这种力量...很危险。"

"要不要...干掉他？"

"不。"第一人摇头。

"现在动手...会引起木叶的愤怒。"

"而且...他在木叶内部。"

"暗杀成功率...很低。"

"那..."

"继续监视。"第一人道。

"等他离开木叶...再动手。"

"明白。"

...

云隐大使馆。

云隐使者们在讨论。

"李武这个少年..."

"很有意思。"

"要不要...招募他？"

"不可能。"另一人摇头。

"他对木叶很忠诚。"

"那..."

"先评估他的威胁等级。"

"如果威胁太大...就除掉。"

"如果威胁可控...就拉拢。"

"明白了。"

...

根组织总部。

团藏坐在黑暗中，独眼在阴影中闪烁着冷光。

"李武..."他喃喃自语。

"这个变数..."

"必须处理。"

"根"成员汇报：

"团藏大人...李武的实力...还在提升。"

"现在的等级...Lv.32。"

"已经是上忍级别了。"

"而且...他的力量...不是查克拉。"

团藏眯起独眼：

"念气..."

"这种力量...从未见过。"

"也许...是血继限界的一种？"

"不。"根成员摇头。

"血继限界...是查克拉的性质变化。"

"但念气...不是查克拉。"

"那是...另一种能量。"

团藏沉默了。

另一种能量...

这意味着...

李武可能掌握了...某种古老的力量。

团藏的手指在桌上轻轻敲击。作为"根"的首领，他见过太多威胁木叶的存在。九尾、宇智波一族、晓组织...每一个都需要被控制，或者被铲除。

但李武不同。

这个少年既不是叛忍，也不是外来者。他是木叶的忍者，拥有保护村子的决心，却掌握着无法理解的力量。

"团藏大人..."根成员犹豫了一下，"我们需要评估李武对木叶的忠诚度。如果他能为我们所用..."

"不。"团藏打断了他，声音冰冷。

"李武的存在本身，就是对根的威胁。他感知了你们的潜入，他挑战了我们的权威。"

"而且..."团藏的眼神中闪过一丝复杂，"他太像那个人了。"

"像谁？"根成员疑惑。

团藏没有回答。

"继续监视。"团藏道。

"而且...要小心。"

"这个少年...不简单。"

"必要时...我会亲自出手。"

...

晓组织基地。

佩恩（长门）和小南在讨论。

"那个李武..."小南道。

"很特别。"佩恩（长门）认真道。

"他的力量...不是查克拉。"

"而且...他的感知...能发现我们的存在。"

"要除掉他吗？"小南问。

"不。"佩恩摇头。

"现在...还不是时候。"

"而且..."

他看向窗外。

"也许...可以利用他。"

"利用？"

"对。"佩恩点头。

"如果李武真的那么强..."

"也许...可以帮我们收集尾兽。"

"你的意思是..."

"让李武...成为我们的棋子。"佩恩道。

"不过...需要观察。"

"看看他...到底有多强。"

...

死亡森林深处。

大蛇丸在疗伤。

"可恶..."他咬牙。

"李武...竟然伤了我..."

"而且...我的双手..."

他看着失去知觉的双手。

"尸鬼封尽..."大蛇丸咬牙。

"三代...这笔账...我记住了。"

"不过..."他冷笑。

"李武..."

"你确实很强。"

"但..."

"总有一天...你会成为我的容器。"

"等着吧。"

...

木叶村。

李武结束了训练。

"各方势力都在关注我..."他若有所思。

"不过..."

系统界面弹出：

【当前实力：Lv.32】
【足以应对：普通上忍】
【难以应对：影级、晓组织、大蛇丸】

"32级还远远不够..."李武判断。

"需要...更快地提升。"

他看向系统商城：

【可购买道具】
- 经验药水（小）：+1000经验，10万两
- 经验药水（中）：+5000经验，50万两
- 经验药水（大）：+20000经验，200万两

"太贵了..."李武皱眉。

"看来...还是要靠实战。"

他闭上眼睛。

未来的路...

会更加危险。

但...

他已经准备好了。
