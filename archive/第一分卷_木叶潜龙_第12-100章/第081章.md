# 第八十一章：成长与历练

木叶村，第二天。

第一卷结束了，第二卷即将开始。

李武继续修行，他需要提升实力到Lv.80，才能达到第二卷的目标。

"Lv.80..."李武喃喃自话。

"从Lv.55到Lv.80，还有很长的路。"

"不过，我有信心。"

李武念气爆发，冲击Lv.56！

【等级提升】
【当前等级：Lv.56】
【经验值：0 / 160000】

"56级了..."李武满意。

"继续努力。"

...

几天后，李武继续修行。

他念气爆发，冲击Lv.57！

【等级提升】
【当前等级：Lv.57】
【经验值：0 / 170000】

"57级了..."李武满意。

"继续努力。"

...

几天后，纲手召集所有影级强者。

"晓组织很快就会再次对木叶出手，"纲手严肃道，"我们需要做好万全准备。"

"李武，你负责训练木叶的忍者，提升他们的实力。"

"明白。"李武点头。

"我会训练木叶的忍者，提升他们的实力。"

"很好。"纲手道。

李武开始训练木叶的忍者。

鸣人、佐助、小樱、雏田、鹿丸、丁次、井野、志乃、牙、天天、小李、宁次...

所有木叶的年轻忍者，都在李武的训练下，提升实力。

"鸣人，你的念气已经达到高级水平。"李武道。

"高级水平..."鸣人惊喜，"太好了！"

"我还能继续提升吗？"

"当然。"李武道，"念气是没有极限的。"

"好！"鸣人兴奋。

"我会继续努力的！"

李武看向其他忍者："你们也都很好。"

"只要你们努力，就能达到更高的境界。"

"明白。"众忍者齐声。

"我们会继续努力的！"

李武微笑："很好。"

"木叶的年轻忍者...会越来越强。"

...

几天后，李武继续修行。

他念气爆发，冲击Lv.58！

【等级提升】
【当前等级：Lv.58】
【经验值：0 / 180000】

"58级了..."李武满意。

"继续努力。"

...

几天后，李武继续修行。

他念气爆发，冲击Lv.59！

【等级提升】
【当前等级：Lv.59】
【经验值：0 / 190000】

"59级了..."李武满意。

"继续努力。"

...

几天后，纲手再次召集所有影级强者。

"晓组织的行动越来越频繁，"纲手严肃道，"我们需要提高警惕。"

"李武，你负责侦查晓组织的行动。"

"明白。"李武点头。

"我会侦查晓组织的行动，保护木叶。"

"很好。"纲手道。

李武开始侦查晓组织的行动。

他来到木叶村外，感知周围的查克拉。

【感知警报：检测到晓组织成员】
【距离：5000米】
【威胁等级：红色（影级）】
【敌人：晓组织，两人】

"晓组织..."李武心中一惊。

"晓组织又来了。"

他深吸一口气，准备迎战。

李武向晓组织的方向前进，很快看到了两个晓组织成员。

他们是迪达拉和蝎。

"你就是李武？"迪达拉问。

"是。"李武冷静道。

"你们是谁？"

"晓组织，"迪达拉道，"我叫迪达拉，他叫蝎。"

"迪达拉...蝎..."李武思考。

"迪达拉使用粘土炸弹，蝎使用傀儡。"

"两个都是影级强者。"

"你们想干什么？"李武问。

"来杀你。"迪达拉道，"晓组织需要你的力量。"

"我的力量..."李武冷笑。

"你们真的以为能杀了我吗？"

"试试看。"迪达拉笑了，"艺术就是爆炸。"

"艺术..."李武道，"那就试试看。"

迪达拉扔出粘土炸弹！

【起爆粘土·炸弹】

轰——！！！

粘土炸弹爆炸！

李武念气爆发，挡住爆炸！

【念气护盾】

当——！！！

爆炸被挡住！

"什么？"迪达拉惊讶。

"念气护盾..."他喃喃自话。

"还没完。"李武道。

他深吸一口气，念气再次爆发！

【寸拳】

轰——！！！

李武的拳头击中迪达拉！

迪达拉被击飞！

"可恶..."迪达拉咬牙。

"艺术...居然被打断了？"

"还没完。"李武道。

就在这时，蝎发动攻击！

【傀儡·绯流琥】

蝎的傀儡绯流琥冲向李武！

【傀儡·毒针】

绯流琥发射毒针！

李武念气爆发！

【念气波】

轰——！！！

毒针被挡住！

"什么？"蝎震惊。

"念气波...居然能挡住毒针？"

"你的忍术，毫无威胁。"李武道。

"可恶..."迪达拉怒。

"一起上！"

迪达拉和蝎同时发动攻击！

【起爆粘土·炸弹】+【傀儡·绯流琥】

两人同时攻击李武！

李武不退反进，冲向两人！

【念气爆发】

轰——！！！

迪达拉和蝎都被击飞！

"可恶..."迪达拉咬牙。

"木叶的忍者...居然这么强..."

"认输吧。"李武道，"你们不是我的对手。"

迪达拉和蝎沉默了片刻，然后道："我们撤退。"

"很好。"李武道。

"记住，"李武道，"晓组织如果再敢对木叶出手，我会亲手消灭你们。"

"是..."迪达拉和蝎低头。

两人转身，向木叶村外离开。

李武看着他们离开，心中是思考。

"晓组织..."他喃喃自话。

"晓组织的目标是捕捉尾兽，而鸣人拥有九尾。"

"晓组织...很快就会全面袭击木叶。"

"我需要做好万全准备。"

...

晚上，李武的房间。

系统界面弹出：

【支线任务完成：击败晓组织成员】
【任务进度：100%】
【奖励：经验值+10000，技能点+10，声望+3000】

【当前等级：Lv.60】
【经验值：0 / 200000】

"60级了..."李武满意。

"又升级了。"

"声望...3000点。"

"木叶的声望，又提升了。"

系统界面弹出：

【主线任务更新】
【当前阶段：继续成长】
【任务目标：继续修行，提升实力到Lv.80】
【下一阶段：觉醒武神】

"继续成长..."李武思考。

"我需要继续提升实力。"

"然后...觉醒武神。"

李武看向窗外，木叶的夜景很美。

"木叶...我的家。"

"我会继续努力的。"
