# 第八十五章：继续成长

木叶村，第二天。

李武继续修行，他需要提升实力到Lv.100。

"Lv.100..."李武喃喃自话。

"从Lv.80到Lv.100，还有20级。"

"不过，我有信心。"

李武念气爆发，冲击Lv.81！

【等级提升】
【当前等级：Lv.81】
【经验值：0 / 410000】

"81级了..."李武满意。

"继续努力。"

...

几天后，纲手召集所有影级强者。

"晓组织撤退了，"纲手道，"但他们会回来的。"

"我们需要做好万全准备。"

"李武，你负责训练木叶的忍者，提升他们的实力。"

"明白。"李武点头。

"我会训练木叶的忍者，提升他们的实力。"

"很好。"纲手道。

李武开始训练木叶的忍者。

鸣人、佐助、小樱、雏田、鹿丸、丁次、井野、志乃、牙、天天、小李、宁次...

所有木叶的年轻忍者，都在李武的训练下，继续提升实力。

"鸣人，你的念气已经达到大师级水平。"李武道。

"大师级水平..."鸣人惊喜，"太好了！"

"我还能继续提升吗？"

"当然。"李武道，"念气是没有极限的。"

"好！"鸣人兴奋。

"我会继续努力的！"

李武看向其他忍者："你们也都很好。"

"只要你们努力，就能达到更高的境界。"

"明白。"众忍者齐声。

"我们会继续努力的！"

李武微笑："很好。"

"木叶的年轻忍者...会越来越强。"

...

几天后，李武继续修行。

他念气爆发，冲击Lv.82！

【等级提升】
【当前等级：Lv.82】
【经验值：0 / 420000】

"82级了..."李武满意。

"继续努力。"

...

几天后，李武继续修行。

他念气爆发，冲击Lv.83！

【等级提升】
【当前等级：Lv.83】
【经验值：0 / 430000】

"83级了..."李武满意。

"继续努力。"

...

几天后，纲手再次召集所有影级强者。

"晓组织的行动越来越频繁，"纲手严肃道，"我们需要提高警惕。"

"李武，你负责侦查晓组织的行动。"

"明白。"李武点头。

"我会侦查晓组织的行动，保护木叶。"

"很好。"纲手道。

李武开始侦查晓组织的行动。

他来到木叶村外，感知周围的查克拉。

【感知警报：检测到强敌】
【距离：4000米】
【威胁等级：红色（影级）】
【敌人：晓组织成员，两人】

"晓组织..."李武心中一惊。

"晓组织又来了。"

他深吸一口气，准备迎战。

李武向晓组织的方向前进，很快看到了两个晓组织成员。

他们是飞段和角都。

"又是你们？"李武冷笑。

"李武..."飞段咬牙。

"这次，我们不会输给你。"

"试试看。"李武道。

飞段拔出镰刀，冲向李武！

【飞镰斩】

飞段的镰刀斩向李武！

李武念气爆发！

【念气罩·改】

当——！！！

镰刀斩在念气罩上，毫无作用！

"什么？"飞段震惊。

"念气罩·改...居然完全挡住了镰刀？"

"还没完。"李武道。

他深吸一口气，念气再次爆发！

【武神拳】

轰——！！！

李武的武神拳击中飞段！

飞段被击飞！

"可恶..."飞段咬牙。

"不死之身...居然这么弱？"

"还没完。"李武道。

就在这时，角都发动攻击！

【地怨虞·黑缠】

角都的触手缠向李武！

李武念气爆发！

【念气炮】

轰——！！！

角都被击飞！

"什么？"角都震惊。

"念气炮...居然这么强？"

"你们的忍术，毫无威胁。"李武道。

"可恶..."飞段怒。

"一起上！"

飞段和角都同时发动攻击！

【飞镰斩】+【地怨虞·黑缠】

两人同时攻击李武！

李武不退反进，念气爆发！

【武神步】

李武使用武神步，速度提升！

【武神拳】

轰——！！！

李武的武神拳击中飞段和角都！

两人都被击飞！

"可恶..."飞段咬牙。

"木叶的忍者...居然这么强..."

"认输吧。"李武道，"你们不是我的对手。"

飞段和角都沉默了片刻，然后道："我们撤退。"

"很好。"李武道。

"记住，"李武道，"晓组织如果再敢对木叶出手，我会亲手消灭你们。"

"是..."飞段和角都低头。

两人转身，向木叶村外离开。

李武看着他们离开，心中是思考。

"晓组织..."他喃喃自话。

"晓组织的目标是捕捉尾兽，而鸣人拥有九尾。"

"晓组织...很快就会再次对木叶出手。"

"我需要继续提升实力。"

...

晚上，李武的房间。

系统界面弹出：

【支线任务完成：击败晓组织成员】
【任务进度：100%】
【奖励：经验值+10000，技能点+10，声望+3000】

【当前等级：Lv.85】
【经验值：0 / 450000】

"85级了..."李武满意。

"连升2级。"

"声望...3000点。"

"木叶的声望，又提升了。"

系统界面弹出：

【主线任务更新】
【当前阶段：继续成长】
【任务目标：继续修行，提升实力到Lv.100】
【下一阶段：第二卷·晓之来袭·完结】

"继续成长..."李武思考。

"我需要继续提升实力。"

"然后...迎战晓组织。"

李武看向窗外，木叶的夜景很美。

"木叶...我的家。"

"我会继续努力的。"

但...

他...犹豫了。

"真的...只是这样吗？"

他...看向系统界面。

"我的判断...真的准确吗？"

"还是说...只是系统的预测？"

李武...坐在屋顶。

星空...璀璨。

"三代火影...用经验判断。"

"卡卡西...用直觉判断。"

"自来也...用智慧判断。"

"而我..."

"用系统判断。"

"谁的判断...更准确？"

他...握紧拳头。

"我不知道..."

"但我开始怀疑了。"

"如果没有系统..."

"我还能做出正确的选择吗？"

"我还能...保护木叶吗？"

系统界面...弹出。

【系统提示：载体自我意识觉醒】
【说明：载体开始质疑系统依赖的真实性】
【效果：开启"直觉判断"模式】
【解锁：自主决策系统】

李武...看着系统界面。

笑了。

"也许...这就是成长。"

"质疑...然后...超越。"

"我想要...相信自己的判断。"

"不是系统的预测..."

"而是...我自己的直觉。"

系统界面...闪烁。

仿佛...在认可。

【系统回应：支持载体的成长】
【说明：系统从辅助工具...转变为伙伴关系】

李武...深吸一口气。

"谢谢你...系统。"

"但从今天起..."

"我会...更加相信...自己。"

他...站起身。

看向星空。

眼神...比之前...更加坚定。
