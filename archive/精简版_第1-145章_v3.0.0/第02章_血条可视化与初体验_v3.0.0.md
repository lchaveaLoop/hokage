# 第二章 血条可视化与初体验

**木叶隐村 · 木叶医院 · 病房**

清晨的阳光透过窗帘洒进病房。

李武站在窗前，活动着身体。

经过一夜的休息，他感觉到身体的虚弱感已经消失了。虽然还没有完全恢复到最佳状态，但至少可以正常行动了。

更重要的，系统也完全融合了。

【状态更新】
【虚弱状态解除】
【当前战斗能力：100%（原身的下忍水平）】

"100%的下忍水平吗..."

李武苦笑了一下。

前世作为Lv.85的巅峰格斗家，这个水平确实不高。但好歹也是正儿八经的忍者等级，不是什么杂兵或平民。

他转身看向房间里的镜子。

镜子里倒映出一个十三岁少年的影像。

黑色的短发，浓眉大眼，绿色的运动服——那是木叶忍者学校的校服。

身材虽然还算结实，但看起来并不起眼。

"这就是...我的新身体..."

李武抬起手，握拳。

力量感虽然没有前世那种摧枯拉朽的程度，但也不差。

然后，他看向镜子里的自己。

在视野的右下角，系统界面依然在安静地运转。

【角色：李武】
【等级：Lv.1（0.1%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"还是Lv.1啊..."

李武叹了口气。

不过也在意料之中。

毕竟他昨天才刚刚穿越，还没来得及战斗，自然没有经验值入账。

"不过..."

他眼中闪过一抹精光。

"至少，系统已经完全融合了。"

"接下来，就是尽快获取经验值的问题了。"

就在这时，房门被推开。

小李走了进来。

"哥，你醒了！"小李的声音里带着惊喜。

李武转过身，看着自己的弟弟。

在视野中，小李的头顶依然显示着【白色】名字，血条大约还有80%的长度。

而在小李身后，李武看到了另一个身影。

那是一个身穿灰色马甲，戴着绿色护额的中年男人。

标志性的毛茸茸眉毛，刺猬一样的黑色头发，以及那个永恒的闪亮牙齿笑容。

迈特凯，小李的导师。

而在凯的头顶，李武看到了一个【蓝色】的名字，血条大约有95%。

"中忍精英级..."李武心中判断。

而且，从凯身上散发出的那种压迫感来看，这个人的实力可能不止中忍精英，甚至接近上忍了。

"李武，你醒了！"凯的笑容依然灿烂，"太好了！医生说你需要休息几天。"

"我没事，凯老师。"李武礼貌地点头。

"真的没事？"凯关切地问道，"你的身体..."

"真的没事。"李武肯定地说，"而且，我感觉...比以前更好了。"

"比以前更好？"凯有些惊讶。

"是的。"李武认真地点头，"我也不知道为什么，但就是感觉...身体变轻了，反应也变快了。"

这倒不是谎话。

经过系统的强化，这具身体的各项素质确实得到了提升。

"看来，你的休息很有成效。"凯笑道，"既然醒了，那就好好休息吧。"

然后，他看向小李。

"小李，我们走吧。不要打扰李武休息。"

"是！"小李点头，然后看向李武，"哥，我先回去了，晚点再来看你。"

"好。"李武微笑道。

"一定要好好休息啊，哥！"小李又叮嘱了一句。

"我知道了。"李武点头。

凯和小李离开了病房。

李武坐在床上，开始思考接下来的行动计划。

"首先，我需要尽快战斗，获取经验值。"

"其次，我需要在战斗中测试系统的各项功能。"

"最后，我需要找到能够高效获取经验的方法。"

他闭上眼睛，开始回忆原身的记忆，寻找可能的战斗机会。

...

十分钟后，李武走出了医院。

木叶村的街道依然繁华，行人来来往往。

李武漫无目的地走着，观察着这个熟悉又陌生的世界。

突然，他的视线中出现了一抹红色。

【警告】

在视野的边缘，李武看到了一个红色的警告光圈。

他顺着光圈看去，发现那是在一条小巷里。

三个穿着黑色制服的少年，正在围殴一个看起来像是平民的老人。

李武皱眉。

"那就是...警告系统的测试吗？"

他思考了片刻，然后迈步向小巷走去。

...

小巷里。

三个少年正围着一个老人，其中一个正举起了拳头。

"把钱交出来！"少年吼道。

老人瑟瑟发抖："我...我没有钱..."

"没有钱？"另一个少年冷笑，"那就用你的命来抵！"

李武的脚步停在了巷口。

在视野中，他看到了三个少年的名字。

都是【白色】。

"杂兵级..."

李武心中判断。

那就是说，这三个少年应该只是普通的平民少年，或者是忍者学校的低年级学生，实力很弱。

"还是 Lv.0 的新手菜鸟啊..."

李武心中自嘲。

"不过，正好，就拿你们来练练手吧。"

他走进巷子。

"住手。"

李武的声音不大，但带着一种不容置疑的威严。

三个少年转过头，看着李武。

"哈？李武？"其中一个少年认出了李武，脸上露出不屑的表情，"那个吊车尾哥哥？哈哈，你要多管闲事吗？"

"就是啊，那个什么都不会的废物？"另一个少年嘲笑道。

"小李的哥哥..."第三个少年也笑了。

李武的脸上没有愤怒，只有冷静。

因为他看到了。

在视野中，这三个少年的头顶都是【白色】名字。

而在他们的身体周围，血条都不超过50%。

"杂兵级。"李武心中判断。

"三个加起来，也就能提供大约150点经验值吧。"

"虽然不多，但蚊子腿也是肉。"

而且，更重要的是...

这是他在这个世界上的第一次战斗。

他需要测试系统，测试这具身体的战斗能力。

"我说，住手。"李武重复道。

"住手？哈哈！"第一个少年大笑，"废物，你以为你是谁啊？还敢命令我们？"

"就是，也不照照镜子！"第二个少年附和道。

"小李都打不过我们，你这个哥哥又能怎么样？"第三个少年也嘲笑道。

李武叹了口气。

"既然你们不听..."

他的身体动了。

...

十秒后。

三个少年躺在地上，呻吟着。

李武站在他们面前，拍了拍手上的灰尘。

"经验+150。"

...

木叶医院门口。

凯和小李正准备离开。

突然，他们看到李武从巷子里走了出来。

"哥？"小李惊讶道，"你怎么出来了？你不是应该在休息吗？"

"我没事了。"李武微笑道。

凯看着李武，有些疑惑。

"李武，你的脸色看起来...很不错啊。"

"是的，凯老师。"李武点头，"我感觉身体已经完全恢复了。"

凯皱眉，"这不太合理啊...正常情况下，查克拉使用过度导致的昏迷，至少需要一周才能恢复..."

他仔细观察李武。

确实，李武的脸色红润，脚步稳健，完全不像是一个刚从昏迷中醒来的人。

"也许..."凯挠了挠头，"你的身体比我想象的更强吧。"

李武微笑不语。

当然比想象中更强。

经过了系统的强化，这具身体虽然表面还是下忍水平，但实际上各项素质都得到了极大提升。

无论是力量、速度、还是反应能力，都比以前好了太多。

"凯老师，我想申请出院。"李武说道。

"出院？"凯有些犹豫，"医生建议你观察一周..."

"我已经完全恢复了。"李武坚定地说，"而且，我感觉身体很好，甚至比以前更好了。"

凯看着李武坚定的眼神，最终点了点头。

"好吧。不过，你需要去火影办公室申请出院证明，然后才能正式出院。"

"我知道了。"李武点头。

"那我们就先走了。"凯说。

"嗯。"李武微笑道。

"哥，我晚点再来看你！"小李挥手告别。

"好。"李武挥手回应。

看着凯和小李离开的背影，李武心中冷静地分析着。

"接下来，我需要做几件事："

"第一，正式出院，回到家里。"

"第二，制定详细的升级计划。"

"第三，尽快开始战斗，获取经验值。"

"第四，在中忍考试前突破Lv.20。"

"第五，转职散打，解锁核心职业技能。"

李武握紧拳头。

"三个月时间..."

"从Lv.1到Lv.20..."

"平均每天需要提升约0.22级..."

"这个难度..."

李武皱眉沉思。

前世在DNF世界里，从Lv.1到Lv.20，即使是顶级玩家刷图，也需要至少两周时间。

而在这个世界...

"不过，也不是完全没有机会。"

李武冷静地分析。

"这个世界充满了战斗。忍者任务、实战考核、甚至是日常训练...每一个都是获取经验的机会。"

"关键是，如何高效地获取经验值。"

"我需要找到能够快速获取大量经验值的方法。"

李武闭上眼睛，开始在大脑中搜索着前世的记忆。

前世作为DNF顶级玩家，他对如何快速升级有着丰富的经验。

"深渊派对..."

"团本..."

"刷图..."

"..."

突然，李武想到了什么。

"对了..."

他睁开眼睛。

"我可以模拟'刷图'。"

"虽然这个世界不是游戏，但战斗的本质是一样的。"

"只要能够找到大量的敌人，进行连续的战斗..."

李武的眼中闪过一抹冷光。

"我需要找火影申请更多的任务。"

"尤其是战斗类任务。"

"C级、B级、A级、S级..."

"任务等级越高，敌人越强，经验值越多。"

"而且..."

李武想到。

"我还可以通过挑战更强的对手来获取更多经验值。"

"比如..."

李武的脑海中闪过几个名字。

"日向宁次、宇智波佐助、春野樱、山中井野、天天..."

"这些原著中的天才忍者..."

"如果能击败他们..."

"那经验值..."

李武的眼睛里闪过一丝兴奋的光芒。

"当然，我不会真的伤害他们。"

"但可以'切磋'的名义进行战斗。"

"既不会造成真正的伤害，又能获得经验值。"

"一举两得。"

李武满意地点了点头。

"就决定这么办。"

...

木叶村，火影办公室。

火影猿飞日看着面前的李武。

"李武，你要申请出院？"

"是的，火影大人。"李武恭敬地回答。

"医生建议你观察一周。"猿飞日说道。

"我感觉身体已经完全恢复了。"李武坚定地说。

猿飞日看着李武，眼中闪过一丝疑惑。

这个少年的气息...

和昨天不一样了。

昨天的时候，他的气息还很虚弱，查克拉流动紊乱。

但现在...

他的查克拉流动稳定，甚至比以前更加强盛。

"而且..."猿飞日心中暗道。

"他的眼神..."

"变了很多。"

"以前的眼神里充满了自卑、愧疚、无力..."

"但现在..."

"那种眼神..."

"冷静、自信，甚至...带着一种...俯视感？"

猿飞日皱眉。

这个少年，真的只是因为"休息好"了吗？

"那我就批准你出院了。"猿飞日最终说道。

"不过，"他补充道，"你需要继续观察。如果身体有任何不适，立即停止修行，来医院检查。"

"是，火影大人。"李武点头。

"另外..."猿飞日看着李武，"你昏迷的原因是查克拉使用过度，这在我们的记录里很少见。能告诉我，你是怎么做到的吗？"

李武心中一惊。

原身的记忆里，他确实尝试过修行查克拉，但每次都是失败的。

即使是最简单的冥想，他也无法成功。

那他是怎么做到"查克拉使用过度"的？

李武心中快速思考。

然后，他有了答案。

"火影大人，"李武恭敬地说道，"其实，我也不太清楚。"

"我只是...按照记忆里的方法，尝试冥想。"

"然后..."李武顿了顿，"我就感觉到有什么东西涌了出来。"

"那种感觉...就像..."

他搜索着合适的词汇。

"就像是身体里有什么东西被打开了。"

"查克拉疯狂涌出..."

"然后我就失去意识了。"

猿飞日若有所思地点头。

"原来如此..."他喃喃道。

"你的查克拉通道...可能一直被某种东西堵住了。"

"而在昨天...那种堵塞被打开了。"

"所以才会有查克拉疯狂涌出的情况。"

"那种情况..."李武继续说道，"就像是...大坝决堤一样。"

"可以理解。"猿飞日点头。

"那你的身体现在..."

"我感觉很好。"李武说道，"比以前任何时候都要好。"

"也许..."李武补充道，"那种堵塞的打开，对我来说，反而是好事。"

猿飞日深深看了李武一眼。

这个少年...

真的发生了改变。

"那你就回去好好修养吧。"猿飞日说道，"至少休息三天。"

"是，火影大人。"李武恭敬地回答。

"另外..."猿飞日补充道，"你的弟弟，李洛克，他最近修行很刻苦。"

"作为哥哥，你...要注意照顾他。"

"不要让他过度透支。"

"我会的。"李武点头。

"小李修行确实很刻苦。"李武认真地说道，"我看过他训练...那种拼命的程度..."

"我必须阻止他。"

"哥..."李武眼中闪过一丝感动。

"我不会阻止他的努力的。"李武坚定地说道。

"但我必须引导他。"

"用正确的方法。"

"而不是无意义的自残。"

猿飞日赞赏地点头。

"很好。你有作为哥哥的自觉。"

"那就这样吧。你去办出院手续吧。"

"是，火影大人。"

李武鞠躬，然后离开了火影办公室。

走出火影办公室，李武长出了一口气。

"第一步，完成了。"

"接下来..."

"制定详细的升级计划。"

...

木叶村，李武家。

李武坐在房间里，拿出一张纸和笔。

"让我好好规划一下。"

"三个月，从Lv.1到Lv.20..."

"平均每天需要提升约0.22级..."

李武在纸上写下几个关键数字。

然后，他开始思考快速升级的方法。

"方法一：接取大量任务。"

"尤其是战斗类任务。"

"C级、B级、A级、S级..."

"任务等级越高，敌人越强，经验值越多。"

"方法二：挑战更强的对手。"

"原著中的天才忍者。"

"宁次、佐助、天天、井野、牙..."

"通过'切磋'的名义进行战斗。"

"既不会造成真正的伤害，又能获得经验值。"

"方法三：日常训练。"

"虽然经验值较少，但累积起来也不少。"

"而且可以提升技能熟练度。"

李武在纸上写下这三个方法。

然后，他开始制定具体的实施计划。

"第1-10天：适应期，主要做C级、B级任务，累积经验值。"

"第11-20天：加速期，开始挑战A级任务，同时挑战原著天才。"

"第21-30天：冲刺期，专攻A级、S级任务，全力冲刺Lv.20。"

"第31-90天：收尾期，确保达成目标。"

李武看着计划，点了点头。

"这个计划...可行。"

"虽然很紧凑，但也并非不可能。"

"关键在于效率。"

"每一场战斗，都要快速结束。"

"每一场战斗，都要追求最高收益。"

"每一场战斗，都要确保安全。"

"完美。"

李武满意地看着计划。

"现在，就是等待机会了。"

"等待第一个任务...等待第一场战斗..."
