# 第三章 毕业考与"血条蒸发"

**木叶隐村 · 忍者学校 · 演武场**

清晨的阳光洒在演武场上。

李武站在场边，看着正在进行毕业考试的同届学生。

今天是毕业考试的日子。通过考试的人，将成为正式的下忍，获得佩戴护额的资格。

"哥，你准备好了吗？"小李关切地问道。

李武点了点头："嗯。"

虽然出院才两天，但他的状态已经调整到了最佳。

经过系统的强化，这具身体的各项素质都得到了极大的提升。力量、速度、反应能力，都比以前好了太多。

更重要的是，他对自己的能力有了清晰的认识。

【角色：李武】
【等级：Lv.1（8.5%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"8.5%..."李武在心中默念。

前天的巷战让他获得了150点经验值，加上出院后的一些基础训练，现在已经接近Lv.9了。

按照这个速度，在中忍考试前突破Lv.20并非不可能。

"下一个，李武！"

监考老师伊鲁卡的声音传来。

李武走上演武场。

站在他对面的，是另一个同届学生——一个看起来很结实的少年，名叫铁次。

在视野中，李武看到了铁次头顶的名字。

【铁次】（白色）

"杂兵级..."李武心中判断。

那就是说，铁次的实力应该只是普通学生水平，比小李还要弱一些。

"李武，我听说你昏迷了三天。"铁次咧嘴一笑，"今天，我会让你知道，什么是差距！"

李武没有回应。

在视野中，他看到了铁次的血条。

大约还有90%的长度。

"90%吗..."李武冷静地分析着。

"虽然不想欺负弱者，但考试就是考试。"

"而且..."

李武的眼中闪过一抹精光。

"我正好需要一场正式的战斗，来测试这具身体的战斗能力。"

"比赛开始！"伊鲁卡宣布。

铁次瞬间冲出，拳头直击李武面门。

速度很快，但...

在李武眼中，这拳慢得像慢动作。

他侧身一闪，轻松避开。

"什么？！"铁次一愣。

然后，李武反击了。

下段踢。

这不是普通的下段踢，而是前世作为DNF顶级格斗家时，经过无数次实战检验的最基础的起手式。

但在这个世界...

在李武的视野中，当他的腿扫过铁次的小腿时，他看到了一个奇怪的现象。

铁次的身体突然失去平衡，整个人向后倾倒。

就像是...被某种力量强行击倒了一样。

"这是..."李武眼中闪过一丝惊讶。

【倒地判定触发】

系统的提示出现在视野中。

"原来如此..."李武心中了然。

"下段踢具有'倒地效果'，能直接让敌人失去平衡，强制倒地。"

"这个机制..."

铁次重重地摔在地上，半天爬不起来。

"怎么会..."铁次难以置信地看着李武。

李武没有给他喘息的机会。

他上前一步，寸拳轰出。

金色念气爆发！

在李武的视野中，铁次的血条瞬间从90%跌到了0%。

【经验+80】

铁次直接昏死过去。

整个演武场陷入了死寂。

所有人都震惊地看着这一幕。

一招。

仅仅一招，铁次就被击败了。

而且，李武甚至没有使用任何忍术，只是最基础的体术。

"这...这是李武？"有人难以置信地说道。

"那个吊车尾的哥哥？"

"他什么时候变得这么强了？"

伊鲁卡也震惊了。

他知道李武最近有些变化，但没想到...

这种程度的实力，已经不是普通下忍能拥有的了。

"李武，胜利！"伊鲁卡宣布道。

李武点了点头，走下演武场。

小李已经在场边等着了，脸上满是兴奋："哥，你刚才太帅了！"

"还行吧。"李武淡淡地说。

但在心中，他已经开始分析刚才的战斗了。

"下段踢触发倒地判定...这是一个很好的起手式。"

"而且，我发现，这个世界的忍者似乎不了解'判定'的概念。"

"他们的攻击，虽然威力不俗，但缺乏'机制'。"

"这就是..."

李武的眼中闪过一抹精光。

"这就是我能够碾压他们的原因。"

"dnf的战斗系统，不是简单的数值比拼，而是机制的碰撞。"

"判定优先级、霸体、抓取、粉碎、倒地、浮空..."

"这些机制，在这个世界，是全新的存在。"

"而我对这些机制的掌控..."

李武握紧拳头。

"是绝对的。"

"下一个，李洛克！"

伊鲁卡的声音传来。

小李走上演武场。

他的对手，是一个看起来很敏捷的少年。

在视野中，李武看到了那个少年的名字。

【油女志乃】（白色）

"油女一族的..."李武心中了然。

志乃使用的是寄坏虫，这种对手需要特别注意。

战斗开始。

志乃的手中出现黑色的虫群，向小李涌去。

小李快速闪避，同时试图接近。

但虫群的数量太多，小李无法近身。

"小李..."李武皱眉。

他看到了小李的血条。

正在缓慢下降。

虽然不多，但这个趋势很危险。

"如果不打破虫群的包围..."

李武冷静地分析着。

然后，他想到了什么。

"小李，用表莲华！"

李武的声音不大，但清晰地传到了小李耳中。

小李一愣。

表莲华？

那是...

那是凯老师教过的招式，但小李还没有完全掌握。

但现在...

小李看向李武，看到了哥哥眼中的坚定。

"我相信你！"

小李深吸一口气，开始结印。

虽然还不熟练，但...

表莲华，发动！

小李的身影消失，瞬间出现在志乃身后。

踢击！

志乃被踢飞，虫群失去控制。

然后，小李追上，继续攻击。

虽然还不够完美，但...

表莲华，完成！

志乃倒在地上，失去了战斗能力。

"李洛克，胜利！"伊鲁卡宣布。

小李喘着气，脸上满是兴奋。

他看向李武，看到了哥哥眼中的赞赏。

"哥，我做到了！"

李武点了点头，微笑道："嗯，做得很好。"

但在心中，他冷静地分析着。

"表莲华...小李已经能用了。"

"而且，从刚才的战斗来看，小李的战斗风格..."

李武想起了前世在DNF世界里，那些顶级玩家的操作。

"小李的战斗风格，很像'连招流'。"

"连续的攻击，快速的动作，让敌人无法反应。"

"这种风格..."

李武的眼中闪过一丝兴奋。

"非常适合散打这个职业。"

"如果能将散打技能与小李的体术结合..."

李武已经开始构思未来的训练计划了。

"小李的潜力，远不止此。"

"而且..."

李武想起了系统任务。

【主线任务：在中忍考试前突破Lv.20】

"如果我能在中忍考试前突破Lv.20，转职散打..."

"那小李..."

李武看向场上的小李，心中已经有了决定。

"我也要让小李变强。"

"用我的系统，用我的方法..."

"让小李能够在这个残酷的忍者世界里，真正地生存下去，甚至，变得强大。"

...

考试继续进行。

一个又一个学生上台战斗。

李武冷静地观察着每一个对手。

在视野中，他看到了所有人的血条和能级。

大部分是【白色】，只有少数几个是【蓝色】。

"蓝色..."李武心中记录着。

"那些蓝色的学生，应该就是天才级别了。"

"比如..."

李武的视线扫过人群，落在几个身影上。

日向宁次、宇智波佐助、春野樱、山中井野...

这些原著中的天才忍者，在视野中都是【蓝色】。

"中忍级..."李武判断。

"而且，从他们身上散发出的气息来看，实际实力可能更强。"

"特别是宁次和佐助..."

李武眯起眼睛。

"宁次的白眼，能看穿查克拉流动，这对我的战斗方式是一个威胁。"

"而佐助的写轮眼，能复制忍术，而且洞察力极强。"

"这两个对手..."

李武握紧拳头。

"将来一定要交手。"

"但现在..."

李武将注意力转回场上的比赛。

考试还在继续。

下一个上台的，是宇智波佐助。

他的对手，是一个普通学生。

战斗只持续了十秒。

佐助用一招豪火球之术，直接击败了对手。

【宇智波佐助】（蓝色）

血条几乎没有下降。

"写轮眼..."李武在心中默念。

"而且，佐助的战斗风格..."

李武想起了原著中的描述。

"佐助擅长火遁和忍术，体术也不弱。"

"这种全能型战士..."

李武冷静地分析着。

"将来会是很难对付的对手。"

"不过..."

李武的眼中闪过一丝自信。

"我有系统。"

"而且，我对机制的掌控..."

"是绝对的。"

"无论佐助的写轮眼有多强，在'判定'面前，都是平等的。"

"抓取判定无视霸体，无敌判定免疫一切..."

"这些机制，是写轮眼无法看穿的。"

"因为..."

李武想起了系统关于"判定"的解释。

"判定是规则级的碰撞。"

"不是简单的物理攻击，而是规则的优先级。"

"而在这个世界..."

"没有人了解'判定'。"

"这就是..."

李武握紧拳头。

"我的绝对优势。"

...

下午，考试结束。

所有通过考试的学生，都获得了下忍护额。

李武和小李都顺利通过了考试。

"哥，我们成功了！"小李兴奋地说道。

李武点了点头，微笑道："嗯。"

但在心中，他已经开始规划下一步的行动了。

"成为下忍只是第一步。"

"接下来，我需要..."

李武在心中列出计划：

"第一，接取更多的任务，获取经验值。"

"第二，挑战更强的对手，测试战斗能力。"

"第三，制定详细的升级计划，确保在中忍考试前突破Lv.20。"

"第四，帮助小李变强，让他能够在这个残酷的世界里生存。"

"第五，找到系统的真相，弄清楚我为什么会穿越，以及这个系统的目的是什么。"

李武睁开眼睛，看向天空。

"木叶的蓝天..."

"这就是火影世界啊。"

"既然来了..."

李武握紧拳头。

"那我就要成为这个世界的最强者。"

"用我的系统，用我的方法..."

"改写所有悲剧的命运。"

"让小李、让宁次、让所有原著中悲剧的角色..."

"都能有一个幸福的结局。"

李武的眼中闪过一丝坚定。

"这是我的承诺。"

"也是我来到这个世界的目的。"

...

回到家中，李武坐在房间里，拿出一张纸。

"让我好好规划一下接下来的行动。"

"首先，我需要尽快达到Lv.20，转职散打。"

"按照目前的经验获取速度..."

李武在纸上写下几个数字。

"巷战+150，毕业考+80..."

"总共230点经验值。"

"从Lv.1到Lv.20，需要的经验值大概是..."

李武回忆起前世DNF的经验值表。

"从Lv.1到Lv.20，大约需要50,000点经验值。"

"我现在有230点，还差49,770点。"

"这个数字..."

李武皱眉。

"确实不少啊。"

"不过..."

李武冷静地分析着。

"这个世界充满了战斗机会。"

"任务、实战考核、甚至是日常训练..."

"只要能高效地获取经验值..."

"达到Lv.20并非不可能。"

"关键是，找到快速获取经验的方法。"

李武闭上眼睛，开始回忆前世DNF世界的升级方法。

"刷图..."

"深渊派对..."

"团本..."

"这些方法，在这个世界里，对应的就是..."

李武睁开眼睛，眼中闪过一抹精光。

"对应的就是：接取大量任务，挑战强敌，参加实战考核。"

"而且..."

李武想到了一个更重要的问题。

"我发现，系统给予的经验值，似乎与敌人的能级有关。"

"巷战中的三个杂兵，总共150经验。"

"毕业考中的铁次，80经验。"

"如果...

如果能挑战更强的对手...

比如，蓝色级别的敌人..."

李武的眼中闪过一丝兴奋。

"那经验值..."

"可能会翻倍，甚至更多。"

"也就是说..."

李武握紧拳头。

"我需要挑战更强的对手。"

"宁次、佐助、天天、井野..."

"这些原著中的天才忍者..."

"如果能击败他们..."

"那经验值..."

李武的眼睛里闪过一丝兴奋的光芒。

"当然，我不会真的伤害他们。"

"但可以'切磋'的名义进行战斗。"

"既不会造成真正的伤害，又能获得经验值。"

"一举两得。"

李武满意地点了点头。

"就决定这么办。"

"明天开始，我就向这些天才忍者发起挑战。"

"用我的系统，用我的方法..."

"证明我的实力。"

"同时，也为小李铺平道路。"

"让他能够在这个残酷的忍者世界里..."

"真正地变强。"

李武站起身，走到窗边。

夕阳西下，木叶村被染上了一层金色的光芒。

"这就是火影世界啊..."

李武喃喃自语。

"既然来了..."

"那我就要成为这个世界的最强者。"

"改写所有悲剧的命运。"

"让所有人都能有一个幸福的结局。"

李武握紧拳头。

"这就是我的目标。"

"也是我来到这个世界的意义。"
