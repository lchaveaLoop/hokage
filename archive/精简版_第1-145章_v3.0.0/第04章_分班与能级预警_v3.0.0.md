# 第四章 分班与"能级预警"

**木叶隐村 · 忍者学校 · 教室**

毕业考试的第二天，分班结果公布。

李武和小李坐在教室里，等待伊鲁卡老师宣布分班结果。

"哥，你说我们会分到哪一班？"小李有些紧张。

"不管哪一班都一样。"李武平静地说，"重要的是，尽快适应下忍的生活。"

他在心中默默计算着。

【角色：李武】
【等级：Lv.9（15.2%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"15.2%..."李武在心中默念。

毕业考击败铁次获得80点经验，加上之后的一些基础训练，现在已经接近Lv.10了。

按照这个速度，突破Lv.20并非不可能。

"好，现在宣布分班结果！"伊鲁卡说道。

"第一班..."

伊鲁卡一个个念着名字。

"第七班，宇智波佐助、春野樱、李武。"

李武愣了一下。

第七班？

原著中，第七班的成员应该是鸣人、佐助和小樱。

但现在...

鸣人不在了？

李武快速思考。

难道是因为他的穿越，改变了某些事情？

或者说...

鸣人根本没有毕业？

"第七班，指导老师，旗木卡卡西。"伊鲁卡继续说道。

旗木卡卡西...

木叶的复制忍者，写轮眼卡卡西。

李武的心中一紧。

这个人...

在原著中，是影级强者。

在视野中，李武看到了教室门口走进来的身影。

银色的头发，戴着面罩，左眼被护额遮住，右手插在裤兜里，整个人看起来懒散无比。

但在李武的视野中...

【旗木卡卡西】（蓝色）

"蓝色..."李武心中判断。

"中忍精英级...不，可能是上忍级。"

"而且，从卡卡西身上散发出的那种压迫感来看..."

李武眯起眼睛。

"这个人的实力，可能不止蓝色。"

"也许..."

李武想起了系统关于"隐藏实力"的说明。

"有些人会刻意压制自己的实力，只在关键时刻爆发。"

"卡卡西，很可能就是这种人。"

"好，第七班的人，跟我来。"卡卡西懒散地说道。

李武站起身，看向佐助和小樱。

佐助冷着一张脸，看起来很不爽。

小樱则有些兴奋，毕竟能和佐助一个班。

李武平静地跟在后面。

在心中，他已经开始分析卡卡西的实力了。

"写轮眼卡卡西..."

"复制了上千种忍术..."

"而且，体术也很强..."

"这种对手..."

李武握紧拳头。

"将来一定要交手。"

"但现在..."

李武冷静地思考着。

"我需要先评估卡卡西的实力，然后制定对策。"

...

天台。

第七班的成员全部到齐。

卡卡西靠在栏杆上，懒散地看着三人。

"自我介绍一下吧。"卡卡西说道，"姓名、喜好、厌恶、梦想、 hobby。"

"先从你开始。"卡卡西指向佐助。

佐助冷冷地说："宇智波佐助，喜好......训练，厌恶......很多，梦想，杀死那个男人，复活一族，hobby......训练。"

卡卡西的眼中闪过一丝复杂。

"嗯，有个性。"

"下一个。"卡卡西指向小樱。

小樱兴奋地说："春野樱，喜好......是......嗯......厌恶......是......梦想......是......"

小樱偷看佐助，脸红。

卡卡西有些无奈。

"好了，下一个。"

他指向李武。

李武平静地说："李武，喜好......变强，厌恶......弱小，梦想......保护重要的人，hobby......战斗训练。"

卡卡西的眼中闪过一丝惊讶。

"保护重要的人？"卡卡西重复道。

"是的。"李武点头，"我有一个弟弟，我要保护他。"

卡卡西看着李武，眼中闪过一丝欣赏。

"很好。"

"现在，我说一下我的规矩。"

卡卡西站直身体，气氛突然变得严肃。

"第一，不要轻视那些看起来简单的东西。"

"第二，在这个世界上，有些人比你想象中更强大。"

"第三..."

卡卡西的眼中闪过一丝寒光。

"不要违抗我的命令。"

"明白了吗？"

三人点头。

"很好。"卡卡西恢复了懒散的样子，"明天早上五点，演习场集合。"

"记得不要吃早饭，不然会吐的。"

说完，卡卡西消失在烟雾中。

...

第二天，清晨五点。

李武准时到达演习场。

佐助和小樱还没有来。

十分钟后，佐助和小樱才到。

"哥，你怎么这么早？"小李从远处跑来。

"你也是第七班的？"李武有些惊讶。

"不是啊。"小李摇头，"我是第三班的，老师是迈特凯。"

"那你来这？"

"凯老师说，让我来给你送东西。"小李递给李武一个便当，"这是凯老师让我给你的，说演习的时候用得上。"

李武接过便当，心中一暖。

"谢了，小李。"

"不客气，哥！"小李笑道，"那我先走了，你加油！"

小李离开。

五分钟后，卡卡西到了。

"你们三个..."卡卡西看了看表，"都准时了。"

"很好。"

他从口袋里拿出两个闹钟。

"放在地上。"

"我要测试的东西很简单。"卡卡西说道，"从这两个闹钟中，抢走它们。"

"如果在中午之前，你们无法从我手中抢走闹钟..."

"那就全部不合格！"

"明白了吗？"

三人点头。

"好，开始！"

话音刚落，佐助就冲了出去。

手里剑、苦无，密集地投向卡卡西。

卡卡西轻松躲过，然后...

"太慢了。"

佐助只觉得眼前一花，卡卡西就已经到了他身后。

"背后。"

卡卡西的手结成了印。

"虎印..."

佐助瞳孔一缩，快速躲避。

但卡卡西的速度更快。

"千年杀！"

佐助被击中，整个人飞了出去。

"可恶！"佐助爬起来，继续攻击。

小樱也在一旁寻找机会。

但卡卡西就像有眼睛一样，总能提前预判他们的动作。

李武站在远处，冷静地观察着。

在视野中，卡卡西的头顶依然显示着【蓝色】。

血条还有98%。

"几乎满血..."李武判断。

"而且，从刚才的战斗来看..."

李武眯起眼睛。

"卡卡西的速度、反应、洞察力..."

"都是顶级的。"

"特别是..."

李武想起了原著中的设定。

"写轮眼。"

"虽然卡卡西的写轮眼不是他的，但他使用得很好。"

"甚至..."

李武心中一凛。

"可能比宇智波一族的人还要熟练。"

"这种对手..."

李武深吸一口气。

"必须小心应对。"

"但是..."

李武的眼中闪过一丝精光。

"我也不是原来的那个'吊车尾'了。"

"我有系统。"

"而且，我对机制的掌控..."

"是绝对的。"

"就算卡卡西是上忍..."

"在'判定'面前，也是平等的。"

李武握紧拳头，准备出手。

...

战斗继续。

佐助和小樱已经疲惫不堪。

卡卡西依然轻松自如。

"你们太弱了。"卡卡西摇头，"完全没有团队配合。"

"而且..."

卡卡西看向一直没有动的李武。

"你，从头到尾都没有出手。"

"为什么？"

李武平静地说："我在观察。"

"观察？"

"观察你的战斗风格，分析你的弱点。"李武毫不隐瞒地说。

卡卡西的眼中闪过一丝惊讶。

"哦？那你发现什么了吗？"

"发现了。"李武点头。

"你的战斗风格，是以写轮眼为核心，结合忍术和体术的全能型。"

"你的速度很快，反应很敏锐，洞察力也很强。"

"但是..."

李武的眼中闪过一丝精光。

"你有一个致命的弱点。"

卡卡西挑眉："哦？说来听听。"

"你太依赖写轮眼了。"李武平静地说。

"虽然你的写轮眼很强，但它不是你的眼睛。"

"长期使用，会对你的身体造成负担。"

"而且..."

李武指着卡卡西的左眼。

"你的写轮眼，一直在消耗你的查克拉。"

"从战斗开始到现在，你的查克拉已经消耗了约15%。"

"如果继续消耗下去..."

李武眯起眼睛。

"你的体力会下降，反应会变慢，最终..."

"败北。"

卡卡西愣住了。

这个少年...

竟然能看出他的查克拉消耗？

这不可能啊！

就算是有白眼的日向一族，也无法精确到百分比...

这个李武...

到底是什么人？

卡卡西的眼中闪过一丝警惕。

"你说得对。"卡卡西承认道，"我的查克拉确实在消耗。"

"但是..."

他突然消失，出现在李武身后。

"在我说完之前，你已经输了。"

李武只觉得背后一凉。

但...

他并没有慌张。

在视野中，他看到了卡卡西的动作。

虽然很快，但...

在系统的加持下，李武能够清晰地"看到"卡卡西的攻击轨迹。

"太快了吗？"李武在心中自问。

"不..."

"正好。"

李武的身体动了。

不是躲避，而是...

反击！

下段踢！

卡卡西只觉得小腿一痛，整个人失去平衡。

"这是..."卡卡西眼中闪过一丝惊讶。

"倒地效果？"

卡卡西勉强稳住身体，看向李武。

"好小子..."卡卡西笑了，"有点意思。"

"你刚才那一招..."

"是什么？"

"下段踢。"李武平静地说。

"这是最基础的体术起手式。"

"但是..."

李武的眼中闪过一丝精光。

"它有一个特殊的效果。"

"能让敌人失去平衡，强制倒地。"

"我们叫它..."

李武想了想，决定用一个简单的解释。

"'倒地效果'。"

"倒地效果？"卡卡西重复道。

"是的。"李武点头，"这是一种特殊的机制，能让敌人强制失去平衡。"

"虽然只是基础体术，但在关键时刻..."

李武看向卡卡西。

"可以逆转局势。"

卡卡西沉默了片刻。

然后，他笑了。

"很好。"

"你通过了。"

"什么？"佐助和小樱都愣住了。

"我说，你们通过了。"卡卡西收起闹钟，"特别是李武。"

"你的观察力和判断力，都很优秀。"

"而且..."

卡卡西看着李武。

"你的战斗风格，很特别。"

"那种'机制'..."

"我在其他人身上，从未见过。"

"你从哪里学的？"

李武平静地说："自己悟出来的。"

"悟出来的？"卡卡西挑眉。

"是的。"李武点头，"我发现，战斗不只是力量的比拼，更是规则的碰撞。"

"谁的规则优先级更高，谁就能赢。"

"这就像..."

李武想了想，决定用一个卡卡西能理解的比喻。

"就像剪刀石头布。"

"剪刀能剪布，布能包石头，石头能砸剪刀。"

"这三个规则，相互克制。"

"但如果..."

李武的眼中闪过一丝精光。

"如果你出的是'剪刀'，而对方出的是'布'..."

"那你就能赢。"

"因为..."

李武握紧拳头。

"你的规则，优先级更高。"

卡卡西愣住了。

规则优先级...

这个概念...

他从未听说过。

但是...

从刚才的战斗来看...

李武确实在用一种不同于普通忍者的战斗方式。

那种"倒地效果"...

以及他对自己查克拉消耗的精确判断...

都说明，李武掌握着某种特殊的"机制"。

"规则优先级..."卡卡西喃喃道。

"有意思。"

"看来，你这个学生..."

卡卡西看着李武，眼中闪过一丝欣赏。

"会给我带来很多惊喜。"

...

中午。

卡卡西带着三人来到纪念碑前。

"这是..."佐助问道。

"木叶的英雄纪念碑。"卡卡西说道，"上面刻着的，都是为村子牺牲的忍者名字。"

"我的朋友，也在上面。"

卡卡西的声音有些低沉。

"卡卡西老师..."小樱关心地说道。

"没事。"卡卡西摆了摆手，"好了，今天解散。"

"明天开始，正式执行任务。"

"记住..."

卡卡西看着三人。

"在下忍的世界里，你们会遇到各种各样的敌人。"

"有些敌人，比你们想象的更强大。"

"有些敌人，会超出你们的认知。"

"但是..."

卡卡西的眼中闪过一丝坚定。

"无论遇到什么困难，都要相信自己的同伴。"

"因为..."

"在这个世界上，没有什么比同伴更重要。"

说完，卡卡西离开了。

李武看着卡卡西的背影，心中若有所思。

"同伴吗..."

李武想起了原著中，第七班的羁绊。

鸣人、佐助、小樱...

他们之间的羁绊，确实很深。

但现在...

鸣人不在了。

第七班的成员，变成了佐助、小樱，和他自己。

"这种改变..."李武在心中思考。

"会产生什么影响？"

"佐助的堕落，还会发生吗？"

"小樱的成长，还会一样吗？"

"而..."

李武握紧拳头。

"我在第七班..."

"又能起到什么作用？"

"这些问题..."

李武抬头看向天空。

"只有时间能给出答案了。"

"但是..."

李武的眼中闪过一丝坚定。

"无论未来如何..."

"我都要变强。"

"强到足以保护所有重要的人。"

"强到足以改写所有悲剧的命运。"

"这是我的承诺。"

"也是我来这个世界的意义。"

...

回到家中，李武坐在房间里，回想今天的战斗。

"卡卡西的实力..."

"确实很强。"

"但..."

李武在心中分析着。

"他太依赖写轮眼了。"

"如果我能针对这一点..."

"比如，制造一个无法用写轮眼预判的局面..."

"或者..."

李武想到了一个更可怕的想法。

"如果我能破解写轮眼的洞察..."

"那卡卡西的实力，会大打折扣。"

"当然..."

李武摇了摇头。

"现在的我，还做不到这一点。"

"但是..."

他看向视野中的系统面板。

【角色：李武】
【等级：Lv.9（18.7%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"18.7%..."李武在心中默念。

"今天的战斗，虽然没有击败卡卡西，但也获得了一些经验值。"

"而且，更重要的是..."

李武的眼中闪过一丝兴奋。

"我验证了一个重要的结论。"

"我的战斗方式..."

"在这个世界，是全新的。"

"没有人了解'机制'，没有人了解'判定'..."

"这就是我的绝对优势。"

"只要我继续强化这个优势..."

李武握紧拳头。

"我就能超越所有人。"

"包括卡卡西，包括影级强者..."

"甚至..."

李武的眼中闪过一丝野心。

"包括使徒。"
