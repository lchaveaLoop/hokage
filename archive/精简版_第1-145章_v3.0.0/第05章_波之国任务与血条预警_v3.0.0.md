# 第五章 波之国任务与"血条预警"

**木叶隐村 · 火影办公室**

分班后的第三天，第七班接到了第一个正式任务。

"护送任务？"佐助皱眉，"这种任务，是不是太简单了？"

"护送建筑设计师达兹纳先生，前往波之国。"卡卡西看着手中的任务卷轴，"C级任务。"

"C级..."小樱有些失望，"我还以为能接到更刺激的任务..."

李武平静地站在一旁。

在心中，他已经开始分析这次任务了。

"波之国任务..."

"原著中，这是第七班的第一个重要任务。"

"也是鸣人开启九尾力量、佐助开启写轮眼的关键战役。"

"但是现在..."

李武眯起眼睛。

"鸣人不在了。"

"那么，这场战斗的结果..."

"会发生什么改变？"

而且...

更重要的是...

在原著中，这场战斗的敌人是...

桃地再不斩。

雾隐鬼人。

上忍级别的强者。

在李武的视野中，系统突然弹出了警告。

【警告】
【检测到高危任务】
【建议等级：Lv.15+】
【当前等级：Lv.9（18.7%）】
【危险度：高】

"危险度高..."李武心中一凛。

"系统在警告我。"

"也就是说..."

"这场任务，可能会超出我的能力范围。"

"但是..."

李武握紧拳头。

"我必须去。"

"因为..."

他想起了原著中，鸣人在这场战斗中的成长。

如果没有这场战斗，鸣人可能无法快速成长。

而现在的李武，同样需要这场战斗。

"而且..."

李武的眼中闪过一丝精光。

"高危险度，意味着高收益。"

"如果能击败再不斩..."

"经验值..."

他看着系统面板。

【角色：李武】
【等级：Lv.9（18.7%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"从Lv.9到Lv.15，需要的经验值大概是20,000点。"

"如果能击败再不斩..."

"可能会一次性获得大量经验值。"

"甚至..."

李武的眼中闪过一丝兴奋。

"可能直接突破Lv.10，甚至Lv.11。"

"这种机会..."

"我不能错过。"

...

波之国。

护送的路上，一切都很平静。

直到...

那座桥。

李武看到了。

一滩血迹。

在血迹旁，躺着两个穿着绿色护额的忍者。

已经死了。

"这是..."佐助瞪大眼睛。

"波之国的护额。"卡卡西蹲下检查，"被杀死的保镖。"

"看来，我们已经接近目标了。"

李武冷静地观察着周围。

在视野中，系统警告再次出现。

【警告】
【敌人接近】
【能级：紫色（影级）】
【数量：1】

"紫色..."李武心中一惊。

"影级强者！"

"而且，数量是1..."

李武快速判断。

"也就是说，再不斩已经来了。"

"这种对手..."

李武深吸一口气。

"必须小心应对。"

就在这时，迷雾中出现了一个身影。

全身缠着绷带，只有眼睛和嘴巴露在外面。

手中握着一把巨大的斩首大刀。

桃地再不斩。

雾隐鬼人。

"果然是你..."卡卡西的眼神变得锐利。

"旗木卡卡西..."再不斩冷笑，"复制忍者。"

"听说你在暗部有'拷贝忍者'的名号..."

"今天，我就要你的人头，来证明我的实力。"

卡卡西拉起护额，露出左眼。

红色的眼睛，三个勾玉。

写轮眼。

"写轮眼..."再不斩眼中闪过一丝兴奋，"太好了。"

"能死在写轮眼之下，也不算冤。"

战斗，一触即发。

李武站在一旁，冷静地观察着。

在视野中，他看到了再不斩的头顶显示着【紫色】。

血条大约还有95%。

"95%..."李武判断。

"而且，从再不斩身上散发出的那种压迫感..."

"这个人的实力，可能是紫色级的巅峰。"

"甚至..."

李武想起了系统关于"隐藏实力"的说明。

"可能已经接近金色级了。"

"这种对手..."

李武握紧拳头。

"不是我一个人能对付的。"

"必须依靠卡卡西。"

"但是..."

李武的眼中闪过一丝精光。

"如果能在战斗中，找到再不斩的破绽..."

"然后..."

他想起了系统的"弱点机制"。

虽然现在还没有完全解锁，但从手册中了解到，当敌人重伤时，血量低于30%，弱点红点会自动出现。

"如果能触发弱点机制..."

"一击必杀..."

李武的眼中闪过一丝野心。

"也不是不可能。"

战斗开始了。

再不斩的斩首大刀横扫，巨大的力量足以将人一刀两断。

卡卡西快速躲避，同时结印。

"水遁·水龙弹之术！"

巨大的水龙从河中冲出，直击再不斩。

再不斩快速结印。

"水遁·水龙弹之术！"

同样的水龙从他手中冲出，两条水龙在空中碰撞，激起巨大的水花。

"复制了我的术？"卡卡西眼中闪过一丝惊讶。

"在我的术完成之前，你就能够预判并复制..."

"这就是写轮眼的力量吗？"

再不斩冷笑："不止如此。"

"我会让你知道，什么是真正的恐惧。"

"雾隐之术！"

浓雾瞬间笼罩了整个战场。

能见度降到几乎为零。

"小心！"卡卡西提醒佐助和小樱，"保护好达兹纳先生！"

"李武，你也退后！"

李武点了点头，退到佐助和小樱身边。

在浓雾中，他"看到"了再不斩的身影。

虽然肉眼看不到，但在系统的加持下，李武能够感知到敌人的位置。

"左前方，三十米..."李武在心中判断。

"正在快速接近..."

"而且..."

李武眯起眼睛。

"再不斩的查克拉，开始聚集。"

"他要发动攻击了。"

就在这时，浓雾中传来了再不斩的声音。

"恐怖...正在蔓延..."

"你什么时候会死？"

"明天？后天？"

"还是..."

"就在此刻？"

八个身影同时出现，包围了卡卡西。

每一个都是再不斩。

"水分身之术..."卡卡西冷静地分析。

"八个分身，每一个都是实体。"

"如果同时攻击..."

卡卡西的手快速结印。

"但是，我也会。"

八个卡卡西同时出现。

"水分身之术，成功复制。"

再不斩冷笑："复制得不错，但你能分辨出哪个是真的我吗？"

"能。"卡卡西平静地说。

"哦？"再不斩挑眉。

"因为..."卡卡西看向其中一个分身，"你的查克拉流动，有些紊乱。"

"那是真正本体的特征。"

"被发现了..."再不斩眼中闪过一丝惊讶。

"不愧是复制忍者。"

"但是..."

再不斩的身影突然消失，出现在卡卡西身后。

"你以为，我只有一个本体吗？"

斩首大刀横扫，卡卡西被击中，整个人飞了出去。

"卡卡西老师！"佐助和小樱惊呼。

李武冷静地看着这一幕。

在视野中，他看到卡卡西的血条下降到了75%。

"75%..."李武判断。

"受伤了，但不是致命伤。"

"而且..."

李武看向再不斩。

"再不斩的查克拉消耗..."

"大概在20%左右。"

"如果继续消耗下去..."

李武的眼中闪过一丝精光。

"再不斩会先耗尽体力。"

"这场战斗的结果..."

"已经很明确了。"

但是...

就在这时，李武看到了再不斩的身体突然变得坚硬。

那种坚硬，不是物理上的坚硬，而是...

某种特殊的防御状态。

在系统视野中，再不斩的身上出现了一层淡淡的光芒。

灰色。

就像...

"霸体..."李武心中一惊。

"这是..."

他想起了手册中的描述。

"霸体，免疫控制状态。"

"也就是说，再不斩进入了'霸体'状态。"

"这种状态下，普通的攻击无法打断他的行动。"

"只有..."

李武想起了"抓取效果"。

"抓取能无视霸体，直接打断。"

"但是..."

李武皱眉。

"我现在还没有掌握'抓取'技能。"

"寸拳虽然具有抓取效果，但距离太近..."

"如果靠近再不斩..."

李武快速计算风险。

"再不斩的斩首大刀，攻击范围很大。"

"如果我贸然接近..."

"可能会被一刀两断。"

"这种风险..."

李武握紧拳头。

"太大了。"

"必须等待时机。"

战斗继续。

卡卡西和再不斩交手，水遁、体术、忍术...

各种招式层出不穷。

李武站在一旁，仔细观察着每一个细节。

"再不斩的战斗风格..."

"是以大刀攻击为主，结合雾隐之术和水分身。"

"他的优势是..."

"力量大、攻击范围广、隐蔽性强。"

"但弱点是..."

李武的眼中闪过一丝精光。

"查克拉消耗大。"

"而且，再不斩似乎不擅长近身格斗。"

"如果能把他拖入近身战..."

李武开始构思战术。

"用速度压制，用体术攻击..."

"再加上..."

他想起了"倒地效果"。

"如果能让再不斩倒地..."

"那就..."

就在这时，战场上的情况发生了变化。

卡卡西被再不斩的斩首大刀压制，险象环生。

"可恶..."卡卡西咬紧牙关。

"这个家伙..."

"力量太大了。"

"而且..."

卡卡西看向自己的左手。

"我的查克拉消耗，已经达到30%了。"

"如果继续这样下去..."

就在这危急时刻，李武动了。

他不是直接攻击再不斩，而是...

快速接近卡卡西。

"卡卡西老师，我有一个战术！"李武快速说道。

"什么战术？"卡卡西问道。

"再不斩的力量很大，但速度不是他的优势。"李武快速分析。

"而且，他的查克拉消耗已经超过25%。"

"如果我们能拖住他，等他体力耗尽..."

"那时候，就是我们反击的机会。"

"所以..."

李武看向卡卡西。

"我建议，转为防守战术。"

"用最小的消耗，拖延时间。"

"等再不斩体力下降到50%以下..."

"那时候，我们再全力反击。"

卡卡西愣了一下。

这个少年...

竟然能精确判断敌人的查克拉消耗？

而且，还能制定出如此合理的战术...

"好。"卡卡西点头，"就按你说的做。"

"佐助、小樱，你们退后。"

"我和李武，牵制再不斩。"

佐助和小樱点了点头，快速退到安全地带。

李武和卡卡西，同时面对再不斩。

"哦？"再不斩挑眉，"两个小鬼，也想挑战我？"

"那就来吧。"

斩首大刀横扫，巨大的力量压来。

李武和卡卡西同时躲避。

然后...

反击！

李武的速度爆发，瞬间接近再不斩。

下段踢！

但是...

再不斩的反应也很快。

大刀一转，挡住了李武的攻击。

"什么？！"李武一惊。

"你的攻击..."再不斩冷笑，"太弱了。"

大刀横扫，李武被迫后退。

"可恶..."李武咬紧牙关。

"力量差距太大了。"

"但是..."

李武的眼中闪过一丝精光。

"我发现了。"

"再不斩的'霸体'状态..."

"虽然能免疫控制，但是..."

"不能完全免疫伤害。"

"也就是说..."

李武快速计算。

"如果我持续攻击，即使不能打断他的行动..."

"也能消耗他的体力。"

"等到体力耗尽..."

"霸体状态就会自动解除。"

"那时候..."

李武的眼中闪过一丝兴奋。

"就是我反击的机会。"

战斗继续。

李武和卡卡西配合，一攻一守，稳稳地牵制着再不斩。

再不斩的查克拉消耗，也在不断增加。

30%、35%、40%...

终于，当再不斩的查克拉消耗达到50%时...

他的身体突然一软。

霸体状态，解除了。

"什么？！"再不斩一惊。

"我的体力..."

"怎么会消耗得这么快？"

"因为..."

李武的眼中闪过一丝精光。

"你在浪费体力。"

"而我们..."

"在保存体力。"

"现在..."

李武上前一步。

"该结束了。"

就在这时，再不斩身后突然出现了针状物。

数十根，直指再不斩。

"什么？！"再不斩大惊。

"谁？！"

一个戴着面具的少年出现在再不斩身后。

追忍。

"目标确认。"追忍的声音冰冷，"桃地再不斩，雾隐叛忍。"

"执行处决。"

针状物射出，再不斩被击中，倒在地上。

"死了吗？"追忍问道。

"不。"卡卡西看着再不斩的"尸体"，"只是假死。"

"那个追忍，用针击中了再不斩的要害。"

"但目的是让他昏迷，而不是杀死。"

"这..."

"应该是某种特殊的术。"

追忍摘下面具，露出一个少年的脸。

大概和李武同龄，脸上有一道独特的印记。

"你是？"佐助问道。

"白。"少年说道，"再不斩的同伴。"

"我来带他回去。"

白扛起再不斩，消失在森林中。

战斗，结束了。

李武站在原地，看着系统面板。

【经验+350】

"350经验..."李武在心中默念。

"虽然没有击败再不斩，但也获得了不少经验值。"

"而且..."

他看向自己的等级。

【角色：李武】
【等级：Lv.9（23.5%）】
【职业：无（可转职）】
【技能点：0】
【状态：正常】

"23.5%..."

"从18.7%提升到23.5%，增加了4.8%。"

"这场战斗..."

"虽然危险，但收益确实不错。"

"而且..."

李武的眼中闪过一丝兴奋。

"我验证了一个重要的结论。"

"我的战术分析能力..."

"在这个世界，是顶级的。"

"能精确判断敌人的查克拉消耗，能制定出合理的战术..."

"这种能力..."

李武握紧拳头。

"会让我在未来的战斗中，占据绝对的优势。"

"而且..."

他想起了刚才战斗中的"霸体"状态。

"再不斩的霸体..."

"虽然我无法破解，但我找到了对抗的方法。"

"消耗他的体力，等霸体解除..."

"这种方法..."

李武的眼中闪过一丝精光。

"虽然简单，但很有效。"

"将来的战斗中，面对具有霸体的敌人..."

"我都可以用这种方法对抗。"

"当然..."

李武想起了"抓取效果"。

"如果我掌握了具有抓取效果的技能..."

"比如，散打的'霸体护甲'和'闪电之舞'..."

"那时候..."

李武的眼中闪过一丝野心。

"我就不用等霸体解除了。"

"直接无视霸体，打断敌人的行动。"

"那种感觉..."

李武握紧拳头。

"一定很爽。"

"所以..."

他看向系统面板。

"必须尽快突破Lv.20，转职散打。"

"只有这样..."

"才能解锁核心技能。"

"才能真正变强。"

...

当晚，第七班在达兹纳家中休息。

卡卡西因为过度使用写轮眼，躺在床上休息。

佐助和小樱在训练。

李武坐在房间里，回想今天的战斗。

"再不斩的霸体..."

"白的追忍身份..."

"以及..."

李武想起了原著中，白的悲剧命运。

"白的悲剧，还要重演吗？"

"不..."

李武握紧拳头。

"我要改写这个命运。"

"白不需要死。"

"再不斩也不需要死。"

"他们可以活下来。"

"找到新的生存意义。"

"这是我的承诺。"

"也是我来这个世界的意义。"

就在这时，系统突然弹出了提示。

【主线任务更新】
【目标：保护达兹纳，完成波之国任务】
【当前进度：50%】
【下一阶段：击败再不斩与白】
【奖励：经验+500，技能点+1】
【时限：3天】

"3天..."李武在心中默念。

"也就是说，再不斩和白会再次来袭。"

"而且，这一次..."

"他们会全力以赴。"

"这种战斗..."

李武握紧拳头。

"必须做好万全的准备。"

"否则..."

"可能会死。"

"但是..."

李武的眼中闪过一丝坚定。

"我不会死。"

"因为我还有太多事情要做。"

"保护小李，改写悲剧，找到回家的路..."

"这些目标..."

"我都要完成。"

"所以..."

李武站起身，走到窗边。

"再不斩，白..."

"你们的悲剧命运..."

"我来改写。"

"用我的力量，用我的方法..."

"让你们都能活下去。"
