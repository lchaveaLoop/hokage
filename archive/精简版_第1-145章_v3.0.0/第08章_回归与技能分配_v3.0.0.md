# 第八章 回归与"技能分配"

**波之国 → 木叶隐村**

波之国任务完成后的第二天，第七班启程返回木叶。

达兹纳和工人们站在桥边，挥手告别。

"李武，佐助，小樱..."达兹纳感激地说道，"谢谢你们保护了我，也保护了这座桥。"

"这是我们的任务。"李武平静地说。

"而且..."

他看向伊那利。

那个曾经失去勇气的孩子，现在眼中重新燃起了希望。

"伊那利，记住。"李武认真地说，"真正的强者，不是没有恐惧，而是恐惧之后，依然选择勇敢。"

"就像..."

李武想起了鸣人。

虽然鸣人不在，但他的精神，应该传承下去。

"就像一个我认识的笨蛋一样。"

伊那利用力点头："我记住了！"

"很好。"李武微笑道。

"那么，再见了。"

"再见，李武大哥哥！"伊那利挥手。

...

回木叶的路上，气氛有些沉默。

佐助一直在思考着什么，小樱担心地看着他。

李武则在心中回想这次任务的收获。

"击败再不斩，获得800点经验值。"李武在心中默念。

"加上之前的修行..."

"从Lv.9突破到Lv.10。"

"而且..."

他看向系统面板。

【角色：李武】
【等级：Lv.10（5.8%）】
【职业：无（可转职）】
【技能点：1】
【状态：恢复中】

"技能点..."

李武的眼中闪过一丝兴奋。

"这是我第一个技能点。"

"虽然只有1点，但也足够解锁一些基础技能了。"

"而且..."

他回忆起前世DNF世界的技能系统。

"每升1级，获得1个技能点。"

"每5级，可以学习新的主动技能。"

"每10级，可以学习更高级的技能。"

"而转职后..."

"还能解锁职业专属技能。"

"这种成长体系..."

李武握紧拳头。

"会让我变得越来越强。"

"但是..."

他冷静地思考。

"技能点很珍贵。"

"必须慎重分配。"

"不能浪费。"

"那么..."

李武开始在心中构思技能加点方案。

"现在我是Lv.10，还没有转职。"

"能学习的技能，都是基础技能。"

"比如..."

"后跳、跃翔、受身蹲伏..."

"这些技能，都是基础中的基础。"

"但是..."

李武想起了前世DNF散打的技能树。

"散打的核心技能，是20级之后才能学习的。"

"比如..."

"霸体护甲、闪电之舞、纷影连环踢..."

"这些技能，才是散打的精髓。"

"所以..."

李武做出决定。

"现在的技能点，先不分配。"

"等到转职散打之后..."

"再加点。"

"这样..."

"技能点不会浪费。"

"而且..."

李武看向系统面板。

"从Lv.10到Lv.20..."

"需要10个技能点。"

"加上现在的1个..."

"总共11个技能点。"

"这些技能点..."

"足够点满散打的核心技能了。"

"很好。"

李武满意地点头。

"就决定这么办。"

...

木叶隐村。

火影办公室。

第七班正在汇报波之国任务。

"任务完成。"卡卡西恭敬地说道，"达兹纳先生已经安全，桥梁也建设完成了。"

"很好。"三代火影点头，"你们的成长，我都听说了。"

"特别是你，李武。"

三代看向李武，眼中闪过一丝欣赏。

"听说你在战斗中，展现了特殊的战斗方式。"

"那种...'机制'..."

"能否详细说说？"

李武沉默了片刻。

然后，他点头。

"可以。"李武认真地说，"我的战斗方式，基于一种特殊的感知能力。"

"我能'看到'敌人的查克拉流动，预判敌人的行动。"

"而且..."

"我能找到敌人的破绽，进行精确打击。"

"这种方法..."

李武想了想，决定用一个三代能理解的比喻。

"就像下棋。"

"普通人下棋，只能看到当前的局面。"

"但我能..."

"看到未来的局面。"

"预判对手的下一步，甚至下几步。"

"然后..."

"制定出完美的应对策略。"

"这种能力..."

三代沉默了。

能预判敌人的行动...

这种能力...

确实很罕见。

甚至...

可以说是"超越忍术"的存在。

"我明白了。"三代点头，"这种能力，确实是你的优势。"

"但是..."

三代认真地说。

"能力越强，责任越大。"

"你一定要谨慎使用这种能力。"

"不要让它成为你傲慢的资本。"

"明白了吗？"

"明白了。"李武点头。

"很好。"三代微笑道，"你们这次任务完成得很好。"

"可以休息几天了。"

"解散。"

第七班鞠躬，离开火影办公室。

...

走出火影办公室，佐助叫住了李武。

"李武，等一下。"

"怎么了？"李武转身。

"你的战斗方式..."佐助认真地说，"能教我吗？"

李武愣了一下。

"教？"

"是的。"佐助点头，"我看到了..."

"你在战斗中的表现。"

"那种预判能力..."

"那种精确打击..."

"如果我能学会..."

佐助握紧拳头。

"我就能更快地变强。"

"然后..."

"杀了那个男人。"

"鼬。"

李武沉默了。

他知道佐助的痛苦。

也知道佐助对力量的渴望。

但是...

"我的战斗方式，你学不会。"李武平静地说。

"为什么？！"佐助愤怒。

"因为..."李武认真地说，"这不是忍术，也不是体术。"

"这是..."

他想了想，决定用一个简单的解释。

"这是'天赋'。"

"或者说..."

"是'本能'。"

"我生来就有这种能力。"

"而你..."

李武看向佐助。

"你的天赋，在别的地方。"

"比如..."

"写轮眼。"

"你的写轮眼，能看穿敌人的动作，复制敌人的忍术。"

"这种能力..."

李武认真地说。

"不比我差。"

"甚至..."

"在某些方面..."

"更强。"

"所以..."

李武拍了拍佐助的肩膀。

"不要羡慕别人的能力。"

"珍惜你自己的。"

"因为..."

"那才是你能变强的关键。"

佐助沉默了。

良久，他点头。

"我明白了。"

"谢谢。"

"不客气。"李武微笑道。

"而且..."

李武的眼中闪过一丝精光。

"虽然我不能教你我的战斗方式..."

"但是..."

"我可以和你切磋。"

"通过实战..."

"让你更快地成长。"

"真的吗？"佐助的眼中闪过一丝兴奋。

"是的。"李武点头，"但是..."

"有个条件。"

"什么条件？"

"不能在小樱和卡卡西老师面前。"李武认真地说。

"因为..."

"我不想让他们知道我们在私自切磋。"

"明白了吗？"

"明白了。"佐助点头。

"很好。"李武笑道，"那么..."

"明天早上，演习场见。"

"好。"佐助转身离开。

李武看着佐助的背影，心中若有所思。

"佐助的堕落..."

"还会发生吗？"

"不..."

李武握紧拳头。

"有我在..."

"不会让佐助堕落的。"

"我会通过切磋..."

"让他感受到..."

"变强的正确方法。"

"不是靠力量，而是..."

李武想起了前世的DNF世界。

"靠机制。"

"靠理解。"

"靠对规则的掌控。"

"这种成长方式..."

"会让佐助变得更强。"

"而且..."

"不会迷失自我。"

...

回到家中，小李已经在等了。

"哥，你回来了！"小李兴奋地跑过来。

"嗯。"李武点头，"任务完成了。"

"太好了！"小李笑道，"我也想听听你的战斗故事。"

"以后再说。"李武摆了摆手，"现在..."

"我有更重要的事情。"

"什么事？"小李好奇。

"分配技能点。"李武认真地说。

"技能点？"小李疑惑。

"这是..."

李武想了想，决定用简单的解释。

"就像..."

"游戏里的技能点。"

"可以用来学习新的技能。"

"比如..."

李武做了一个踢击的动作。

"这种技能，需要技能点才能学习。"

"原来如此！"小李恍然大悟，"那么..."

"哥你要学什么技能？"

"暂时不学。"李武摇头。

"为什么？"小李不解。

"因为..."李武认真地说，"技能点很珍贵。"

"不能浪费。"

"我要等到转职之后..."

"再分配技能点。"

"这样才能..."

"学到最强的技能。"

"原来如此..."小李点头。

"那么..."

"哥你什么时候转职？"

"Lv.20。"李武平静地说。

"还有10级..."

"大概需要..."

李武快速计算。

"两个月吧。"

"两个月..."小李喃喃道。

"那么..."

"我也..."

李武看着小李，眼中闪过一丝惊讶。

"你也想转职？"

"不..."小李摇头，"我是说..."

"我也要在两个月内..."

"变强。"

"强到能追上哥哥。"

李武愣了一下。

然后...

他笑了。

"很好。"李武认真地说。

"但是..."

"变强不是一朝一夕的事。"

"需要日复一日的修行。"

"而且..."

"不能盲目修行。"

"必须..."

"找到正确的方法。"

"就像..."

李武想起了凯的教学方式。

"体术的本质..."

"不是力量，也不是速度。"

"而是..."

"对身体的掌控。"

"对时机的把握。"

"对距离的判断。"

"这些..."

"才是体术的真谛。"

"所以..."

李武看着小李。

"从明天开始..."

"我会指导你的修行。"

"用我的方法..."

"让你更快地成长。"

"真的吗？！"小李兴奋。

"是的。"李武点头，"但是..."

"有个条件。"

"什么条件？"

"必须听我的。"李武认真地说。

"不能偷懒，不能勉强。"

"一切按照我的计划来。"

"明白了吗？"

"明白了！"小李大声回答。

"很好。"李武笑道。

"那么..."

"从明天开始..."

"我们一起变强。"

"好！"小李握紧拳头。

"我一定会..."

"追上哥哥的！"

看着小李兴奋的样子，李武心中一暖。

"小李..."

李武在心中默念。

"我会保护你。"

"也会让你变强。"

"强到足以在这个残酷的世界里..."

"真正地生存下去。"

"甚至..."

"变得强大。"

"这是我的承诺。"

"也是我来这个世界的意义。"

...

晚上，李武坐在房间里，仔细研究技能系统。

他打开系统面板，查看可学习的技能列表。

【可学习技能列表】
- 后跳：1级（消耗1技能点）
- 跃翔：1级（消耗1技能点）
- 受身蹲伏：1级（消耗1技能点）
- 旋风腿：1级（消耗1技能点）
- 上勾拳：1级（消耗1技能点）
- 前踢：1级（消耗1技能点）

"6个基础技能..."李武在心中默念。

"但是..."

他摇头。

"现在不适合学习。"

"因为..."

"转职散打后，这些技能的作用会大打折扣。"

"所以..."

"还是等转职后再加点。"

"这样..."

"技能点不会浪费。"

"而且..."

李武看向等级。

【角色：李武】
【等级：Lv.10（5.8%）】
【职业：无（可转职）】
【技能点：1】
【状态：正常】

"Lv.10..."

"从Lv.10到Lv.20..."

"还需要10级。"

"每级大约需要5,000点经验值。"

"也就是说..."

"总共需要50,000点经验值。"

"这个数字..."

李武皱眉。

"确实不少。"

"但是..."

他冷静地思考。

"如果接取更多的任务..."

"挑战更强的对手..."

"获取经验值的速度..."

"会大幅提升。"

"特别是..."

李武想起了原著中的剧情。

"中忍考试..."

"那是一个绝佳的机会。"

"如果能击败那些天才忍者..."

"宁次、佐助、天天、牙..."

"经验值..."

李武的眼中闪过一丝兴奋。

"可能会大幅提升。"

"甚至..."

"一次性突破多个等级。"

"所以..."

李武握紧拳头。

"必须做好准备。"

"在中忍考试前..."

"突破Lv.20。"

"只有这样..."

"才能在考试中..."

"取得好成绩。"

"而且..."

"转职散打后..."

"实力会大幅提升。"

"那时候..."

"我就能..."

李武的眼中闪过一丝野心。

"挑战更强的对手。"

"比如..."

"影级强者。"

"甚至..."

"使徒。"

"当然..."

李武摇头。

"现在想这些还太早。"

"当务之急..."

"是快速提升等级。"

"方法..."

李武在心中列出计划：

"第一，接取大量任务。"
"第二，挑战强敌，通过切磋提升经验。"
"第三，指导小李修行，从教学相长中学习。"
"第四，深入研究战斗机制，优化战斗方式。"

"这些方法..."

李武满意地点头。

"应该能让我快速提升等级。"

"而且..."

他看向窗外。

"明天就要和佐助切磋了。"

"那是一个好机会。"

"通过切磋..."

"既能获得经验值..."

"又能帮助佐助成长。"

"一举两得。"

"很好。"

李武站起身，走到窗边。

"木叶的夜空..."

"还是那么美。"

"但是..."

李武的眼中闪过一丝坚定。

"我必须变强。"

"强到足以保护所有重要的人。"

"强到足以改写所有悲剧的命运。"

"这是我的承诺。"

"也是我来这个世界的意义。"
