# 第九章 切磋与"战斗直觉"

**木叶隐村 · 演武场**

清晨五点。

天还没亮，演武场已经空无一人。

李武准时到达。

五分钟后，佐助也到了。

"你来得早。"佐助说道。

"你也一样。"李武点头。

两人站在演武场中央，相距十米。

"那么..."李武认真地说，"切磋的规则是什么？"

"没有规则。"佐助冷冷地说，"全力战斗。"

"那么..."

"开始。"

速度快，力量大。

而且...

"这种状态下，佐助的洞察力会大幅提升。"

"他能看穿我的动作，预判我的攻击。"

"但是..."

"我也能预判他的动作。"

"查克拉感知..."

李武在心中默念。

"让我能'看到'佐助的查克拉流动。"

"我就已经知道他要攻击哪里。"

"这种能力..."

"就是'战斗直觉'。"

"或者更准确地说..."

"是'机制理解'。"

"对敌人行动规律的掌控。"
