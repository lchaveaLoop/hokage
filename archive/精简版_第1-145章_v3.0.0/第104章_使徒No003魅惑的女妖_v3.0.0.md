- 效果:操控被魅惑者的行动

- 破解:需要强大意志力

- 效果:操控目标情感
- 强度:可操控任何人
- 限制:需要接触

1. 物理防御极低
2. 没有近战能力
3. 魅惑需要时间
4. 精神攻击可被反制
5. 本体受伤会影响能力
