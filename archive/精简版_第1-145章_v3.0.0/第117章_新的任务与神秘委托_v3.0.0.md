# 第117章：新的任务与神秘委托

## 正文

第二天清晨，李武早早来到了火影大楼。纲手刚刚处理完公务，看到李武来访，立刻露出了笑容。

"小李子，这么早来找我有事？"纲手问道，递给李武一杯茶。

"纲手大人，关于昨天晚上的异常能量波动，我想向您了解一下。"李武开门见山地说道。

纲手的表情瞬间变得严肃起来。

"你也发现了？看来我的感觉没错。确实有事情发生。"

李武点点头，将昨晚的发现详细地告诉了纲手。

"能量波动很微弱，但确实存在，而且速度极快，我没能追上。"李武皱眉道，"而且，系统提示这可能和使徒有关。"

使徒战争已经过去了一个月，所有人都以为事情已经结束，但现在看来，背后还有更大的秘密。

"确实有异常。"纲手放下茶杯，"最近村子周围确实出现了几次不明能量波动，但我们一直以为是自然现象。"

【系统提示：检测到任务线索，是否接受调查任务？】

李武立刻在脑海中回应："接受！"

【任务系统】
主线任务：调查异常能量波动
任务等级：S级
任务描述：调查木叶村周围的异常能量波动，查明真相
任务奖励：经验值+5000、特殊技能点+100、可能获得使徒相关信息

"纲手大人，我想亲自调查这件事。"李武说道，"使徒的威胁还没有完全解除，我必须查清楚。"

纲手看着李武坚定的眼神，点了点头。

"我正有此意。而且，还有一件事。"

纲手从抽屉里拿出一封信，递给李武。

"这是什么？"

"一封神秘委托信。"纲手解释道，"昨天晚上放在火影大楼门口的，没有署名，只有这个任务委托。"

李武接过信，展开一看，顿时愣住了。

【委托任务：寻找失落遗迹】
委托等级：SS级
委托内容：寻找位于火之国边境的失落遗迹，据说与古代忍者文明有关
委托报酬：1,000,000两、稀有好物、传承忍术
委托人：神秘人

"SS级的任务？"李武惊讶道，"而且报酬这么丰厚，委托人却隐藏身份，这不寻常啊。"

"确实很奇怪。"纲手点头，"但这种任务往往是危机与机遇并存的。特别是与古代遗迹相关的任务，往往伴随着巨大的危险。"

【探索发现：古代遗迹线索】
古代遗迹位于火之国边境山脉深处
据说记载着失落的忍术文明
可能存在使徒力量的源头
危险等级：极高

"系统提示古代遗迹可能与使徒力量有关。"李武立刻将这个信息告诉了纲手。

纲手的眉头皱得更紧了。

"看来这两件事有联系。小李子，这个任务你愿意接吗？"

李武思考了一会儿，然后坚定地点头。

"我愿意。使徒的威胁还没有完全解除，我必须查清楚。而且，这种古代遗迹中很可能记载着对付使徒的关键信息。"

"好！"纲手满意地拍拍李武的肩膀，"准备一下吧，这个任务可能会很危险。"

【系统提示：接受委托任务，获得基础装备奖励】

李武感觉到一股暖流涌入体内，他的装备得到了提升。

"这是委托任务的初始装备。"纲手解释道，"应该能帮到你。"

李武检查了一下自己的装备，发现多了一把特制的苦无，上面刻着封印术式，专门用来对付异常能量。

"谢谢纲手大人。"李武感激地说道。

"不用客气。对了，你可以叫上鸣人他们一起。"纲手建议道，"人多力量大，而且鸣人那小子最近也成长了不少。"

李武想了想，觉得有道理。鸣人、佐助和小樱都是可靠的伙伴，特别是鸣人，他的九尾之力在关键时刻往往能发挥意想不到的作用。

"我会的。"李武点头，"那我先去准备一下。"

离开火影大楼后，李武直接来到了训练场。鸣人正在那里练习螺旋丸，看到李武来了，立刻跑了过来。

"李武老师！您找我？"鸣人兴奋地问道，最近李武教导他们让他学到了很多东西。

"鸣人，有个任务要你们一起参加。"李武说道，"可能和使徒有关，需要你们的帮助。"

提到使徒，鸣人的表情立刻变得严肃起来。

"使徒？他们不是已经被消灭了吗？"

"恐怕没有。"李武摇头，"昨天晚上我发现了一股异常能量波动，可能和使徒有关。而且，我接到了一个寻找古代遗迹的任务，据说那里可能记载着使徒的真相。"

鸣人听得入了迷，眼中闪烁着兴奋的光芒。

"真的吗？那太好了！我一定要参加！"

"还有你们。"李武看向正在一旁练习的小樱和佐助。

"我也要参加！"小樱立刻举手，"我的医疗忍术在关键时刻可能会派上用场。"

佐助虽然沉默，但也点了点头："虽然我讨厌麻烦，但如果和使徒有关，我必须参加。"

看到大家都有参与的热情，李武欣慰地点头。

"好，那我们明天就出发。记得做好准备，这个任务可能会很危险。"

【团队组建：木叶小队】
队长：李武（Lv.60）
成员：漩涡鸣人（Lv.45）宇智波佐助（Lv.43）春野樱（Lv.42）
特殊能力：九尾之力、写轮眼、医疗忍术
任务目标：寻找失落遗迹，调查异常能量

回到家里，李武开始准备这次任务需要的物品。除了纲手给的特制苦无外，他还准备了一些卷轴和药物。

【装备准备完成】
特制苦无*1（封印属性）
查克拉恢复药*10
治愈卷轴*5
侦查卷轴*3
能量探测器*1（系统赠送）

"应该差不多了。"李武检查着装备，满意地点点头。

就在这时，系统突然发出提示。

【系统提示：检测到特殊任务触发，是否接受？】

"什么特殊任务？"李武好奇地问道。

【隐藏任务：遗迹守护者】
任务等级：SSS级
任务描述：失落遗迹中存在着古老的力量，需要特殊的守护者才能激活
任务条件：李武的使徒之力与遗迹能量产生共鸣
任务奖励：未知传承、力量突破

SSS级的任务？李武心中一动。这个任务显然不简单，而且与自己体内的使徒之力有关。

"我接受！"李武毫不犹豫地答应了。

【任务绑定成功，获得遗迹感应能力】

李武感觉到自己的身体发生了一些变化，能够隐约感知到遗迹的大致位置。

"看来这次任务比我想象的还要复杂。"李武握紧了拳头，"不过，越是危险，收获可能就越大。"

第二天一早，木叶小队集合在了村口鸣人家里。鸣人已经迫不及待地准备好了，佐助则是一副冷静的样子，小樱则在仔细检查医疗用品。

"都准备好了吗？"李武问道。

"准备好了！"三人异口同声地回答。

"那我们出发吧。"李武拿出能量探测器，"先向这片区域出发，根据探测器的指引寻找遗迹。"

李武启动能量探测器，指针开始缓慢转动，最终指向了东北方向。

"东北方向，火之国边境。"李武说道，"看来我们要穿越一片山脉。"

"好嘞！那就出发吧！"鸣人兴奋地喊道，九尾查克拉开始涌动。

四人踏上了征程。对于木叶的年轻忍者来说，这不仅是一次任务，更是一次成长的历练。而对于李武来说，这可能是揭开使徒真相的关键。

【任务进行中：探索未知区域】

沿途风景秀丽，但李武知道，平静的表面下可能隐藏着危险。使徒的力量远比他们想象的更加复杂，而这次的任务，可能只是冰山一角。

"不管前面有什么，我都会保护好大家。"李武暗下决心，"为了木叶的和平，为了所有人的安全。"

队伍向着未知前进，新的冒险即将开始。而那神秘的委托人，究竟是谁？古代遗迹中又隐藏着什么秘密？这一切，都需要他们去揭开。

【第117章 完】
