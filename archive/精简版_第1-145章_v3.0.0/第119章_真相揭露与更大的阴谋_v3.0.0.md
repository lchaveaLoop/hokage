# 第119章：真相揭露与更大的阴谋

## 正文

李武全力催动遗迹守护者模式，强大的能量从体内涌出，化作一道光束射向位面裂缝。裂缝中的异界生物感受到了威胁，发出阵阵低吼，变得更加狂暴。

"吼——！"

一声震天动地的咆哮从裂缝中传出，一个巨大的身影开始浮现。那是一个类似人类但更加扭曲的生物，全身覆盖着黑色的鳞片，眼睛闪烁着红色的光芒。

【敌人出现：位面领主】
等级：未知（远超当前地图Boss）
特点：位面掌控者，能够撕裂空间
弱点：能量核心，但位置未知

"这...这是什么怪物？"鸣人震惊地看着眼前的景象，"这绝对不是普通的忍者能够对付的！"

李武也感到一阵心悸。这种级别的敌人，已经超出了他们的认知范围。就算是他全力发动遗迹守护者模式，也没有绝对的把握。

"必须想办法找到它的弱点！"李武冷静地分析道，"但它的核心在哪里？"

位面领主似乎感受到了众人的威胁，猛地挥动爪子，一道空间裂痕朝着众人射来。

"小心！"李武立刻吼道，"空间攻击！"

众人立刻分散躲避，但空间裂痕的速度太快了，佐助虽然反应迅速，但还是被擦中了肩膀。

"啊！"佐助闷哼一声，肩膀上出现了一道深可见骨的伤口。

"佐助！"小樱立刻冲过去治疗，"伤口太深了，我的医疗忍术可能需要时间！"

"没关系，我能撑住。"佐助咬牙说道，但脸色已经变得苍白。

位面领主没有给他们太多时间，继续发动攻击。它的攻击方式非常奇特，能够撕裂空间，制造出各种空间陷阱。

"这家伙太强了，"鸣人一边躲避攻击，一边说道，"我们根本打不过它！"

"李武老师，你的能力能支撑多久？"小樱焦急地问道。

李武感受着体内的能量，眉头紧锁。

"遗迹守护者模式只能维持10分钟，而且消耗极大。在这之前，必须找到解决办法！"

就在这时，系统突然发出了提示。

【系统提示：检测到古代遗物能量，是否融合？】

古代遗物？李武立刻想起了神殿中央石台上的那个神秘物品。

"融合遗物？那会有什么后果？"

【系统提示：融合后可获得位面掌控力，但风险极大，可能导致永久性力量提升或失控】

风险极大，但可能是唯一的希望。李武没有犹豫。

"融合！"

【遗物融合开始】
融合度：10%
力量提升：50%
副作用：未知

李武伸出手，触摸到了石台上的遗物。瞬间，一股强大的能量涌入他的体内，与他的使徒之力产生共鸣。

"啊——！"李武忍不住惨叫一声，感觉自己的身体快要被撕裂了。

"李武老师！"鸣人担忧地喊道。

"我没事，"李武咬牙坚持道，"继续！"

【融合度：30%】
力量提升：100%
副作用：身体开始承受巨大压力

随着融合的进行，李武的身体开始发生变化。他的皮肤上浮现出古老的符文，眼睛也变成了金色。这股力量虽然强大，但也非常危险。

"吼——！"

位面领主似乎感受到了李武的变化，变得更加狂暴。它张开大嘴，一道巨大的空间能量球朝着众人射来。

"完了！"小樱惊呼道。

就在这危急时刻，李武睁开了眼睛。

"遗迹之力，位面掌控！"

一道金色的光罩笼罩在众人周围，空间能量球撞在光罩上，竟然被弹开了。

"成功了！"鸣人惊喜地喊道。

"只是暂时的，"李武面色凝重地说道，"融合还在继续，但这样下去我会失控的！"

【融合度：60%】
力量提升：200%
副作用：精神开始混乱

李武感觉到自己的意识开始模糊，脑海里浮现出各种奇怪的景象。那是古代遗迹的记忆，是关于使徒的真相。

"啊——！"李武痛苦地抱着头，"这些记忆...到底是什么？"

就在这时，李武的脑海中浮现出了一个惊人的真相。

【记忆回溯：使徒的起源】

原来，使徒并不是简单的入侵者，而是古代文明的守护者。他们拥有强大的力量，用以维护位面的平衡。但是，随着时间推移，使徒的力量变得越来越不稳定，最终失去了控制，变成了威胁。

"什么？使徒是守护者？"李武震惊不已，"那他们为什么要攻击木叶？"

【记忆回溯：位面失衡】

原因很简单——位面失衡。由于某些未知的原因，多个位面的平衡被打破，导致使徒的力量变得狂暴。他们本能地寻找稳定的空间，但在这个过程中，造成了巨大的破坏。

"所以，真正的敌人不是使徒，而是位面失衡？"李武恍然大悟。

就在这时，系统的声音再次响起。

【系统提示：融合完成，获得位面掌控能力，同时发现真相】

【真相揭露：使徒计划】

使徒们并非单纯想要毁灭，而是在寻找一种方法来恢复位面平衡。他们选择了木叶，是因为木叶的查克拉浓度异常，可能成为修复位面的关键。

"所以，使徒战争只是开始，真正的任务是修复位面平衡？"李武惊讶地问道。

【系统提示：正确，但还有一个更大的阴谋】

更大的阴谋？李武心中一紧。

【阴谋揭示：幕后黑手】

有人在利用使徒，试图通过位面失衡来实现某种目的。这个人隐藏在暗处，操控着一切，包括委托他们寻找遗迹的任务。

"是那个神秘的委托人！"李武瞬间明白了，"这一切都是他的计划！"

【系统提示：位面领主就是幕后黑手创造的怪物，用来测试众人的能力】

果然，位面领主突然停止了攻击，冷漠地看着众人。

"呵呵，终于明白了？"一个声音从裂缝中传出，"没想到你们竟然能够发现真相。"

"你是谁？"李武冷冷地问道，"为什么要这么做？"

"我是谁不重要，重要的是，你们的计划即将成功。"声音神秘地说道，"位面失衡是不可避免的，但我会利用这个机会，获得前所未有的力量！"

【阴谋升级：位面融合计划】

幕后黑手想要通过位面失衡，将多个位面融合在一起，创造一个全新的世界。而木叶，将成为融合的中心。

"什么？他要融合位面？"李武震惊不已，"这会带来什么后果？"

【系统提示：位面融合将导致所有被融合位面的毁灭，只有幕后黑手能够在新世界中存活】

太过分了！这个人不仅要毁灭木叶，还要毁灭无数位面的生命！

"我们不能让他得逞！"李武坚定地说道，"必须阻止他！"

但问题是，他们该怎么阻止？位面领主的实力太过强大，就连融合了古代遗物的李武也没有绝对的把握。

就在这时，系统再次发出了提示。

【系统提示：检测到特殊能量波动，是否激活最终形态？】

最终形态？李武心中一动。这可能就是最后的希望了。

"激活！"

【最终形态激活】
形态名称：使徒守护者
力量提升：500%
持续时间：5分钟
代价：生命能量消耗50%

随着最终形态的激活，李武的身体发生了翻天覆地的变化。他的背后展开了一对巨大的能量翅膀，全身被金色的能量包裹。现在的他，已经超越了忍者的范畴，成为了真正的使徒守护者。

"这就是使徒守护者的力量吗？"李武感受着体内涌动的力量，"太强大了！"

位面领主感受到了威胁，变得更加疯狂。它张开大嘴，准备发动最终攻击。

"吼——！"

一道巨大的空间能量波朝着众人射来，威力比之前强了十倍不止。

"完了！"小樱绝望地喊道。

但李武没有放弃。他张开双翼，挡在了众人面前。

"使徒之力，位面屏障！"

一道强大的能量屏障出现在众人面前，空间能量波撞击在屏障上，竟然被完全阻挡了。

"竟然...挡住了..."位面领主震惊地看着眼前的景象，"你...你到底是谁？"

"我是木叶的守护者，"李武坚定地说道，"我不会让你伤害任何人！"

"呵呵，就凭你？"位面领主冷笑道，"你虽然强大，但想要阻止位面融合，还早了一百年！"

"是吗？"李武微笑道，"那你就试试看！"

李武瞬间移动到位面领主面前，一拳挥出。

【技能升级：使徒之拳 Lv.1 → Lv.5】

位面领主反应不及，被击中了腹部。但它也不是等闲之辈，立刻反手一爪，划向李武的胸口。

"砰！"

两人硬碰了一招，各自后退了几步。

"你的力量确实很强，"位面领主说道，"但还不够！"

"是吗？"李武眼中闪过精光，"那让你见识一下真正的力量！"

李武双手结印，全身的查克拉涌动。

"使徒奥义·位面崩灭！"

一道金色的能量柱冲天而起，朝着位面领主射去。这一击，蕴含了李武所有的力量！

位面领主感受到了致命的威胁，不再保留，全力迎击。

"空间裂灭！"

一道黑色的能量柱与金色的能量柱在空中相撞，产生了剧烈的爆炸。

"轰——！"

爆炸的余波席卷了整个神殿，石壁纷纷倒塌。但众人都没有受伤，因为李武早就做好了防护。

烟尘散去，位面领主的状态很不乐观。它的身体多处受伤，能量也在迅速流失。

"不可能...你竟然...这么强..."位面领主难以置信地说道。

"这只是开始，"李武冷冷地说道，"说出幕后黑手的真实身份，我可以考虑给你一个痛快。"

位面领主冷笑一声，"你想要知道？那自己去问吧！"

说完，它的身体开始崩解，化为了无数的光点消散在空气中。

"它...它自爆了？"鸣人惊讶地说道。

不，不是自爆，位面领主选择了同归于尽。但在消散的最后一刻，李武感受到了一股特殊的能量波动。

【系统提示：检测到线索，指向幕后黑手的位置】

李武立刻闭上眼睛，感知着那股能量波动。很快，他就锁定了位置。

"在...位面夹层中？"李武惊讶地说道，"那个家伙竟然躲在位面夹层里？"

【真相升级：幕后黑手身份】

位面夹层是位面之间的缓冲地带，是理论上最安全的地方。但那个家伙竟然选择躲在那里，说明他有绝对的自信。

"我们必须去阻止他！"李武坚定地说道，"否则，整个世界都会被他毁灭！"

但问题是，位面夹层不是一个普通人能够进入的地方。即使是现在的李武，也没有绝对的把握。

就在这时，系统再次发出了提示。

【系统提示：检测到特殊通道，可通往位面夹层】

李武眼前一亮，立刻说道："有了！古代遗物可以开启通往位面夹层的通道！"

"那我们还等什么？"鸣人立刻说道，"快走！"

众人立刻离开了神殿，带着古代遗物，准备前往位面夹层。

但就在这时，李武突然感觉一阵头晕。

"怎么了，李武老师？"小樱关切地问道。

"没事...只是使用最终形态的代价开始显现了，"李武苦笑道，"消耗太大了..."

"那我们真的要去位面夹层吗？"佐助冷静地分析道，"那里太危险了，我们可能回不来。"

"必须去，"李武坚定地说道，"这是我们的责任。不管付出什么代价，都要阻止那个家伙！"

众人看着李武坚定的眼神，都点了点头。

"那我们一起去吧！"鸣人说道，"我们是伙伴，要一起面对！"

"好！"李武欣慰地点头，"那我们准备出发吧！"

新的征程即将开始，位面夹层中的终极对决，将是决定世界命运的关键！
