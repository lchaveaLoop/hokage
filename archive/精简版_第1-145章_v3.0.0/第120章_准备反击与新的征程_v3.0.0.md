# 第120章：准备反击与新的征程

## 正文

离开古代遗迹后，木叶小队在附近找了一个安全的地方休整。李武因为使用了最终形态，身体非常虚弱，需要时间恢复。

"李武老师，你感觉怎么样？"小樱关切地问道，一边为李武处理伤口。

"还好，就是消耗太大，"李武苦笑道，"最终形态的代价比我想象的还要大。"

"我们应该先回木叶，佐助冷静地分析道，"你的状态不适合立刻进入位面夹层。"

鸣人立刻反对道："不行！那个家伙随时可能毁灭世界，我们必须立刻行动！"

"但李武老师现在这个状态，怎么去位面夹层？"小樱担忧地说道。

众人陷入了沉默。确实，李武现在的状态很差，贸然进入位面夹层只会送死。但如果不尽快行动，那个幕后黑手就可能完成位面融合的计划。

【系统提示：检测到恢复机会，是否接受？】

李武心中一动："什么恢复机会？"

【系统提示：可利用古代遗物的能量进行恢复，但需要时间】

古代遗物？李武立刻想起了他们带回来的那个神秘物品。

"有了！"李武眼睛一亮，"我们可以利用古代遗物的能量进行恢复！"

众人立刻围了过来，"真的吗？真的可以恢复？"

"应该可以，"李武说道，"古代遗物蕴含着强大的能量，而且与我的使徒之力相容，应该可以帮助我恢复。"

"那还等什么？快用啊！"鸣人迫不及待地说道。

李武点点头，小心翼翼地拿出了古代遗物。那是一块散发着神秘光芒的石头，上面刻着无数古老的符文。

"让我试试。"李武将手掌贴在遗物上，开始引导其中的能量。

【能量恢复开始】
恢复速度：10%/分钟
副作用：可能吸收过多能量导致失控

随着能量的流动，李武感觉到自己的体力在慢慢恢复。虽然速度不快，但确实有效。

"太好了！"鸣人高兴地说道，"这样李武老师就能恢复战斗力了！"

"别高兴得太早，"佐助冷静地分析道，"恢复需要时间，而且那个家伙可能已经察觉到了我们。"

确实，幕后黑手绝对不是等闲之辈，他们刚刚摧毁了位面领主，那个家伙不可能没有察觉。

"我们必须做好最坏的打算，"李武说道，"在恢复的同时，也要制定作战计划。"

众人立刻开始讨论作战计划。

"根据之前的情报，位面夹层是一个特殊的维度，介于各个位面之间，"李武分析道，"在那里，常规的忍术可能会受到限制。"

"那我们该怎么办？"小樱问道。

"我们需要发挥各自的优势，"李武说道，"鸣人的九尾之力在特殊维度中可能会有意想不到的效果，佐助的写轮眼能够看穿幻术和陷阱，小樱的医疗忍术则是我们的保障。"

"那你呢？"鸣人问道，"你是最重要的战斗力。"

"我会尽快恢复，"李武说道，"但即使恢复了，我也不敢保证能够单独战胜那个家伙。我们需要配合！"

就在这时，系统突然发出了提示。

【系统提示：检测到特殊能力升级，是否接受？】

"什么能力升级？"李武好奇地问道。

【系统提示：位面掌控力升级，可获得新的技能】

"当然要接受！"李武立刻说道。

【位面掌控力升级】
新技能：位面感知（可感知位面中的异常能量）
新技能：位面穿梭（可在位面间短距离移动）
新技能：位面守护（可创造小型位面屏障）

这些技能太有用了！特别是在位面夹层中，这些技能可能会成为决定胜负的关键。

"太好了！"李武兴奋地说道，"我现在有了更好的能力来对付那个家伙！"

"那我们现在就出发吧？"鸣人跃跃欲试地说道。

"不，还要等等，"李武摆摆手，"我的恢复还没有完成，而且我们需要准备一些东西。"

接下来的一天，木叶小队开始为进入位面夹层做准备。李武利用古代遗物的能量进行恢复，同时练习新获得的技能。

【位面感知掌握】
可以感知位面中的能量波动
范围：100米
精度：高

【位面穿梭掌握】
可在位面间短距离移动
消耗：中等
冷却时间：5分钟

【位面守护掌握】
可创造小型位面屏障
防护等级：中
持续时间：10分钟

这些技能虽然强大，但也需要时间来熟练掌握。李武花了整整一天的时间来练习，才勉强掌握了基本的使用方法。

"终于差不多了，"李武擦了擦汗，"虽然还不能完全熟练，但应该够用了。"

"那我们什么时候出发？"鸣人问道。

"今晚，"李武看了看天色，"趁夜色掩护，我们悄悄前往位面夹层。"

众人点点头，开始最后的准备。每个人都带上了自己最强的武器和道具，尤其是小樱，准备了大量的医疗用品和解毒药。

"都准备好了吗？"李武问道。

"准备好了！"众人异口同声地回答。

"那我们出发吧！"

夜深人静，木叶村已经进入了梦乡。木叶小队悄悄离开了村子，向着位面夹层的方向前进。

根据系统的指引，位面夹层的入口在一片特殊的山谷中。那里是位面之间的薄弱点，是最容易进入位面夹层的地方。

"就是这里了，"李武停下脚步，"前面就是位面夹层的入口。"

众人抬头看去，只见山谷中有一道闪烁着光芒的裂缝，这就是通往位面夹层的通道。

"我们要从这里进去？"鸣人问道。

"是的，"李武点头，"但要做好准备，里面可能很危险。"

"我们已经准备好了！"众人坚定地说道。

李武深吸一口气，说道："那我们进去吧！"

四人小心翼翼地进入位面裂缝。眼前的一切都变得模糊起来，仿佛穿越了无数个空间。当他们再次睁开眼睛时，已经来到了一个完全不同的世界。

【进入位面夹层】

眼前的景象让所有人都惊呆了。这里是一片灰色的空间，看不到尽头，到处都是漂浮的岩石和破碎的能量碎片。天空中是紫色的闪电，时不时地划过天空。

"这里...就是位面夹层？"鸣人震惊地看着周围的景象。

"是的，"李武皱眉道，"比我想象的还要危险。"

【系统提示：检测到多处能量波动，可能是幕后黑手的陷阱】

"果然有陷阱，"李武立刻说道，"大家小心！"

突然，周围的岩石开始移动，组合成各种奇怪的形状。

"是陷阱！"佐助立刻警觉道。

但已经晚了，周围的岩石突然爆炸，无数的碎石向众人射来。

"防御！"李武立刻喊道。

众人立刻做出反应。李武使用位面守护创造了一个屏障，小樱使用医疗忍术强化了大家的防御，鸣人则使用螺旋丸反击，佐助则用火遁术掩护。

"轰！"

爆炸的威力很强，但众人的防御也很顽强。虽然有些受伤，但都不严重。

"这些只是前哨战，"李武分析道，"真正的主使还在后面。"

就在这时，一个神秘的声音从天空中传来。

"呵呵，没想到你们竟然能够来到这里。"声音很飘渺，似乎来自四面八方。

"是谁？出来！"鸣人喊道。

"我是谁不重要，重要的是，你们的计划即将失败。"声音继续说道，"位面融合是不可逆转的，你们阻止不了！"

"是吗？"李武冷冷地说道，"那我们就试试看！"

"呵呵，不自量力。"声音轻蔑地说道，"不过，我倒是可以给你们一个机会。"

"什么机会？"

"加入我，成为新世界的统治者。"声音诱惑地说道，"新世界将更加美好，你们可以获得无尽的力量！"

"做梦！"鸣人立刻拒绝道，"我们才不会和你这种家伙合作！"

"那就别怪我不客气了！"声音变得愤怒，"既然你们不识抬举，那就一起毁灭吧！"

随着声音落下，周围的能量开始变得狂暴。无数的能量碎片开始聚集，形成了一个巨大的能量球，朝着众人射来。

"不好！是能量攻击！"小樱惊呼道。

"分散躲避！"李武立刻喊道，"这种攻击范围很大！"

众人立刻分散，但能量球的速度太快了，还是被擦中了。

"啊！"众人同时惨叫一声，都被击中了。

"伤得不轻，"李武咬牙站起，"这种攻击太强了！"

"我们必须反击！"鸣人喊道，"不能一直被动挨打！"

"说得对，"李武点头，"使用组合技！"

"我明白了！"佐助立刻说道，"火遁·豪火球之术+螺旋丸的组合！"

"好！我来辅助！"小樱说道，"医术·掌仙术，麻痹敌人！"

众人立刻开始结印。佐助和鸣人同时发动攻击，一个巨大的火球和螺旋丸融合在一起，形成了一个火焰螺旋丸，朝着能量球射去。

"轰！"

火焰螺旋丸与能量球相撞，产生了剧烈的爆炸。但这次的效果不错，能量球被击碎了。

"成功了！"鸣人兴奋地喊道。

"但只是暂时的，"李武皱眉道，"那个家伙肯定还有更强的攻击。"

果然，天空中再次传来声音。

"有点意思，但还不够！"声音说道，"那就让你们见识真正的力量！"

随着声音落下，整个位面夹层开始震动。天空中的紫色闪电变得更加密集，能量碎片也开始疯狂移动。

"不好！他在准备大招！"李武立刻说道，"我们必须找到他的本体！"

"怎么找？这里这么大！"鸣人问道。

"我有办法，"李武说道，"使用位面感知！"

李武闭上眼睛，全力催动位面感知能力。在这个特殊的位面中，位面感知的效果比想象中还要好。

【位面感知发动】
范围扩大到1000米
发现多处能量源
但本体能量最强

找到了！李武睁开了眼睛，"在他的方向！"

"大家一起上！"李武喊道，"使用最强的攻击！"

众人立刻朝着李武指出的方向冲去。但当他们到达那里时，却发现什么都没有。

"人呢？"鸣人惊讶地问道。

"陷阱！"佐助立刻警觉道。

果然，在他们到达的瞬间，周围的能量碎片开始聚集，形成了一个巨大的能量怪兽。

【最终敌人：位面聚合体】
等级：未知
特点：可吸收位面能量
弱点：能量核心不稳定

"这是...什么？"小樱震惊地看着眼前的景象。

"是位面聚合体，"李武分析道，"那个家伙用位面能量创造出来的怪物！"

"那我们怎么办？"鸣人问道，"这种怪物看起来很强！"

"找到它的核心！"李武立刻说道，"这种怪物的核心通常是不稳定的！"

众人立刻开始攻击位面聚合体。但聚合体的防御力很强，普通的攻击根本无法造成有效的伤害。

"我们需要更强的攻击！"李武说道，"使用合体忍术！"

"合体忍术？"小樱惊讶地问道。

"对，我们的合体忍术！"李武说道，"螺旋仙法·风遁·螺旋手里剑！"

李武想到了鸣人在仙人模式下使用的忍术，但现在的他们没有仙人模式，但可以创造自己的合体忍术。

"明白了！"鸣人立刻说道，"我来准备螺旋手里剑！"

佐助也点头道："我来提供火遁加持！"

小樱则说道："我来强化你们的查克拉！"

三人立刻开始配合。鸣人制造出螺旋手里剑，佐助的火遁加持在上面，小樱则强化了他们的查克拉。最后，李武使用位面之力，将所有的力量都注入其中。

"螺旋仙法·风火遁·超螺旋手里剑！"

一个巨大的螺旋手里剑呼啸而出，直接命中了位面聚合体的核心。

"轰——！"

剧烈的爆炸让整个位面夹层都在震动。爆炸过后，位面聚合体终于崩溃了，化为了无数的能量碎片消散在空气中。

"成功了！"鸣人兴奋地喊道。

但就在这时，那个声音再次响起。

"呵呵，没想到你们竟然能够摧毁我的聚合体。"声音说道，"不过，这只是开始！"

"你到底是谁？为什么要做这种事？"李武冷冷地问道。

"我是谁？"声音轻笑道，"我就是新世界的创造者！而你们，将成为新世界的第一批居民！"

"你做梦！"鸣人立刻说道，"我们才不会让你毁灭这个世界！"

"那就别怪我不客气了！"声音变得愤怒，"既然你们不识抬举，那就一起毁灭吧！"

随着声音落下，整个位面夹层开始崩溃。天空中的紫色闪电变成了紫色的火焰，到处都是毁灭的气息。

"位面要崩塌了！"小樱惊呼道。

"我们必须离开这里！"李武立刻说道，"不然会被困在这里！"

"怎么离开？"鸣人问道。

"使用位面穿梭！"李武说道，"但只能传送一个人！"

"那我们怎么办？"佐助问道。

"我先走，"李武说道，"然后在外面接应你们！"

"不行！"鸣人立刻反对道，"我们要一起走！"

"没时间了！"李武着急地说道，"位面崩塌的速度太快了！"

就在这时，系统突然发出了提示。

【系统提示：检测到特殊通道，可一次性传送所有人】

"有了！"李武眼睛一亮，"古代遗物可以开启紧急通道！"

李武立刻拿出古代遗物，全力催动其中的能量。

"位面紧急通道，开启！"

一道金色的光芒从遗物中射出，形成了一个稳定的通道。

"快进去！"李武喊道，"从这里离开！"

四人立刻冲进了通道。就在他们离开的瞬间，整个位面夹层彻底崩塌了。

【位面崩塌完成】

当四人再次睁开眼睛时，已经回到了原来的山谷中。但他们的脸色都非常凝重。

"我们成功了？"鸣人不敢相信地问道。

"暂时，"李武摇头道，"但那个家伙还没有出现。"

就在这时，山谷中突然出现了一个黑影。那是一个穿着黑色长袍的人，脸上戴着面具，看不清真实面目。

"终于见面了，"黑影说道，"没想到你们竟然能够逃脱。"

"你到底是谁？"李武冷冷地问道，"为什么要做这种事？"

黑影沉默了一会儿，然后说道："我是新世界的创造者，而你们，将成为我新世界的垫脚石。"

"我绝对不会让你得逞的！"李武坚定地说道。

"是吗？"黑影轻笑道，"那就让我们看看，谁才是真正的强者！"

黑影张开双手，一股强大的能量波动从体内涌出。

"决战，现在开始！"

新的征程，最终的对决，即将展开！

【第120章 完】
