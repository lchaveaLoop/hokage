# 第121章：进入位面夹层与异界风光

## 正文

决战在即，木叶小队站在山谷之中，与那个神秘的黑影对峙着。空气中弥漫着紧张的气息，仿佛连风都停止了流动。

"你就是幕后黑手？"李武冷冷地盯着眼前的黑影，眼神中充满了警惕。

黑影缓缓抬起头，面具下的双眼闪烁着诡异的光芒："没错，我一直在等待着你们的出现。"

"你到底是谁？为什么要做这种事？"鸣人质问道，九尾的查克拉在体内躁动不安。

"我是谁并不重要，"黑影轻笑道，"重要的是，新世界的即将到来。"

"你所谓的'新世界'，就是要毁灭现在的世界吗？"佐助冷冷地问道，写轮眼已经开启，准备随时发动攻击。

黑影沉默了一会儿，然后说道："不是毁灭，而是进化。这个世界已经腐朽了，需要被重塑。"

"胡说！"小樱愤怒地说道，"这个世界有很多美好的东西，为什么要毁了它？"

黑影摇头道："你们不懂，你们只看到了表面的美好，却没有看到深层的腐朽。"

"够了！"李武打断了他的话，"不管你的理由是什么，我都不会让你伤害这个世界！"

"是吗？"黑影轻笑道，"那我们就来一场真正的对决吧！"

黑影突然动了！只见他抬手一挥，一道黑色的能量波朝着众人射来。

"小心！"李武立刻喊道，"大家散开！"

众人立刻分散躲避，但能量波的速度太快了，还是擦中了边缘。

"轰！"

能量波击中地面，炸出了一个巨大的坑洞。众人稳住身形，脸上都露出了凝重的表情。

"这个人的力量比想象中还要强！"李武心中暗道，"必须想办法拖住他，直到我们恢复全部力量。"

"位面穿梭！"李武立刻使用新获得的能力，身影瞬间消失在原地。

"想逃？没那么容易！"黑影冷哼一声，伸手一抓，一道黑色的锁链从天而降，朝着李武的位置射去。

李武刚刚出现，就看到了迎面射来的黑色锁链，心中一惊，立刻使用位面守护创造了一个屏障。

"砰！"

锁链击中屏障，发出一声巨响。李武被震得后退了几步，胸口一阵发闷。

"有点意思。"黑影轻笑道，"竟然能够躲过我的攻击。"

"你的力量确实很强，"李武承认道，"但你不可能战胜我们整个团队！"

"是吗？"黑影说道，"那就让我看看你们的真正实力！"

黑影双手结印，周围的空气开始扭曲，一股强大的压力从天而降。

"这是...什么忍术？"鸣人震惊地看着眼前的景象。

"不是忍术，"李武皱眉道，"是位面之力！他在动用位面夹层的力量！"

随着黑影的动作，整个山谷都在震动。天空中的云层开始聚集，形成了一个巨大的漩涡。

"不好！他在准备更大的攻击！"李武立刻说道，"我们必须阻止他！"

"怎么阻止？这种力量太强了！"佐助冷静地分析道。

"我有办法，"李武说道，"使用位面感知！"

李武闭上眼睛，全力催动位面感知能力。在这个特殊的地方，位面感知的效果比想象中还要好。

【位面感知发动】
范围：500米
发现多处能量源
正在快速聚集

"找到了！"李武睁开眼睛，"在他的核心位置！"

"那是什么？"小樱问道。

"是他的能量核心，"李武分析道，"可能是他的弱点！"

"那我们怎么办？"鸣人问道。

"我来创造机会，"李武说道，"你们负责攻击核心！"

"明白！"三人异口同声地回答。

李武深吸一口气，说道："位面守护！"

一道金色的屏障出现在众人面前，形成一个完美的防护罩。

"现在！"李武喊道。

三人立刻冲了出去。鸣人制造出螺旋丸，佐助使用火遁·豪火球之术，小樱则用医疗忍术强化了他们的攻击。

"螺旋丸！"
"火遁·豪火球之术！"
"医术·查克拉强化！"

三道攻击同时射向黑影的核心位置。

黑影似乎没有料到他们会突然攻击，被击中了核心位置。

"啊！"黑影发出一声惨叫，身上的黑气开始消散。

"成功了？"鸣人兴奋地说道。

但就在这时，黑影突然大笑起来。

"呵呵，没想到你们竟然能够伤到我。"黑影说道，"但你们太小看我了！"

随着他的话音，消散的黑气再次聚集，变得更加浓烈。

"这只是...分身？"佐助震惊地说道。

"没错，"黑影轻笑道，"真正的我还在后面。"

就在这时，山谷突然震动起来。天空中的漩涡开始扩大，整个空间都在扭曲。

"他在打开真正的通道！"李武立刻说道，"我们必须阻止他！"

但已经晚了，漩涡中传来了一股强大的吸力，试图将众人吸进去。

"抓紧了！"李武喊道，"大家一起使用查克拉固定！"

四人立刻将查克拉注入地面，抵抗着强大的吸力。但吸力越来越强，他们的情况越来越危险。

"不行了！"小樱惊呼道，"我们的查克拉快耗尽了！"

"必须想办法离开这里！"李武说道，"使用位面穿梭！"

但位面穿梭只能传送一个人，根本无法解决所有人的问题。

就在这时，李武突然想起了什么。

"古代遗物！"李武眼睛一亮，"也许它可以帮我们！"

李武立刻拿出古代遗物，全力催动其中的能量。

"位面稳定！"

一道金色的光芒从遗物中射出，暂时稳定了空间。但这也消耗了遗物的大部分能量。

"只能撑一会儿了，"李武说道，"我们必须赶快找到出路！"

就在这时，李武的位面感知突然发现了一个异常。

"等等！"李武喊道，"我发现了什么！"

"什么？"众人问道。

"那边有通道！"李武指向一个方向，"似乎是通往另一个位面的！"

"那是什么地方？"鸣人问道。

"不知道，"李武摇头道，"但总比被吸进漩涡要好！"

"那就快走吧！"佐助冷静地说道。

四人立刻朝着李武指的方向跑去。很快，他们就发现了一个闪烁着光芒的裂缝。

"就是这里了！"李武说道，"大家跟我进去！"

四人毫不犹豫地跳进了裂缝。就在他们离开的瞬间，整个山谷彻底被漩涡吞噬，消失在了原地。

【进入未知位面】

当四人再次睁开眼睛时，眼前的景象让他们都惊呆了。

这里是一个完全不同的世界。天空是紫色的，漂浮着无数的光点。地面是灰色的，生长着一些奇异的植物。远处有巨大的山脉悬浮在空中，还有瀑布从山顶流下，却永远不会落地。

"这里...是什么地方？"小樱震惊地看着周围的景象。

"位面夹层！"李武立刻说道，"我们进入了位面夹层！"

"这就是传说中的位面夹层？"鸣人好奇地打量着周围的一切，"太不可思议了！"

"这里很危险，"李武皱眉道，"我们必须保持警惕。"

就在这时，系统突然发出了提示。

【位面探索模式开启】
可以探测位面中的能量波动
可以分析环境特征
可以寻找资源点

"系统升级了，"李武说道，"现在我们可以更好地探索这个位面了。"

"这有什么用？"鸣人问道。

"用处很大，"李武解释道，"我们可以探测危险，寻找资源，还可以分析环境特征来制定作战计划。"

"那我们快试试！"鸣人兴奋地说道。

李武闭上眼睛，全力催动位面探索能力。

【位面感知结果】
探测到多处能量源
其中三个为高威胁等级
一个为中威胁等级
其余为低威胁等级

"这里有很多能量源，"李武睁开眼睛说道，"有些是危险的，有些可能有价值。"

"那我们应该怎么做？"小樱问道。

"先找一个安全的地方休整，"李武说道，"然后制定探索计划。"

四人开始寻找安全的地方。很快，他们就发现了一个山洞，里面似乎很安全。

"就这里吧，"李武说道，"我们可以暂时在这里休整。"

众人进入山洞，开始恢复体力。李武则利用这个时间，详细地探索这个位面的情况。

【位面分析结果】
位面类型：夹层位面
能量浓度：极高
环境特征：重力异常，空间不稳定
资源点：3个（高价值）
威胁点：5个（高威胁）

"这个位面的能量浓度很高，"李武分析道，"对我们的修炼很有帮助。"

"那我们是不是可以在这里修炼？"鸣人问道。

"当然可以，"李武点头道，"但要注意安全。这个位面很不稳定，随时可能有危险。"

"那个黑影呢？他会不会追到这里？"小樱担忧地问道。

"不知道，"李武摇头道，"但我们必须做好最坏的打算。"

就在这时，山洞外突然传来了一阵骚动。

"什么声音？"佐助警觉地问道。

"好像是有什么东西在外面，"李武说道，"大家小心！"

四人立刻戒备起来，准备随时应对可能的威胁。

李武使用位面感知探测外面的情况。

【威胁感知】
检测到生物活动
等级：中等
数量：多
特征：未识别

"外面有未知的生物，"李武说道，"可能是这个位面的原住民。"

"它们会攻击我们吗？"小樱问道。

"不知道，"李武摇头道，"但我们还是小心为上。"

就在这时，一个生物走进了山洞。那是一个人形的生物，全身覆盖着灰色的鳞片，眼睛是红色的，看起来很凶猛。

【生物分析】
名称：夹层守卫
等级：中等
特点：守护位面节点
弱点：能量核心

众人立刻摆出战斗姿态，准备迎战。

但那个夹层守卫并没有攻击他们，只是好奇地看着众人，然后转身离开了。

"它没有攻击我们，"鸣人惊讶地说道，"这是为什么？"

"也许它认为我们没有威胁，"李武分析道，"或者它有自己的任务。"

不管怎样，这个夹层守卫没有攻击他们，让他们松了一口气。

"看来这个位面并不像想象中那么危险，"小樱说道。

"不能掉以轻心，"李武提醒道，"这只是开始。真正的挑战还在后面。"

确实，他们刚刚进入位面夹层，还有很长的路要走。那个神秘的黑影随时可能出现，而位面夹层中可能还有更多的危险等待着他们。

但无论如何，木叶小队已经做好了准备。他们要在这个陌生的位面中寻找机会，提升实力，最终阻止那个神秘黑图的计划。

"我们要在这里好好探索一下，"李武说道，"也许能够找到对付那个黑影的方法。"

"好！"众人坚定地回答。

新的篇章已经开启，位面夹层中的冒险，即将开始！
