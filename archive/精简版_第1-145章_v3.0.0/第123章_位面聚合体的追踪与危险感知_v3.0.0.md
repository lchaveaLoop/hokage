"我们需要更多的情报，"李武说道，"关于这个位面和那个黑影的情报。"

就在这时，李武的位面感知突然发现了一个异常的能量波动。

"等等！"李武喊道，"我发现了什么！"

"什么？"众人立刻警觉起来。

"熟悉？"佐助疑惑地问道，"是什么？"

"是那个黑影的能量特征！"李武震惊地说道，"他果然在附近！"

"什么？他竟然追到这里来了？"鸣人震惊地说道。

"不，也许他本来就住在这里，"李武分析道，"这个位面夹层可能是他的老巢！"

"那我们怎么办？"小樱担忧地问道。

李武小心翼翼地接近能量波动的源头，其他人则跟在后面保持警戒。

"大家保持安静，"李武低声说道，"不要被发现。"

【威胁感知】
检测到强烈能量波动

"那是什么东西？"鸣人小声问道。

"不知道，但肯定不是好东西，"李武皱眉道，"它的能量波动太强了。"

就在这时，系统突然发出了警告。

【系统警告：检测到高威胁目标】

但就在他们准备离开时，洞穴中突然传来了一阵骚动。

"吼！"

"不好！被发现了！"李武惊呼道。

【生物分析】
名称：位面聚合体守卫
等级：高级

"攻击！"李武喊道。

"轰！"

"什么？我们的攻击竟然无效？"鸣人震惊地说道。

"不对，是它的防御太强了，"李武分析道，"我们需要更强的攻击！"

"那怎么办？"小樱担忧地问道。

"我来试试！"李武说道，"位面穿梭！"

李武的身影瞬间出现在位面聚合体守卫的侧面，发动了偷袭。

"砰！"

李武的攻击命中了守卫的侧面，但依然没有造成有效的伤害。

"这个家伙的防御力太强了！"李武震惊地说道。

位面聚合体守卫似乎被激怒了，它张开大口，一道黑色的能量波朝着众人射来。

"小心！"李武立刻喊道，"防御！"

众人立刻做出反应。李武使用位面守护创造了一个屏障，其他人则用各种忍术进行防御。

"轰！"

能量波击中屏障，发出一声巨响。屏障虽然抵挡住了攻击，但也消耗了大量的能量。

"这个家伙比我想象中还要强！"李武咬牙说道。

"我们不能一直这样被动挨打，"鸣人道，"必须想办法反击！"

"但怎么反击？"佐助冷静地分析道，"它的防御太强了，普通攻击无效。"

李武皱眉思考着对策。位面聚合体守卫虽然强大，但肯定有它的弱点。

等等...李武突然想起了什么。

"怎么利用？"小樱问道。

"我试试看！"李武说道，"位面感知！"

李武闭上眼睛，全力催动位面感知能力。他要详细分析位面聚合体守卫的能量流动情况。

李武双手结印，开始催动位面之力。他要创造一个机会，让其他人能够攻击到核心位置。

"火遁·豪火球之术！"
"医术·查克拉冲击！"

"有效果！"鸣人兴奋地说道。

确实，他们在这里大闹了一场，肯定会被那个黑影注意到。
