# 第124章：遭遇战与力量测试

## 正文

离开山谷后，木叶小队朝着神庙的方向前进。经过几天的修炼和探索，他们的实力都有了明显的提升，但李武心中依然充满了警惕。

"那个神庙就在前面了，"李武指着远处说道，"我们终于到了。"

远处的神庙看起来很古老，全身由灰色的岩石建成，上面刻满了神秘的符文。神庙周围弥漫着一股神秘的气息，让人感到既好奇又敬畏。

"这里看起来很有历史感，"小樱说道，"不知道是什么时候建成的。"

"可能是在位面夹层形成的时候，"李武分析道，"这个神庙说不定见证了位面的历史。"

"那我们快进去看看吧！"鸣人迫不及待地说道。

"等等，"李武摆摆手，"我们要先做好准备。"

李武使用位面感知探测神庙周围的情况。

【位面感知结果】
检测到多处能量波动
其中三个为高威胁
一个为中威胁
其余为低威胁

"神庙周围有很多能量源，"李武说道，"有些可能是守卫。"

"那我们怎么进去？"佐助冷静地问道。

"先试试能不能和平进入，"李武说道，"如果不行再硬闯。"

四人小心翼翼地接近神庙。当他们走到神庙门口时，突然出现了几个身影。

【生物分析】
名称：神庙守卫
等级：中级
特点：守护神庙
弱点：能量核心

"是神庙守卫，"李武说道，"看来他们不欢迎我们。"

神庙守卫全身覆盖着银色的铠甲，手持长矛，看起来很威猛。他们拦住了众人的去路，似乎不让他们进入神庙。

"你们是什么人？为何来此闯我神庙？"一个领头的守卫质问道。

"我们是来寻求答案的，"李武说道，"关于这个位面和那个神秘黑影的答案。"

"神秘黑影？"守卫似乎很惊讶，"你们也遇到了那个家伙？"

"你知道什么？"李武敏锐地察觉到了异常。

"你们跟我来吧，"守卫说道，"大祭祀想要见你们。"

"大祭祀？"小好奇地问道。

"神庙的守护者，"守卫解释道，"他可能知道你们想知道的事情。"

四人跟着守卫进入了神庙。神庙内部很宽敞，墙壁上刻满了壁画，讲述着位面夹层的历史。

"这些壁画很有意思，"李武说道，"似乎在讲述什么故事。"

"等一下你们就知道了，"守卫神秘地说道。

很快，他们就来到了一个巨大的大厅前。大厅中央，有一个身穿白色长袍的老人正在等待他们。

【生物分析】
名称：大祭祀
等级：高级
特点：知晓位面秘密
弱点：未知

"欢迎你们，远道而来的客人，"大祭祀微笑着说道，"我知道你们是谁，也知道你们想要什么。"

"你知道我们？"李武惊讶地问道。

"当然，"大祭祀点头道，"在这个位面夹层中，能够来到这里的，都是不简单的人物。"

"那你一定知道关于那个神秘黑影的事情，"李武直接说道。

大祭祀的表情变得严肃起来："你们说的是那个想要融合所有位面的家伙吧。"

"你知道他的计划？"佐助冷静地问道。

"当然知道，"大祭祀说道，"他想要利用位面夹层作为跳板，融合所有的位面，创造一个全新的世界。"

"为什么要这么做？"鸣人问道。

"因为他的力量太弱了，"大祭祀解释道，"在一个位面中，他的力量有限。但如果融合所有位面，他就能获得无穷的力量。"

"这太疯狂了！"小樱震惊地说道。

"确实疯狂，但这也是他唯一的出路，"大祭祀叹气道，"他受到了某种诅咒，必须不断吸收位面的力量才能生存。"

"诅咒？"李武好奇地问道。

"那是很久以前的故事了，"大祭祀说道，"涉及到这个位面夹层的起源。"

"你能告诉我们更多吗？"李武问道。

"可以，但你们要先证明自己的实力，"大祭祀说道，"只有强者才有资格知道这些秘密。"

"你要比试？"鸣人跃跃欲试地说道。

"差不多吧，"大祭祀笑道，"让我看看你们有没有对抗那个家伙的实力。"

"好吧，"李武点头道，"我们接受挑战。"

"很好，"大祭祀说道，"跟我来。"

大祭祀带着他们来到了一个训练场。训练场中央，有一个巨大的能量球在旋转。

"这是位面能量训练场，"大祭祀解释道，"你们可以在里面测试自己的实力。"

"那我们什么时候开始？"鸣人兴奋地问道。

"现在就可以，"大祭祀说道，"我先看看你们的水平。"

李武深吸一口气，说道："我先来！"

李武走进训练场，开始催动体内的查克拉。他要在这个特殊的训练场中测试自己的实力。

【位面能量检测】
能量等级：高级
适应性：良好
控制力：中等

"你的能量水平不错，"大祭祀说道，"但还有提升的空间。"

李武点点头，开始进行更深入的测试。他尝试使用新获得的位面融合能力。

"位面融合！"

李武开始与位面能量融合，他的气息开始飙升。

【位面融合启动】
属性提升：200%
能量等级：顶级
控制力：优秀

"不错，你已经掌握了位面融合的基本技巧，"大祭祀满意地说道，"但还不够熟练。"

"我还能更强！"李武说道，他要测试自己的极限。

"位面掌控！"

李武开始全面掌控位面能量，他的周围出现了无数的能量粒子。

【位面掌控掌握】
范围：500米
精度：极高
控制力：完美

"很好，你已经达到了这个位面的极限，"大祭祀惊讶地说道，"很少有外来者能够达到这个程度。"

"那我呢？"鸣人迫不及待地想要测试自己的实力。

"你也来试试，"大祭祀说道，"让我看看九尾之力在这个位面的表现。"

鸣人走进训练场，开始释放九尾的查克拉。

"九尾模式！"

鸣人的身体被九尾的查克拉包围，出现了狐狸耳朵和尾巴。

【位面能量检测】
能量等级：顶级
适应性：极佳
控制力：中等

"你的九尾之力很特别，"大祭祀说道，"似乎与这个位面有某种共鸣。"

"是吗？"鸣人好奇地问道。

"是的，"大祭祀解释道，"九尾本身就是位面生物，它的力量与这个位面很契合。"

"那佐助和小樱呢？"李武问道。

"他们也来试试吧，"大祭祀说道，"让我们看看整个团队的水平。"

佐助和小樱也依次进入训练场测试。佐助的写轮眼在这个位面中表现出了特殊的能力，而小樱的医疗忍术也达到了新的高度。

"你们的团队配合很不错，"大祭祀满意地说道，"但面对真正的敌人，还不够。"

"那我们要怎么做？"李武问道。

"你们需要更强的力量，"大祭祀说道，"而这个神庙中，就有你们需要的东西。"

"什么东西？"众人好奇地问道。

"位面圣物，"大祭祀神秘地说道，"能够大幅提升实力的神器。"

"在哪里？"李武迫不及待地问道。

"跟我来，"大祭祀说道，"我带你们去。"

大祭祀带着他们来到了神庙的最深处。那里有一个祭坛，上面放着一个散发着光芒的物体。

【物品分析】
名称：位面圣物
等级：传说级
效果：大幅提升实力
副作用：可能失去自我

"这就是位面圣物，"大祭祀说道，"传说中能够提升使用者实力的神器。"

"我们要怎么使用它？"小樱问道。

"直接接触它，"大祭祀解释道，"但要做好心理准备，它的力量很强大。"

李武深吸一口气，走向祭坛。他要亲自试试这个位面圣物的力量。

就在李武即将接触到圣物时，突然传来了一个熟悉的声音。

"呵呵，没想到你们竟然在这里找到了位面圣物。"

众人立刻转身，只见一个黑影出现在他们面前。

"是你！"李武震惊地说道，"那个神秘黑影！"

"没错，"黑影轻笑道，"我来这里就是为了这个位面圣物。"

"你休想得到它！"李武坚定地说道。

"是吗？"黑影冷笑道，"那就看看你们有没有这个本事！"

黑影突然发动了攻击，一道黑色的能量波朝着众人射来。

"小心！"李武立刻喊道，"防御！"

众人立刻做出反应。李武使用位面守护创造了一个屏障，其他人则用各种忍术进行防御。

"轰！"

能量波击中屏障，发出一声巨响。屏障虽然抵挡住了攻击，但也消耗了大量的能量。

"有点意思，"黑影轻笑道，"看来你们有些实力。"

"但还不够，"黑影继续说道，"让我真正见识一下你们的实力！"

黑影双手结印，周围的空气开始扭曲，一股强大的压力从天而降。

"这是...位面之力！"李武震惊地说道，"他在使用位面之力！"

"位面融合！"黑影喊道，他的身体开始与位面能量融合。

【位面融合检测】
能量等级：顶级
适应性：完美
控制力：完美

"这个家伙竟然掌握了完美的位面融合！"李武震惊地说道。

"看来你们遇到了真正的对手，"大祭祀严肃地说道，"你们要小心了。"

"没关系，"李武坚定地说道，"我们有信心战胜他！"

"信心？"黑影轻笑道，"那我就让你们见识一下，什么叫做真正的绝望！"

黑影猛地一挥手，一个巨大的能量漩涡出现在众人面前。

"位面湮灭！"

能量漩涡开始扩大，要将所有人都吸进去。

"不好！这是强大的位面攻击！"李武惊呼道。

"我们必须阻止他！"鸣人喊道。

"合力攻击！"李武喊道，"使用最强的力量！"

四人立刻摆出战斗姿态，准备发动最强的攻击。

"这次，我们要测试一下真正的力量！"

【第124章 完】
