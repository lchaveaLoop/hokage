# 第125章：聚合体本部与最终对决前夜

## 正文

面对黑影的位面湮灭攻击，木叶小队摆出了战斗姿态。黑影的强大超出了他们的想象，但他们并没有退缩。

"大家，"李武坚定地说道，"这是我们真正的考验！"

"我们知道，"众人异口同声地回答。

"那就让我们看看，谁才是真正的强者！"李武喊道。

"位面融合！"

李武首先发动了位面融合，他的身体开始与位面能量融合。周围的空气开始扭曲，无数的能量粒子朝着他涌来。

【位面融合启动】
属性提升：200%
能量等级：顶级
控制力：优秀

"不错，"黑影轻笑道，"但还不够！"

"螺旋仙法·风遁·螺旋手里剑！"

鸣人也进入了九尾模式，制造出了螺旋手里剑。在他的身后，巨大的九尾虚影若隐若现。

【九尾之力爆发】
查克拉量：无限
能量等级：顶级
特性：位面共鸣

"火遁·豪火球之术！"

佐助的写轮眼进化到了新的高度，他的火遁术也变得更加猛烈。周围的温度急剧升高，连空气都在燃烧。

"医术·生命之术！"

小樱的医疗忍术也达到了新的高度，她不仅能够治疗伤势，还能够增强生命力和战斗力。

"很好，"黑影满意地说道，"这才有点意思！"

"现在！"李武喊道，"合力攻击！"

四人同时发动了最强的攻击。螺旋手里剑、火遁豪火球之术、生命之术，再加上李武的位面之力，形成了一个巨大的能量团，朝着黑影射去。

"轰——！"

能量团与黑影的位面湮灭攻击相撞，产生了剧烈的爆炸。整个神庙都在震动，墙壁上的壁画纷纷脱落。

"有点意思，"黑影擦了擦嘴角，"但你们以为这样就能战胜我吗？"

黑影突然变得极其恐怖，他的全身开始膨胀，无数的黑气从体内涌出。

"位面聚合！"

黑影开始与整个位面的能量融合，他的身体变得巨大无比，几乎充满了整个空间。

【位面聚合检测】
能量等级：神级
适应性：完美
控制力：完美

"这是什么？"众人震惊地看着眼前的景象。

"这就是我的真正力量！"黑影咆哮道，"位面聚合，我就是这个位面的化身！"

"太强了..."李武震惊地说道，"这种力量已经超出了我们的想象！"

"不能放弃！"鸣人坚定地说道，"我们一定要战胜他！"

"说得对，"李武点头道，"即使面对再强的敌人，我们也不能退缩！"

"那我们怎么办？"小樱担忧地问道，"这种力量根本不是我们能够对抗的！"

李武皱眉思考着对策。黑影的位面聚合确实太强大了，他们的攻击似乎根本无法伤到他。

等等...李武突然想起了什么。

"位面圣物！"李武眼睛一亮，"也许它可以帮我们！"

李武立刻看向祭坛上的位面圣物。那是一件散发着神秘光芒的物体，似乎蕴含着无穷的力量。

"你想用那个东西？"大祭祀惊讶地问道。

"是的，"李武坚定地说道，"这是我们唯一的机会！"

"但你知道它的副作用吗？"大祭祀担忧地问道。

"我知道，"李武点头道，"但现在顾不了那么多了！"

李武走向祭坛，他要取到位面圣物。但就在他即将接触到圣物时，黑影突然发动了攻击。

"想拿圣物？问过我了吗！"

一道黑色的能量波朝着李武射来。李武立刻躲开，但能量波击中了祭坛。

"轰！"

祭坛破碎，位面圣物飞了出去，落在了众人的中间。

"正好，"黑影轻笑道，"省得我动手了！"

黑影伸出大手，想要抓住位面圣物。但就在这时，李武抢先一步抓住了圣物。

"现在，让我们看看谁才是真正的强者！"

李武开始接触位面圣物。一股强大的能量立刻涌入他的体内，他的身体开始发光。

【位面圣物启动】
能量转化中...
位面掌控力MAX...
新技能解锁：位面主宰

【位面主宰掌握】
掌控整个位面
可修改位面规则
副作用：可能失去自我

"位面主宰？"李武震惊地说道，"这是传说中的能力！"

"不错，"大祭祀惊讶地说道，"你竟然能够启动位面圣物！"

"但它的副作用很大，"大祭祀担忧地说道，"你可能会迷失自我！"

"我知道，"李武坚定地说道，"但现在必须这么做！"

李武开始全力催动位面圣物的力量。他的身体周围出现了无数的空间裂缝，整个位面都在他的掌控之中。

"位面主宰！启动！"

随着李武的喊声，整个位面夹层开始震动。天空中的紫色闪电变成了金色的，空间开始扭曲。

"什么？他在掌控位面？"黑影震惊地说道，"不可能！"

"没有什么不可能的，"李武冷冷地说道，"当你想要毁灭这个世界的时候，我就要保护它！"

"你以为凭这个就能战胜我吗？"黑影怒吼道，"我可是位面聚合体！"

"位面聚合？"李武冷笑道，"在这个位面中，我就是主宰！"

李武猛地一挥手，无数的空间屏障出现在黑影周围，限制了他的行动。

"不可能！"黑影惊呼道，"我的力量被压制了！"

"这就是位面主宰的力量，"李武解释道，"在这个位面中，我可以修改规则！"

李武开始修改位面规则。他削弱了黑影的力量，增强了队友们的实力，创造了一个有利于战斗的环境。

"现在，让我们做个了结！"

李武朝着黑影发起了攻击。这一次，他的攻击中蕴含了整个位面的力量，威力无穷。

"轰——！"

黑影被击中了，他的身体开始崩溃。位面聚合的状态被打破，他变回了原来的样子。

"不可能..."黑影震惊地说道，"你怎么可能掌握位面主宰？"

"因为我要保护这个世界，"李武坚定地说道，"我不会让你伤害它！"

"那你杀了我吧，"黑影苦笑道，"反正我也已经输了。"

"我不会杀你，"李武说道，"但你要承诺，不再做这种事。"

"承诺？"黑影苦笑道，"你知道我为什么会变成这样吗？"

"为什么？"众人好奇地问道。

"因为诅咒..."黑影说道，"我被位面诅咒了，必须不断吸收位面力量才能生存。"

"诅咒？"大祭祀惊讶地说道，"难道是..."

"是的，"黑影点头道，"我就是位面夹层的创造者。"

"什么？"众人震惊地说道。

"很久以前，我试图创造一个完美的位面，"黑影解释道，"但失败了，反而被位面诅咒了。"

"所以你想要融合所有位面？"李武问道。

"是的，"黑影点头道，"只有这样才能摆脱诅咒。"

"但这种方法太极端了，"李武说道，"伤害了无数的人。"

"我知道，"黑影苦笑道，"但已经回不去了。"

"也许有办法，"大祭祀突然说道，"古老的文献中记载过，位面诅咒是可以解除的。"

"真的？"黑影惊喜地问道。

"是的，"大祭祀点头道，"但需要付出很大的代价。"

"什么代价？"众人问道。

"牺牲自己的力量，"大祭祀解释道，"变成一个普通人。"

"我愿意，"黑影坚定地说道，"只要能够摆脱诅咒，做什么都值得。"

"那我们帮你，"李武说道，"我们一起想办法。"

就这样，原本的敌人变成了战友。木叶小队和大祭祀一起，帮助黑影寻找解除诅咒的方法。

经过几天的努力，他们终于找到了方法。黑影成功解除了诅咒，变回了一个普通人。

"谢谢你们，"黑影感激地说道，"是你们给了我重新开始的机会。"

"不用谢，"李武说道，"这都是我们应该做的。"

"那接下来我们要做什么？"鸣人问道。

"回去原来的世界，"李武说道，"那里还有很多事情要做。"

"是的，"大祭祀点头道，"位面夹层只是开始，你们的冒险还远未结束。"

"我们知道了，"众人坚定地说道。

在这个位面夹层中，木叶小队经历了前所未有的冒险。他们不仅提升了实力，还找到了真正的朋友。

而他们的故事，才刚刚开始...
