# 第127章 位面掌控与新的能力

"前方就是位面核心了。"黑影的声音在众人脑海中回响，虽然它已经离开，但留下了一些空间标记，指引着大家前往正确的位置。

位面夹层已经因为诅咒的解除而变得稳定，但中央区域依然散发着强大的能量波动。这里的时空结构异常复杂，到处都是空间裂缝和时间断层。

"位面掌控者..."宇智波佐助握紧了拳头，"这个神秘的家伙一直在操控着整个位面夹层，甚至包括我们所有人。"

"他为什么要这么做？"日向雏田疑惑地问道，"操控一个位面对他有什么好处？"

"位面掌控者通常是为了获得某种特殊的力量。"漩涡鸣人分析道，"位面本身的能量对他来说可能是补品，或者是实验材料。"

"无论他的目的是什么，我们都要阻止他。"旗木卡卡西冷静地说道，"这个位面已经够乱了，不能再让他继续下去。"

六位忍者小心翼翼地前进着，位面核心区域的空间极不稳定，随时都可能发生扭曲。漩涡鸣人使用六道仙人的空间感知能力，为大家开辟安全的道路。

"小心，前面有空间陷阱！"漩涡鸣人警告道，"这些空间裂缝能够吞噬一切进入的物质。"

宇智波佐助开启白眼，仔细观察着周围的空间结构："漩涡鸣人说得对，这些空间裂缝看起来就像深渊，一旦掉进去恐怕连查克拉都无法维持。"

"我来处理。"李洛克主动请缨，"表莲华！"他用极快的速度在空间裂缝周围布置体术陷阱，形成一个临时的安全区域。

"表莲华能够稳定周围的空间波动，为我们争取时间。"李洛克解释道，"虽然效果有限，但足够我们通过了。"

在李洛克的帮助下，众人安全地穿过了空间裂缝区域。继续前进，他们终于到达了位面核心——一个巨大的球形空间，中心悬浮着一个透明的能量球。

"那就是位面掌控者！"漩涡鸣人指着能量球说道。

能量球中的人影缓缓转过头，露出了令人震惊的面孔——那是一个看起来只有十多岁的少年，面容俊美，但眼神中却充满了冷漠和傲慢。

"欢迎来到位面核心。"少年用平淡的语气说道，"终于有人能够到达这里了。"

"你就是位面掌控者？"宇智波佐助警惕地问道。

"没错。"少年点头道，"我是这个位面的掌控者，也是这里的唯一统治者。"

"你为什么要操控整个位面？"旗木卡卡西质问道，"这些无辜的人都被你困在这里，你到底想做什么？"

"无辜？"少年轻笑一声，"在这个世界上，哪里有什么真正的无辜？每个人都在追逐力量，只是方式不同而已。"

"你的方式就是牺牲他人？"日向雏田愤怒地说道。

"这是最快的方式。"少年冷冷地说道，"我需要足够的能量来突破自己的极限，而这个位面...就是我的养料。"

"我不会让你继续下去！"漩涡鸣人大声道，"我们一定要阻止你的计划！"

少年看着漩涡鸣人，眼中闪过一丝兴趣："哦？就凭你们几个？"

"就凭我们！"宇智波佐助展开须佐能乎，"须佐能乎·完全体！"

巨大的须佐能乎身影出现在位面核心中，给少年带来了不小的压力。但少年的脸上依然保持着平静，甚至有些嘲弄的表情。

"须佐能乎...不错的力量。"少年评价道，"但在位面掌控者面前，这种力量根本不值一提。"

"那就试试看！"宇智波佐助发动攻击，"天照！"黑色的不灭黑炎从须佐能乎的手掌中喷出，想要吞噬少年。

然而，黑炎在距离少年还有几米远的地方就消失了，仿佛被什么力量阻挡了。

"空间屏障。"少年淡淡地说道，"在我的领域里，我的意志就是法则。"

"什么？"宇智波佐助震惊地看着自己的攻击失效。

"在我的位面中，我可以随意操控空间和时间。"少年解释道，"包括改变物理法则。你的火忍术...在这里行不通。"

宇智波佐助立刻意识到情况的严重性。在位面掌控者的领域内，常规的忍术可能会受到很大限制。

"那我们只能使用更强的力量了。"漩涡鸣人说道，"六道仙人模式！"

漩涡鸣人全身散发出金色的光芒，六道仙人的力量完全爆发。在这个位面中，他依然能够感受到六道仙人的本源力量。

"六道仙人模式...有意思。"少年的眼中闪过一丝兴趣，"看来你确实有些特殊。"

"特殊？"漩涡鸣人疑惑地问道。

"在这个位面中，只有你能够抵抗我的空间操控。"少年惊讶地说道，"这是怎么回事？"

漩涡鸣人立刻明白了原因。六道仙人的力量本身就超越了位面法则，所以即使是在位面掌控者的领域中，他依然能够保持一定的力量。

"看来你确实是个不错的对手。"少年说道，"那么，就让我认真对待吧。"

少年抬起手，位面核心中的能量开始疯狂涌动。"空间压缩！"

整个位面核心的空间开始向内收缩，强大的压力让所有人都感到窒息。宇智波佐助的须佐能乎开始出现裂痕，其他人更是直接被压得跪倒在地。

"大家坚持住！"漩涡鸣人大声道，"六道仙人的空间防护！"

金色的查克拉形成了一个巨大的保护罩，勉强抵挡住了空间压缩的压力。但保护罩正在不断缩小，看来最多只能再坚持几秒钟。

"必须想办法反击！"宇智波佐助喊道，"这样下去我们会被压成肉饼的！"

"我知道！"漩涡鸣人咬紧牙关，"六道仙人的空间跳跃！"

在保护罩即将破碎的瞬间，漩涡鸣人发动了空间跳跃能力，带着所有人瞬间移动到了位面核心的另一端，暂时避开了空间压缩的压力。

"不错的能力。"少年赞叹道，"能够在我控制的位面中进行空间跳跃，这确实很厉害。"

"你的能力虽然强大，但并非无敌。"漩涡鸣人冷静地说道，"只要找到你的弱点，我们就能打败你！"

"弱点？"少年轻笑一声，"在位面掌控者的领域中，哪里还有什么弱点可言？"

"一定有的！"宇智波佐助说道，"任何力量都有其极限，即使你是位面掌控者也不例外。"

少年沉默了片刻，然后说道："你们说得对，确实有弱点。"

"什么弱点？"众人立刻问道。

"位面掌控者的力量来自于位面本身。"少年解释道，"如果位面出现问题，我的力量也会受到影响。"

"那就是说，如果我们能够破坏这个位面的稳定性，就能削弱你的力量？"漩涡鸣人立刻理解了。

"没错。"少年点头道，"但这几乎是不可能的任务。位面的稳定性是由位面法则维持的，除非..."

"除非什么？"日向雏田追问道。

"除非你们能够获得位面掌控的能力。"少年说道，"只有掌握了一部分位面法则，才能影响位面的稳定性。"

"位面掌控能力..."漩涡鸣人重复道，"我们该如何获得？"

少年看着漩涡鸣人，眼中闪过一丝复杂："其实...你们已经在一开始就获得了这种能力的种子。"

"种子？"宇智波佐助疑惑地问道。

"还记得进入位面夹层时，你们获得的那些特殊能力吗？"少年问道，"那些就是位面掌控能力的雏形。"

漩涡鸣人立刻回想起来，进入位面夹层时，他们确实获得了一些特殊的空间操控能力。

"你的意思是，只要我们进一步开发这些能力，就能获得位面掌控的力量？"

"没错。"少年点头道，"但这需要时间和经验，而且..."

"而且什么？"日向雏田追问道。

"而且这种能力非常危险。"严肃地说道，"位面掌控之力足以让任何人迷失自我，变成像我这一样的怪物。"

少年的话语中透露出一丝悔恨，这让众人都感到意外。原来，这位面掌控者也有自己的苦衷。

"你...也是被力量所困？"漩涡鸣人试探性地问道。

少年沉默了片刻，然后缓缓点头："我被困在这个位面中已经很久了，渐渐地...我就只记得追求力量，忘记了最初的目的。"

"最初的目的是什么？"旗木卡卡西轻声问道。

"寻找回家的路..."少年低声说道，"我原本只是个普通的学生，因为意外被卷入这个位面，为了活下去，我不得不掌握位面之力..."

少年的故事让所有人都陷入了沉默。他们原本以为这是个纯粹的恶人，没想到背后还有这样的故事。

"那...你为什么不让其他人离开？"日向雏田轻声问道。

"因为我害怕..."少年的声音中带着颤抖，"如果只有我一个人留在这里，我会彻底迷失自我。所以...我把其他人也困在这里，让我觉得自己不是孤单一人。"

这个真相让所有人都感到震惊和同情。原来，位面掌控者并非纯粹的邪恶，只是一个被孤独和恐惧所困的可怜人。

"我们可以帮你。"漩涡鸣人真诚地说道，"我们不会抛下你不管的。"

少年看着漩涡鸣人，眼中闪过一丝暖意："你们...真的愿意帮我？"

"当然！"宇智波佐助说道，"每个人都有被救赎的机会，包括你。"

"但...我做了这么多错事..."少年低头道，"那些被我困住的人..."

"我们可以解除位面夹层，让所有人都能离开。"旗木卡卡西说道，"然后我们一起想办法，帮你找到回家的路。"

少年抬起头，眼中闪烁着泪光："谢谢...谢谢你们..."

随着少年情绪的稳定，位面核心中的能量也开始变得平和。看来，内心的救赎比外在的力量更加重要。

"看来，我们不必战斗了。"漩涡鸣人微笑道，"我们可以成为朋友。"

"朋友..."少年轻声重复着这个词，脸上露出了久违的笑容，"我已经很久没有朋友了..."

"从今天开始，你就有我们了。"日向雏田温柔地说道。

在众人的帮助下，少年开始学习如何正确使用位面掌控之力，而不是用它来囚禁他人。渐渐地，位面夹层开始变得稳定，被困的生灵都能够离开这里。

"谢谢你们..."少年感动地说道，"是你们让我重新找到了人性的温暖。"

"这是我们应该做的。"漩涡鸣人说道，"每个人都有回家的权利，每个人都有重新开始的机会。"

随着位面夹层的解除，所有人都能够回到原来的世界。而这位面掌控者，也终于找到了自己的救赎之路。

"我...可以跟你们一起离开吗？"少年小心翼翼地问道。

"当然可以！"漩涡鸣人点头道，"欢迎加入我们的队伍。"

少年高兴地点头："太好了！我终于可以...回家了..."

夕阳下，七个人影并肩站立，准备离开这个充满回忆的位面夹层。虽然经历了无数的困难和挑战，但最终，他们收获了友谊和救赎。

"出发吧！"漩涡鸣人大声道，"我们一起回家！"

七道光芒划破虚空，消失在夕阳的余晖中。位面夹层的旅程结束了，但新的冒险，才刚刚开始...
