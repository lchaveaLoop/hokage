# 第135章：集结伙伴与新的征程

李武和卡卡西回到木叶后，立刻向纲手汇报了情况。当听到虚空之主计划吞噬整个世界时，纲手的脸色变得异常凝重。

"虚空入侵..."纲手沉思道，"看来我们面临的敌人，比想象中的更加可怕。"

"是的，"李武点头道，"而且它们已经开始行动了。系统检测到全球各地都有虚空能量反应，包括五大国的首都。"

"五大国..."纲手眉头紧锁，"这意味着整个忍界都面临着威胁。我们必须联合所有力量，共同对抗这个敌人。"

"我同意。"卡卡西说道，"这种级别的威胁，已经不是木叶能够单独应对的了。"

"那么，我们该如何行动？"李武问道。

纲手沉思片刻，然后说道："首先，我们需要召集所有可以信赖的力量。木叶的精英必须全部动员起来，同时还要与其他四国建立联盟。"

"说到联盟，"卡卡西突然说道，"我想起了一个人。风之国的我爱罗，他现在已经是风影了，而且实力强大，应该能够帮助我们。"

"我爱罗..."李武若有所思，"确实，他现在是影级强者，而且我与他有过交情，应该能够说服他。"

"还有云之国的雷影，土之国的土影，水之国的水影，"纲手一个个数着，"他们都是各自国家的最强者，必须拉入我们的联盟。"

"但是，如何说服他们相信虚空入侵的存在呢？"卡卡西问道，"这种事情太过不可思议，恐怕不会轻易相信。"

"我们可以用事实说话。"李武说道，"我已经让系统收集了一些证据，包括虚空能量的样本和被污染的生物样本。这些证据应该能够说服他们。"

"很好。"纲手点头道，"那么，我们就分头行动。卡卡西，你负责联系我爱罗和雷影；李武，你负责联系土影和水影；我则准备召开五影会谈。"

"明白。"两人点头应道。

会议结束后，众人立刻行动起来。李武首先来到水之国的雾隐村，找到了现任水影——长门。

长门在失去轮回眼后，实力有所下降，但依然是影级强者。当他看到李武带来的虚空能量样本时，脸色变得异常凝重。

"这就是你们说的虚空能量？"长门谨慎地观察着样本，"确实很危险，如果大规模扩散，整个水之国都会被污染。"

"长门大人，我们必须相信这不是开玩笑的事情。"李武严肃地说道，"我已经在火之国亲眼见过它的威力。"

"我相信你。"长门点头道，"毕竟你帮助我们解决了那么多问题。我愿意加入联盟，对抗这个共同的敌人。"

得到长门的承诺后，李武又前往土之国的岩隐村，找到了现任土影——大野木。大野木虽然年纪大了，但实力依然强大，而且经验丰富。

"虚空入侵？"大野木听完后，沉思道，"这种事情我以前从未听说过，但根据你描述的情况，确实有可能发生。"

"大野木大人，这不是危言耸听。"李武将证据交给大野木，"我已经在火之国和海中亲眼见过它们的行动。如果不及早防范，整个忍界都会面临灭顶之灾。"

"我明白了。"大野木看完证据，脸色变得严肃，"这种威胁，我们必须认真对待。我同意加入联盟。"

同时，卡卡西也成功说服了风之国的我爱罗和云之国的雷影。当李武和卡卡西带着所有影级强者回到木叶时，纲手已经准备好了五影会谈的场地。

"大家都到齐了。"纲手看着面前的四位影级强者，正式宣布道，"今天召集大家来，是有重要的事情要宣布。"

"火影大人，到底是什么事情？"我爱罗开门见山地问道。

纲手将虚空入侵的情况详细说明了一遍，并展示了相关的证据。四位影听完，脸上都露出了凝重的表情。

"这就是你们说的虚空之主？"雷影皱眉道，"一个想要吞噬整个世界的存在？"

"是的。"李武点头道，"而且它已经开始行动了。我们必须尽快制定应对策略。"

"但是，我们该如何对抗这种敌人？"大野木问道，"普通的忍术对它似乎效果不大。"

"我有一些想法。"李武说道，"从之前的战斗来看，虚空生物最怕的是系统能量和空间技术。我们可以重点发展这些方面。"

"空间技术？"我爱罗好奇地问道，"这东西我们以前从未接触过。"

"是的，"李武解释道，"这是我从古代文明遗迹中获得的技术。虽然还不是很完善，但已经能够对虚空生物造成有效伤害。"

"古代文明..."长门若有所思，"看来这个世界的秘密还有很多。"

"不管怎样，"雷影打断道，"既然这种威胁真实存在，我们就必须认真对待。我同意组成联盟，共同对抗这个敌人。"

"我也同意。"大野木点头道，"为了保护我们的世界，这点牺牲是值得的。"

"既然大家都同意了，"纲手说道，"那我们就正式成立'虚空对抗联盟'，由五大国共同组成，共同应对这次危机。"

"很好。"李武点头道，"接下来，我们需要制定具体的行动计划。"

经过几天的讨论，五影终于制定了一个完整的应对策略。首先，各国要加强边境防御，防止虚空生物渗透；其次，要建立情报共享机制，及时发现和处理虚空能量反应；最后，要组建一支由各国精英组成的特别行动队，专门负责处理大规模的虚空入侵事件。

"李武，"纲手看向李武，"我想请你担任特别行动队的队长，负责统一指挥所有行动。"

"为什么是我？"李武惊讶地问道，"我虽然有对付虚空生物的经验，但指挥这么多国家的人，恐怕不太合适。"

"因为你是唯一对付过虚空生物的人，"纲手坚定地说道，"而且你的实力最强，能够应对各种突发情况。我相信你有这个能力。"

听到纲手的话，其他几影也纷纷表示支持。

"李武确实是最合适的人选。"我爱罗说道，"他不仅有实力，还有丰富的经验。"

"没错。"雷影也点头道，"我们就把指挥权交给他吧。"

既然大家都这么说，李武也不好再推辞："那好吧，我接受这个任务。"

"太好了。"纲手高兴地说道，"那么，我们需要尽快组建队伍。"

"我推荐几个合适的人选。"李武说道，"漩涡鸣人、宇智波佐助、旗木卡卡西、日向雏田、春野樱、我爱罗手下的勘九郎和手鞠..."

"等等，"卡卡西突然打断道，"佐助不是在外游历吗？他能回来吗？"

"我会联系他。"李武说道，"这种事情，他一定会回来的。"

果然，当李武联系宇智波佐助时，佐助立刻表示愿意加入。很快，所有被选中的人都聚集在木叶的训练场上，等待着李武的指示。

"大家好。"李武站在众人面前，"今天召集大家来，是有重要的事情要宣布。"

"李武大哥，是什么事？"鸣人好奇地问道。

李武将虚空入侵的情况简单说明了一遍。当听到有敌人想要吞噬整个世界时，众人都露出了惊讶的表情。

"吞噬世界？"佐助皱眉道，"这种敌人到底有多强？"

"很强，"李武严肃地说道，"比我之前遇到的任何敌人都要强大。但也不是没有战胜的可能。"

"那我们需要做什么？"雏田问道。

"我们需要组成一支特别行动队，负责处理各地的虚空能量反应。"李武解释道，"同时，各国也会加强边境防御，防止虚空生物渗透。"

"这种事情，我们必须全力以赴。"鸣人坚定地说道，"为了保护我们的家园和亲人。"

"没错。"佐助也点头道，"不管敌人多强大，我们都要战斗到底。"

看到大家的决心，李武心中感到一阵温暖："谢谢大家的信任。接下来的日子，我们可能会面临很多危险，但我相信，只要我们团结一心，就一定能够战胜这个敌人。"

"那我们还等什么？"鸣人兴奋地说道，"不如现在就开始训练吧！"

"好主意。"李武点头道，"从今天开始，我们将进行特殊的训练，重点是提升对抗虚空生物的能力。"

接下来的几天，李武开始教导大家如何使用空间技术和对抗虚空能量。虽然这些技术很难掌握，但众人都表现出了惊人的学习能力。

"次元斩！"佐助尝试发动技能，一道空间裂缝出现在前方，将一棵大树切成两半。

"不错。"李武满意地点头，"继续练习。"

"空间屏障！"雏田双手结印，面前出现了一层空间屏障，成功挡住了鸣人的螺旋丸攻击。

"很好！"鸣人赞叹道，"雏田，你学得真快！"

众人都在努力练习着新技能，整个训练场充满了紧张的气氛。李武看着大家的进步，心中充满了信心。

"看来我们的队伍已经准备好了。"李武心想，"虽然前方的道路还很艰难，但我相信，只要我们团结一心，就一定能够战胜虚空之主。"

几天后，特别行动队正式成立。由李武担任队长，鸣人、佐助、卡卡西、雏田、樱、勘九郎、手鞠等人担任核心成员。各国的精英忍者也陆续加入，形成了一支强大的队伍。

"各位，"李武站在众人面前，正式宣布道，"我们的征程即将开始。前方等待我们的，将是未知的挑战和危险。但我相信，只要我们团结一心，就一定能够守护这个我们深爱着的火影世界！"

"为了守护家园！"众人齐声高呼，声音响彻云霄。

李武看着眼前这些充满朝气的面孔，心中充满了坚定的信念。无论前方的敌人多么强大，他们都会战斗到底，直到胜利的到来。

"出发！"李武大手一挥，"我们的第一站，是风之国的边境，那里检测到了强烈的虚空能量反应！"

随着李武的命令，特别行动队正式踏上了征程。新的挑战在等待着他们，但他们毫不畏惧，因为他们知道，自己不是一个人在战斗。

整个忍界的命运，都将掌握在他们手中。

而这场对抗虚空入侵的战争，才刚刚开始...

（第135章完）
