# 第137章：虚空之主现身与绝望力量

天空中，那个巨大的漩涡突然裂开了一道缝隙。一个高大的身影从中缓缓走出，每一步都让大地为之颤抖。那身影完全由纯粹的暗能量构成，但却散发着一种令人窒息的威压。

"是虚空之主！"李武震惊地说道，"它的气息比之前遇到的任何敌人都要强大千倍！"

虚空之主身高超过二十米，全身流淌着紫黑色的暗能量，背后展开一对遮天蔽日的能量翅膀。它的脸上没有五官，只有一片虚无，却让人感到它在注视着每一个人。

"终于见面了，世界的蝼蚁们。"一个冰冷的声音在每个人脑海中响起，"我是虚空之主，这个世界的终结者。"

"狂妄之徒！"雷影怒喝一声，"雷遁·雷光千本！"

无数雷光如雨点般射向虚空之主。但令人震惊的是，所有的雷光在接近虚空之主时，都被一股无形的力量吞噬了。

"真是无力的抵抗..."虚空之主轻蔑地说道，"在我的力量面前，你们的忍术就像萤火虫般微弱。"

"那就让我们见识见识萤火虫的力量！"纲手怒吼道，"禁术·百豪之术！"

纲手身上的查克拉瞬间暴涨，无数白蛇般的查克拉从她身上涌出。这是她最强的状态，开启百豪之术后，实力提升到了极致。

"八万拳！"

纲手化作一道残影，朝着虚空之主发起了狂暴的攻击。每一拳都带着强大的破坏力，拳风呼啸，将周围的空气都撕裂了。

然而，所有攻击都打在了虚空之主的能量体上，就像打在棉花上一样，没有造成任何伤害。

"就这点力量吗？"虚空之主冷笑一声，"看来你们让我失望了。"

说着，它轻轻一挥手，一股无形的冲击波朝着纲手袭去。纲手全力防御，但依然被击飞出去，撞毁了数座山丘。

"纲手大人！"鸣人大惊失色。

"各位，看来硬碰硬不是办法。"卡卡西冷静分析道，"我们需要找出它的弱点。"

"弱点？"虚空领主在一旁冷笑道，"伟大的虚空之主根本没有弱点！它是完美的存在！"

"完美？"李武冷笑一声，"任何存在都会有弱点。系统，分析虚空之主的能量构成！"

李武立刻启动了系统的全面分析功能。无数的符文在虚空中闪烁，开始分析虚空之主的能量数据。

"分析中..."系统机械的声音响起，"检测到虚空之主由99%的暗能量构成，1%的未知能量...正在解析..."

就在这时，虚空之主突然动了。它并没有使用任何华丽的招式，只是简单地向前踏出一步。这一步，却让整个时空都发生了扭曲。

"时空扭曲！"佐助惊呼道，"它改变了周围的空间结构！"

虚空之主周围的时空变得极不稳定，连光线都被扭曲了。靠近它的虚空生物也受到了影响，形态都发生了变化。

"这就是虚空之主的可怕之处..."大野木凝重地说道，"它不仅能控制暗能量，还能扭曲时空。"

"连时空都能扭曲..."长门若有所思，"这已经超出了普通忍术的范畴。"

"各位，我们必须小心。"李武提醒道，"它的力量已经超越了我们的理解范围。"

虚空之主似乎对众人的讨论很感兴趣，它缓缓开口："你们在讨论什么？是在讨论如何反抗我吗？"

"不，我们是在讨论如何消灭你！"雷影怒吼道，"土遁·超重岩之术！"

大野木配合雷影的攻击，无数的岩石从地面升起，形成了一个巨大的岩石牢笼，将虚空之主困在其中。

"土水联手！水遁·水断波！"

长门释放出巨大的水波，与岩石牢笼形成了一体。这是大野木和长门的联合忍术，威力巨大。

"火遁·豪火灭却之阵！"

纲手和樱同时释放火焰，在岩石牢笼周围形成了一道火墙。五位影级强者联手，使出了最强的禁术！

"风遁·风雷螺旋手里剑！"

手鞠和鸣人联手，风螺旋雷丸呼啸着射向火墙。这是五大国忍者的最强一击！

轰隆！

惊天动地的爆炸发生了。岩石牢笼、水波、火焰、雷光交织在一起，形成了一个巨大的能量爆炸。所有人都屏住了呼吸，等待着结果。

烟尘缓缓散去，虚空之主依然站在原地，虽然身上有一些裂痕，但整体完好无损。

"你们的攻击..."虚空之主缓缓说道，"就像蚊子叮咬一样，不痛不痒。"

"什么？！"所有人都震惊了，"五大影级强者的最强一击，竟然只能造成这么点伤害？"

"这就是虚空之主的力量..."纲手喘息着说道，"我们已经用出了全部力量，但依然无法伤到它分毫。"

"绝望了吗？"虚空之主的声音充满了恶意，"这才刚刚开始。让我让你们见识什么是真正的绝望！"

说着，虚空之主张开双臂，周围的暗能量开始疯狂涌动。天空中的漩涡越来越多，更多的虚空生物从中涌出。

"虚空召唤术！"虚空领主在一旁兴奋地说道，"伟大的虚空之主正在召唤更多的同胞！"

只见无数虚空生物从天而降，覆盖了整个天空。它们像黑色的潮水，朝着地面涌来。每一个虚空生物都散发着令人恐惧的气息，数量之多，让人绝望。

"这...这怎么可能？"勘九郎惊恐地说道，"它们的数量太多了！"

"这不是召唤，"李武震惊地说道，"这是虚空之主在抽取这个世界的能量！你看，大地的颜色正在变暗！"

众人低头一看，发现脚下的土地正在逐渐失去颜色，变成一片死气沉沉的黑色。花草树木都在迅速枯萎，整个世界都在被虚空侵蚀。

"它在吞噬这个世界..."长门震惊地说道，"一旦这个世界被完全吞噬，就再也没有恢复的可能了。"

"我们必须阻止它！"纲手坚定地说道，"不管付出什么代价！"

"怎么阻止？"雷影怒吼道，"我们的攻击对它根本没用！"

就在这时，李武的系统突然发出警报："警告！检测到虚空之主的能量正在急剧增强！它的下一波攻击即将到来！"

众人立刻紧张起来，握紧了手中的武器。虚空之主冷漠地看着众人，然后缓缓举起了右手。

"终结吧，蝼蚁们。"

随着它的话音落下，一个巨大的黑色能量球在它手中凝聚。那能量球不断旋转，散发出毁灭的气息。周围的空气都被压缩了，连呼吸都变得困难。

"那是..."李武震惊地说道，"它在聚集能量，准备发动毁灭性的攻击！"

"大家快躲开！"纲手立刻下令。

但能量球已经成型了，它缓缓朝着地面压来。所过之处，空间都被扭曲，地面寸草不生。更可怕的是，能量球还在不断吸收周围的能量，让它的威力越来越大。

"跑不了了..."卡卡西苦涩地说道，"它的范围太广了。"

"看来...这就是我们的结局了吗？"鸣人握紧了拳头，"我还想成为火影呢..."

"我还想和佐助决出胜负..."佐助冷着脸说道。

"我还想勘九郎和我一起成为最强的..."手鞠轻声说道。

"雏田..."樱轻声呼唤道。

雏田只是微笑着，双手结印："我准备好了。"

勘九郎和手鞠也站到了众人身边。五大影级强者也做好了战斗的准备。

"看来...我们只能战斗到底了。"纲手握紧了拳头。

就在这时，李武突然站了出来："各位，我有一个想法！"

"队长，什么想法？"鸣人问道。

李武指着虚空之主手中的能量球："它的能量虽然强大，但有一个弱点！"

"什么弱点？"众人齐声问道。

"它的能量正在不断聚集，但还没有完全稳定！这就是机会！"李武眼神坚定地说道，"我们需要在它完全成型之前，攻击它的核心！"

"可是怎么攻击？"佐助皱眉道，"它的防御太强了。"

"使用空间技术！"李武说道，"佐助，你的次元斩可以切开空间；雏田，你的空间屏障可以保护我们；我系统的时间暂停能力可以争取时间！"

"可是这样太危险了..."樱担忧地说道。

"危险也要试试！"李武坚定地说道，"这是我们现在唯一的机会！"

虚空之主手中的能量球越来越大了，周围的暗能量都在疯狂涌向它。一旦完全成型，整个风之国边境都将被夷为平地。

"时间不多了！"李武急切地说道，"大家准备！"

"明白了！"众人点头道。

李武深吸一口气，启动了系统的时间暂停能力："时间静止！"

周围的一切都仿佛静止了，只有李武能够活动。他立刻朝着虚空之主冲去。

"次元斩！"佐助同时发动技能，空间撕裂的锋芒朝着虚空之主斩去。

"空间屏障！"雏田的屏障保护着众人。

李武在时间静止的状态下，发动了最强的一击："系统·数据洪流！"

无数数据符文从李武身上涌出，化作一条条数据链，朝着虚空之主的核心缠绕而去。数据链与暗能量发生了激烈的碰撞。

时间静止状态很快结束了，虚空之主注意到了李武的攻击。

"竟然敢攻击我？"它怒吼一声，手上的能量球猛地朝着李武砸来。

"不好！"众人惊呼道。

李武立刻启动了防御系统："数据屏障！"

黑色能量球撞在数据屏障上，发出了剧烈的爆炸。烟尘散去后，李武倒在地上，嘴角流出了鲜血。

"队长！"鸣人立刻冲了过去。

"我没事..."李武擦掉嘴角的血，"但我的攻击似乎伤到了它！"

众人看向虚空之主，发现它的胸口出现了一道裂痕，暗能量正在不断从中泄露出来。

"可恶...竟然敢伤到我..."虚空之主的声音中带着愤怒，"我要让你们付出代价！"

它胸口的裂痕越来越大，暗能量泄露得越来越厉害。虚空领主在一旁焦急地说道："主人，您的能量在泄露！"

"没关系..."虚空之主冷笑道，"虽然受了伤，但你们也付出了代价。"

虚空之主身上的暗能量开始变得不稳定，周围的虚空生物也受到了影响，发出了痛苦的嘶吼。

"看来我们的攻击有效果！"李武兴奋地说道。

"是吗？"虚空之主突然冷笑起来，"你们认为这样就赢了吗？"

说着，它胸口的裂痕突然爆发出更加强大的暗能量。那能量形成了一个巨大的黑洞，将周围的虚空生物都吸了进去。

"什么？它在吸收虚空生物的能量！"李武震惊地说道。

虚空之主胸口的黑洞越来越大，最后变成了一道漆黑的能量柱，直冲云霄。那能量柱所过之处，空间都被撕裂，时间都发生了扭曲。

"这就是...虚空之主的绝望之力..."长门震惊地说道。

虚空之主身上的气息变得更加恐怖，虽然胸口有一道裂痕，但整体实力却提升到了一个新的高度。

"现在...才是真正的绝望！"虚空之主的声音充满了恶意，"感受虚空的力量吧！"

它朝着众人一指，那道漆黑的能量柱朝着众人席卷而来。速度之快，让众人根本来不及反应。

"糟了..."李武绝望地说道。

看来，即使他们伤到了虚空之主，也无法阻止它的攻击。这场战斗的结果，似乎已经注定了...
