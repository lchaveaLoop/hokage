# 第138章：联盟的团结与联合最强技再现

面对那道毁灭性的能量柱，所有人都在绝望中等待死亡的降临。然而，就在那道能量柱即将击中众人的瞬间，五位影级强者同时挡在了众人面前。

"火之国的火影，让我来保护大家！"纲手全身百豪之术全开，无数白蛇般的查克拉在她身上涌动。

"风之国的风影，我的沙子不会让任何人受到伤害！"我爱罗操控着沙子，在众人面前形成了一道沙之壁垒。

"水之国的水影，水遁·水阵壁！"长门结印，巨大的水壁在众人面前升起。

"土之国的土影，土遁·土流壁！"大野木双手拍地，无数岩石升起，形成了一道土壁。

"云之国的雷影，雷遁·雷光壁！"雷影双手合十，雷光形成了半透明的屏障。

五大影级强者联手，将最强的防御力施展出来，挡在了众人面前。

"各位，拜托了！"李武对着五位影级强者说道。

"交给我们吧！"纲手坚定地说道。

漆黑的能量柱重重地撞在了五大影级强者的防御上。轰隆！惊天动地的爆炸发生了。冲击波席卷了整个战场，地面都出现了巨大的裂痕。

"可恶...竟然能挡住我的攻击..."虚空之主的声音中带着愤怒。

"你不要太小看我们了！"纲手怒吼道，"我们虽然伤到了你，但你也别想轻易伤害我们的人！"

五位影级强者的防御力被完全消耗，但他们依然坚持着。虚空之主的力量虽然强大，但五大国的最强者也不是吃素的。

"各位..."李武看着五位拼命的影级强者，心中充满了感动，"我们也要战斗到底！"

"队长说得对！"鸣人握紧了拳头，"我们不能坐视不管！"

"是啊，"佐助点头道，"既然我们都选择战斗，就应该战斗到最后！"

众人重新站了起来，虽然身上都带着伤，但眼神中却充满了坚定。

"各位，"李武看着自己的伙伴们，"我知道我们的实力可能不足以对抗虚空之主，但只要我们团结一心，就一定能找到胜利的希望！"

"说得对！"卡卡西点头道，"单打独斗我们可能不是对手，但只要我们联手，就一定能创造奇迹！"

"那我们该怎么联手？"樱问道。

李武沉思片刻，然后说道："还记得我们在古代遗迹中发现的羁绊系统吗？它可以让我们之间的力量产生共鸣，产生更大的威力。"

"羁绊系统？"众人好奇地问道。

"是的，"李武解释道，"只要我们之间的羁绊足够强大，就能激活系统的最高级别，释放出联合的最强技！"

"那我们试试吧！"鸣人兴奋地说道。

"好！"李武点头道，"大家，让我们回想一下我们一起走过的路，那些战斗，那些感动，那些羁绊..."

众人闭上眼睛，开始回忆起过去的点点滴滴。从第一次见面，到一起战斗，到成为生死与共的伙伴。每一个回忆都充满了情感，每一个瞬间都值得珍惜。

"我想起了..."鸣人轻声说道，"第一次遇见佐助时的那种愤怒和不甘，还有后来一起成长的过程。"

"我想起了..."佐助冷着脸说道，"与鸣人最后一次决斗时的那种释然，还有后来一起战斗的默契。"

"我想起了..."雏田轻声说道，"第一次见到鸣人时的那种心动，还有后来一起经历的那些风雨。"

"我想起了..."樱握紧了拳头，"与卡卡西老师学习时的那种努力，还有与佐助、鸣人一起成长的过程。"

"我想起了..."勘九郎和手鞠对视一眼，"一起成长的那种羁绊，还有相互支持的那种温暖。"

五位影级强者也被众人的情感所感染，他们也回忆起了过去的点点滴滴。

"我想起了..."纲手说道，"三代目教导我时的那种期望，还有木叶村对我的那种包容。"

"我想起了..."我爱罗说道，"与李武相遇时的那种改变，还有后来成为风影的那种责任。"

"我想起了..."长门说道，"与弥彦、小南一起时的那种理想，还有后来寻找和平的那种执着。"

"我想起了..."大野木说道，"与三代目影的较量，还有后来成为土影的那种担当。"

"我想起了..."雷影说道，"与八尾人柱力的那种羁绊，还有守护云隐村的那种决心。"

每个人的回忆都充满了情感，每个人心中都涌动着强烈的羁绊。这些羁绊交织在一起，形成了一股强大的力量。

"感受到了吗？"李武微笑着说道，"这就是羁绊的力量！"

众人睁开眼睛，发现自己的身上都散发出了不同的光芒。鸣人身上是金色的查克拉光芒，佐助身上是银色的写轮眼光芒，雏田身上是柔弱的查克拉光芒，樱身上是温暖的查克拉光芒，勘九郎和手鞠身上是沙子和查克拉交织的光芒。

而五位影级强者的身上也散发出了强大的光芒：纲手是金色的查克拉，我爱罗是金色的沙子，长门是蓝色的查克拉，大野木是褐色的岩石，雷影是紫色的雷光。

"这就是羁绊的光芒..."长门震惊地说道，"从未见过如此强大的力量。"

"羁绊系统正在被激活..."李武说道，"它在响应我们的情感！"

突然，众人的光芒连接在了一起，形成了一个巨大的光球。光球中包含了所有人的力量，形成了一个完美的整体。

"这就是...联合的最强技..."李武激动地说道。

虚空之主看着这神奇的一幕，眼中闪过一丝惊讶："竟然能将这么多人的力量融合在一起？这确实很让我惊讶。"

"但惊讶是没有用的，"纲手坚定地说道，"我们的力量已经觉醒了！"

光球越来越大，最后形成了一个巨大的漩涡。漩涡中，所有人的力量都被完美地融合在一起，产生了一种全新的力量。

"羁绊·联合最强技·无缺之阵！"

李武大喝一声，光球爆发出了前所未有的光芒。那光芒照亮了整个天空，将所有的黑暗都驱散了。

虚空之主感受到了这股力量的威胁，它皱眉道："这是什么力量？竟然能让我感到威胁..."

"这就是羁绊的力量！"众人齐声说道。

光球化作无数道光线，朝着虚空之主射去。每道光线都包含着不同的力量，但它们的目标只有一个——消灭虚空之主！

虚空之主立刻防御起来，它胸口的黑洞再次出现，想要吞噬那些光线。但那些光线竟然穿过了黑洞，继续前进。

"竟然能穿透我的黑洞..."虚空之主惊讶地说道。

光线击中了虚空之主的胸口，那里的裂痕被进一步扩大。暗能量疯狂地泄露出来，虚空之主发出了痛苦的嘶吼。

"可恶...竟然伤到我这么深..."虚空之主愤怒地说道。

但光线的攻击还没有结束，它们在击中虚空之主后，分散开来，形成了一个巨大的结界。结界将虚空之主困在其中，开始不断地压缩它的能量。

"这是什么忍术？竟然能困住我！"虚空之主挣扎着，但它发现，结界的力量正在不断地吸收它的能量。

"这就是无缺之阵的力量..."李武解释道，"它能困住敌人，并不断地吸收对方的力量。"

"而且..."卡卡西补充道，"结界会随着我们力量的增强而变强！"

"你们的攻击确实很厉害，"虚空之主冷笑道，"但你们认为这样就赢了吗？"

说着，它身上的暗能量突然爆发，竟然强行突破了结界的束缚。结界被撕开了一个缺口，虚空之主从中冲了出来。

"看来低估你了..."李武皱眉道。

"我承认，你们的羁绊力量确实很强大，"虚空之主说道，"但想要依靠这点力量战胜我，还不够。"

"那我们就用更强的羁绊力量来击败你！"鸣人坚定地说道。

"对，我们还有更多的羁绊！"佐助点头道。

"我们的羁绊不仅仅是同伴之间，"樱微笑着说道，"还有我们对这个世界的守护之心！"

"还有我们对亲人的思念！"雏田轻声说道。

"还有我们对未来的希望！"勘九郎和手鞠齐声说道。

五位影级强者也加入了进来："还有我们对村民的责任！"

"还有对和平的渴望！"

"还有对朋友的守护！"

"还有对家人的爱！"

"还有对伙伴的信任！"

所有的羁绊交织在一起，形成了一股更加强大的力量。光球再次出现，但这一次，它的光芒更加耀眼，力量更加恐怖。

"羁绊·联合最强技·无缺之阵·最终形态！"

光球爆发出了前所未有的光芒，这一次，它不再是分散的光线，而是一道凝聚的光柱。光柱直冲云霄，将整个天空都照亮了。

"这...这是什么力量？"虚空之主震惊地说道，"它竟然比刚才的力量强大十倍！"

光柱击中了虚空之主的胸口，这一次，虚空之主再也无法承受。胸口的裂痕被完全撕裂，暗能量疯狂地泄露出来。

"不...这不可能..."虚空之主的声音中充满了恐惧，"我竟然会被这些蝼蚁伤到这种程度..."

"你不是蝼蚁，"李武坚定地说道，"但你确实做错了。你不能随意破坏这个世界，不能随意伤害无辜的人！"

"是你们逼我的..."虚空之主愤怒地说道，"是你们先拒绝我的和平！"

"所谓的和平，"纲手冷笑道，"就是将世界变成你的东西吗？这种和平，我们不要！"

"说得对！"众人齐声说道，"真正的和平，是建立在相互尊重和理解的基础上的！"

光柱的力量越来越强，虚空之主身上的暗能量被不断地吸收。它变得越来越虚弱，最后，整个身体都开始崩溃。

"可恶...我竟然会败给你们这些蝼蚁..."虚空之主不甘地说道，"但你们也别想好过..."

说着，它身上最后的暗能量开始自爆，准备与众人同归于尽。

"不好！"李武惊呼道，"它在自爆！"

"大家快躲开！"卡卡西立刻说道。

但已经来不及了，虚空之主的能量已经达到了临界点，随时都可能爆炸。

"看来我们必须阻止它..."李武坚定地说道，"即使要付出代价！"

"我们一起上！"众人齐声说道。

五位影级强者再次挡在了众人面前，他们准备承受这自爆的冲击。

"不行！"李武摇头道，"这样你们会受伤的！"

"那你说怎么办？"雷影怒吼道，"难道看着大家被炸死吗？"

李武沉思片刻，然后说道："或许...我们可以用羁绊之力来中和它的能量！"

"怎么做？"众人问道。

"无缺之阵不是能吸收能量吗？"李武解释道，"我们可以将所有的羁绊之力都注入进去，形成一个完美的吸收体！"

"那我们试试吧！"众人点头道。

光球再次出现，这一次，它变得更加完美。所有人的力量都融入其中，形成了一个完美的平衡。

"羁绊·联合最强技·无缺之阵·完美吸收！"

光球朝着虚空之主的自爆能量飞去，两者相遇的瞬间，发生了剧烈的碰撞。

光球在吸收虚空之主的能量，而虚空之主的自爆能量也在破坏光球。两者在激烈地对抗着。

"看来我们的力量还是不够..."李武皱眉道。

"那就用更强的羁绊！"鸣人坚定地说道，"我们还有更多的羁绊没有用！"

"对，我们还有对整个世界的守护！"佐助说道。

"还有对所有生命的尊重！"雏田说道。

"还有对未来的憧憬！"樱说道。

"还有对伙伴的信任！"勘九郎和手鞠说道。

五位影级强者也加入进来："还有对所有村民的责任！"

"还有对和平的渴望！"

"还有对朋友的守护！"

"还有对家人的爱！"

所有的羁绊交织在一起，形成了一股前所未有的力量。光球变得越来越大，最后形成了一个完美的球形，将虚空之主的自爆能量完全吸收了。

"成功了！"众人齐声欢呼道。

虚空之主的自爆能量被完全吸收，而它的身体也彻底崩溃了。最后，它的身体化作无数光点，消散在空气中。

"看来...我们赢了..."李武微笑着说道。

但就在这时，天空中最大的那个漩涡突然变得更加恐怖。一股更加强大的威压从中散发出来。

"什么？"众人震惊地说道，"虚空之主不是被消灭了吗？"

"不..."李武皱眉道，"那不是虚空之主..."

只见漩涡中，一双更加巨大的血红色眼睛缓缓睁开。那双眼睛充满了无尽的恶意和毁灭的气息，比之前的虚空之主还要强大十倍。

"这是..."众人惊恐地说道，"虚空之主的真正本体！"

原来，他们之前击败的，只是虚空之主的化身而已。真正的虚空之主，现在才刚刚降临...

（第138章完）
