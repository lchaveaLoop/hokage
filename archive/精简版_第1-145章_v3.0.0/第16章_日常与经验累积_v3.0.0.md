# 第十六章 日常与"经验累积"

**木叶隐村**

凯的特训开始后的第三周。

李武的生活变得规律而充实。

清晨五点，演武场。

与凯进行快速攻防训练，锻炼反应速度和判断能力。

上午，第七班执行任务。

C级、偶尔B级， steadily 获取经验值。

下午，指导小李修行。

教授寸劲的技巧，帮助小李理解机制。

晚上，研究系统。

查看技能树，分析战斗回放，优化战斗方式。

【角色：李武】
【等级：Lv.10（35.8%）】
【职业：无（可转职）】
【技能点：1】
【状态：正常】

"35.8%..."李武在心中默念。

"三周时间，从Lv.10（28.5%）提升到Lv.10（35.8%）。"

"增加了7.3%。"

"平均每周2.4%左右。"

"这个速度..."

李武皱眉。

"还不够快。"

"如果要在中忍考试前突破Lv.20..."

"平均每天需要提升约0.2级。"

"而现在..."

"每天只能提升约0.35%。"

"虽然还算可以，但距离目标还有差距。"

"必须找到更高效的方法。"

李武在心中列出目前的经验来源：

"1. 任务执行：C级任务+200-300经验，B级任务+400-500经验。"
"2. 与佐助切磋：每次+50-80经验。"
"3. 指导小李：每次+20-30经验。"
"4. 凯的特训：每次+30-40经验。"

"这些来源，加起来..."

"每天大概能获得150-200经验值。"

"从Lv.10到Lv.11，需要5000经验。"

"也就是说，需要25-33天。"

"太慢了。"

李武握紧拳头。

"必须优化。"

"方法一：增加任务数量。"

"如果每天能执行2-3个任务，经验值会大幅提升。"

"但是..."

"任务数量有限制，不能太频繁接取。"

"方法二：挑战更强的对手。"

"如果与佐助的切磋，能使用全力..."

"经验值会翻倍。"

"但是..."

"佐助可能会受伤。"

"方法三：参加对战考试。"

"如果参加忍者学校的对战考试..."

"一次可能获得500+经验。"

"但是..."

"这种考试不是每天都有。"

"方法四：深入险地。"

"如果进入死亡森林，遭遇强敌..."

"一次可能获得1000+经验。"

"但是..."

"风险太大，可能会死。"

李武冷静地分析。

"这些方法，都有各自的限制。"

"必须找到平衡点。"

"既能快速获得经验值，又不会太过危险。"

"那么..."

他做出决定。

"从明天开始，调整计划。"

"第一，主动申请更多任务。"
"第二，与佐助切磋时，适度使用全力。"
"第三，寻找机会参加对战考试。"
"第四，在保证安全的前提下，尝试深入险地。"

"这些方法..."

李武满意地点头。

"应该能让我快速提升等级。"

...

第二天，火影办公室。

"火影大人。"李武恭敬地说。

"我想申请更多的任务。"

三代火影看着李武，眼中闪过一丝惊讶。

"更多的任务？"

"是的。"李武认真地说，"我想要快速成长。"

"而成长，需要实战。"

"所以..."

"我希望能接取更多的任务。"

三代沉默了片刻。

然后，他笑了。

"很好。"三代点头，"这种上进心，很值得赞赏。"

"但是..."

"李武，你要明白。"

"任务不是越多越好。"

"更重要的是..."

"质量。"

"选择适合自己的任务，稳步成长。"

"这样，才能走得更远。"

"我明白了。"李武点头。

"但是..."他补充道，"我已经做好了准备。"

"而且..."

"我有自己的方法。"

"能确保安全的同时，快速成长。"

三代看着李武坚定的眼神，最终点头。

"好。"三代说，"既然你这么说，那就试试吧。"

"不过，你要记住。"

"不要勉强自己。"

"如果遇到危险，立即撤退。"

"明白了吗？"

"明白了。"李武恭敬地回答。

"很好。"三代微笑道，"那么..."

"今天的任务，是护送商队到波之国。"

"C级任务。"

"但是..."

三代认真地说。

"最近，波之国附近有流浪忍者出没。"

"可能比较危险。"

"你要小心。"

"我会的。"李武点头。

"那么，出发吧。"

李武离开火影办公室。

在心中，他快速分析。

"波之国任务..."

"上次已经去过一次了。"

"那次遭遇了再不斩和白。"

"这次..."

李武的眼中闪过一丝期待。

"可能还会遇到强敌。"

"如果...

能击败强敌...

经验值会很多。"

"而且..."

他想到。

"这次任务，是第七班执行。"

"也就是说，佐助和小樱也会去。"

"团队配合...

会更成熟。"

"很好。"

李武满意地点头。

"那么...

开始吧。"
