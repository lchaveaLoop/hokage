# 第二十四章 七日任务与"经验爆发"

**木叶隐村 · 周边**

日常任务发布后的第一天。

李武按照计划，连续执行任务。

【日常任务进度：1/7】

第一天，李武接取了B级任务：护送商队到波之国。

路上没有遇到强敌，只有一些小打小闹。

但是...

【任务完成】
【奖励：经验+200】

"200经验..."

李武在心中判断。

"虽然不多，但是累积起来也不少。"

"而且..."

他看向系统。

【日常任务进度：2/7】

第二天，李武接取了C级任务：清理村庄附近的野兽。

任务很简单，但是...

李武发现了系统的一个特殊功能。

【经验加成触发】
【连续任务经验加成：+10%】

"经验加成..."李武眼中闪过一丝惊喜。

"连续执行任务，会有经验加成。"

"这很好。"

第三天，李武接取了B级任务：护送重要物品。

这次遭遇了强敌。

【敌人出现】
【能级：蓝色（精英）×2】

"两个蓝色级敌人..."李武冷静判断。

"不过，现在的我已经Lv.12了。"

"应对两个蓝色级，应该没问题。"

战斗开始。

李武使用旋风腿，快速击败两个敌人。

【任务完成】
【奖励：经验+400（含加成）】

"400经验，含加成..."李武满意地点头。

"加成效果很明显。"

第四天，李武接取了B级任务：调查失踪的忍者。

这次，李武遭遇了一个小型基地。

里面有不少敌人。

【敌人出现】
【能级：白色（杂兵）×10，蓝色（精英）×1】

"10个白色级，1个蓝色级..."李武判断。

"正好，可以测试范围攻击的效果。"

旋风腿！

李武释放技能，同时攻击多个敌人。

效果很好，10个杂兵瞬间被击败。

然后，李武与蓝色级敌人单挑。

几分钟内，击败所有敌人。

【任务完成】
【奖励：经验+450（含加成）】

"450经验..."李武满意地点头。

"而且，系统的战斗经验加成也在生效。"

第五天，李武接取了B级任务：护送使节团。

路上遭遇了袭击。

但是...

使节团的安全，李武成功保障。

【任务完成】
【奖励：经验+300（含加成）】

第六天，李武接取了C级任务：帮助村民收割庄稼。

虽然是简单的农活，但是...

李武发现，系统也会给经验值。

【任务完成】
【奖励：经验+100（含加成）】

"即使是农活，只要是有意义的工作，系统也会给经验..."李武在心中思考。

"这个设定很有趣。"

"说明系统不只是奖励战斗，也奖励对社会的贡献。"

第七天，李武接取了最后一个任务。

B级任务：护送学者到研究基地。

这次任务很顺利，没有遭遇敌人。

【任务完成】
【奖励：经验+250（含加成）】

【日常任务完成】
【连续执行7天任务：成功】
【奖励：经验+500，技能点+1】

"连续7天任务完成..."李武眼中闪过一丝兴奋。

"而且，获得了1个技能点。"

"现在，我有3个技能点了。"

"虽然还是决定保留到转职后使用，但是..."

"有更多的选择总是好的。"

李武看向系统面板。

【角色：李武】
【等级：Lv.12（35.8%）】
【职业：无（可转职）】
【技能点：3】
【状态：正常】

"35.8%..."

"从10.5%提升到35.8%，增加了25.3%。"

"这7天的任务，让我大幅提升。"

"而且..."

李武满意地点头。

"按照这个速度，中忍考试前突破Lv.20..."

"并非不可能。"

"当然..."

他冷静地思考。

"还需要更多的实战经验。"

"特别是..."

"与强者的战斗。"

"比如..."

李武想到了中忍考试。

"宁次、佐助、天天..."

"这些人，都是很好的练手对象。"

"如果能与他们切磋..."

"不仅能获得经验值，还能帮助他们成长。"

"一举两得。"

"很好。"

李武满意地点头。

"接下来的三周..."

"会更有挑战。"

"但是..."

"他眼中闪过一丝决心。

"我已经做好了准备。"

"无论遇到什么困难..."

"我都会坚持下去。"

"为了..."

"变强。"

"保护..."

"所有...

"重...

"要...

"的...

"人...
