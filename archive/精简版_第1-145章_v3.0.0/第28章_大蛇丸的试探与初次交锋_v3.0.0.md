# 第二十八章 大蛇丸的试探与"初次交锋"

**死亡森林内部**

李武和小李进入森林已经半小时。

周围的环境阴暗潮湿，高大的树木遮挡了大部分阳光。

"哥..."小李警惕地环顾四周。

"这地方...给人很不舒服的感觉。"

李武点头。

"死亡森林...确实名不虚传。"

【任务进度：中忍考试·第二场】
【目标：拿到卷轴，到达中央塔】

【威胁等级：高】

"威胁等级高..."李武在心中默念。

"确实...这森林里有很多强敌。"

而且...

【查克拉感知全开】
【感知范围：200米】
【敌我识别：已启用】

在200米范围内...

其中有...

"佐助..."李武感受到熟悉的气息。

"在3点钟方向，大约150米。"

"宁次..."在6点钟方向，100米。

"还有..."

李武皱眉。

"一个很陌生的气息..."

"在9点钟方向，80米左右..."

"那个气息..."

李武感到系统发出警告。

【系统警报：发现S级威胁】
【目标：???】
【能级：紫色（上忍级）】
【威胁度：极高】
【建议：立即远离】

"S级威胁..."李武握紧拳头。

"这个级别..."

"只有在面对大蛇丸时才会有。"

"但是..."

他冷静思考。

"大蛇丸应该在森林深处..."

"怎么会在这里..."

而且...

李武发现那个气息正在...

"移动..."

"朝我们方向..."

他看向小李。

"弟弟..."

"嗯？"小李看向他。

"有强敌靠近。"李武冷静地说。

"虽然不知道是谁，但是..."

"我们需要做好准备。"

小李点头。

"明白!"

他摆出战斗架势。

"不管是谁..."

"我都会迎战!"

李武满意地点头。

"很好..."

就在这时...

一道身影从树林阴影中出现。

草绿色长发...

苍白的皮肤...

诡异的笑容...

大蛇丸。

"哦..."大蛇丸看着李武和小李。

"两个木叶的小鬼..."

他舔了舔嘴唇。

"你们的查克拉...很有趣呢。"

李武冷静地看着大蛇丸。

系统疯狂报警。

【系统警报：确认目标身份】
【大蛇丸：三忍之一，S级叛忍】
【能级：紫色（上忍级）】
【威胁度：极高】
【胜率预测：李武（Lv.12）vs 大蛇丸（？级）】
【胜率：3.2%】

"3.2%..."李武在心中默念。

"确实..."

"以我现在的实力..."

"很难战胜他。"

但是...

他冷静分析。

"大蛇丸现在的状态..."

"似乎不是完全体..."

而且...

李武注意到大蛇丸的查克拉流动有些紊乱。

"他在..."

"压制实力？"

"还是..."

"身体出了问题？"

不管怎样...

李武做出判断。

"不能硬拼。"

"需要用智慧和机制来应对。"

大蛇丸看着李武。

"你..."

"能感觉到我的强大吗？"

他微笑。

"很好..."

"这说明你有资格..."

"成为我的容器。"

李武皱眉。

"容器..."

"果然..."

"大蛇丸的目标是佐助..."

"但是..."

他看着大蛇丸的目光。

"好像对我也很有兴趣。"

大蛇丸走向李武。

"你的眼睛..."

"能看到很多有趣的东西吧？"

李武心中一惊。

"他察觉到..."

"我的系统？"

不对...

李武冷静判断。

"应该是..."

"我的查克拉感知..."

"被他注意到了。"

大蛇丸伸出舌头。

"让我看看..."

"你的身体..."

"有多适合我..."

就在大蛇丸即将触碰到李武的瞬间...

李武武神步发动!

瞬间闪避!

大蛇丸的手抓空了。

"哦..."大蛇丸眼中闪过一丝惊讶。

"不错的速度..."

"但是..."

他转身，再次攻击!

这一次...

他的速度更快!

小李冲上来!

"不许伤害哥哥!"

八门遁甲...

第一门·开!

小李的速度瞬间提升!

但是...

大蛇丸只是轻轻一挥手...

"太弱了..."

小李被击飞!

"弟弟!"李武大喊。

他查看小李的状态。

【角色：小李】
【状态：轻度眩晕】
【血量：75%】
"还好..."

"只是轻度眩晕..."

李武松了一口气。

但是...

大蛇丸已经出现在他面前!

"你的眼神..."

"我更喜欢..."

大蛇丸的手即将碰到李武的脖子...

就在这时...

李武系统触发特殊机制!

【系统警报：生死危机】
【紧急方案已激活】
【判定优先级临时提升】

李武眼中闪过一丝金光。

寸拳!

轰!

大蛇丸被击退三步!

"哦...?"大蛇丸惊讶地看着李武。

"刚才那..."

"是什么力量？"

李武喘息。

"那是..."

"体术。"

他不会暴露系统。

大蛇丸冷笑。

"体术...?"

"不..."

"那不是普通的体术..."

"你的拳头..."

"有一种特殊的查克拉..."

"金色..."

"前所未见..."

他眯起眼睛。

"你到底是什么人？"

李武冷静地回答。

"木叶忍者..."

"李武。"

"李武..."大蛇丸品味着这个名字。

"很好..."

"我记住你了。"

"下次见面..."

"希望你能成为更合适的容器..."

说完...

大蛇丸的身影消失在阴影中。

李武松了一口气。

"走了..."

【战斗结算】
【敌人大蛇丸（不完全体）】
【击退成功】
【经验+1500】

"1500经验..."李武眼中闪过惊喜。

"真是意外收获。"

他看向经验条。

【等级：Lv.12（88.5%）】
【获得经验：+1500】
【等级提升！】
【当前等级：Lv.13（0.0%）】

"Lv.13..."李武满意地点头。

"终于突破了。"

"而且..."

【技能点：3 → 4】

"又获得1个技能点。"

"现在有4个了。"

"弟弟..."

"你没事吧？"

小李晃了晃头。

"我..."

"我没事..."

"哥..."

"那个人..."

"太强了..."

李武点头。

"是的..."

"那是传说中的三忍之一..."

"大蛇丸。"

小李震惊。

"大蛇丸？!"

"那个S级叛忍？"

李武确认。

"是的。"

"所以..."

"我们必须更加小心。"

"接下来的考试..."

"会很危险。"

小李握紧拳头。

"我明白了!"

"我会更加努力!"

"为了..."

"不再拖哥哥的后腿!"

李武微笑。

"不..."

"弟弟..."

"你保护了我的决心..."

"我已经看到了。"

"你做得很好。"

小李眼中闪过一丝感动。

"哥哥..."

李武看向森林深处。

"但是..."

"事情不会这么简单..."

"大蛇丸出现..."

"说明..."

"木叶崩溃计划..."

"我必须..."

"想办法..."

"阻止..."

"减少伤亡。"

这是他来这个世界的原因。

改写悲剧。

拯救所有人。

"不能改变太多。"

"否则..."

"三代火影..."

"宁次..."

"小李..."

"佐助..."

"我爱罗..."

"我都会尽全力保护。"

李武看向天空。

"能救多少..."

"这就是我能做的。"

"走吧，弟弟。"

"我们需要卷轴。"

"是!"

两人继续前进。
