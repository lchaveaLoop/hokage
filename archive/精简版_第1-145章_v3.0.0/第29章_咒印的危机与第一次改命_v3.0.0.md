# 第二十九章 咒印的危机与"第一次改命"

**死亡森林另一处**

李武和小李正在搜寻卷轴。

突然...

李武感受到一股强烈的查克拉波动。

"那是..."他皱眉。

"佐助的查克拉？!"

而且...

查克拉感知显示...

【系统警报：佐助遭遇致命威胁】
【威胁源：大蛇丸】
【距离：350米】
【预计接触时间：45秒】

"45秒..."李武在心中计算。

"来不及..."

他看向小李。

"弟弟，你有幻术卷轴吗？"

小李点头。

"有的!"

"好..."李武说。

"你拿着卷轴先走。"

"我去救佐助。"

小李担忧。

"哥..."

"但是..."

李武摇头。

"没有但是。"

"佐助有危险..."

"我必须去。"

"这是..."

"作为朋友的承诺。"

小李咬唇。

"我明白了!"

"哥哥小心!"

李武武神步全力发动!

瞬间消失!

---

**死亡森林某处**

佐助倒在地上。

大蛇丸站在他面前。

"很好..."大蛇丸微笑。

"你..."

"确实很有天赋。"

"而且..."

"你的身体..."

"很适合我..."

他张开嘴。

蛇一般的舌头伸出。

在佐助脖子上...

咬下。

咒印。

"啊啊啊啊——!"

佐助发出痛苦的惨叫。

黑色的咒印开始蔓延。

【系统提示】
【目标：宇智波佐助】
【状态：咒印发作】
【生命力：极速下降】
【预测：若无干预，10秒内死亡】

"10秒..."大蛇丸冷笑。

"那么弱..."

"也许..."

"我不该在你身上浪费..."

就在大蛇丸准备离开时...

一道身影出现!

李武!

"不许碰他!"

李武一拳轰向大蛇丸!

但是...

大蛇丸只是轻松闪避。

"哦...?"他看着李武。

"又是你..."

"那个金色查克拉的小鬼。"

李武查看佐助的状态。

【角色：宇智波佐助】
【状态：咒印侵蚀（致死级）】
【生命力：12%】
【死亡倒计时：8秒】

"8秒..."李武握紧拳头。

"必须..."

"救他..."

突然...

系统触发特殊提示!

【系统警报：改命机会】
【目标：宇智波佐助】
【当前命运：死亡（咒印完全侵蚀）】
【可改写为：存活（咒印削弱版）】
【代价选项：】
  A. 消耗20%当前经验值
  B. 等级-2
  C. 暂时封印一个技能（7天）
【警告：此为强制改命，不可跳过】

"改命..."李武在心中默念。

"系统真的有这个功能..."

他看向佐助。

佐助的生命力已经降到8%。

"没有时间犹豫..."

李武做出选择。

"我选B..."

"等级-2。"

【确认：代价支付】
【等级扣除：Lv.13 → Lv.11】
【命运改变度：30%】
【目标：宇智波佐助】
【改变内容：咒印削弱至可控制程度】

一股强烈的虚弱感袭来。

李武差点倒下。

"这是..."

"等级下降的副作用..."

但是...

他看向佐助。

咒印停止了蔓延。

原本黑色的咒印...

现在变成了浅灰色。

【系统提示】
【目标：宇智波佐助】
【状态：稳定】
【咒印等级：削弱版（原版30%强度）】
【生命力：15%（停止下降）】

"成功了..."李武松了一口气。

大蛇丸皱眉。

"奇怪..."

"咒印怎么..."

"停止了..."

他看向李武。

"你做了什么？"

李武站起来。

尽管身体虚弱...

但是...

他的眼神坚定。

"我只是..."

"不希望看到朋友死。"

大蛇丸冷笑。

"朋友...?"

"有趣..."

"那么..."

"你也一起死吧。"

他冲向李武!

李武查看自己的状态。

【角色：李武】
【等级：Lv.11（0.0%）】
【状态：虚弱（副作用持续中）】
【战斗力：临时下降50%】

"50%..."李武在心中判断。

"以我现在的状态..."

"很难对抗他。"

但是...

他看向佐助。

"必须..."

"保护他..."

突然...

小李冲了出来!

"不许伤害我哥!"

小李...

八门遁甲...

第二门·休!

开启!

速度和力量大幅提升!

小李冲向大蛇丸!

但是...

大蛇丸只是冷笑。

"还是你..."

"太弱了..."

他准备反击。

就在这时...

李武系统再次提示!

【系统警报：隐藏任务触发】
【任务名称：兄弟的羁绊】
【目标：保护佐助和小李，击退大蛇丸】
【奖励：经验+2000，技能点+2，临时能力提升】
【时限：60秒】

"临时能力提升..."李武眼中闪过一丝希望。

【临时能力：武神附体（Lv.1）】
【效果：全属性+50%，持续30秒】
【是否激活？】

"激活!"李武在心中大喊。

【武神附体已激活】
【全属性+50%】
【持续时间：30秒】

一股强大的力量涌遍全身!

李武的虚弱感消失!

取而代之的是...

前所未有的力量!

金色念气在他周围爆发!

大蛇丸震惊。

"这..."

"这是什么力量？!"

李武武神步发动!

速度比之前更快!

他出现在大蛇丸面前!

旋风腿!

轰!

大蛇丸被踢飞!

"咳..."大蛇丸吐血。

"你..."

李武乘胜追击!

寸拳!

金刚碎!

连续技能!

大蛇丸节节败退!

"该死..."

"这两个小鬼..."

大蛇丸咬牙。

"看来..."

"今天不是好时机..."

他看向李武。

"你..."

"我记住你了..."

"下次见面..."

"我会夺取你的身体..."

说完...

大蛇丸消失!

【任务完成】
【击退大蛇丸】
【奖励：经验+2000，技能点+2】

李武松了一口气。

武神附体的效果消失。

他再次感到虚弱。

"但是..."

"至少..."

他看向佐助。

佐助的情况稳定了。

"救下来了。"

李武倒在地上。

小李冲过来。

"哥!"

"你没事吧？"

李武微笑。

"我没事..."

"只是..."

"有点累。"

他看向佐助。

佐助慢慢睁开眼睛。

"我..."

"我还活着？"

李武点头。

"是的..."

"你还活着。"

佐助看向自己的脖子。

"咒印..."

"怎么..."

"变成了灰色？"

李武说。

"我削弱了它。"

"代价是..."

"我自己的等级下降了。"

佐助震惊。

"等级...?"

他看着李武。

"你..."

"为了救我..."

"牺牲了自己的力量？"

李武摇头。

"不算牺牲..."

"只是..."

"我不希望看到朋友死。"

"仅此而已。"

佐助沉默。

然后...

他说。

"为什么..."

"为什么要救我？"

李武看着天空。

"因为..."

"我们是朋友。"

"而且..."

"我知道..."

"你背负着很重的命运。"

"复仇..."

"孤独..."

"但是..."

"你不是一个人。"

"你有..."

"鸣人..."

"小李..."

"我..."

"还有很多关心你的人。"

佐助眼中闪过一丝复杂的情感。

"我..."

"谢谢。"

李武微笑。

"不客气。"

他查看自己的状态。

【角色：李武】
【等级：Lv.11（0.0%）】
【技能点：6】
【状态：虚弱（恢复中）】

"6个技能点..."李武在心中想。

"虽然等级降低了..."

"但是..."

"技能点变多了。"

"而且..."

他看向经验条。

【经验+2000】

"从Lv.11开始..."

"还有2000经验。"

"很快就能回到Lv.13。"

李武站起身。

"走吧..."

"我们需要拿到卷轴。"

"还要..."

"前往中央塔。"

小李扶住他。

"哥..."

"你还能走吗？"

李武点头。

"可以。"

"虽然虚弱..."

"但是..."

"还能坚持。"

他看向佐助。

"佐助..."

"你还能战斗吗？"

佐助站起来。

"当然..."

"我可是..."

"要复兴宇智波的人。"

李武微笑。

"很好..."

"那么..."

"我们一起..."

"完成这场考试。"

三人继续前进。

而李武心中...

有一个明确的想法。

"改命..."

"确实需要代价。"

"但是..."

"这些代价..."

"都是值得的。"

"因为..."

"每一个被拯救的生命..."

"都是无价的。"

他握紧拳头。

"接下来..."

"我要保护更多的人。"

"三代火影..."

"宁次..."

"小李..."

"所有人..."

"我都不会让他们死。"

"这就是我来这个世界的意义。"

"改写悲剧。"

"创造..."

"一个更好的结局。"
