# 第五十七章 纲手回归与"新的火影"

**木叶隐村 · 村口**

阳光明媚的下午。

木叶村口聚集了大量的忍者。

消息传来——纲手姬回来了。

李武站在人群中，看着远处走来的人影。

金发，身材高挑，背后的"赌"字格外显眼。

"那就是传说中的三忍..."

李武心中暗道。

他的视野中，纲手的状态清晰地显示。

【系统提示】

【目标：纲手姬】

【等级：Lv.35】

【称号：传说中的三忍】

【能级：紫色（影级）】

【状态：查克拉雄厚】

【评估：影级强者，木叶的新火影候选人】

"影级强者..."

李武点了点头。

"这个实力，确实有资格成为火影。"

纲手走进村子，许多忍者向她行礼。

她微笑着回应，但眼神中带着一丝疲惫。

李武注意到，她的查克拉虽然雄厚，但有些不稳定。

"看来，三代火影的死对她打击很大。"

他心中分析着。

---

**火影办公室 · 数小时后**

三代召集了所有上忍和特别上忍。

李武作为特别上忍，也在列席人员中。

办公室内气氛凝重。

三代坐在办公桌后，脸色苍白但精神尚好。

"各位，我有一件重要的事情要宣布。"

三代的声音虽然虚弱，但依然有力。

"我决定，辞去火影之位。"

全场哗然。

"火影大人..."

"这怎么能..."

三代抬手制止众人的议论。

"我的年龄已经大了，木叶需要新的力量。"

"我已经找到了继承人。"

他看向纲手。

"纲手，你是初代火影的孙女。"

"你的实力和智慧，都足以胜任火影之位。"

"我希望你能接受这个职位。"

纲手沉默了片刻。

"三代爷爷..."

她的声音有些哽咽。

"您真的决定了吗？"

"是的。"

三代微笑着点头。

"木叶需要你。"

纲手深吸一口气。

"好。我接受。"

她转向众人。

"我会成为木叶的第五代火影。"

众人鼓掌，但李武注意到，纲手的眼中有泪光。

---

**火影办公室 · 仪式结束后**

众人离开后，三代叫住了李武。

"李武，你留一下。"

李武停下脚步。

"火影大人，您找我？"

"是的。"

三代看着李武。

"我听说你在追寻佐助的任务中表现出色。"

"虽然任务失败了，但你的实力得到了证明。"

李武点头。

"是的。我很遗憾没能带佐助回来。"

"不用自责。"

三代说。

"佐助的离开是他自己的选择。"

"你已经尽力了。"

他顿了顿。

"现在，纲手成为火影。"

"我想介绍你和她认识。"

"她会对你的实力感兴趣的。"

就在这时，纲手走了进来。

"三代爷爷，您找我？"

她看到李武，眼神一凝。

"这个孩子是..."

"这是李武，李洛克的哥哥。"

三代介绍道。

"他是最近崛起的天才忍者。"

"虽然只是下忍，但实力已经达到了上忍级别。"

纲手挑眉。

"上忍级别？"

她走到李武面前，上下打量。

"小小年纪就有这种实力..."

"让我看看你的本事。"

她突然出手。

纲手的拳头快如闪电，直取李武面门。

【系统警告】

【检测到敌意攻击】

【目标：纲手姬】

【等级：Lv.35】

【能级：紫色（影级）】

【攻击强度：高】

【建议：使用防御技能】

李武反应迅速。

"念气罩！"

金色的念气在他周围形成护盾。

**嘭！**

纲手的拳头打在念气罩上，发出巨响。

念气罩震动，但没有破碎。

纲手眼中闪过惊讶。

"哦？能挡住我的攻击？"

她收起拳头，笑了。

"不错，确实有上忍的实力。"

"但是还不够。"

"你的查克拉量还有待提升。"

李武撤去念气罩。

"纲手大人说得对。"

"我还需要更多的训练。"

纲手点头。

"很好。你有自知之明。"

她转向三代。

"三代爷爷，这个孩子确实是个天才。"

"我会关注他的成长。"

三代微笑。

"很好。"

他看向李武。

"李武，纲手大人成为火影后。"

"会有新的任务给你。"

"你要做好准备。"

李武点头。

"是，我明白了。"

---

**木叶隐村 · 火影就任仪式**

三天后，纲手正式就任五代火影。

全村的忍者都参加了仪式。

李武站在人群中，看着纲手站在火影岩上。

她穿着火影袍，神情庄重。

"我，纲手姬，在此宣誓。"

"成为木叶隐村的第五代火影。"

"我将用我的生命，守护这个村子。"

"守护村子里的每一个人。"

"火之意志，代代相传。"

"我绝不会让它断绝！"

她的声音响彻全村。

众人欢呼。

"五代火影大人！"

"纲手大人！"

李武也跟着鼓掌。

他的视野中，纲手的状态显示。

【系统提示】

【目标：纲手姬】

【称号：第五代火影】

【状态：查克拉稳定、信心坚定】

【评估：合格的火影，木叶的守护者】

"第五代火影..."

李武心中暗道。

"木叶迎来了新的时代。"

---

**火影办公室 · 仪式结束后**

纲手叫来了李武。

"李武，我有任务给你。"

她坐在火影的位置上，神情严肃。

"火影大人，请说。"

李武站在办公桌前。

"最近，我们收到了一些情报。"

纲手拿出一份文件。

"有一个名为'晓'的神秘组织开始活动。"

"他们的成员都是S级叛忍。"

"而且，他们的目标似乎是尾兽。"

李武心中一动。

"晓组织..."

他想起系统的提示。

"是的，晓组织。"

纲手继续说。

"我们需要收集他们的情报。"

"你是我们村子最有潜力的忍者。"

"我希望你能承担这个任务。"

李武点头。

"我明白了。"

"但是，火影大人。"

"这个任务很危险。"

"晓组织的成员都是影级强者。"

"我担心..."

"不用担心。"

纲手打断他。

"我知道晓组织的危险。"

"但是，我们需要情报。"

"而且，我相信你的实力。"

她站起身，走到李武面前。

"你虽然只是下忍。"

"但你的实力已经超越了上忍。"

"甚至，接近了影级。"

李武心中一惊。

她是怎么知道的？

"不要惊讶。"

纲手微笑。

"我可是医疗忍者。"

"我能感知到你体内的查克拉。"

"你的查克拉量，虽然不如我。"

"但是质量非常高。"

"而且，你的身体强度也远超常人。"

"这种规格，已经不是普通忍者能达到的了。"

李武沉默。

"火影大人，您想让我做什么？"

"很简单。"

纲手说。

"继续提升实力。"

"同时，保持对晓组织的警惕。"

"如果他们出现在木叶附近..."

"你要第一时间报告。"

"明白吗？"

李武点头。

"是，我明白了。"

"很好。"

纲手满意地点头。

"你可以走了。"

"记住，木叶需要你。"

李武离开火影办公室。

---

**木叶隐村 · 夜晚**

李武走在回家的路上。

他的心中思考着纲手的话。

"晓组织...S级叛忍..."

"系统早就警告过我。"

"看来，大战即将来临。"

他打开系统面板。

【系统提示】

【当前任务：收集晓组织情报】

【目标：了解晓组织的成员构成、活动模式、最终目的】

【进度：0%】

【奖励：经验值、技能点、特殊道具】

【警告：此任务危险等级极高】

"危险等级极高..."

李武苦笑。

"但是，我必须做。"

"为了木叶，为了鸣人，为了所有人。"

他握紧拳头。

"我会变强的。"

"强到足以对抗晓组织。"

"强到足以保护我想保护的人。"

就在这时，系统的提示音响起。

【系统提示：等级提升】

【当前等级：Lv.18 → Lv.19】

【经验值：85% → 100% → 15%】

【提升原因：完成重要事件（纲手就任）】

【获得技能点：1个】

【总技能点：9个】

"升级了？"

李武惊喜。

"原来完成重要事件也能获得经验值。"

他查看技能点。

"9个技能点..."

"应该怎么分配呢？"

他想了想。

"我还需要提升螺旋念气场和纷影连环踢。"

"还有，我应该学习念兽：龙虎啸。"

"召唤技能可以牵制敌人。"

他分配技能点。

【技能点分配】

- 螺旋念气场：Lv.5 → Lv.7（+2点）
- 纷影连环踢：Lv.3 → Lv.5（+2点）
- 念兽：龙虎啸：Lv.1（+5点）

"很好。"

李武满意地点头。

"现在我有了召唤技能。"

"战斗方式更加多样化了。"

"念兽：龙虎啸..."

"可以召唤龙和虎帮助战斗。"

"这个技能非常适合对付多个敌人。"

他站起身，看向火影岩。

"纲手大人已经就任火影。"

"新的时代开始了。"

"我会继续成长。"

"为了保护这个村子。"
