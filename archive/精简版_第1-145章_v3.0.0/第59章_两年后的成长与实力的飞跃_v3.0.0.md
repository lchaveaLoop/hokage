# 第五十九章 两年后的成长与"实力的飞跃"

**木叶隐村 · 时间跳跃**

时间飞逝。

转眼间，两年过去了。

木叶隐村发生了很大的变化。

新的建筑拔地而起，村子更加繁荣。

火影岩上，纲手的雕像已经完成。

五代火影，正式成为木叶的象征。

---

**木叶隐村 · 第七训练场**

清晨。

李武和小李正在训练。

两年过去了，两人的实力都有了巨大的提升。

"好了，今天就到这里。"

李武收起架势。

"哥哥，等等！"

小李喊道。

"我想再试一次。"

李武转头，看着弟弟。

两年前，小李还是个只能开启三门的少年。

现在，他已经能够开启五门了。

而且，体术更加精湛，速度更快，力量更强。

"好吧。"

李武微笑。

"让我们看看你的进步。"

两人再次对峙。

"八门遁甲，第一门，开！"

"第二门，休！"

"第三门，生！"

"第四门，伤！"

"第五门，杜！"

小李一口气打开了五门。

绿色的查克拉从他身上爆发，气势惊人。

李武的视野中，小李的状态清晰地显示。

【系统提示】

【角色：洛克·李】

【等级：Lv.12 → Lv.18】

【称号：木叶的美丽苍蓝猛兽】

【状态：八门遁甲（五门全开）】

【查克拉：绿色→淡紫色（大幅提升）】

【能级：蓝色→紫色（接近影级）】

【评估：巨大的进步，已经达到上忍级别】

"五门..."

李武点了点头。

"很好，小李。你的进步让我惊讶。"

"谢谢哥哥！"

小李激动地说。

"都是哥哥的指导。"

"如果不是哥哥一直帮助我训练。"

"我不可能有这样的进步。"

李武拍了拍小李的肩膀。

"这是你自己的努力。"

"我只是给你指明了方向。"

"接下来的路，要靠你自己走。"

小李认真地点头。

"是！我会继续努力的！"

---

**木叶隐村 · 火影办公室**

李武向纲手报告。

"李武，你回来了。"

纲手微笑。

"两年的修行，辛苦你了。"

"不辛苦。"

李武摇头。

"这是必要的修行。"

"而且，我的实力确实提升了很多。"

他打开系统面板。

【系统提示】

【角色：李武】

【等级：Lv.19 → Lv.25】

【职业：散打（主）、气功师（副）】

【称号：木叶的天才忍者】

【查克拉：蓝色→紫色（质变）】

【能级：紫色（影级）】

【技能熟练度：】

- 闪电之舞：Lv.4 → Lv.10（满级）
- 念气弹：Lv.5 → Lv.10（满级）
- 狮子啸：Lv.5 → Lv.10（满级）
- 螺旋念气场：Lv.7 → Lv.10（满级）
- 纷影连环踢：Lv.5 → Lv.10（满级）
- 念兽：龙虎啸：Lv.1 → Lv.8
- 寸拳：Lv.10（满级）
- 武神强踢：Lv.1（觉醒技）

"Lv.25..."

纲手惊讶地看着李武。

"两年时间，从Lv.19提升到Lv.25。"

"这种成长速度，简直不可思议。"

"而且，你已经达到影级实力了。"

李武点头。

"是的。"

"这两年的修行，让我理解了很多。"

"不仅是实力的提升。"

"还有对忍道、对战斗的理解。"

纲手满意地点头。

"很好。"

"现在的你，已经可以独当一面了。"

"甚至...可以担任上忍导师了。"

李武眼睛一亮。

"上忍导师？"

"是的。"

纲手说。

"我想让你带领一支小队。"

"指导新一代的下忍。"

"就像卡卡西带领第七班那样。"

李武思考片刻。

"我明白了。"

"我会接受这个任务的。"

"很好。"

纲手微笑。

"那么，明天开始。"

"你就要带领你的小队了。"

---

**木叶隐村 · 村外**

李武走出村子，来到一片开阔地。

他想测试一下自己现在的实力。

"两年了..."

李武喃喃自语。

"让我看看，我达到了什么程度。"

他深吸一口气。

"武神强踢！"

巨大的能量从他身上爆发。

金色的查克拉形成一只巨大的脚。

李武跳到空中，然后猛地踢下。

**轰！！！**

巨大的冲击波向四周扩散。

地面被踢出一个巨大的坑。

威力惊人。

"这就是觉醒技..."

李武看着眼前的破坏。

"武神强踢的威力，远超我的想象。"

"这就是...影级的实力吗？"

他打开系统面板。

【系统提示】

【技能：武神强踢】

【等级：Lv.1】

【类型：觉醒技】

【判定等级：神级】

【效果：大范围毁灭性打击，强制清空视野内所有低阶单位】

【消耗：大量查克拉】

【冷却时间：长】

"神级判定..."

李武眼中闪过兴奋。

"这个技能，可以对抗影级强者。"

"甚至...可以对抗更强的敌人。"

就在这时，他感受到一个熟悉的气息。

"鸣人？"

李武转身。

鸣人从树林中走出。

"李武！"

鸣人兴奋地跑过来。

"我看到了！你刚才那一招！"

"太强了！"

李武微笑。

"你也看到了。"

"是啊！"

鸣人激动地说。

"两年了，我们都变强了很多。"

"我掌握了仙人模式的基础！"

"而且，螺旋丸也强化了！"

"现在是我的'风遁·螺旋手里剑'！"

李武点头。

"很好。"

"让我看看你的实力。"

两人对视，同时笑了。

---

**木叶隐村 · 傍晚**

李武和鸣人、佐井一起吃饭。

"对了，李武。"

鸣人说。

"你有佐助的消息吗？"

李武摇头。

"没有。"

"这两年来，佐助音讯全无。"

"但是..."

他顿了顿。

"最近，我们收到了一些情报。"

"情报？"

鸣人眼睛一亮。

"什么情报？"

"佐助...投靠了大蛇丸。"

李武沉重地说。

"而且，他杀了大蛇丸。"

"现在，他组建了自己的队伍。"

"蛇小队。"

"目标是...鼬。"

"鼬..."

鸣人咬牙。

"佐助还是执意要杀鼬吗？"

"是的。"

李武点头。

"复仇的火焰，依然在他心中燃烧。"

"但是，鸣人。"

"鼬的事情，没有那么简单。"

"什么意思？"

"这两年来，我收集了一些情报。"

李武说。

"鼬当年灭族，似乎有隐情。"

"而且，他对佐助...似乎没有杀意。"

鸣人震惊。

"你是说...鼬不是为了杀佐助才灭族的？"

"我不知道。"

李武摇头。

"但是，我感觉事情没有那么简单。"

"所以，当佐助和鼬相遇的时候。"

"我们要在场。"

"我们要阻止悲剧的发生。"

鸣人点头。

"我明白了。"

"我也要变强。"

"强到足以阻止他们。"

---

**木叶隐村 · 夜晚**

李武回到家中。

他打开系统面板，查看自己的成长。

【系统提示】

【成长总结：两年修行】

【时间：2年】

【等级提升：Lv.19 → Lv.25】

【查克拉提升：蓝色 → 紫色（质变）】

【能级提升：蓝色（上忍级） → 紫色（影级）】

【技能提升：】

- 闪电之舞：Lv.4 → Lv.10（满级）
- 念气弹：Lv.5 → Lv.10（满级）
- 狮子啸：Lv.5 → Lv.10（满级）
- 螺旋念气场：Lv.7 → Lv.10（满级）
- 纷影连环踢：Lv.5 → Lv.10（满级）
- 念兽：龙虎啸：Lv.1 → Lv.8
- 寸拳：Lv.10（满级）
- 武神强踢：觉醒技（Lv.1）

【新获得技能：】

- 念气波：远程念气攻击
- 念气环绕：防御技能
- 蓄念炮：强力念气炮

【获得称号：】

- 木叶的天才忍者
- 影级强者
- 气功大师

【系统升级】

【当前版本：v3.0.0 → v3.5.0】

【新功能：】

- 战斗预测系统
- 弱点精准定位
- 团队协作分析

"两年时间，我达到了影级。"

李武满意地点头。

"而且，系统也升级了。"

"战斗预测系统..."

"可以预测战斗的走向。"

"弱点精准定位..."

"可以更准确地找到敌人的弱点。"

"团队协作分析..."

"可以分析团队的配合情况。"

"这些功能，都很有用。"

他站起身，看向窗外。

"两年了..."

"佐助，你在哪里？"

"鼬...你为什么要接近木叶？"

"晓组织...你们的最终目的是什么？"

李武握紧拳头。

"不管答案是什么。"

"我都会保护木叶。"

"保护鸣人。"

"保护...我想保护的一切。"

就在这时，系统的提示音响起。

【系统提示：重大事件预警】

【事件：晓组织正式来袭】

【目标：漩涡鸣人（九尾人柱力）】

【执行者：宇智波鼬、干柿鬼鲛】

【预计时间：1-2周】

【建议：做好战斗准备】

"1-2周..."

李武眼中闪过决意。

"鼬和鬼鲛要来了。"

"这次，我不会逃跑。"

"我要正面迎战。"

"为了木叶，为了鸣人。"

"为了...所有人的未来。"
