# 第六十一章 恢复训练与"气功师转职准备"

**木叶隐村 · 医院**

三天后。

李武躺在病床上，望着窗外。

"三天的治疗，伤势基本恢复了。"

他活动了一下肩膀。

伤口已经愈合，只剩下淡淡的疤痕。

纲手的医疗技术确实高超。

"但是..."

李武心中分析。

"这次与鼬和鬼鲛的战斗，暴露了很多问题。"

"1对2，而且都是影级强者。"

"我的实力还不够。"

"虽然达到了影级，但是...只是影级的入门。"

"鼬和鬼鲛，都是影级的巅峰。"

"差距...依然存在。"

他打开系统面板，查看自己的状态。

【系统提示】

【角色：李武】

【等级：Lv.26】

【职业：散打】

【查克拉：紫色（影级）】

【能级：紫色（影级入门）】

【状态：恢复中（90%）】

【技能熟练度：】

- 闪电之舞：Lv.10（满级）
- 念气弹：Lv.10（满级）
- 狮子啸：Lv.10（满级）
- 螺旋念气场：Lv.10（满级）
- 纷影连环踢：Lv.10（满级）
- 念兽：龙虎啸：Lv.8
- 寸拳：Lv.10（满级）
- 武神强踢：Lv.3
- 幻术抵抗：Lv.2

"Lv.26..."

李武眼中闪过思索。

"距离Lv.30，还有4级的差距。"

"但是，我有一个想法。"

"系统一直提示我可以转职气功师。"

"两年前，我就已经具备了转职条件。"

"但是，当时我选择了专注于散打。"

"现在...是时候双职业发展了。"

他打开转职界面。

【系统提示】

【转职系统】

【当前职业：散打（主）】

【可转职职业：气功师】

【转职要求：】

1. 等级达到Lv.25（已完成）
2. 掌握至少5个念气系技能（已完成）
3. 念气亲和度达到S级（已完成）
4. 完成转职任务（未完成）

【转职任务：】

- 任务名称：念气的真谛
- 目标：理解念气的本质，创造新的念气应用
- 奖励：转职成功，解锁气功师专属技能
- 惩罚：无

"转职任务..."

李武思考。

"理解念气的本质，创造新的念气应用。"

"这个任务...很有意思。"

他闭上眼睛，回忆这两年对念气的理解。

念气，是DNF世界中格斗家特有的力量。

它不同于查克拉。

查克拉是身体能量和精神能量的结合。

而念气，是纯粹的精神力量的具象化。

或者说，是意志力的外放。

"念气的本质..."

李武喃喃自语。

"这两年来，我一直在使用念气。"

"念气弹、念气罩、螺旋念气场..."

"但是，我真的理解念气的本质吗？"

"不，我只是在使用它。"

"没有真正理解它。"

他站起身，走出病房。

---

**木叶隐村 · 训练场**

李武来到训练场。

这里没有人。

清晨的训练场很安静。

"让我重新认识念气。"

李武深吸一口气。

他闭上眼睛，感受体内的念气。

金色的能量在体内流动。

温暖，柔和，但又强大。

"念气...不是查克拉。"

"查克拉是工具，可以用来释放忍术。"

"但念气...是意志的延伸。"

"我的意志有多强，念气就有多强。"

他伸出右手。

金色的念气在手掌中凝聚。

"念气弹。"

金色的能量球形成。

但是，李武没有释放。

他控制着念气弹，改变它的形态。

从圆形，到方形，再到不规则形状。

"原来如此..."

李武眼睛一亮。

"念气的本质，是控制。"

"不是单纯的释放，而是精细的控制。"

"我可以控制念气的形态、密度、方向。"

"甚至...可以让念气产生不同的效果。"

他尝试着让念气弹旋转。

"念气弹...旋转版。"

金色的能量球开始旋转，产生小型的旋风。

"这就是新的应用。"

李武满意地点头。

"但是，还不够。"

"转职任务要求的是'理解念气的本质'。"

"我还需要更深的理解。"

他闭上眼睛，进入冥想状态。

---

**木叶隐村 · 训练场**

一个小时后。

李武睁开眼睛。

"我明白了。"

"念气的本质，不是攻击，不是防御。"

"而是...连接。"

"连接我和外部世界。"

"通过念气，我可以感知周围的一切。"

"可以影响周围的一切。"

"甚至...可以治愈他人。"

他伸出手，掌心向上。

金色的念气缓缓升起，形成一个小型的念气场。

"念气...不仅是武器。"

"它是力量，但也是保护。"

"是攻击，但也是治愈。"

"这就是念气的真谛。"

就在这时，系统的提示音响起。

【系统提示】

【转职任务完成：念气的真谛】

【评价：完美】

【你理解了念气的本质：念气是意志的延伸，是连接内外的桥梁】

【奖励：】

- 转职资格已解锁
- 气功师专属技能树已开启
- 念气控制力大幅提升
- 新技能预览：念气波、念气罩、蓄念炮、念兽：龙虎啸（强化版）

"转职资格已解锁..."

李武眼中闪过兴奋。

"现在，我可以转职了。"

他打开转职界面。

【系统提示】

【转职仪式准备】

【是否开始转职气功师？】

【警告：转职过程中，身体会承受巨大压力，请确保身体状况良好】

【当前身体状况：90%（建议：休息1天后转职）】

"身体状况90%..."

李武思考。

"虽然可以转职，但是最好还是等身体完全恢复。"

"毕竟，转职过程可能会对身体造成负担。"

他决定。

"明天，开始转职。"

"今天，先做一些准备工作。"

---

**木叶隐村 · 家中**

李武回到家中。

他整理这段时间的战利品。

与鼬和鬼鲛的战斗，虽然受伤，但也获得了很多奖励。

【系统提示】

【战斗奖励：晓组织来袭】

【经验值：大量】

【等级提升：Lv.26】

【技能点：10点】

【特殊道具：】

- 晓组织情报碎片（可拼凑完整情报）
- 写轮眼研究数据（稀有）
- 鲛肌查克拉样本（稀有）
- 影级强者战斗录像（珍贵）

【金钱：500万两】

"这么多奖励..."

李武满意地点头。

"特别是情报碎片和战斗录像。"

"这些，对我了解晓组织很有帮助。"

他打开情报碎片。

【系统提示】

【晓组织情报碎片】

【内容：】

1. 晓组织的最终目标是捕获所有尾兽
2. 晓组织有10名成员，每人都是影级强者
3. 晓组织的首领，身份神秘
4. 宇智波鼬在晓组织中似乎有特殊目的
5. 晓组织正在计划大规模行动

"最终目标是捕获所有尾兽..."

李武眼中闪过担忧。

"鸣人是九尾人柱力。"

"晓组织...一定会来找鸣人。"

"我必须做好准备。"

他查看战斗录像。

【系统提示】

【影级强者战斗录像】

【内容：】

- 宇智波鼬 vs 李武（完整版）
- 干柿鬼鲛 vs 李武（完整版）
- 配合战术分析
- 弱点分析

"弱点分析..."

李武打开分析报告。

【系统提示】

【弱点分析报告】

【目标：宇智波鼬】

【弱点：】

1. 身体状况不佳，查克拉在衰退
2. 过度依赖写轮眼
3. 对佐助有特殊的感情
【建议：】

- 利用他的身体弱点，进行持久战
- 防止他被月读命中
- 利用佐助作为筹码

【目标：干柿鬼鲛】

【弱点：】

1. 鲛肌被破坏后，实力会下降
2. 近战能力强，但远程能力弱
3. 容易被激怒

【建议：】

- 保持距离，使用远程技能
- 优先破坏鲛肌
- 利用他的怒气，引诱他犯错

"详细的弱点分析..."

李武满意地点头。

"下次见面，我就能更好地对付他们了。"

---

**木叶隐村 · 夜晚**

李武躺在床上，准备休息。

但是，他睡不着。

"明天，就要转职气功师了。"

"双职业...散打+气功师。"

"这样的组合，会有多强？"

他打开系统面板，查看气功师的技能树。

【系统提示】

【气功师技能树（预览）】

【核心技能：】

- 念气波：远程念气攻击，可蓄力
- 念气罩：群体防御技能，可保护队友
- 蓄念炮：超强力念气炮，需要蓄力
- 念兽：龙虎啸（强化版）：召唤念兽攻击敌人
- 念气环绕：被动技能，提高念气亲和度
- 幻影爆破：幻影+念气爆炸的组合技
- 风之气息：Buff技能，提高攻速和移速
- 光之兵刃：Buff技能，增加光属性攻击
- 念气归元：终极技能，大范围念气爆发

"这么多技能..."

李武眼中闪过兴奋。

"而且，都是辅助+攻击的组合。"

"念气罩可以保护队友。"

"风之气息和光之兵刃可以提高战斗力。"

"这样的技能组合...适合团队作战。"

"散打是单体爆发，气功师是团队辅助。"

"两者结合...就是完美的战斗体系。"

他闭上眼睛，想象转职后的实力。

"双职业切换..."

"我可以根据战况，切换不同的战斗模式。"

"需要单体输出时，切换散打。"

"需要团队辅助时，切换气功师。"

"甚至...可以同时使用两个职业的技能。"

"这样的实力..."

李武握紧拳头。

"足以对抗任何敌人。"

就在这时，系统的提示音响起。

【系统提示：升级】

【等级提升：Lv.26 → Lv.27】

【恭喜！你达到了转职的推荐等级】

【转职成功率：100%】

【建议：立即进行转职】

"Lv.27..."

李武微笑。

"不知不觉，又升级了。"

"可能是这段时间的思考，也算一种修行吧。"

"明天，转职气功师。"

"然后，迎接新的挑战。"

他闭上眼睛，进入梦乡。
