# 第六十三章 晓组织的阴谋与"尾兽捕获计划"

**木叶隐村 · 火影办公室**

第二天清晨。

李武来到火影办公室。

纲手正在看情报文件。

"李武，你来了。"

纲手抬起头。

"身体恢复了吗？"

"是的，火影大人。"

李武点头。

"而且...我有重要的事情要报告。"

"什么？"

纲手放下文件。

"我转职了。"

李武说。

"现在，我是散打和气功师的双职业。"

"什么？！"

纲手震惊地站起来。

"双职业？这怎么可能？"

"在DNF世界中，转职是永久的选择。"

"一旦转职，就无法改变。"

"你怎么可能同时拥有两个职业？"

"这是系统的特殊功能。"

李武解释。

"双职业模式。"

"我可以同时使用散打和气功师的技能。"

"甚至可以在战斗中切换主职业。"

纲手沉默片刻。

"这样的话...你的实力..."

"提升了80%。"

李武说。

"现在的我，已经达到影级中阶。"

"甚至可以对抗影级巅峰的强者。"

纲手深深地看着李武。

"你的成长速度...真是惊人。"

"短短几年，从下忍到影级。"

"而且还是双职业。"

"木叶因为有你...真是幸运。"

就在这时，一名暗部冲进办公室。

"火影大人！砂隐村急报！"

"什么？"

纲手皱眉。

"说。"

"砂隐村...我爱罗风影被袭击了！"

"什么？！"

纲手震惊。

"怎么回事？"

"根据情报...晓组织袭击了砂隐村。"

"目标是我爱罗。"

"现在...我爱罗已经被抓走。"

"什么..."

纲手脸色沉重。

"一尾...被捕获了。"

李武心中一沉。

果然，晓组织开始了。

尾兽捕获计划。

"火影大人。"

李武说。

"我申请前往砂隐村。"

"营救我爱罗。"

纲手看着李武。

"李武...晓组织这次出动的人，肯定是影级强者。"

"甚至...可能是影级巅峰。"

"你确定要去吗？"

"我确定。"

李武点头。

"而且...我有信心。"

"现在的我，已经不是原来的我了。"

"双职业的力量...可以对抗晓组织。"

纲手沉默片刻，然后点头。

"好。"

"我批准你的请求。"

"但是，你不会一个人去。"

"我会组织一支救援队。"

"你担任副队长。"

"队长...是卡卡西。"

"卡卡西..."

李武点头。

"很好。"

"卡卡西的实力，我很清楚。"

"我们配合...一定可以成功。"

---

**木叶隐村 · 集合广场**

一小时后。

木叶救援队集结。

队长：旗木卡卡西

副队长：李武

队员：漩涡鸣人、春野樱、日向宁次、李洛克、天天

"这次的任务..."

卡卡西看着众人。

"是营救砂隐村的风影，我爱罗。"

"他被晓组织抓走了。"

"晓组织...就是那个S级叛忍组织吗？"

鸣人震惊。

"是的。"

卡卡西点头。

"晓组织的成员，都是影级强者。"

"这次的任务，非常危险。"

"但是...我们一定要成功。"

"因为我爱罗不仅是风影。"

"也是一尾的人柱力。"

"如果晓组织成功提取尾兽..."

"我爱罗会死。"

"什么..."

鸣人咬牙。

"我绝对不会让这种事发生！"

"我爱罗...和我一样是人柱力。"

"我理解他的痛苦。"

"我一定要救他！"

李武看着鸣人。

鸣人的决心，让他很满意。

"鸣人..."

他拍了拍鸣人的肩膀。

"这次的任务，我们一起努力。"

"一定会成功。"

"李武..."

鸣人看着李武。

"你...也变强了吧。"

"两年前，你就已经很强了。"

"现在...一定更强了。"

"是的。"

李武微笑。

"我转职了气功师。"

"现在是双职业。"

"双职业？！"

鸣人震惊。

"对。"

李武说。

"散打+气功师。"

"可以同时使用两个职业的技能。"

"甚至可以在战斗中切换。"

"太强了！"

鸣人激动地说。

"这样的话，我们一定会赢！"

卡卡西看着众人。

"好了，出发。"

"目标：砂隐村。"

众人点头，出发。

---

**火之国 · 前往砂隐村的路上**

李武和鸣人走在一起。

"李武，你说晓组织为什么要抓我爱罗？"

鸣人问。

"因为他们要收集尾兽。"

李武说。

"晓组织的最终目标，是捕获所有尾兽。"

"然后...利用尾兽的力量，发动战争。"

"战争..."

鸣人咬牙。

"他们为什么要发动战争？"

"我不知道。"

李武摇头。

"但是，我推测，他们的目标是颠覆忍界。"

"或者...实现某种'和平'。"

"和平？用战争实现和平？"

鸣人困惑。

"扭曲的和平。"

李武说。

"通过绝对的武力，强制实现和平。"

"这种方式...我认为是错误的。"

"真正的和平，应该建立在理解和信任的基础上。"

"而不是恐惧和武力。"

鸣人认真地点头。

"我也是这么想的。"

"所以...我要阻止他们。"

"不仅是救我爱罗。"

"还要...阻止他们的计划。"

"保护忍界的和平。"

李武看着鸣人。

鸣人...真的成长了很多。

以前的鸣人，只会冲动。

现在的鸣人，有了使命感。

"鸣人..."

李武说。

"你说得对。"

"我们一定要阻止晓组织。"

"不仅是这次。"

"以后，晓组织还会来抓你。"

"你是九尾人柱力。"

"他们的最终目标...是你。"

"我知道。"

鸣人点头。

"所以...我要变强。"

"强到足以保护自己。"

"强到足以保护木叶。"

"强到足以...实现我的梦想。"

"火影的梦想吗？"

李武问。

"是的。"

鸣人眼中闪过光芒。

"我要成为火影。"

"让所有人都认可我。"

"然后...实现真正的和平。"

李武微笑。

"我相信你能做到。"

"而且...我会帮你。"

"李武..."

鸣人感激地看着李武。

"谢谢你。"

"我们...是朋友。"

李武说。

"朋友之间，不需要谢谢。"

两人相视一笑。

---

**火之国 · 夜晚**

救援队在野外露营。

卡卡西和李武正在商量战术。

"李武，这次敌人的实力..."

卡卡西说。

"根据情报，晓组织这次出动的是迪达拉和蝎。"

"迪达拉...是爆炸艺术家。"

"擅长使用粘土炸弹。"

"蝎...是傀儡师。"

"擅长使用傀儡术。"

"两人都是影级强者。"

李武点头。

"我了解。"

"迪达拉的爆炸艺术，威力巨大。"

"但是...我有办法对付。"

"什么办法？"

"气功师的念气罩。"

李武说。

"念气罩可以防御爆炸。"

"而且，我可以用念气波远程攻击。"

"保持距离，对付迪达拉。"

"至于蝎..."

李武思考。

"傀儡师...最怕的是感知。"

"我可以用念气感知，找到蝎的本体。"

"然后...集中攻击。"

"很好。"

卡卡西点头。

"你的双职业，确实很有用。"

"散打对抗蝎的近战傀儡。"

"气功师对抗迪达拉的远程爆炸。"

"这样的战术...很完美。"

"但是，还有一个问题。"

李武说。

"我爱罗...被捕获了。"

"晓组织可能会开始提取尾兽。"

"我们必须尽快。"

"否则...我爱罗会有生命危险。"

卡卡西脸色沉重。

"是的。"

"所以，我们分两路。"

"一路追踪晓组织。"

"一路...直奔他们的据点。"

"我建议..."

李武说。

"主力追踪晓组织。"

"我...单独行动。"

"什么？"

卡卡西皱眉。

"太危险了。"

"不危险。"

李武摇头。

"我的速度很快。"

"而且...我有念气感知。"

"我可以更快地找到晓组织的位置。"

"然后...拖延时间。"

"等待主力赶到。"

卡卡西沉默片刻。

"好。"

"但是，你一定要小心。"

"如果遇到危险，立即撤退。"

"不要逞强。"

"我明白。"

李武点头。

---

**火之国 · 前往砂隐村的路上**

第二天清晨。

李武单独行动。

他展开念气感知。

金色的念气向四周扩散。

范围达到5公里。

"这么大的范围..."

李武满意地点头。

"我可以快速搜索晓组织的位置。"

他快速前进。

闪电之舞启动。

金色的闪光在森林中穿梭。

速度极快。

"按照这个速度..."

李武计算。

"半天时间，就能找到晓组织。"

就在这时，系统的提示音响起。

【系统提示】

【检测到前方有战斗痕迹】

【距离：3公里】

【特征：爆炸痕迹、傀儡线】

【推测：晓组织（迪达拉、蝎）】

"找到了..."

李武眼中闪过决意。

"比我想象的还要快。"

"那么...出发。"

他加速前进。
