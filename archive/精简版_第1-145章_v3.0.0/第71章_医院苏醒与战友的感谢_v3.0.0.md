# 第七十一章 医院苏醒与"战友的感谢"

**木叶医院 · 特护病房**

三天后。

李武...缓缓睁开眼睛。

白色的天花板。

消毒水的味道。

这是...医院。

"你醒了？"

一个温柔的声音响起。

李武转头。

是...红。

"红..."

李武虚弱地微笑。

"我...睡了多久？"

"三天。"

红说。

"你...昏迷了三天。"

"三天..."

李武心想。

"我...真的睡了很久。"

"你...生命力太低了。"

红说。

"纲手大人说...你差点死了。"

"差点死了..."

李武微笑。

"还好...我活下来了。"

"是的。"

红点头。

"你...活下来了。"

"而且..."

她看着李武。

"阿斯玛...也活下来了。"

"阿斯玛..."

李武心中一松。

"他...还好吗？"

"他很好。"

红说。

"他的伤...已经好了。"

"生命力...也恢复了。"

"谢谢..."

李武微笑。

"太好了。"

"李武..."

红流泪。

"谢谢...你救了阿斯玛。"

"如果不是你..."

"阿斯玛...已经死了。"

"不用谢。"

李武摇头。

"我只是...做了该做的事。"

"保护朋友...是应该的。"

"但是..."

红说。

"你付出了太多。"

"你的生命力...几乎耗尽。"

"如果再晚一点..."

"你...真的会死的。"

"我知道。"

李武点头。

"但是...我不后悔。"

"因为..."

他微笑。

"阿斯玛...是重要的朋友。"

"我...不会让他死的。"

"李武..."

红流泪。

"你...真的很了不起。"

"不。"

李武摇头。

"我...只是普通的人。"

"只是...做了普通的事。"

"保护朋友...很普通。"

"不普通。"

红摇头。

"很多人...不会这样做。"

"很多人...不会付出生命保护朋友。"

"但是...你做到了。"

"这...很了不起。"

"李武...谢谢。"

"不用谢。"

李武微笑。

就在这时...

病房门被推开。

阿斯玛走了进来。

"李武！"

阿斯玛激动。

"你醒了！"

"阿斯玛..."

李武微笑。

"你来了。"

"我...我等了三天。"

阿斯玛说。

"一直在等你醒来。"

"担心..."

"担心你...不会醒来。"

"别担心。"

李武说。

"我...不会死的。"

"我...还有很多事要做。"

"所以...我不会死。"

"李武..."

阿斯玛流泪。

"谢谢...你救了我。"

"如果不是你..."

"我...已经被飞段杀了。"

"不用谢。"

李武微笑。

"我们是朋友。"

"朋友...就该互相帮助。"

"但是..."

阿斯玛说。

"你付出的...太多了。"

"你的生命力...只有20%。"

"如果再晚一点..."

"你...真的会死的。"

"我知道。"

李武点头。

"但是...我不后悔。"

"因为..."

他看着阿斯玛。

"你活下来了。"

"这...就够了。"

"李武..."

阿斯玛流泪。

"我...不知道该怎么感谢你。"

"你...救了我的命。"

"我...这辈子...都欠你的。"

"不欠。"

李武摇头。

"你救过我...很多次。"

"这次...只是我回报你。"

"而且..."

他微笑。

"我们是朋友。"

"朋友之间...不需要计较这些。"

"朋友..."

阿斯玛流泪。

"是的...我们是朋友。"

"永远的朋友。"

"永远的朋友。"

李武微笑。

就在这时...

又有人推门进来。

是小葵。

"李武哥哥！"

小葵跑过来。

"你醒了！"

"小葵..."

李武微笑。

"你也来了。"

"我...我听说了。"

小葵说。

"你...为了救阿斯玛..."

"差点死了。"

"我...我真的很担心。"

"别担心。"

李武说。

"我...不会死的。"

"我...还要看着你长大。"

"看着你...成为优秀的忍者。"

"李武哥哥..."

小葵流泪。

"谢谢...你保护了爸爸。"

"如果不是你..."

"爸爸...已经不在了。"

"不用谢。"

李武微笑。

"我只是...做了该做的事。"

"而且..."

他看着小葵。

"你爸爸...也是重要的人。"

"我...会保护他的。"

"因为..."

"他...是你的爸爸。"

"李武哥哥..."

小葵抱住李武。

"谢谢...谢谢你。"

"不用谢。"

李武摸了摸小葵的头。

"你是个好孩子。"

"你爸爸...一定很骄傲。"

"是的。"

阿斯玛流泪。

"我...很骄傲。"

"有...这样的女儿。"

"有...这样的朋友。"

"我...真的很幸福。"

"幸福..."

李武微笑。

"是的...幸福。"

"我们...都很幸福。"

就在这时...

纲手走了进来。

"你醒了。"

纲手说。

"纲手大人..."

李武点头。

"我...醒了。"

"你知道...你有多危险吗？"

纲手严厉。

"你的生命力...只有20%。"

"如果再低一点..."

"你...就真的死了。"

"我知道。"

李武点头。

"但是...我不后悔。"

"不后悔？"

纲手生气。

"你以为...生命是儿戏吗？"

"你以为...你可以随便牺牲吗？"

"你...太天真了。"

"不。"

李武摇头。

"我...不天真。"

"我...知道生命很重要。"

"但是..."

他看着纲手。

"朋友的生命...更重要。"

"如果...我不救阿斯玛..."

"他...会死的。"

"所以...我必须救他。"

"即使...付出我的生命。"

"你..."

纲手沉默。

"你...这个家伙。"

"总是...这样。"

"总是...为了别人...牺牲自己。"

"但是..."

她叹气。

"我理解你的心情。"

"我理解...你想保护朋友的心情。"

"但是..."

她看着李武。

"你必须...保护好自己。"

"因为..."

"如果你死了..."

"你的朋友...会很伤心。"

"你保护的人...会很痛苦。"

"所以..."

"你必须...活下去。"

"活下去..."

李武点头。

"我...会活下去的。"

"因为..."

"我还有...要保护的人。"

"还有...要改变的世界。"

"所以...我不会死。"

"很好。"

纲手微笑。

"你...要记住。"

"保护自己...也是保护朋友。"

"如果...你死了..."

"没有人...能代替你。"

"所以...你必须活下去。"

"我会的。"

李武点头。

"我...会活下去的。"

"为了...所有朋友。"

"为了...要改变的世界。"

"很好。"

纲手点头。

"那...开始康复训练吧。"

"康复训练？"

李武问。

"是的。"

纲手说。

"你的生命力...虽然恢复了。"

"但是...你的身体...还很虚弱。"

"你需要...康复训练。"

"恢复...所有的实力。"

"恢复...所有的力量。"

"我...明白。"

李武点头。

"我会...努力康复的。"

"很快...我就能恢复实力。"

"很快...我就能达到Lv.30。"

"到时候...我就能觉醒了。"

"觉醒..."

纲手点头。

"我听说...你有特殊的系统。"

"可以...变强。"

"是的。"

李武说。

"命运改写系统。"

"可以...改写命运。"

"可以...让我变强。"

"很好。"

纲手微笑。

"那...我期待你的觉醒。"

"期待...你更强的力量。"

"谢谢。"

李武微笑。

李武心想。

"很好。"

"通过康复训练..."

"我可以恢复实力。"

"而且...还能领悟新技能。"

"这...很重要。"

他看向纲手。

"纲手大人..."

"什么？"

"康复训练...要多久？"

"取决于你的努力。"

纲手说。

"如果...你努力的话..."

"一周...就能恢复80%。"

"两周...就能完全恢复。"

"一周80%..."

李武点头。

"我...会努力的。"

"我会在一周内...恢复80%。"

"两周内...完全恢复。"

"很好。"

纲手微笑。

"我...期待你的表现。"

"我...不会让你失望的。"

李武坚定地说。

他看向窗外。

阳光...洒进病房。

这是...新的一天。

也是...新的开始。

李武知道...

他还有很多事要做。

他还有很多...要保护的人。

他还要...改变这个世界。

所以...

他必须...变强。

必须...觉醒。

必须...达到Lv.30。

"很快..."

李武心想。

"很快...我就能觉醒了。"

"到时候...我就能拥有...更强的力量。"

"就能...更好地保护所有人。"

"阿斯玛..."

他看向阿斯玛。

"下周...一起康复训练吧。"

"我们...一起变强。"

"好。"

阿斯玛微笑。

"一起变强。"

"一起...变得更强。"

李武点头。

他的眼中...闪过光芒。

那是...坚定的光芒。

也是...希望的光芒。

康复训练...

开始了。
