## 主角

### 李武 (Li Wu)

**基本信息**:
- 身份: 李洛克的哥哥,穿越者
- 年龄: 比小李大2岁
- 职业: 散打(Lv.18) → 武神(Lv.50) → 极武圣(Lv.75) → 归元(Lv.110)
- 系统: 位面舞王系统

**核心人设**:
- 数值暴力 + 兄弟情深
- 用绝对的力量保护弟弟
- 改变火影世界的悲剧命运

**性格特点**:
- 冷静、理性、分析型
- 对敌人冷酷,对亲人温柔
- 有原则,不滥杀
- 腹黑,偶尔毒舌

**语言风格**:
- 简洁有力,不废话
- 常用语: "数值,就是一切。" "太弱了。" "不够。"
- 内心独白: 理性分析,偶尔吐槽
- 对小李: 温和、鼓励、守护

**成长轨迹**:
- 第1-100章: Lv.1→50,散打→武神,建立守护决心
- 第101-300章: Lv.50→75,武神→极武圣,改命多人
- 第301-456章: Lv.75→110,极武圣→归元,终极对决,归家

**战斗风格**:
- 近战碾压,判定优先
- 念气感知,锁定弱点
- 技能连招,流畅衔接
- 冷静分析,战术智慧

**写作要点**:
- 战斗时展示数值碾压和机制优势
- 日常时展现对小李的守护
- 改命时展现价值观和代价承受
- 避免过度冷酷,保持人性温度

## 核心角色

### 李洛克 (Rock Lee)

**基本信息**:
- 身份: 李武的弟弟,第四班成员
- 年龄: 比李武小2岁
- 特点: 不会忍术和幻术,只会体术
- 绝招: 八门遁甲

**核心人设**:
- 极致努力 + 永不放弃
- 用努力证明自己
- 热血、执着、感恩

**性格特点**:
- 热血、乐观、积极
- 执着于修炼,永不放弃
- 对哥哥无比信任和依赖
- 对凯老师尊敬崇拜

**语言风格**:
- 热情洋溢,充满激情
- 常用语: "哥哥!" "我会努力的!" "青春!" "燃烧吧!"
- 口头禅: "这就是青春!"
- 对李武: 崇拜、依赖、感激

**成长轨迹**:
- 第1-100章: 八门第三门→第五门,自残→自信
- 第101-300章: 八门第六门,念气融合,实力飞跃
- 第301-456章: 八门第八门不死,达到神话级

**战斗风格**:
- 纯体术,速度型
- 八门遁甲,爆发力强
- 后期融合念气,威力倍增

**写作要点**:
- 展现努力和成长
- 与李武的兄弟情感
- 热血时刻和励志场景
- 避免过度弱化,给予高光时刻

### 日向宁次 (Hyuga Neji)

**基本信息**:
- 身份: 第四班成员,日向分家
- 血继限界: 白眼
- 绝招: 柔拳、回天、八卦六十四掌

**核心人设**:
- 天才 + 宿命论(前期) → 自由意志(后期)

**性格特点**:
- 冷静、理性、高傲(前期)
- 对宿命无奈 → 对自由感激(后期)
- 对李武从怀疑到信任到感激

**语言风格**:
- 冷静、简洁、理性
- 前期: "宿命无法改变。" "这就是命运。"
- 后期: "多谢你,李武。" "我终于自由了。"

**成长轨迹**:
- 第1-100章: 宿命论者,对李武怀疑
- 第101-200章: 被李武破解笼中鸟,获得自由
- 第201-456章: 创造念气柔拳,实力飞跃

**写作要点**:
- 展现从宿命论到自由意志的转变
- 与李武的关系从怀疑到感激
- 破解笼中鸟是重要情感节点

- 对敌人: "太弱了。" "数值,就是一切。"
- 对小李: "相信我,我会守护你。" "你做得很好。"
- 对队友: "跟上。" "交给我。"
- 内心独白: "这个世界的悲剧,我会改变。"

**小李**:
- 对李武: "哥哥!" "我会努力的!" "谢谢你,哥哥!"
- 对凯: "凯老师!" "青春!"
- 对敌人: "我不会放弃!" "这就是努力的力量!"

**鸣人**:
- "相信我!" "我要成为火影!" "绝不放弃!"

**佐助**:
- "哼。" "无聊。" "我要变强。"

**卡卡西**:
- "啊啊..." "真是麻烦..." "有趣..."

**凯**:
- "青春!" "燃烧吧!" "李武!小李!这就是青春的力量!"

### 角色互动

**李武 + 小李**:
- 温馨互动,兄弟情深
- 李武指导,小李努力
- 危机时刻,李武守护

**李武 + 鸣人**:
- 互相认可,好友关系
- 鸣人热情,李武冷静
- 战斗配合,默契十足

**李武 + 佐助**:
- 从冷漠到认可
- 李武改变佐助命运
- 互相尊重,实力认可

**李武 + 卡卡西**:
- 从试探到认可
- 卡卡西好奇李武的秘密
- 后期成为火影,支持李武

## 快速查询

**需要兄弟情感** → 李武 + 小李互动
**需要热血场景** → 小李 + 凯的青春时刻
**需要战术智慧** → 李武 + 宁次的配合
**需要改命高潮** → 李武改命三代/千代/鼬
**需要敌对冲突** → 李武 vs 大蛇丸/晓组织
**需要终极对决** → 李武 vs 使徒/大筒木
