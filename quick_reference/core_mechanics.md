# 核心机制速查

> 快速参考DNF系统的核心机制,用于写作时查阅。

---

## 系统名称

**位面舞王系统** (Planar Dance King System)
- 前身: DNF男格斗技能系统 / 直觉UI系统
- 特点: 游戏化机制在火影世界的具现化

---

## 三大核心机制

### 1. 数值暴力

**定义**: 不只是简单的数值碾压,而是机制强度、基础数值、成长潜力构成的三位一体碾压

**三个维度**:
- **机制强度**: 判定优先级、技能特性、系统规则
- **基础数值**: 力量、体力、智力、精神等属性
- **成长潜力**: 等级上限、技能进阶、觉醒突破

**写作应用**:
- 战斗时展示属性对比: 力量2000 vs 力量500
- 强调判定优先级: 抓取判定 > 霸体 > 普通攻击
- 展示成长速度: 等级快速提升,技能熟练度增长

---

### 2. 绝对判定

**定义**: 游戏化的优先级系统,高优先级判定可以无视低优先级防御

**判定等级** (从高到低):
1. **规则级判定** (极武霸皇拳) - 无视一切防御,强制执行
2. **抓取判定** (背摔、抛投) - 无视霸体,强制控制
3. **霸体判定** (钢筋铁骨) - 免疫普通控制和打断
4. **普通判定** (常规攻击) - 可被霸体无效化

**写作应用**:
- 展示判定碾压: "【抓取判定】【优先级:超越霸体】"
- 敌人震惊: "怎么可能?我的霸体居然被破了?!"
- 系统提示: "【判定成功】【强制控制】"

---

### 3. 直觉UI

**定义**: 可视化系统界面,显示敌人信息、血条、弱点等

**显示内容**:
- **血条**: 实时显示敌人生命值
- **能级颜色**: 白色/蓝色/紫色/金色
- **等级**: Lv.XX
- **弱点**: 红色标记的要害部位
- **技能冷却**: 自身技能的冷却时间

**写作应用**:
```
【系统扫描】
【目标: 桃地再不斩】
【能级: 蓝色·精英级】
【等级: Lv.35】
【生命值: 8500/8500】
【弱点: 颈部、心脏】
【威胁度: 中等】
```

## 念气体系

### 念气特性

**基础能力**:
- **念气感知**: 500米范围内感知所有生命体
- **念气护体**: 形成防护罩,减伤/格挡
- **念气攻击**: 附加在攻击上,增强威力
- **念气净化**: 净化负面状态、诅咒、控制

**进阶能力**:
- **念气罩·绝对**: 绝对防御,无视攻击
- **念气爆发**: 范围攻击,震退敌人
- **念气锁定**: 锁定目标,必中效果

### 写作应用

**念气感知**:
```
李武闭上眼睛,念气感知展开。

【念气感知】
【范围: 500米】
【锁定目标: 15】
【最强目标: 紫色·BOSS级 Lv.???】
```

**念气净化**:
```
李武的念气涌入佐助体内,包裹住咒印。

【念气净化】
【目标: 天之咒印】
【净化进度: 35%...68%...100%】
【净化成功】
```

## 技能系统

### 技能分类

**主动技能**:
- 寸拳、背摔、抛投、闪电之舞、武神强踢等
- 需要主动释放,有冷却时间

**被动技能**:
- 钢筋铁骨、霸体护甲、念气感知等
- 持续生效,无需释放

**觉醒技能**:
- 极武霸皇拳(一觉)、武神强踢·终(二觉)等
- 威力巨大,冷却时间长

### 技能熟练度

| 等级 | 名称 | 效果 |
|------|------|------|
| Lv.1-5 | 初学 | 基础威力 |
| Lv.6-10 | 熟练 | 威力+20% |
| Lv.11-15 | 精通 | 威力+50%,冷却-20% |
| Lv.16-20 | 大师 | 威力+100%,冷却-40% |
| Lv.MAX | 极限 | 威力+200%,技能进阶 |

### 写作应用

**技能升级**:
```
【系统提示】
【寸拳熟练度提升】
【当前等级: Lv.15 → Lv.16】
【达成: 大师级】
【威力提升: +100%】
【冷却时间: -40%】
```

## 改命机制

### 改命条件

1. **触发时机**: 原著角色生命垂危
2. **系统提示**: 【检测到改命机会】
3. **代价显示**: 【改命代价: 等级-1/-2/-5】
4. **选择权**: 玩家可以选择是否执行

### 代价等级

| 角色重要性 | 代价 | 示例 |
|------------|------|------|
| 次要角色 | 等级-1 | 千代、自来也 |
| 重要角色 | 等级-2 | 鼬、三代 |
| 核心角色 | 等级-5 | 白(未执行) |

### 写作应用

**改命流程**:
```
【系统提示】
【检测到改命机会】
【目标: 猿飞日斩】
【状态: 生命垂危】
【改命代价: 等级-1】
【是否执行?】

[主角选择]

【改命执行中...】
【消耗: 等级-1】
【当前等级: Lv.35 → Lv.34】
【念气涌入目标体内...】
【修复致命伤...】
【改命成功!】
【猿飞日斩: 存活】
```

## 战斗公式

### 伤害计算

**基础伤害** = 技能威力 × (力量/100) × 技能熟练度加成

**暴击伤害** = 基础伤害 × 2.5

**最终伤害** = 暴击伤害 - 敌人防御

### 写作应用

```
【寸拳】
【基础威力: 500】
【力量加成: 2000/100 = ×20】
【熟练度加成: Lv.16大师级 = ×2】
【基础伤害: 500×20×2 = 20000】
【暴击!】
【最终伤害: 20000×2.5 = 50000】
【敌人防御: 1000】
【实际伤害: 49000】

再不斩的血条瞬间清空!
```

## 系统提示格式

### 标准格式

使用【】或「」包裹系统提示:
- 【系统扫描】
- 【技能释放】
- 【等级提升】
- 【改命机会】

### 频率控制

- 每章3-5次系统提示
- 不超过8次
- 避免刷屏感

### 写作建议

**好的系统提示**:
```
【系统扫描】
【目标: 桃地再不斩】
【能级: 蓝色·精英级】
【等级: Lv.35】
```

**避免的系统提示**:
```
【系统提示: 你看到了再不斩】
【系统提示: 再不斩很强】
【系统提示: 你要小心】
【系统提示: 建议使用寸拳】
【系统提示: ...】(过于频繁)
```

## 快速查询

**需要展示数值碾压** → 属性对比 + 伤害计算
**需要展示机制优势** → 判定说明 + 系统提示
**需要展示成长** → 等级提升 + 技能进阶
**需要情感高潮** → 改命机制 + 代价承受
**需要战术智慧** → 念气感知 + 弱点锁定
