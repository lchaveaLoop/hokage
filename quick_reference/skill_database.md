## 常用技能速查

### 基础技能(Lv.1-20)

#### 寸拳
- **类型**: 主动攻击
- **判定**: 普通判定
- **威力**: 中等
- **特点**: 快速、精准、可连击
- **冷却**: 3秒
- **写作应用**: 最常用的攻击技能,适合碾压战斗

**示例**:
```
李武转身,寸拳轰出!

【寸拳】
【判定: 命中】
【暴击!】
【伤害: 9999】

再不斩倒飞而出!
```

---

#### 背摔
- **类型**: 主动控制
- **判定**: 抓取判定(无视霸体)
- **威力**: 中等
- **特点**: 强制控制,摔投敌人
- **冷却**: 5秒
- **写作应用**: 展示判定优势,破解霸体

**示例**:
```
【背摔】
【判定: 抓取判定】
【优先级: 超越霸体】
【强制控制!】

李武抓住敌人,狠狠摔在地上!

"怎么可能?!我的霸体居然被破了?!"
```

---

#### 抛投

### 被动技能

#### 钢筋铁骨
- **类型**: 被动防御
- **效果**: 增加物理防御和体力
- **等级**: Lv.1-20
- **特点**: 持续生效,减伤
- **写作应用**: 展示防御力,硬抗攻击

**示例**:


---

#### 霸体护甲
- **类型**: 被动防御
- **效果**: 免疫控制和打断
- **等级**: Lv.1-20
- **特点**: 持续生效,霸体状态
- **写作应用**: 展示霸体优势,无视控制

## 技能连招

### 基础连招(Lv.1-20)

**连招1: 寸拳连击**
```
寸拳 → 寸拳 → 寸拳 → 金刚碎
```
- 适用: 碾压型战斗
- 效果: 快速击杀

**连招2: 控制连招**
```
背摔 → 抛投 → 寸拳 → 金刚碎
```
- 适用: 技巧型战斗
- 效果: 控制+伤害

**连招3: 清场连招**
```
闪电之舞 → 金刚碎
```
- 适用: 多敌人
- 效果: 范围清场

---

### 进阶连招(Lv.20-50)

**连招4: 散打连招**
```
铁山靠 → 碎骨 → 寸拳连击 → 闪电之舞
```
- 适用: BOSS战
- 效果: 高伤害输出

**连招5: 破甲连招**

| 等级 | 名称 | 威力加成 | 冷却减少 |
|------|------|----------|----------|
| Lv.1-5 | 初学 | +0% | 0% |
| Lv.6-10 | 熟练 | +20% | -10% |
| Lv.11-15 | 精通 | +50% | -20% |
| Lv.16-20 | 大师 | +100% | -40% |
| Lv.MAX | 极限 | +200% | -50% |

## 技能组合建议

### 碾压型战斗
- 主力: 寸拳、闪电之舞
- 辅助: 霸体护甲、钢筋铁骨
- 特点: 快速、高效、视觉震撼

### 技巧型战斗
- 主力: 背摔、抛投、碎骨
- 辅助: 念气感知
- 特点: 控制、破甲、战术智慧

### 苦战型战斗
- 主力: 极武霸皇拳、武神强踢
- 辅助: 念气罩·绝对
- 特点: 规则级判定、绝对防御

### 终极对决
- 主力: 归元·万法归一
- 辅助: 所有技能融合
- 特点: 超越规则级、一击必杀

## 快速查询

**需要快速击杀** → 寸拳连击
**需要控制敌人** → 背摔、抛投
**需要范围清场** → 闪电之舞、金刚碎
**需要破甲** → 碎骨
**需要防御** → 霸体护甲、念气罩·绝对
**需要感知** → 念气感知
**需要规则级判定** → 极武霸皇拳、武神强踢·终
**需要终极一击** → 归元·万法归一
