# 场景模板库

> 本模板库提取自《创作手册》第2.2节,提供战斗、日常、情感场景的标准模板。

---

## 战斗场景模板

### 模板1: 碾压型战斗 (800-1,000字)

**适用场景**: 主角实力远超敌人,展示数值碾压

**结构**:

```
[开场] (200字)
- 敌人登场,展示威胁(技能/气势/背景)
- 系统扫描: 【蓝色·精英级 Lv.XX】
- 敌人轻视主角,放狠话
- 主角冷静分析,内心独白: "太弱了。"

[主体] (500字)
- 敌人率先攻击,展示技能
- 主角霸体硬抗或闪避,毫发无伤
- 系统提示: 【霸体护甲】【减伤95%】
- 主角反击: 技能名称+动作描写+视觉效果
- 判定说明: 【抓取判定】【强制打断】
- 伤害计算: 暴击!XX技能×2.5倍!9999伤害!
- 敌人震惊: "怎么可能?!"
- 主角追击: 连招组合,流畅衔接
- 敌人血条清空,倒地

[结尾] (300字)
- 围观者震撼反应(对话/内心独白)
- 敌人绝望/认输/逃跑
- 主角淡定离场,留下一句话
- 系统提示: 【获得经验XXX】【等级提升】(可选)
```

**示例片段**:

```
"雾隐之术!"再不斩的声音在浓雾中回荡。

【系统扫描】
【目标: 桃地再不斩】
【能级: 蓝色·精英级】
【等级: Lv.35】
【威胁度: 中等】

"小鬼,在我的雾隐术中,你连我在哪里都看不到!"再不斩的声音充满嘲讽。

李武闭上眼睛,念气感知展开。

【念气感知】
【范围: 500米】
【锁定目标: 1】
【位置: 正后方3米】

"找到了。"

李武转身,寸拳轰出!

【寸拳】
【判定: 命中】
【暴击!】
【伤害: 9999】

再不斩的身影从雾中倒飞而出,重重砸在地上。

"怎...怎么可能?!"再不斩震惊地看着李武,"你怎么看穿的?!"

"数值,就是一切。"李武淡淡地说。
```

### 模板2: 技巧型战斗 (1,200-1,500字)

**适用场景**: 敌人有特殊能力,需要展示主角的战术智慧

**结构**:

```
[开场] (300字)
- 敌人登场,系统扫描: 【紫色·BOSS级 Lv.XX】
- 主角认真对待: "这个对手...有点意思。"
- 双方试探性交手,展示基础能力
- 敌人展示特殊能力(血继限界/秘术/特殊体质)

[主体·第一阶段] (400字)
- 敌人占据上风,压制主角
- 主角分析敌人能力,寻找弱点
- 系统提示: 【分析中...】【弱点锁定】
- 主角内心独白: "原来如此..."

[主体·第二阶段] (400字)
- 主角改变战术,针对弱点
- 展示DNF机制优势(判定/数值/技能)
- 敌人惊讶: "你看穿了?!"
- 主角反击,逐渐占据上风

[结尾] (300字)
- 主角最终一击,敌人败北
- 敌人认可主角实力
- 围观者震撼反应
- 主角收获(经验/道具/情报)
```

### 模板3: 苦战型战斗 (1,500-2,000字)

**适用场景**: 遭遇强敌,展示主角的成长和突破

**结构**:

```
[开场] (400字)
- 强敌登场,系统扫描: 【紫色·BOSS级 Lv.???】
- 主角警觉: "这个气息...很危险!"
- 双方试探,势均力敌
- 敌人展示强大能力

[主体·僵持阶段] (600字)
- 激烈交锋,难分胜负
- 主角技能全开,敌人也不示弱
- 环境破坏,视觉冲击
- 围观者紧张观战
- 主角体力/查克拉消耗

[主体·突破阶段] (400字)
- 主角陷入困境,濒临败北
- 关键时刻,新技能领悟/等级突破/觉醒
- 系统提示: 【技能进阶】【等级提升】
- 主角实力暴涨

[结尾] (400字)
- 主角反败为胜,一击制敌
- 敌人震撼: "这就是...你的力量!"
- 围观者欢呼/震撼
- 主角疲惫但胜利
```

## 日常场景模板

### 模板1: 兄弟互动 (600-800字)

**适用场景**: 展现李武与小李的兄弟情感

**结构**:

```
[开场] (200字)
- 时间地点: 清晨/训练场/家中
- 小李的日常(训练/吃饭/聊天)
- 李武的观察

[主体] (400字)
- 兄弟对话,展现性格差异
- 小李的努力/热血/执着
- 李武的冷静/守护/关心
- 温馨互动(指导/鼓励/玩笑)

[结尾] (200字)
- 情感升华时刻
- 李武内心独白: "我会守护你的。"
- 小李的笑容/感激
```

**示例片段**:

```
清晨,训练场。

小李已经完成了第五百次俯卧撑,汗水浸透了衣服。

"哥哥!"小李看到李武,眼睛发亮,"你来了!"

"嗯。"李武点点头,"休息一下吧。"

"不行!"小李摇头,"凯老师说,青春就是要燃烧!如果今天不能完成一千次俯卧撑,我就倒立跑一百圈!"

李武叹了口气,走到小李身边,蹲下来。

"小李,努力是好事,但也要注意方法。"李武说,"过度训练会损伤身体,反而影响成长。"

"可是..."小李犹豫了。

"相信我。"李武拍了拍小李的肩膀,"我会帮你变强的。"

小李看着哥哥坚定的眼神,重重点头: "嗯!"

李武心中默念: "我会守护你的,小李。不会让你走上原著的悲剧之路。"
```

### 模板2: 训练场景 (800-1,000字)

**适用场景**: 展示主角的训练和成长

**结构**:

```
[开场] (200字)
- 训练目标: 技能熟练度/属性提升/新技能领悟
- 训练地点: 训练场/森林/秘密基地
- 训练准备

[主体] (600字)
- 训练过程描写
- 技能练习: 动作+效果+系统提示
- 失败→调整→成功
- 属性提升: 系统提示数值增长
- 新技能领悟: 灵光一现,技能解锁

[结尾] (200字)
- 训练成果: 系统提示
- 主角满意/期待
- 为后续战斗铺垫
```

```
[开场] (200字)
- 信息来源: 任务/传闻/间谍/偶遇

- 决策: 接取任务/调查/观望

- 主角行动计划
```

**适用场景**: 过渡章节,保持节奏

**结构**:

```
[开场] (300字)

- 主角继续前进

## 情感场景模板

### 模板1: 改命时刻 (1,200-1,500字)

**适用场景**: 改变原著悲剧,情感爆发

**结构**:

```
[开场·危机] (400字)
- 原著悲剧即将发生
- 角色生命垂危
- 主角赶到现场
- 系统提示: 【改命机会】【代价: 等级-1】

[主体·挣扎] (500字)
- 主角内心挣扎: 权衡代价
- 回忆与该角色的互动
- 情感铺垫: 为什么要救
- 最终决定: "我不能眼睁睁看着你死!"

[主体·改命] (400字)
- 主角行动: 技能/道具/代价
- 系统提示: 【改命成功】【等级-1】
- 角色存活,众人震撼
- 主角虚弱但坚定

[结尾·升华] (300字)
- 被救角色的感激/震撼
- 围观者的反应
- 主角内心: "值得。"
- 情感余韵
```

**示例片段**:

```
三代火影倒在血泊中,生命气息微弱。

【系统提示】
【检测到改命机会】
【目标: 猿飞日斩】
【改命代价: 等级-1】
【是否执行?】

李武看着三代苍老的面容,脑海中闪过无数画面。

第一次见面,三代温和的笑容。
任务汇报时,三代的认可。
木叶的守护者,火之意志的传承者。

"我不能眼睁睁看着你死!"李武咬牙,选择【是】。

【改命执行中...】
【消耗: 等级-1】
【当前等级: Lv.49】

李武的念气涌入三代体内,修复着致命伤。

"李武...你..."三代震惊地看着李武。

"三代大人,木叶还需要你。"李武虚弱地说,"火之意志,不能在这里断绝。"

三代的伤口开始愈合,生命气息逐渐稳定。

"值得。"李武心中默念。
```

## 过渡场景模板

### 模板1: 时间过渡 (200-400字)

**适用场景**: 跳过无关紧要的时间

**结构**:

```
- 时间流逝: 几天后/一周后/一个月后
- 期间发生的事: 简要概括
- 主角的变化: 等级/技能/关系
- 引入下一个场景
```

**适用场景**: 场景转换

**结构**:

```
- 离开当前地点
- 前往目的地的过程(简要)
- 到达新地点
- 新场景描写
```

```

## 使用建议

1. **灵活运用**: 模板是参考,不是限制,根据实际情况调整
2. **组合使用**: 一章可以包含多个场景模板
3. **保持新鲜**: 同类型场景变换角度和细节,避免重复
4. **注意节奏**: 战斗、日常、情感场景合理搭配
5. **突出特色**: 每个场景都要体现DNF机制和数值暴力特色

---

## 快速选择指南

**需要爽点时** → 碾压型战斗模板
**需要智慧时** → 技巧型战斗模板
**需要成长时** → 苦战型战斗模板
**需要温情时** → 兄弟互动模板
**需要铺垫时** → 信息收集模板
**需要高潮时** → 改命时刻模板
**需要过渡时** → 小冲突模板/过渡场景模板
