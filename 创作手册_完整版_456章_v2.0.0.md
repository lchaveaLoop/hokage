## 第一章：作品核心

### 1.1 核心创意

**一句话概括**：
> 穿越火影的DNF玩家，用绝对的判定位格碾压一切敌人，守护弟弟与伙伴，最终踏上归家之路。

**三个关键词**：
1. **数值暴力** - 不只是简单的数值碾压，而是机制强度、基础数值、成长潜力构成的三位一体碾压
2. **兄弟情深** - 李洛克的哥哥，用绝对的力量保护弟弟的努力
3. **双职业** - 散打（近战绝对判定）+ 气功师（中远距离压制）→ 极武圣 → 归元

**核心冲突设计**：
- **理念层**：机制 vs 技巧，直觉 vs 分析，系统 vs 自我
- **世界层**：念气体系 vs 查克拉体系，可视化直觉 vs 传统感知
- **宇宙层**：使徒位格 vs 大筒木神格，主角定位为破格者

### 1.4 写作核心原则

**日常写作的五个关键原则**：

1. **爽点优先** - 每章至少2个爽点（数值碾压、改命成功、技能震撼、情感升华）
2. **节奏控制** - 3章战斗 + 1章日常 + 1章情感，避免单一节奏疲劳
3. **兄弟为核心** - 所有剧情围绕"守护弟弟"展开，这是情感主线
4. **机制展现** - 每次战斗必须展现DNF机制特色（判定、数值、技能），不是简单的力量碾压
5. **语境一致** - 土著角色使用火影世界话术，系统术语只在系统提示或主角认知层出现

**写作时的自我检查**：
- 本章是否有明确的爽点？
- 本章是否推进了主线剧情或角色成长？
- 本章是否体现了"数值暴力"的核心特色？
- 本章是否让读者感受到主角的成长或情感变化？
- 本章是否存在“土著角色口播游戏面板”的穿帮？

### 1.5 设定一致性铁律

**术语分层（必须执行）**：

1. **系统层术语**（仅系统提示/主角内心可见）：`Lv`、伤害数值、判定优先级、经验值、词条
2. **世界层术语**（土著角色常用）：下忍/中忍/上忍、影级、查克拉量、血继限界、战术克制

**表达禁区（严格避免）**：

- 卡卡西、三代、团藏等土著角色直接说“Lv.50”“BOSS级”“数值碾压”
- 旁观者直接读取系统面板内容（除非剧情明确“共享感知/系统投影”）

**替换规则（统一用法）**：

- `Lv.50精英上忍` → `再不斩这种级别的上忍`
- `BOSS级敌人` → `头目级/影级威胁`
- `面板压制` → `查克拉与战斗经验的代差`

## 第二章：实用写作指南

> **本章目的**：提供日常写作的实用指导，包括写作流程、场景构建、对话技巧、战斗描写、情感节奏控制等。这是本手册的核心价值所在。

### 2.1 日常写作流程

#### 每日写作检查清单

**写作前（5分钟）**：
- [ ] 确认本章在整体剧情中的位置（查看第四章章节规划）
- [ ] 查看上一章结尾和本章开头的衔接点
- [ ] 确定本章核心剧情点（1-2个主要事件）
- [ ] 确定本章爽点设计（2-3个爽点类型）
- [ ] 确定本章情感节点（1个情感高潮或铺垫）

**写作中（90-120分钟）**：
- [ ] 开场（100-200字）：承接上章或设置悬念
- [ ] 主体（1,600-2,000字）：核心剧情展开，包含2-3个场景
- [ ] 结尾（200-300字）：留钩子或情感升华
- [ ] 系统提示：每章3-5次，不超过8次，避免刷屏感

**写作后（10分钟）**：
- [ ] 检查字数（目标：2,200±300字，范围：1,900-2,500字）
- [ ] 检查爽点密度（至少2个明确爽点）
- [ ] 检查对话比例（30-40%为佳）
- [ ] 检查系统提示格式统一（使用【】或「」）

#### 章节规划模板

```
【章节号】：第XX章
【标题】：[简短有力的标题，5-8字]
【字数目标】：2,200字（±300）
【核心剧情点】：
  1. [主要事件1，1句话]
  2. [主要事件2，1句话]

【场景清单】：
  场景1：[地点] - [事件] - [字数：200-400字]
  场景2：[地点] - [事件] - [字数：800-1,200字]
  场景3：[地点] - [事件] - [字数：400-600字]

【爽点设计】：
  爽点1：[类型] - [具体表现]
  爽点2：[类型] - [具体表现]
  爽点3：[类型] - [具体表现]（可选）

【情感节点】：[情感类型] - [具体表现]

【技能展示】：[本章展示哪些DNF技能]

【结尾钩子】：[如何吸引读者继续阅读下一章]
```

#### 爽点类型速查

**数值碾压类**（最常用）：
- 等级差距展示（Lv.50 vs Lv.20）
- 属性对比（力量2000 vs 力量500）
- 伤害计算过程（暴击！9999伤害！）
- 秒杀效果（一击必杀）

**技能震撼类**：
- 新技能首次展示
- 觉醒技能登场
- 技能连招组合
- 判定优先级碾压

**改命成功类**（情感爆发）：
- 原著角色存活
- 悲剧被改写
- 代价承受展示
- 众人震惊反应

**情感升华类**：
- 兄弟情深时刻
- 守护决心展现
- 价值观体现
- 角色成长瞬间

**围观震撼类**：
- 旁观者视角
- 众人议论
- 名声传播
- 实力认可

### 2.2 场景构建指南

#### 战斗场景结构（3-5-8原则）

**3段式结构**：
1. **开场试探**（300字）- 展示敌我差距，敌人轻视→主角冷静分析
2. **主体碾压**（800字）- 技能连招+数值展示+判定说明+敌人心理变化
3. **结束震撼**（200字）- 围观者反应+敌人绝望+主角淡定离场

**5个必备要素**：
1. **敌人能级显示** - 系统扫描：【蓝色·精英级】【紫色·BOSS级】
2. **数值对比展示** - 属性差距：力量2000 vs 力量500
3. **技能判定说明** - 机制解释：【抓取判定】【优先级：超越霸体】
4. **伤害计算过程** - 数值爽点：暴击！寸拳×2.5倍！9999伤害！
5. **敌人心理变化** - 轻视→震惊→恐惧→绝望

**8个避免事项**：
- 避免纯力量对轰，必须体现DNF机制特色
- 避免战斗过长（单场战斗不超过1,500字）
- 避免重复描写（同一技能变换角度描写）
- 避免忽略围观者（旁观者视角增强代入感）
- 避免敌人无脑送（给敌人合理的战术和反应）
- 避免系统提示刷屏（每场战斗3-5次系统提示）
- 避免忽略环境（战斗对环境的破坏增强视觉冲击）
- 避免结束仓促（给足围观者震撼反应的篇幅）

#### 战斗场景模板

**模板1：碾压型战斗**（800-1,000字）
```
[开场]（200字）
- 敌人登场，展示威胁（技能/气势/背景）
- 系统扫描：【蓝色·精英级 Lv.XX】
- 敌人轻视主角，放狠话
- 主角冷静分析，内心独白："太弱了。"

[主体]（500字）
- 敌人率先攻击，展示技能
- 主角霸体硬抗或闪避，毫发无伤
- 系统提示：【霸体护甲】【减伤95%】
- 主角反击：技能名称+动作描写+视觉效果
- 判定说明：【抓取判定】【强制打断】
- 伤害计算：暴击！XX技能×2.5倍！9999伤害！
- 敌人震惊："怎么可能？！"
- 主角追击：连招组合，流畅衔接
- 敌人血条清空，倒地

[结尾]（300字）
- 围观者震撼反应（对话/内心独白）
- 敌人绝望/认输/逃跑
- 主角淡定离场，留下一句话
- 系统提示：【获得经验XXX】【等级提升】（可选）
```

**模板2：技巧型战斗**（1,200-1,500字）
```
[开场]（300字）
- 敌人登场，系统扫描：【紫色·BOSS级 Lv.XX】
- 主角认真对待："这个对手...有点意思。"
- 双方试探性交手，展示基础能力
- 敌人展示特殊能力（血继限界/秘术/特殊体质）

[主体·第一阶段]（400字）
- 敌人占据上风，压制主角
- 主角防御/闪避，观察敌人弱点
- 系统分析：【弱点洞察】【发现破绽】
- 内心独白：分析敌人能力的本质和破解方法

[主体·第二阶段]（400字）
- 主角反击，针对弱点
- 技能组合：技能A破防→技能B控制→技能C输出
- 判定优先级展示：【抓取判定】>【霸体】
- 敌人应对，展示底牌
- 双方激烈交锋，环境破坏描写

[主体·第三阶段]（300字）
- 主角使用觉醒技能/强力技能
- 技能详细描写：动作+视觉效果+判定说明
- 敌人绝招对抗，但被判定压制
- 伤害爆发：连续暴击！总伤害XXXXX！
- 敌人战败

[结尾]（300字）
- 围观者震撼（多角度反应）
- 敌人认可主角实力
- 主角评价："你很强，但我更强。"
- 战后影响（名声传播/关系变化）
```

#### 日常场景节奏

**日常场景公式：对话60% + 描写30% + 内心独白10%**

**场景类型1：兄弟互动**（600-800字）
- 开场：日常生活场景（训练场/家中/街道）
- 对话：小李的热血+李武的冷静，性格对比
- 互动：小李展示进步/李武指导/兄弟日常
- 情感：温馨/搞笑/感动，强化兄弟情感线
- 结尾：小李的感激/李武的守护决心

**场景类型2：训练场景**（800-1,000字）
- 开场：训练目标设定
- 过程：技能练习+系统提示+数值提升
- 细节：训练方法的DNF特色（判定练习/连招熟练度）
- 成果：属性提升/技能熟练度增加/新技能领悟
- 结尾：为后续战斗埋下伏笔

**场景类型3：社交场景**（600-800字）
- 开场：社交场合（任务/聚会/偶遇）
- 对话：多角色互动，展现性格
- 信息：传递剧情信息/埋下伏笔/推进关系
- 冲突：小冲突/误会/挑衅（可选）
- 结尾：关系变化/信息获取/引出下一场景

### 2.3 对话写作指南

#### 网文对话基线（番茄/起点风格）

1. 直接写对白，不要连续使用“某某说：”广播式标签
2. 每2-4句对白插入动作、神态、视线或环境反馈，避免空口对话
3. 解释信息时拆成多轮攻防，不要单段讲设定
4. 同一段对白句长尽量控制，优先短句推进节奏
5. 让角色“带着目的说话”：施压、试探、回避、诱导，而不是作者解释

**❌ 生硬写法**：
```
卡卡西说："你的能力很特殊。你拥有一种叫做念气的力量，这种力量可以形成护盾，还可以增强攻击力，而且你的判定优先级很高。"
```

**✅ 推荐写法**：
```
卡卡西盯着李武，目光微沉。

"刚才那层金色护盾，不是查克拉。"

李武没有否认："念气。"

"从没听过。"

卡卡西停了一拍，又补了一句："而且你的出手，能强行打断对方动作。"
```

#### 对话推进剧情技巧

**信息传递型对话（避免说明文）**：

❌ **错误示范**：
```
卡卡西说："李武，你的能力很特殊。你拥有一种叫做'念气'的力量，这种力量可以形成护盾，还可以增强攻击力，而且你的判定优先级很高，可以压制大部分忍术。"
```

✅ **正确示范**：
```
卡卡西盯着李武，眼神复杂："刚才那层金色的护盾...不是查克拉。"

"念气罩。"李武平静回答。

"念气？"卡卡西皱眉，"从未听说过的力量体系。而且..."他顿了顿，"你的攻击，似乎无视了对方的防御姿态。"

"判定优先级。"李武简短解释，"抓取判定，强制打断。"

卡卡西沉默片刻："...你到底是什么人？"
```

**冲突型对话（展现性格）**：

```
佐助冷笑："就凭你？一个体术忍者？"
```

#### 战斗节奏控制

**短战斗（600-800字）**：
- 结构：开场100字 + 主体400字 + 结尾200字
- 适用：杂兵清场、碾压型战斗、快节奏剧情
- 技巧：一个技能解决，重点在围观者反应

**中战斗（1,000-1,200字）**：
- 结构：开场200字 + 主体700字 + 结尾300字
- 适用：精英级敌人、展示新技能、推进剧情
- 技巧：2-3个技能组合，展现战术思路

**长战斗（1,500-2,000字）**：
- 结构：开场300字 + 主体1,200字 + 结尾500字
- 适用：BOSS级敌人、觉醒技能首秀、重要剧情节点
- 技巧：多阶段战斗，敌人有反击，展现完整战术博弈

**避免战斗疲劳**：
- 连续战斗不超过3章
- 战斗后必有日常缓冲
- 同一技能变换描写角度
- 控制单场战斗长度（不超过2,000字）

- 兄弟情感线每10章1次互动（保持情感主线热度）
- 配角高光每15-20章1次（丰富情感层次）

#### 情感节点设计原则

**原则1：情感必须有铺垫**
- 改命场景前3-5章铺垫角色关系
- 觉醒场景前5-10章铺垫力量渴望
- 重大决定前展现价值观冲突

**原则2：情感必须有代价**
- 改命：等级下降/物品消耗/身体负担
- 守护：受伤/暴露实力/树敌
- 成长：痛苦/挣扎/自我怀疑

**原则3：情感必须有回响**
- 改命后角色关系变化
- 守护后被守护者成长
- 成长后实力提升展现

**原则4：情感不能廉价**
- 改命不能频繁（每50章最多2次）
- 兄弟情感不能重复（变换表现形式）
- 价值观体现要有深度（不是简单的"我要守护"）

#### 改命时刻情感爆发

**完整改命场景流程**（参考2.2节模板，这里强调情感处理）：

**阶段1：危机降临**（情感：紧张）
- 原著角色濒死，详细描写伤势
- 周围人的绝望反应
- 时间紧迫感营造

**阶段2：主角挣扎**（情感：冲突）
- 内心独白：理性分析代价
- 情感冲突：想救但代价很大
- 回忆闪回：与该角色的互动（铺垫情感）
- 价值观体现：最终决定的理由

**阶段3：系统提示**（情感：抉择）
- 改命选项出现
- 代价明确展示
- 倒计时压力

**阶段4：决定时刻**（情感：爆发）
- 主角做出选择
- 情感爆发："我不能让他死！"
- 行动描写：不顾代价的救援

**阶段5：改命成功**（情感：震撼）
- 角色存活，伤势恢复
- 众人震撼："怎么可能？！"

### 2.6 常见问题解决

#### 问题1：如何避免重复描写

**问题表现**：
- 寸拳用了50次，每次描写都一样
- 念气罩总是"金色护盾展开"
- 战斗场景千篇一律

**解决方案**：

**方法1：变换描写角度**
- 主角视角："寸拳轰出，念气凝聚。"
- 敌人视角："他看到一道金光袭来，然后失去了意识。"
- 围观者视角："那个金发少年只是轻轻一拳，敌人就倒下了？！"

**方法2：变换描写重点**
- 重点动作："右拳后撤，念气凝聚，寸拳轰出！"
- 重点效果："金色拳劲轰出，空气爆鸣，敌人倒飞！"
- 重点机制："【抓取判定】【强制命中】【暴击9999】"

**方法3：变换描写详细度**
- 简洁版（快节奏）："寸拳！"金光闪过，敌人倒地。
- 详细版（重要战斗）：参考2.4节版本2
- 机制版（首次展示）：参考2.4节版本3

#### 问题2：如何控制章节长度

**问题表现**：
- 写着写着就超过3,000字
- 或者写不到2,000字就没内容了

**解决方案**：

**方法1：使用章节规划模板**（参考2.1节）
- 提前分配字数：开场200字+主体1,800字+结尾200字
- 每个场景设定字数上限
- 写作中途检查进度（写到1,000字时检查是否完成一半内容）

**方法2：场景字数控制表**
| 场景类型 | 字数范围 | 适用情况 |
|---------|---------|---------|
| 开场场景 | 100-200字 | 承接上章/设置悬念 |
| 战斗场景 | 600-1,500字 | 根据敌人等级调整 |
| 对话场景 | 400-800字 | 信息传递/关系推进 |
| 训练场景 | 600-1,000字 | 技能提升/属性增长 |
| 情感场景 | 800-1,200字 | 兄弟互动/改命时刻 |
| 过渡场景 | 200-400字 | 时间/空间/情绪过渡 |
| 结尾场景 | 200-300字 | 留钩子/情感升华 |

**方法3：内容扩展/压缩技巧**

**内容不足时（<2,000字）**：
- 增加围观者反应（200-300字）
- 增加敌人心理活动（100-200字）
- 增加环境描写（100-200字）
- 增加系统提示详细信息（100字）
- 增加主角内心独白（200-300字）

**内容过多时（>2,500字）**：
- 删减重复描写
- 简化战斗过程（多个技能合并描写）
- 删减次要角色对话
- 简化环境描写
- 将部分内容移到下一章

#### 问题3：如何处理过渡章节

**问题表现**：
- 两个高潮之间的过渡章节很难写
- 不知道写什么内容
- 写出来很无聊

**解决方案**：

**过渡章节的三个作用**：
1. 缓冲情感（避免高潮疲劳）
2. 铺垫剧情（为下一个高潮做准备）
3. 丰富角色（展现日常生活和关系）

**过渡章节内容模板**：

**模板1：日常互动**（600-800字）
- 兄弟日常（训练/吃饭/聊天）
- 队友互动（任务/聚会/切磋）
- 村民互动（购物/委托/传闻）
- 作用：展现角色关系，增加温馨感

**模板2：训练场景**（800-1,000字）
- 技能练习（熟练度提升）
- 属性增长（系统提示）
- 新技能领悟（为后续战斗铺垫）
- 作用：展现成长，合理化实力提升

**模板3：信息收集**（600-800字）
- 情报获取（任务/传闻/间谍）
- 敌人动向（晓组织/大蛇丸/其他势力）
- 世界局势（各国关系/政治变化）
- 作用：推进剧情，埋下伏笔

**模板4：小冲突**（1,000-1,200字）
- 遭遇小敌人（杂兵/流浪忍者）
- 快速解决（展示实力）
- 引出信息（敌人背后的势力）
- 作用：保持节奏，不完全是日常

#### 问题4：如何平衡战斗与日常

**问题表现**：
- 连续战斗读者疲劳
- 连续日常读者无聊
- 不知道如何安排节奏

**解决方案**：

**黄金节奏比例：3战斗 + 1日常 + 1情感**

**具体安排示例**（每5章为一个节奏单元）：
```
第1章：战斗章（遭遇敌人，展示实力）
第2章：战斗章（激烈交锋，技能展示）
第3章：战斗章（战斗结束，围观震撼）
第4章：日常章（战后恢复，日常互动）
第5章：情感章（兄弟互动，情感升华）

[循环]

第6章：战斗章（新敌人登场）
第7章：战斗章（战术博弈）
第8章：战斗章（觉醒技能首秀）
第9章：日常章（训练提升）
第10章：情感章（改命时刻）
```

**节奏调整原则**：
- 重大战斗（BOSS级）可以连续5-7章
- 但战斗后必须有3-5章日常缓冲
- 情感高潮后必须有5-10章缓冲
- 每25章至少1个情感高潮

**节奏检查清单**：
- [ ] 最近5章是否都是战斗？（是→加入日常章）
- [ ] 最近5章是否都是日常？（是→加入战斗章）
- [ ] 最近25章是否有情感高潮？（否→安排改命/觉醒）
- [ ] 最近10章是否有兄弟互动？（否→安排兄弟场景）

## 第五章：质量标准

### 5.1 硬性发布标准（红线）

每章发布前必须满足以下红线，不满足则返工：
- 字数：1,900-2,500字（目标2,200字）
- 结构：开场承接明确，中段有冲突升级，结尾有钩子
- 爽点：每章至少2个有效爽点（战斗章建议3个）
- 情感：至少1个可感知情绪转折（不是纯说明）
- 术语：土著角色不得口播系统面板术语

### 5.2 单章验收评分（100分）

- 剧情推进（25分）：是否推进主线或关键关系
- 战斗/机制可读性（20分）：动作清晰、机制可见、节奏不乱
- 对话自然度（20分）：不播报、不堆叠、角色口吻稳定
- 情感有效性（15分）：读者能感知代价、动机、情绪变化
- 钩子强度（10分）：是否驱动下一章点击
- 术语与设定一致性（10分）：无穿帮、无OOC

**判定规则**：
- 85分及以上：可发布
- 75-84分：小修后发布
- 75分以下：必须重写关键段落

### 5.3 人物与对话一致性

**李武（主角）**：冷静、克制、行动优先于口号。台词短句为主，不长篇说教。  
**李洛克（弟弟）**：热血、直给、执行力强，情绪表达更外放。  
**卡卡西/三代/团藏等土著**：使用火影语境，不使用`Lv/BOSS/面板`类词。  
**配角群像**：反应要有差异，不写成同一个说话模板。

对话写作规则：
- 每2-4轮对话插入动作、表情或心理锚点
- 避免连续“某某说：”播报式句型
- 争执场景优先短句，情感场景允许留白

### 5.4 战斗章与日常章差异标准

**战斗章**：
- 至少1次战术变化（不能从头到尾单一碾压）
- 至少1次敌我认知反转（如误判、破局、反制）
- 系统提示建议3-5次，避免刷屏

**日常章**：
- 至少推进1条关系线（兄弟线/队友线/政治线）
- 至少埋1个后续可回收信息点
- 避免“纯过渡章”无事件状态变化

### 5.5 连载执行标准

- 更新频率：日更1章为基准，周更不少于6章
- 库存策略：常备2-3章可发布库存，防止断更
- 复盘节奏：每完成10章做一次结构复盘（节奏、术语、钩子）
- 大节点复盘：改命章、觉醒章必须做发布前二审

### 5.6 返工触发条件（新增）

满足任一条件即触发返工：
- 土著角色出现系统术语穿帮
- 单章没有明确冲突或目标变化
- 结尾无法形成下一章阅读动机
- 关键战斗读不清“谁在何时做了什么”
- 情感高潮仅靠说明，没有行为支撑
