# 第十八章：暗部盯了我一天，我也记住了他们

第二天，暗部的监视更明显了。

不是更近。

而是更专业。

清晨训练场，一个。

中午回家路上，一个。

傍晚屋顶上，又换了一个。

连晚上小李去院子里倒水，墙外都多了一道更轻的气息，像专门负责补视野死角。

李武没去拆，也没去躲。

他只是继续练，继续买菜，继续陪小李吃饭，像什么都不知道。

可念气感知已经把这些人的位置、换位节奏和惯用盯梢角度记了个七七八八。

他们在看他。

他也在看他们。

区别只在于，对方以为自己藏得住。

而李武知道，他们根本没意识到自己早就被反标记了。

上午练拳时，李武故意改了三次节奏。

第一次突然停手喝水。

树上的那股气息立刻往前压了半米。

第二次，他假装起身要离开训练场，墙外那股更轻的气息马上跟着换位。

第三次，他索性把出拳节奏全打乱，明明应该收势，却故意多往前踏了一步。

树上的人没动。

墙外那个却明显把注意力压向了他的落脚点。

这说明对方不只是在看他会不会暴走。

还在记他平时的出拳习惯、发力方向和临场反应。

很好。

一个偏谨慎。

一个更像负责接手跟踪。

还有一个藏得最远，不怎么看动作，只盯路线和接触对象。

这说明盯他的人不是随便扔出来的暗部，而是带着明确分工来的。

中午回家时，他还故意绕过标准路线。

先从木叶口岸那排新建宅子之后的菜巷走过去。

那里的屋檐低，巷口的水缸里还冒着热气。

屋顶那股气息立刻慢慢挪向他身后的方向。

他没有加速。

只是把视野切成几个扇区，一边留意自己的节奏，一边确认对方怎么在拐角处贴紧他的路线。

凯一上午都在狠狠干他们的体能，小李被练得趴在地上起不来，李武却还分出一半心神去记那些视线。

等训练结束，他故意比平时晚走了一刻钟。

树上那股气息也跟着多停了一刻钟。

这让他彻底确认了一件事。

对方不是在巡逻时顺便看一眼。

而是在专门盯他。

更准确点说，是在给他建立一份越来越细的记录。

他几点训练。

练多久。

出拳偏重左脚还是右脚。

回家会不会绕路。

中午回家路上，李武又绕了一段远路。

他先去了平民区最乱的菜市场，又故意在卖鱼摊前多停了一会儿，接着转进一条满是晾衣绳和木桶的小巷，最后才绕去药铺。

空气里全是腥味、药味和骂价声，普通感知在这种地方最容易被冲散。

可那几股暗部气息还是没丢。

一个留在屋顶。

一个落在街角。

还有一个干脆借着人群掩护，从更远的地方盯着他进出药铺的时间。

不远不近，刚好卡在“普通学生绝对察觉不到”的安全线。

李武买完药，故意站在门口翻了一会儿钱袋。

那股最轻的气息立刻往这边多偏了一点。

他没抬头，心里却已经有了判断。

这些人盯的，不只是他的日常。

更像是在观察，他到底能长多快。

这种感觉很差。

像被人拿着尺子量骨头。

可李武并不打算被这种感觉拖住。

监视摆脱不了，那就适应。

盯梢躲不开，那就反利用。

反正只要他们还没决定动手，监视本身就说明一件事。

他有价值。

既然有价值，那在真正变成威胁前，他就还有时间。

而时间，就是他现在最需要的东西。

想到这里，李武甚至没有再躲那几道视线。

他只是照常走、照常买、照常把药塞进袋子里，像默认了这场沉默的对峙。

傍晚回家后，小李正蹲在院子里绑新护腿。

“哥哥，这副真的比之前那双紧好多。”

“紧一点好，省得你再把自己踢废。”

“我不会了。”小李立刻抬头，“我记住了。”

他说得太认真，李武“嗯”了一声，没再说重话。

屋顶上那股气息却一直没挪。

对方离他们大概三十米。

很能藏。

但还不够。

吃饭时，小李一边扒饭一边抬头。

“哥哥，你今天怎么老看屋顶？”

“看风。”

“啊？”

“最近风向变了。”

小李愣了两秒，居然一本正经地点头。

“那我明天训练多穿一件。”

李武差点被这句话逗笑。

屋顶上那股气息却在这一刻轻轻绷了一下。

显然，对方也在听。

李武忽然意识到，这些人盯的其实不只是现在的他。

他们在量三天后的他、十天后的他、一个月后的他会长成什么样。

李武心里反而更稳了。

既然这些人想看，那他就让他们看。

看他怎么练。

看他怎么长。

看他怎么在他们的记录里，一点点变成不该被轻视的东西。

这笔账，他也记得很清楚。

夜里，李武躺回床上，闭着眼重新推演接下来的训练顺序。

念气不能停。

视野边缘忽然闪出一句系统提示：

> 【外部观察：精度++】

这行字一晃即逝。

他没慌。

只是把它记在心底：他们正在把他记录成“越来越值钱”的异常。

对小李的发力修正不能停。

护脉的思路也不能停。

屋顶上那股气息始终没撤。

可李武已经不在意了。

他只是把拳头慢慢握紧，在心里给自己定下一句更硬的底线。

你们可以盯。

但别妨碍我变强。

谁敢伸手拦，我就先记谁。

明天训练再过一次那些节奏。

让他们知道，记录再多，也只是把他从“有没有价值”一路推到“必须被重视”的节点。
