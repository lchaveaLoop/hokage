# 第二十一章：暗线

突破之后，监视立刻变了。

以前是暗部。

现在，根部也下场了。

区别很明显。

暗部再怎么藏，身上多少还带一点“人”的波动。

警惕、判断、偶尔的犹豫。

根部没有。

李武第三次绕训练场跑圈时，念气感知精准捕到两股几乎没有情绪的气息。

一个藏在树冠里。

一个压在出口方向的灌木后。

专业得像两只提前架好的眼睛。

他没有抬头，也没有乱看，只是照常跑完第五圈，照常喝水，照常在木桩前练拳。

可心里已经把两人的位置、分工和换位习惯记清了。

一个负责看。

一个负责跟。

更远一点的围墙外，还有第三个气息若隐若现，像专门等着收结果。

根部最麻烦的地方，不在于强。

而在于他们把“盯人”这件事，做得像记账一样稳定。

李武训练了两个小时，他们就看了两个小时。

他回家，他们就换人继续盯。

没有情绪。

没有失误。

像三台只负责记录的机器。

更麻烦的是，他们不只记“李武做了什么”。

还在记“李武什么时候会回头”。

上午凯让所有人做变向冲刺时，小李因为新护腿还没完全磨开，落地慢了半拍。

只是这么半拍，李武的视线就本能往那边偏了一下。

树上那股最稳的气息没有动。

围墙外那股更轻的，却立刻跟着把注意力压向了小李。

只这一瞬，李武心里就沉了下去。

他们果然在试。

试他的注意力会不会先落到弟弟身上。

为了确认得更细，李武下午还特意做了个小测试。

他故意把木桩旁的一块碎石踢远，转身去捡时突然变线，从训练场东侧绕了出去。

树上的那股气息先稳住。

灌木后的那股却立刻补位，直接卡到了他回家的必经路线上。

连这个都算到了。

李武又故意在水井边多停了十息，再突然折去另一边的坡道。

第三股最淡的气息这次也跟着动了。

说明对方盯的根本不是他今天练了什么。

而是他每一天的成长曲线。

李武这次没有立刻往家走。

他反而拎着药袋，在巷口一棵老树下多站了一会儿，像是在等什么人。

十息。

二十息。

巷口那股最冷的气息果然也稳着没动。

可更远处另一个负责收尾的点，却悄悄往弟弟放学会经过的那条路偏了偏。

不是杀意。

不是要动手。

可这种试探本身，比直接翻脸更恶心。

因为这代表他们已经把“小李”三个字，正式写进了那份观察里。

傍晚回家时，李武又在门口多停了一会儿。

他一边低头系鞋带，一边把感知铺到最薄。

果然，巷口尽头还有一道更冷的气息。

不进。

不退。

像在专门量这对兄弟和他之间的距离。

李武回屋后，没有急着坐下。

他先把今天训练用过的木桩位置、自己绕开的路线和三股根部气息出现的时间全在心里过了一遍。

上午固定。

下午补位。

傍晚收尾。

这不是临时起意的监视。

是已经开始形成记录表的跟踪。

换句话说，团藏那边大概已经有人在专门写他的成长报告了。

李武心里一点点发沉。

团藏现在盯着他的，不只是力量本身。

更是成长速度。

三个月前，他还只能用念气止痛。

现在，已经能感知三十米内隐藏气息，还能把念气推出体外。

这种速度，放在木叶，是会让高层不安的。

而不安，就会催生更严密的监控。

可李武没有因为这个焦躁，反而更快冷静下来。

根部越盯，说明他越值钱。

越值钱，就越不能在没摸透前轻易下死手。

这就是他还能利用的时间差。

想到这里，李武忽然不再只是被动记位置。

他开始反过来记他们的规矩。

谁负责白天。

谁负责傍晚。

谁在他和小李靠近时，会先把注意力分过去。

谁最像在替团藏判断“哪根线最适合拿来拽他”。

这不是反杀的机会。

至少现在不是。

但这是以后下手前必须先摸清的地图。

吃饭时，小李一边扒饭一边念叨今天凯又让他多跑了多少圈。

“哥哥，凯老师说我现在起步快多了！”

“嗯。”

“那我是不是很快就能追上宁次？”

“先别做梦，先把饭吃完。”

小李顿时蔫了一下，随后又立刻精神。

“那我吃完再做梦。”

李武低头喝了口汤，嘴角终于有了一点笑意。

可笑意刚落，窗外那股最淡的气息又轻轻挪了一寸。

像是有人在确认，这对兄弟到底是不是他的软肋。

李武眼神一下冷了。

他可以被盯。

但小李不行。

这一下，他终于彻底明白了。

团藏的人不是只在量“李武能长多快”。

也在量“李武为了什么会失控”。

他没有立刻关窗。

只是抬手把桌上的油灯往旁边挪了一寸，让窗纸上的影子刚好把自己和小李错开。

下一秒，外面那股气息果然停住了。

对方在重新确认谁是谁。

这让李武彻底明白，根部今天已经走到哪一步了。

他们不是在看一个孩子。

是在拆一件武器的说明书。

隔壁房间里，小李夜里翻了个身，梦里还在小声念着训练次数。

李武坐在窗边，借着那点动静把心里最后一丝浮躁也压了下去。

团藏可以盯他。

但不能碰小李。

这是底线。

真有人敢顺着这条线摸进来，那他之后练的就不只是变强了。

还得是怎么把伸进门里的手狠狠干断。

至少在他们写完这份报告前，他还有继续往上长的空档。

这就够了。

而且还不止一点。

远远不止。

谁踩过来，他就先记谁。

李武靠在窗边，直到外面那几股气息彻底换过一轮，才缓缓闭眼。

从今天开始，他要做的已经不只是变强。

还得让根部明白一件事。

他们可以看他。

但别想顺着他，看见小李。
