# 第五十二章：百豪之术

李武和鸣人再次冲向兜。

两人的配合越来越默契。

李武用念气波远程攻击，鸣人用影分身近战。

兜虽然强大，但面对两人的配合，也开始感到吃力。

"可恶...这两个小鬼..."他咬牙。

"明明等级差距那么大，为什么能撑这么久..."

李武在战斗中思考。

"兜的细胞活性化，应该快结束了..."他计算。

"已经过去4分钟了，还有1分钟..."

"只要撑过这1分钟，兜的力量就会下降..."

"到时候，就是我们的机会！"

他看向鸣人："鸣人！再撑1分钟！"

"好！"鸣人大喊。

两人继续战斗，不断躲避兜的攻击。


1分钟后——

兜的身体突然一颤。

绿色的查克拉消失，肌肉恢复正常。

"呼...呼..."他喘着粗气。

"细胞活性化...结束了..."

"现在！"李武大喊。

他和鸣人同时冲向兜！

"疾风连拳！"

李武的拳头如疾风般连续击出！

砰砰砰砰砰！

五拳连击，全部击中兜！

"螺旋丸！"

鸣人的螺旋丸紧随其后，击中兜的腹部！

轰！

兜被击飞，重重摔在地上！

"啊！"他惨叫。

这次，他的伤势很重，无法立刻恢复。

"呼...呼..."李武喘着粗气。

【念气值：2,600→1,800】

"赢了..."他松了一口气。

"终于击败兜了..."

鸣人兴奋地大喊："我们赢了！李武！"

"嗯。"李武笑了笑。

但就在这时——

大蛇丸的声音响起。

"兜，你真是让我失望。"他冷冷道。

"居然被两个小鬼击败了。"

兜挣扎着站起来："对不起...大蛇丸大人..."

"算了。"大蛇丸说道，"撤退吧。"

"纲手比我想象的更强，而且自来也也在这里。"

"我们赢不了。"

他看向纲手："纲手，这次算你赢了。"

"但下次，我一定会得到你的医疗忍术。"

"做梦吧。"纲手冷冷道。

大蛇丸和兜的身影开始消失。

"下次见，自来也。"大蛇丸说道。

"还有那两个小鬼，你们的力量很有趣。"

"我会记住你们的。"

他们消失在黑暗中。

战斗，结束了。

李武跪在地上，喘着粗气。

【当前状态】
【等级：Lv.32】
【生命值：5,200/6,100】
【念气值：1,800/4,600】
【状态：中度疲劳】

"终于...结束了..."他思考。

"兜Lv.65，我和鸣人联手才勉强击败..."

"而大蛇丸Lv.78，纲手Lv.73，自来也Lv.75..."

"他们的战斗，我完全插不上手..."

"我还是太弱了..."

纲手走过来，看着李武和鸣人。

"你们做得很好。"她说道，"能击败兜，已经很了不起了。"

"谢谢，纲手婆婆。"鸣人笑道。

"不要叫我婆婆！"纲手不满道。

但她的脸上，露出了笑容。

自来也也走过来："李武，鸣人，你们的配合很好。"

"这次战斗，你们学到了很多东西吧？"

"是的。"李武点头，"我学到了团队配合的重要性。"

"还有，我和影级强者的差距。"

"很好。"自来也说道，"认识到差距，才能更好地成长。"

"你们还年轻，还有很大的成长空间。"

"嗯。"李武用力点头。

纲手看着两人，眼中闪过复杂的情绪。

"你们...让我想起了绳树和断。"她低声道。

"他们也是这样，充满梦想，想要保护同伴。"

"但最终，他们都死在了战场上。"

"纲手大人..."静音担心地看着纲手。

"没事。"纲手摇头，"我已经想明白了。"

"正是因为有他们的牺牲，才有现在的木叶。"

"而我，作为他们的继承者，应该继承他们的意志。"

"保护木叶，保护这些年轻人。"

她看向自来也："自来也，我答应你。"

"我会回木叶，担任第五代火影。"

"太好了！"鸣人兴奋地大喊。

"纲手婆婆要当火影了！"

"不要叫我婆婆！"纲手再次不满道。

但她的脸上，露出了久违的笑容。

自来也也笑了："欢迎回来，纲手。"

"木叶，需要你。"

"嗯。"纲手点头。

李武看着这一幕，心中感慨。

"纲手...终于决定回木叶了..."他思考。

"这样的话，木叶就有了新的火影。"

"村子会变得更强。"

"而我，也要继续变强。"

"总有一天，我要达到影级，甚至超越影级。"

"保护木叶，保护小李，保护所有重要的人。"

他握紧拳头，眼神坚定。

...

三天后。

李武、鸣人、自来也、纲手、静音，一行五人回到了木叶。

村子的大门前，已经有很多人在等待。

"纲手大人！"

"第五代火影大人！"

村民们欢呼着。

纲手看着熟悉的村子，眼中涌出泪水。

"木叶...我回来了..."她低声道。

"绳树，断，你们看到了吗？"

"我回来了，我会保护这个村子。"

"我会继承你们的意志。"

她擦去眼泪，露出笑容。

"大家，我回来了。"她大声道。

"从今天开始，我是木叶的第五代火影！"

"我会保护这个村子，保护所有人！"

"这就是我的承诺！"

村民们欢呼着，掌声雷动。

李武站在人群中，看着纲手。

"第五代火影...纲手..."他思考。

"木叶，终于有了新的领导者。"

"接下来，村子会重建，会变得更强。"

"而我，也要继续修炼。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.32】
【职业：散打(Striker)】
【生命值：6,100/6,100】
【念气值：4,600/4,600】
【状态：最佳】

【经验值：5,000/13,000】

"从Lv.32升到Lv.33，需要11,000经验值。"他计算。

"还差6,000经验值。"

"必须继续狩猎，继续训练。"

"目标是Lv.50，达到影级的门槛。"

就在这时——

系统提示突然响起。

【系统提示：主线任务完成】
【任务：协助自来也寻找纲手】
【奖励：经验值+5,000，技能点+3】

【当前经验值：5,000→10,000/11,000】

"主线任务完成了！"李武眼神一亮。

"获得5,000经验值和3技能点！"

"经验值达到10,000了，只差1,000就能升级！"

"技能点也有3点，可以提升技能等级。"

他思考着如何分配技能点。

【当前技能点：3】

【可提升技能：】
- 霸体护甲·Lv.2 → Lv.3（需要1技能点）
- 念气罩·Lv.2 → Lv.3（需要1技能点）
- 念气波·Lv.2 → Lv.3（需要1技能点）
- 寸拳·Lv.3 → Lv.4（需要2技能点）
- 念气感知·Lv.4 → Lv.5（需要2技能点）
- 念气爆发·Lv.2 → Lv.3（需要1技能点）
- 闪电之舞·Lv.2 → Lv.3（需要1技能点）
- 疾风连拳·Lv.2 → Lv.3（需要1技能点）
- 旋风腿·Lv.2 → Lv.3（需要1技能点）
- 碎骨·Lv.2 → Lv.3（需要1技能点）
- 铁山靠·Lv.2 → Lv.3（需要1技能点）
- 影分身之术·Lv.1 → Lv.2（需要1技能点）
- 念气螺旋·Lv.1 → Lv.2（需要1技能点）

"技能很多，但技能点只有3点。"他思考。

"必须选择最重要的技能提升。"

"念气感知是核心技能，但需要2技能点，提升到Lv.5。"

"寸拳也是核心攻击技能，需要2技能点，提升到Lv.4。"

"但我只有3点..."

"如果提升念气感知，就只剩1点，只能提升一个Lv.2技能。"

"如果提升寸拳，也只剩1点。"

"还是先提升念气感知吧，这是最重要的。"

"然后用剩下的1点提升念气螺旋，这是新技能，很有潜力。"

他做出决定。

【使用技能点：2】
【念气感知·Lv.4 → Lv.5】

【技能：念气感知·Lv.5】
【效果：感知范围1,000米，可探测Lv.90以下目标，可分析目标弱点】
【消耗：念气值100/分钟】

"感知范围提升到1,000米了！"李武眼神一亮。

"而且可以探测Lv.90以下目标，还能分析弱点！"

"这个技能太强了！"

【使用技能点：1】
【念气螺旋·Lv.1 → Lv.2】

【技能：念气螺旋·Lv.2】
【效果：高速旋转的念气球体，威力提升50%】
【消耗：念气值600】

"念气螺旋威力提升了，但消耗也从800降到600。"李武满意地点头。

"效率提升了。"

【剩余技能点：0】

"技能点用完了。"他思考。

"接下来，要继续狩猎升级。"

"目标是Lv.33，然后是Lv.40，最终是Lv.50。"

他看向木叶村，眼神坚定。

"木叶，我会保护你。"

"小李，我会保护你。"

"所有重要的人，我都会保护。"

"这就是我的忍道。"
