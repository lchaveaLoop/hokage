# 第六十三章：护送完成

战斗结束后。

四人休息了半小时，恢复体力。

李武用冥想恢复念气值。

金色的念气在体内流转，缓慢恢复。

【念气值：0→200→400→600】

半小时后，念气值恢复到600左右。

"差不多了。"李武站起来。

"我们继续赶路。"

"是。"三人点头。

四人继续前进，向火之国首都前进。

...

路上。

宁次突然说道："李武，刚才那一战，你的实力真的很强。"

"Lv.42的叛忍头目，你居然能击败。"

"而且，最后那一拳，威力太惊人了。"

"那是念气全开。"李武解释道，"消耗全部念气值，换取30秒的全属性提升150%。"

"是一个临时技能，只能在关键时刻使用。"

"全属性提升150%..."宁次震惊，"这个技能太强了。"

"但代价也很大。"李武说道，"消耗全部念气值，而且使用后会陷入虚弱状态。"

"如果不是你们帮忙，我可能无法击败他。"

"我们是队友。"天天笑道，"互相帮助是应该的。"

"嗯。"李武点头。

小李在旁边兴奋道："哥哥好强！"

"我也要变得那么强！"

"你会的。"李武鼓励道，"只要继续努力。"

"嗯！"小李用力点头。


...

傍晚。

四人终于抵达了一个小镇。

这是火之国大名顾问的所在地。

"这里就是目标地点。"李武说道，"我们要找到那位顾问。"

"是。"三人点头。

四人来到镇长家，询问顾问的位置。

镇长是一个中年男人，看到他们，眼神一亮。

"你们是木叶的忍者？"他问道。

"是的。"李武说道，"我们是来护送火之国大名顾问的。"

"太好了！"镇长激动道，"顾问大人已经等了很久了。"

"请跟我来。"

镇长带领四人来到一个大宅院。

宅院里，一个老人坐在椅子上，正在喝茶。

他看到四人，放下茶杯，说道："你们就是木叶派来的忍者？"

"是的。"李武行礼，"我是李武，这是宁次、天天、小李。"

"我们是来护送您到火之国首都的。"

"很好。"老人点头，"我是火之国大名的顾问，名叫山田太郎。"

"请多关照。"

"请放心。"李武说道，"我们会保护您的安全。"

"那就拜托了。"山田太郎说道，"我们明天早上出发。"

"今晚，你们就在这里休息吧。"

"谢谢。"李武说道。

...

晚上。

四人住在宅院的客房里。

李武躺在床上,回顾今天的战斗。

"今天击败了叛忍头目Lv.50。"他思考。

"获得了1,500经验值。"

"现在经验值是10,000/20,000。"

"还差10,000经验值就能升级到Lv.41了。"

"如果顺利的话，这次任务结束后，应该能升级。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.40】
【职业：散打(Striker)】
【生命值：8,700/8,700】（已恢复）
【念气值：2,000/6,900】（恢复中）
【经验值：10,000/20,000】

"生命值已经恢复了，但念气值还在恢复中。"他思考。

"需要继续冥想。"

他闭上眼睛，进入冥想。

金色的念气在体内流转，缓慢恢复。

一个小时后，冥想结束。

【念气值：2,000→2,600】
【念气值上限：5,900→5,920】

"念气值恢复到2,600了，上限也提升了20点。"李武满意地点头。

"明天继续护送任务。"

"从这里到火之国首都，大概需要三天。"

"希望不要再遇到敌人。"

...

第二天。

清晨。

四人和山田太郎出发，向火之国首都前进。

山田太郎坐在马车上，四人在周围警戒。

李武用念气探测扫描周围，警戒敌人。

【念气探测·Lv.1】
【感知范围：2,000米】

"2,000米范围内，暂时没有发现敌人。"他思考。

"看来昨天的战斗，吓退了其他叛忍。"

"这样也好，可以顺利完成任务。"

宁次在旁边说道："李武，你的感知能力真的很强。"

"有你在，我们的安全就有保障了。"

"不要大意。"李武说道，"叛忍组织可能还有其他成员。"

"我们必须保持警惕。"

"是。"宁次点头。

四人继续前进。

...

中午。

四人在一个小溪边休息，吃午饭。

山田太郎从马车上下来，说道："谢谢你们，木叶的忍者们。"

"有你们保护，我感到很安心。"

"这是我们应该做的。"李武说道。

"对了，"山田太郎突然说道，"我听说，昨天你们遇到了叛忍组织的袭击？"

"是的。"李武点头，"六个叛忍，头目是Lv.42的强者。"

"但我们成功击败了他们。"

"太好了。"山田太郎松了一口气，"我一直很担心会遇到袭击。"

"现在看来，你们的实力很强。"

"我们会继续保护您的。"李武说道。

"谢谢。"山田太郎感激道。

就在这时——

李武的念气探测突然发出警报！

【系统提示：检测到敌对目标】
【距离：1,800米】
【数量：3】
【威胁等级：中】

"有敌人！"李武立刻站起来。

"距离1,800米，3个人，威胁等级中等。"

"什么？！"三人脸色一变。

"又有敌人？"

"是的。"李武眼神凝重，"而且，他们正在向我们接近。"

"速度很快。"

"准备战斗。"

"是！"三人立刻警戒。

很快，三个身影出现在他们面前。

三个穿着黑色斗篷的忍者，额头上的护额有划痕。

又是叛忍。

李武用念气探测扫描三人。

【目标：叛忍 x 3】
【等级：Lv.32-34】
【威胁等级：中】
【当前生命值：蓝色（精英级）】
【特殊能力：各种忍术】

"三个叛忍，Lv.32-34，蓝色精英级。"李武思考。

"比昨天的弱一些，但也不能大意。"

叛忍头目冷笑："木叶的忍者...我们是来为我们的老大报仇的。"

"昨天，你们杀了我们的老大。"

"今天，我们要杀了你们。"

"做梦。"李武冷冷道。

"那就试试吧。"叛忍头目说道。

"动手！"

三个叛忍同时攻击！

"宁次、天天、小李！"李武大喊。

"每人对付一个！"

"是！"三人点头。

战斗，再次开始！


...

宁次冲向一个Lv.34的叛忍。

"白眼！"

他的眼睛变成白眼，血管凸起。

"柔拳·八卦六十四掌！"

他的双手快速击出，攻击叛忍的穴道。

叛忍想要躲避，但宁次的速度太快了！

砰砰砰砰！

连续数掌击中，叛忍的查克拉流动被封锁。

"可恶..."叛忍咬牙。

"我的查克拉被封了..."

"结束了。"宁次冷冷道。

"柔拳·八卦一百二十八掌！"

他的攻击速度暴涨，连续一百二十八掌击出！

砰砰砰砰砰！

叛忍被击飞，重重摔在地上，失去意识！

【击败叛忍Lv.34】

...

天天这边。

她面对一个Lv.33的叛忍。

"忍具·双升龙！"

她从卷轴中召唤出大量忍具，射向叛忍！

叛忍立刻躲避，但忍具太多了！

"水遁·水阵壁！"

他结印，召唤水墙防御。

但天天早有准备。

"忍具·操手里剑之术·连锁！"

她用钢丝控制手里剑，绕过水墙，从背后攻击叛忍！

"什么？！"叛忍脸色一变。

砰砰砰！

手里剑击中叛忍，他受伤了。

"忍具·爆破符·连环！"

天天扔出大量起爆符，附着在叛忍身上！

"不好！"叛忍想要逃跑。

但已经来不及了。

轰轰轰！

起爆符爆炸，叛忍被炸飞，失去意识！

【击败叛忍Lv.33】

...

小李这边。

他面对一个Lv.32的叛忍。

"木叶旋风！"

小李一脚踢向叛忍！

叛忍用手臂格挡，但小李的力量很大！

砰！

叛忍被踢退。

"木叶烈风！"

小李继续攻击，一脚踢向叛忍的头部！

叛忍想要躲避，但小李的速度太快了！

砰！

一脚击中，叛忍被踢飞！

"木叶刚力旋风！"

小李全力一脚，踢向叛忍的腹部！

轰！

叛忍被踢飞，重重摔在地上，失去意识！

【击败叛忍Lv.32】

...

三场战斗，同时结束。

宁次、天天、小李都成功击败了对手。

李武满意地点头："做得很好。"

"大家的实力都提升了。"

"谢谢。"三人说道。

"对了，"李武突然说道，"这些叛忍，应该是昨天那个头目的手下。"

"他们是来报仇的。"

"但他们的实力，比昨天的弱很多。"

"看来，昨天的头目是他们组织中最强的。"

"现在他死了，这个组织应该也瓦解了。"

"那就好。"宁次说道，"这样，我们的任务就更安全了。"

"嗯。"李武点头。

"继续赶路吧。"

四人继续前进。

...

三天后。

四人终于抵达了火之国首都。

这是一个繁华的城市，街道上人来人往。

"终于到了。"李武松了一口气。

"任务快要完成了。"

山田太郎从马车上下来，说道："谢谢你们，木叶的忍者们。"

"有你们保护，我才能安全到达。"

"这是我们应该做的。"李武说道。

"我会向火影大人汇报你们的功绩。"山田太郎说道。

"谢谢。"李武说道。

山田太郎离开后，四人在首都休息了一晚。

第二天，他们开始返程。

...

回程路上。

四人轻松了很多，没有遇到敌人。

宁次说道："这次任务，我们做得很好。"

"成功护送顾问到首都，而且击败了叛忍组织。"

"是的。"李武点头，"这是一次成功的任务。"

"而且，大家的实力都提升了。"

"尤其是团队合作，做得很好。"

"谢谢你，李武。"天天说道，"你是一个很好的队长。"

"在你的指挥下，我们才能成功完成任务。"

"不用谢。"李武说道，"这是团队的功劳。"

"哥哥最棒了！"小李兴奋道。

四人继续前进，向木叶返回。

...

三天后。

四人终于回到了木叶。

村口，纲手已经在等待。

"你们回来了。"她说道，"任务完成了吗？"

"是的，纲手大人。"李武说道，"我们成功护送顾问到首都。"

"而且，击败了叛忍组织。"

"很好。"纲手满意地点头。

"山田太郎已经向我汇报了。"

"他说，你们做得很好。"

"尤其是李武，作为队长，指挥得当，保护了所有人。"

"谢谢，纲手大人。"李武说道。

"这是你们的任务报酬。"纲手递给李武一个钱袋。

"A级任务，每人20,000两。"

"谢谢。"李武接过钱袋。

"还有，"纲手继续说道，"李武，你通过了考验。"

"从今天开始，你被提升为特别上忍。"

"特别上忍？"李武眼神一亮。

"是的。"纲手点头，"你的实力和领导能力，都达到了特别上忍的水平。"

"这是你应得的。"

"谢谢，纲手大人。"李武认真道。

"好了，你们去休息吧。"纲手说道。

"是。"四人行礼，离开。

...

回到家中。

李武躺在床上，回顾这次任务。

"A级任务，成功完成。"他思考。

"击败了叛忍组织，保护了顾问。"

"而且，被提升为特别上忍。"

"这是一个重要的里程碑。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.31】
【职业：散打(Striker)】
【生命值：7,700/7,700】
【念气值：5,920/5,920】
【经验值：10,000/17,000】

【身份：特别上忍】

"特别上忍..."李武满意地点头。

"这意味着，我的实力和地位都提升了。"

"接下来，要继续努力，冲击Lv.40。"

"按照计划，三个月内达到Lv.40。"

"现在是Lv.31，还差9级。"

"如果保持现在的速度，应该能达成。"

他闭上眼睛，进入梦乡。

明天，又是新的一天。

又是新的挑战。

他会继续变强。

一直到达到目标。

这就是他的忍道。
