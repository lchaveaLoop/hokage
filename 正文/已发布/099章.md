# 第九十九章:武神之力

现实世界。

木叶村,训练场。

李武站在训练场中央,准备测试新的力量。

"武神...第一次觉醒。"他心中暗道。

"全属性提升10%,而且解锁了三个新技能。"

"让我看看,这些技能的效果。"

他打开系统界面,查看新技能。

【技能:武神强拳·Lv.1】
【类型:主动技能】
【效果:将念气集中在拳头,释放强力一击】
【威力:攻击力×3.0】
【消耗:1,200念气值】
【冷却:10秒】

【技能:念气场·Lv.1】
【类型:主动技能】
【效果:在周围释放念气场,对范围内的敌人造成持续伤害】
【威力:攻击力×0.5/秒】
【范围:20米】
【持续时间:10秒】
【消耗:1,500念气值】
【冷却:30秒】

【技能:霸体护甲·Lv.1】
【类型:被动技能】
【效果:受到攻击时,自动激活霸体状态,免疫控制效果】
【持续时间:3秒】
【触发条件:受到攻击】
【冷却:20秒】

"三个新技能..."李武思考。

"武神强拳,威力×3.0,比念气怪力的×3.5稍弱,但冷却时间更短。"

"念气场,范围持续伤害技能,适合对付多个敌人。"

"霸体护甲,被动技能,可以免疫控制效果,非常实用。"

"这三个技能,都很强大!"

"让我测试一下!"

他走向训练木桩。

【技能:念气爆发·Lv.3】

轰!

金色的念气爆发!

【攻击力:2,550→4,590】

但这次,念气的颜色更加明亮,威力也更强!

"武神状态下,念气爆发的效果也提升了!"李武惊喜道。

【技能:武神强拳·Lv.1】

他将念气集中在拳头!

金色的念气包裹着拳头,形成一个巨大的拳影!

"去!"

武神强拳轰向训练木桩!

轰!

【威力:4,590×3.0=13,770】

训练木桩被瞬间轰碎!

巨大的冲击波扩散,周围的地面都被震裂!

"好强的威力!"李武兴奋道。

"而且,冷却时间只有10秒!"

"这意味着,我可以频繁使用这个技能!"

他继续测试。

【技能:念气场·Lv.1】

金色的念气从李武身上扩散,形成一个直径20米的念气场!

念气场中,所有的训练木桩都开始震动!

【威力:4,590×0.5=2,295/秒】

10秒后,所有的训练木桩都被摧毁!

"念气场,持续10秒,每秒造成2,295点伤害。"李武计算。

"总计22,950点伤害!"

"而且是范围伤害,可以同时攻击多个敌人!"

"这个技能,对付群体敌人非常有效!"

最后,是霸体护甲。

"这是被动技能,需要受到攻击才能触发。"李武思考。

"我需要找个人测试。"

就在这时——

"李武!"

凯冲了过来,满脸兴奋。

"凯前辈。"李武道。

"我听说你达到Lv.50了!"凯兴奋道,"而且,你的气息变得更强了!"

"你是不是...突破了?"

"是的。"李武点头,"我完成了职业进阶。"

"从散打,进阶到武神。"

"武神?!"凯震惊,"这是什么职业?"

"是散打的觉醒职业。"李武解释,"实力比散打强很多。"

"太厉害了!"凯惊叹,"那么,让我见识一下你的新力量!"

"来一场友谊赛吧!"

"好。"李武点头,"正好,我需要测试新技能。"

两人走到训练场中央,摆出战斗姿态。

"那么,开始!"凯道。

"第五门·杜门,开!"

轰!

绿色的查克拉从凯身上爆发!

他的速度和力量大幅提升!

凯冲向李武,速度极快!

"木叶旋风!"

一脚踢向李武!

李武没有闪避,而是站在原地!

"来吧,让我测试霸体护甲!"

凯的攻击击中李武!

但就在这时——

【系统提示:霸体护甲触发】

金色的光芒从李武身上爆发!

他的身体被金色的护甲包裹!

凯的攻击被完全挡住!

而且,李武没有被击退,也没有受到任何控制效果!

"什么?!"凯震惊,"你的防御...这么强?!"

"这是霸体护甲。"李武解释,"被动技能,受到攻击时自动触发。"

"可以免疫控制效果,而且大幅提升防御。"

"太强了!"凯惊叹。

"继续!"李武道。

他立即反击!

【技能:武神强拳·Lv.1】

金色的拳头轰向凯!

凯立即闪避!

但武神强拳的速度太快了!

轰!

凯被击中,被轰飞出去!

"好强的力量!"凯喘息道,"你的实力,提升了很多!"

"而且,这个拳头的威力...至少是之前的1.5倍!"

"是的。"李武点头,"武神的力量,远超散打。"

"太好了!"凯兴奋道,"李武,你真的很强!"

"木叶有你这样的天才,真是太好了!"

"谢谢凯前辈。"李武道。

两人结束了友谊赛。

...

训练结束后。

李武回到家中,思考接下来的计划。

"我现在是Lv.50,武神职业。"他心中暗道。

"实力大幅提升,应该能对抗Lv.62-65的敌人了。"

"但是,距离Lv.60,还差10级。"

"而且,大蛇丸是Lv.65。"

"三年后,他会再来找我。"

"我必须在三年内,达到Lv.60,甚至更高。"

"那么,接下来的计划是..."

"继续完成任务,积累经验值。"

"同时,开启修炼模式,加速提升。"

"目标:一年内达到Lv.55。"

"两年内达到Lv.58。"

"三年内达到Lv.60,并完成第二次觉醒。"

就在这时——

咚咚咚。

有人敲门。

"李武,在吗?"

是纲手的声音。

"在。"李武打开门。

纲手站在门外,脸色严肃。

"纲手大人。"李武道。

"李武。"纲手道,"我有一个重要任务给你。"

"什么任务?"李武问道。

"护送任务。"纲手道,"我需要你护送鸣人,前往短册街。"

"短册街?"李武一愣。

"是的。"纲手点头,"自来也在那里,他会带鸣人修行。"

"为期三年。"

"三年修行..."李武恍然大悟。

"这是原作中的剧情。"他心中暗道。

"鸣人会跟随自来也修行三年,学习仙术和螺旋丸的变化。"

"而我,也有三年时间提升实力。"

"这是一个很好的安排。"

"我接受。"李武点头。

"很好。"纲手道,"明天出发。"

"而且,"她认真道,"这三年,你也要全力提升实力。"

"晓组织的威胁,越来越大。"

"三年后,可能会有大战。"

"我们必须做好准备。"

"我明白。"李武点头。

...

第二天。

木叶村,村口。

李武和鸣人站在村口,准备出发。

"李武哥,我们要去找自来也老师!"鸣人兴奋道。

"他会教我很多厉害的术!"

"是的。"李武点头,"自来也是三忍之一,实力很强。"

"跟着他修行,你会变得更强。"

"嗯!"鸣人用力点头。

两人立即出发,前往短册街。

...

三天后。

短册街。

李武和鸣人抵达了短册街。

"这里就是短册街?"鸣人看着繁华的街道,"好热闹啊!"

"是的。"李武道,"自来也应该在这里。"

"我们去找他。"

两人在街上搜索。

十分钟后,他们在一家赌场找到了自来也。

"自来也大人!"鸣人大喊。

自来也转头,看到鸣人和李武。

"鸣人?李武?"自来也一愣,"你们怎么来了?"

"纲手大人派我们来的。"李武道,"她说,你会带鸣人修行。"

"哦,是这样啊。"自来也点头,"纲手那家伙,终于想起我了。"

"那么,鸣人,你准备好了吗?"

"准备好了!"鸣人兴奋道。

"很好。"自来也笑道,"那么,从今天开始,你就是我的徒弟了。"

"我会教你很多厉害的术!"

"太好了!"鸣人欢呼。

自来也看向李武:"李武,你呢?"

"我要回木叶。"李武道,"继续训练。"

"三年后,我们再见。"

"好。"自来也点头,"那么,保重。"

"你也是。"李武道。

他转身离开,返回木叶。

...

回程路上。

李武思考着接下来的三年。

"鸣人跟随自来也修行,我也要全力提升。"他心中暗道。

"三年时间,从Lv.50升到Lv.60。"

"需要大约400,000经验。"

"平均每年133,000经验。"

"每个月11,000经验。"

"如果开启修炼模式,每次能获得25,000-30,000经验。"

"我现在有8个技能点,可以开启一次修炼模式。"

"之后,每升1级,获得2个技能点。"

"从Lv.50升到Lv.60,可以获得20个技能点。"

"加上现在的8个,总共28个。"

"可以开启5次修炼模式。"

"5次修炼模式,能获得125,000-150,000经验。"

"加上任务,应该能达到400,000经验。"

"而且,"他想起了武神的力量,"我现在是武神,实力大幅提升。"

"可以接受更高难度的任务,获得更多经验。"

"那么,计划是..."

"第一年:从Lv.50升到Lv.53-54。"

"第二年:从Lv.53-54升到Lv.57-58。"

"第三年:从Lv.57-58升到Lv.60,并完成第二次觉醒。"

"目标明确!"

他加快脚步,返回木叶。

...

一周后。

木叶村。

李武回到村子,立即向纲手汇报。

"任务完成。"李武道,"鸣人已经交给自来也了。"

"很好。"纲手点头,"那么,接下来的三年,你有什么计划?"

"全力训练。"李武道,"目标是达到Lv.60。"

"Lv.60..."纲手思考,"那是影级的实力。"

"如果你能达到Lv.60,就能成为木叶的顶尖战力。"

"而且,"她认真道,"三年后,晓组织可能会有大动作。"

"我们需要更多的强者。"

"所以,李武,全力提升吧。"

"木叶,需要你的力量。"

"我明白。"李武点头。

"那么,去吧。"纲手道,"我会给你安排高难度的任务。"

"让你快速成长。"

"谢谢纲手大人。"李武道。

他离开火影办公室,回到家中。

"三年时间,从Lv.50到Lv.60。"他握紧拳头。

"我一定会做到的!"

"而且,我会变得更强!"

"强到...可以保护所有人!"

他打开系统界面,查看当前状态。

【当前状态】
【等级:Lv.50】
【职业:武神(Brawler God)】
【身份:上忍】
【生命值:17,050/17,050】
【念气值:13,552/13,552】
【经验值:12,120/36,000】
【技能点:8】

【主要技能】
【念气爆发·Lv.3】
【疾风连拳·Lv.6】
【寸拳·Lv.6】
【旋风腿·Lv.6】
【闪电之舞·Lv.5】
【念气怪力·Lv.2】
【念气螺旋·Lv.3】
【武神强拳·Lv.1】(新)
【念气场·Lv.1】(新)
【霸体护甲·Lv.1】(新,被动)

"很好。"李武满意地点头。

"现在的我,已经是武神。"

"实力远超之前。"

"接下来,就是不断提升,冲击更高的境界!"

"三年后,我会让所有人看到。"

"我,李武,会成为最强的武神!"

他闭上眼睛,开始冥想,调整状态。

准备迎接接下来的三年修行。
