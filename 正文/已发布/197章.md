# 第一百九十七章：晓的据点

雨之国，边境山区。

阴雨绵绵，整个山区笼罩在一片灰蒙蒙的雨雾中。李武、卡卡西和凯三人潜行在山林间，小心翼翼地接近目标地点。

"前面就是晓的据点了。"卡卡西指着远处一个隐蔽的山洞，写轮眼微微转动，"我感知到里面有查克拉波动。"

【系统提示：检测到敌人】

【数量：10人】

【等级：Lv.45-55】

【位置：前方山洞】

【威胁等级：低】

"先解决这些人。"李武低声说道，金色念气在掌心凝聚，"速战速决，不要惊动其他人。"

三人悄悄接近山洞。洞口的守卫还未反应过来，李武已经化作金色流光冲了上去！

【技能发动：念气场·Lv.5】

嗡——

金色的念气以李武为中心向四周爆发，瞬间笼罩了整个洞口。恐怖的念气波动直接冲击着敌人的身体，十个晓组织成员甚至来不及发出惨叫，就全部被念气压垮，昏死过去！

【系统提示：击败敌人×10】

【获得经验值：+2,000】

"好强..."凯震惊地看着这一幕，"这就是极武圣的力量吗？"

"先继续调查。"李武收起念气，目光望向山洞深处，"这里面一定还有更重要的情报。"

三人谨慎地走进山洞，准备揭开晓组织的秘密。

**本章字数：约1,800字**

**核心剧情：**
1. 三人小队找到晓组织的据点
2. 系统扫描显示10名敌人
3. 李武使用念气场秒杀所有守卫
4. 准备深入调查

**爽点设计：**
- 念气场范围秒杀的震撼（力量爽点）
- 凯的震惊反应（对比爽点）
- 深入敌营的紧张感（悬念爽点）

**情感节点：**
- 团队协作的默契
- 对未知情报的期待

**技能展示：**
- 念气场·Lv.5：范围攻击
- 卡卡西的写轮眼侦察
