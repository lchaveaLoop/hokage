# 第二百章：第二次觉醒·极武圣降临

火之国边境，前往雨隐村的路上。

李武正在树冠间高速穿梭，金色的念气在他周身流转，推动着他的身体以惊人的速度前进。雨隐村距离木叶村有三天的路程，但他已经用【闪电之舞】将这个时间缩短到了一天。

"老师，坚持住..."李武在心中默念，"我马上就到。"

突然——

【系统提示：检测到特殊条件达成】
【条件1：等级达到Lv.78（武神）】
【条件2：面临SS级生死危机】
【条件3：守护意志达到临界值】
【隐藏任务触发：第二次觉醒试炼】

李武的身形猛然停下，落在一棵大树的枝干上。

"第二次觉醒？"他眉头一挑。

在DNF的世界观中，每个职业都有两次觉醒。第一次觉醒是从基础职业进化为高级职业（如散打→武神），第二次觉醒则是从高级职业进化为终极职业（如武神→极武圣）。

【系统提示：第二次觉醒为职业终极进化】
【觉醒后职业：极武圣】
【觉醒要求：通过觉醒试炼】
【试炼内容：在限定时间内击败觉醒守护者】
【是否接受试炼？】

李武沉默了。

自来也还在雨隐村等待救援，每一分每一秒都至关重要。但如果能在救援之前完成第二次觉醒，他的实力将会有一个质的飞跃，救援成功的几率也会大大增加。

"接受试炼。"李武做出了决定。

【系统提示：试炼开始】
【传送中...】

一道金色的光芒将李武笼罩，他的身影瞬间从原地消失。

...

觉醒空间。

这是一片无边无际的虚空，脚下是如镜般的金色水面，头顶是浩瀚的星空。在这片空间的中央，矗立着一道巨大的金色身影。

那身影与李武有七分相似，但更加高大，浑身散发着令人窒息的威压。金色的念气在他周身形成了实质化的铠甲，身后隐约可见一个巨大的武神虚影。

【觉醒守护者】
【等级：Lv.80】
【职业：极武圣（完全体）】
【生命值：50,000/50,000】
【试炼要求：在10分钟内对其造成超过30,000点伤害】

"Lv.80的极武圣完全体..."李武感受着对方身上那股恐怖的气息，心中既震惊又兴奋。

这就是他即将达到的境界！

【试炼开始！】
【倒计时：10:00】

觉醒守护者动了。

他的速度快得不可思议，几乎在瞬间就到了李武面前。金色的拳头带着毁天灭地的气势轰向李武！

【技能：极武霸皇拳】
【威力：攻击力×5.0】

李武瞳孔一缩，立即启动【霸体护甲】！

金色的护甲在他体表形成，但面对极武圣的拳头——

砰！！！

李武整个人被轰飞出去，在水面上滑行了近百米才停下！

【生命值：11,200→8,200】
【霸体护甲：破碎】

"只是一拳就..."李武擦去嘴角的血迹，眼中闪过一丝凝重。

这就是极武圣的力量！比现在的他强了不止一个档次！

"不能硬抗，必须找到他的弱点。"李武迅速冷静下来。

他启动【念气探测】，扫描觉醒守护者的状态。

【扫描结果】
【弱点1：每次使用大招后有2秒硬直】
【弱点2：左侧防御稍弱】
【弱点3：对念气罩的破防需要3秒】

"原来如此。"李武嘴角微微上扬，"有弱点就好办。"

【倒计时：9:20】

觉醒守护者再次冲来，这次他使用了【闪电之舞】，身形化作金色闪电，速度快到肉眼难以捕捉！

但李武早有准备。

【技能：念气罩·Lv.5】

金色的半球形护罩瞬间展开，将李武笼罩其中。

觉醒守护者的拳头轰在念气罩上，发出巨大的轰鸣声！

【念气罩耐久度：100%→60%】

"还能撑3秒..."李武在心中默数。

3...

2...

1...

【念气罩破碎！】

就在念气罩破碎的瞬间，李武动了！

【技能：闪电之舞·Lv.5】

他同样化作金色闪电，但目标不是觉醒守护者，而是——绕到了对方的左侧！

"就是这里！"

【技能：武神强拳·Lv.5】
【念气爆发·Lv.5】

双重buff叠加！

金色的拳头轰向觉醒守护者的左侧！

轰！！！

【造成伤害：12,500】
【觉醒守护者HP：50,000→37,500】

"有效！"李武眼中闪过一丝喜色。

觉醒守护者被击中后，身形微微一顿——就是那2秒的硬直！

李武抓住这个机会，连续攻击！

【技能：疾风连拳·Lv.6】

砰砰砰砰砰！

五拳连击，全部命中！

【造成伤害：18,750】
【觉醒守护者HP：37,500→18,750】

"还差11,250伤害！"李武计算着。

觉醒守护者从硬直中恢复，立即反击。但李武早有预料，提前使用【闪电之舞】拉开距离。

【倒计时：7:30】

"时间充裕。"李武深吸一口气，"再来一轮！"

他再次启动【念气罩】，引诱觉醒守护者攻击。

轰！轰！轰！

念气罩在三次攻击后破碎。

但这一次，李武没有逃跑，而是——

【技能：霸体护甲·Lv.5】

硬抗！

觉醒守护者的拳头轰在李武身上，霸体护甲剧烈震颤，但并未破碎！

"霸体状态下，我可以反击！"

【技能：寸拳·Lv.6】

金色的念气凝聚在拳头上，李武一拳击中觉醒守护者的腹部！

【破甲效果触发！】
【造成伤害：15,000】
【觉醒守护者HP：18,750→3,750】

"还差3,750！"

李武没有任何犹豫，趁着觉醒守护者硬直的瞬间，发动了最后一击！

【技能：武神强拳·Lv.5】

轰！！！

金色的拳影贯穿觉醒守护者的身体！

【造成伤害：12,500】
【觉醒守护者HP：3,750→0（溢出伤害）】

【试炼完成！】
【用时：4分32秒】
【造成伤害：58,750】
【评价：完美】

觉醒守护者的身影渐渐消散，化作无数金色光点融入李武的身体。

【系统提示：第二次觉醒完成！】
【职业进化：武神→极武圣】
【等级提升：Lv.78→Lv.80】
【全属性大幅提升】
【获得极武圣专属技能】

轰——————！！！

金色的光柱从李武身上冲天而起，整个觉醒空间都在震颤！

他感觉自己的身体正在发生质变。每一条肌肉纤维都在重组，每一滴血液都在沸腾，每一根骨头都在变得更加坚韧。

这是生命层次的跃迁！

从"武神"到"极武圣"，不仅仅是等级的提升，更是位格的升华！

【当前状态】
【等级：Lv.80】
【职业：极武圣】
【生命值：11,200→15,000】
【念气值：30,400→40,000】
【力量：180→220】
【体质：179→215】
【敏捷：177→210】
【智力：77→85】

【获得技能】
【极武霸皇拳·Lv.1：威力×5.0，冷却15秒】
【极武领域·Lv.1：领域范围内全属性+30%，持续20秒】
【极限突破·Lv.1：临时突破等级限制，全属性+50%，持续30秒，冷却1小时】

李武握紧了拳头，感受着体内那股澎湃的力量。

"这就是...极武圣的力量..."

他随手一挥，一道金色的拳劲破空而出，在虚空中留下一道长长的裂痕。

"比之前的武神状态强了至少三倍！"

【系统提示：即将返回现实世界】
【倒计时：10...9...8...】

李武深吸一口气，眼中闪过坚定的光芒。

"老师，等我。"

"这一次，我一定会把你安全带回来！"

...

火之国边境，大树上。

李武的身影重新出现，仿佛从未离开过。

但他的气势已经完全变了。

金色的念气在他周身缭绕，形成了一层肉眼可见的能量场。那双眼睛中蕴含的力量，让人不敢直视。

"雨隐村..."李武看向远方，"我来了。"

嗖——

他的身形化作一道金色流光，以比之前快了三倍的速度向雨隐村飞去！
