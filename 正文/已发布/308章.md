# 第三百零八章：休养

接下来的日子里，李武进入了长时间的休养。

与十三使徒和大筒木桃式的战斗，让他的身体达到了极限。若不是归元境界的力量支撑，他恐怕已经倒下了。

木叶村开始重建。

忍者们忙碌的身影随处可见。李武站在废墟边缘，看着正在重建的村子，心中感慨万千。

“哥哥！”小李跑了过来，“你身体还没好，怎么出来了？”

“躺太久了，想出来走走。”李武笑了笑。

现在的他，已经恢复了部分实力。但想要完全恢复，还需要一段时间。

“村子重建的进度很快啊。”李武说道。

“嗯！”小李点头，“大家都很有干劲。毕竟...我们赢了嘛！”

想到那场战斗，小李的眼中满是崇拜。

“哥哥，你当时真是太帅了！一个人就打败了那些什么使徒...”

“打住。”李武打断了他，“那场战斗...伤亡很大。”

小李的表情也变得沉重：“嗯...很多同伴都...”

“正因为如此。”李武说道，“我们才要变得更强。只有足够强大，才能保护重要的人。”

“我明白了！”小李握紧拳头，“我一定会更努力的！”

看着弟弟充满干劲的样子，李武欣慰地笑了。

就在这时，宁次走了过来。

“李武前辈。”宁次恭敬地说道，“火影大人请您过去一趟。”

“知道了。”

李武跟着宁次，来到了临时的火影办公室。

纲手姬正在那里等着他。

“李武，这次多亏了你。”纲手说道，“若不是你...整个忍界恐怕都已经完了。”

“职责所在。”李武淡淡地说道。

纲手看着李武，欲言又止。

“李武...你现在的实力...到底达到了什么程度？”

这个问题，不仅是纲手，整个忍界都很想知道答案。

李武沉默了片刻，然后说道：“如果按照忍界的标准...我应该已经超越六道仙人了。”

“六道仙人？！”纲手惊呼出声。

“六道仙人能封印辉夜姬。”李武继续说道，“但如果是大筒木本家...或者更强的存在，我也未必对手。”

纲手的脸色变得极其复杂。

“你...”她深吸一口气，“到底是谁？”

李武没有回答，只是转身离开。

有些秘密...还是永远埋在心里的好。
