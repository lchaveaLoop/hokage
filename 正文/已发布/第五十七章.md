# 第五十七章：任务完成

傍晚。

四人终于抵达了目标小镇。

这是一个边境小镇，在战争中受到了严重破坏。

很多建筑被摧毁，街道上满是废墟。

但居民们还在努力重建，没有放弃。

"这里就是目标小镇。"卡卡西说道，"我们要把医疗物资送到镇长那里。"

"是。"三人点头。

四人来到镇长家。

镇长是一个年迈的老人，看到他们，眼中涌出泪水。

"木叶的忍者...你们终于来了..."他哽咽道。

"我们等了很久..."

"镇长大人，这是医疗物资。"卡卡西递上卷轴。

"谢谢...谢谢..."镇长接过卷轴，激动不已。

"有了这些物资，我们的伤员就能得到治疗了..."

"这是我们应该做的。"卡卡西说道。

镇长擦去眼泪，说道："请你们在这里休息一晚吧。"

"明天再回木叶。"

"好的，谢谢。"卡卡西点头。

晚上。

镇长安排四人住在一个旅馆里。

虽然旅馆也受到了战争的破坏，但还能住人。

李武躺在床上，回顾这次任务。

"B级任务，护送医疗物资。"他思考。

"遭遇了山贼和叛忍，但都成功击败了。"

"任务完成，没有人受伤。"

"这是一次成功的任务。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.28】
【生命值：6,900/6,900】（已恢复）
【念气值：5,240/5,240】（已恢复）
【经验值：10,300/14,000】

"距离Lv.29还差3,700经验值。"他计算。

"如果明天回程路上再遇到一些敌人，应该能升级。"

"但希望不要遇到太强的敌人。"

"尤其是晓组织的成员。"

他想起纲手的警告。

晓组织...鼬Lv.82，鬼鲛Lv.76...

如果遇到他们，以我们现在的实力，完全不是对手。

必须避免遭遇。

就在这时——

敲门声响起。

"李武，是我。"鸣人的声音。

"进来。"李武说道。

鸣人推门进入，坐在李武旁边。

"李武，你睡了吗？"他问道。

"还没有。"李武说道，"怎么了？"

"我...想和你聊聊。"鸣人说道。

"好啊。"李武笑了笑，"聊什么？"

"我...觉得自己太弱了。"鸣人低声道。

"这次任务，我什么都没做。"

"都是你和卡卡西老师在战斗。"

"我只能在旁边看着..."

"我...很没用..."

"不要这么说。"李武拍了拍鸣人的肩膀。

"你不是没用，你只是还在成长。"

"我也是从弱小开始的。"

"一步一步，慢慢变强。"

"但是..."鸣人咬牙，"我想快点变强..."

"我想保护大家..."

"我想成为火影..."

"我知道。"李武说道，"但成长需要时间。"

"你不能急于求成。"

"螺旋丸快要完成了，对吧？"

"嗯。"鸣人点头，"好色仙人说，我只差最后一步了。"

"那就继续努力。"李武鼓励道，"完成螺旋丸，你就会变得更强。"

"而且，你有九尾的查克拉，潜力巨大。"

"总有一天，你会超越我的。"

"真的吗？"鸣人眼神一亮。

"真的。"李武认真道，"我相信你。"

"谢谢，李武。"鸣人笑了，"我会继续努力的！"

"嗯。"李武点头。

鸣人离开后，李武继续思考。

"鸣人...他的成长速度会很快。"他想。

"原著中，他在几年内就从下忍成长到影级。"

"而我，虽然有系统，但也需要持续努力。"

"不能松懈。"

他闭上眼睛，进入冥想。

金色的念气在体内流转，越来越纯粹。

...

第二天。

清晨。

四人告别镇长，开始返程。

"谢谢你们，木叶的忍者们。"镇长感激道。

"有了这些医疗物资，我们的小镇会重建得更好。"

"这是我们应该做的。"卡卡西说道。

"再见。"

四人离开小镇，向木叶前进。

回程路上，一切平静。

没有遇到山贼，也没有遇到叛忍。

李武用念气感知扫描周围，也没有发现敌人。

"看来回程很顺利。"他思考。

"没有遇到危险。"

"这样也好，大家都很累了。"

...

中午。

四人在一个小溪边休息，吃午饭。

小樱突然说道："李武，我能问你一个问题吗？"

"当然。"李武说道。

"你...是怎么变得那么强的？"小樱问道。

"我...也想变强..."

"但我不知道该怎么做..."

李武思考了一会儿，说道："我的方法，可能不适合你。"

"我用的是念气，和查克拉不同。"

"但有一点是相同的。"

"那就是持续训练，永不放弃。"

"我每天都在训练，狩猎、技能、体能、实战、冥想。"

"从不间断。"

"这样的话，实力就会慢慢提升。"

"持续训练..."小樱若有所思。

"我明白了。"

"而且，"李武继续说道，"你有医疗忍术的天赋。"

"纲手大人说过，你的天赋很好。"

"如果你向纲手大人学习，一定能成为强大的医疗忍者。"

"嗯！"小樱用力点头，"我会努力的！"

"我要成为像纲手大人一样强大的医疗忍者！"

"加油。"李武鼓励道。

卡卡西在旁边看着这一幕，满意地笑了。

"李武，你很会鼓励队友。"他说道。

"这是一个优秀忍者应该具备的品质。"

"谢谢，卡卡西老师。"李武说道。

四人继续前进。

...

下午。

四人终于回到了木叶。

村口，纲手已经在等待。

"你们回来了。"她说道，"任务完成了吗？"

"是的，纲手大人。"卡卡西说道，"医疗物资已经送到了。"

"很好。"纲手满意地点头。

"这次任务，你们做得很好。"

"尤其是李武，表现出色。"

"谢谢，纲手大人。"李武说道。

"这是你们的任务报酬。"纲手递给卡卡西一个钱袋。

"B级任务，每人10,000两。"

"谢谢。"卡卡西接过钱袋。

"好了，你们去休息吧。"纲手说道。

"是。"四人行礼，离开。

...

回到家中。

李武躺在床上，松了一口气。

"任务完成了。"他思考。

"B级任务，护送医疗物资，成功完成。"

"获得了10,000两的报酬。"

"而且，获得了1,400经验值。"

"虽然没有升级，但也不错。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.28】
【职业：散打(Striker)】
【生命值：6,900/6,900】
【念气值：5,240/5,240】
【经验值：10,300/14,000】

"距离Lv.29还差3,700经验值。"他计算。

"如果继续狩猎，大概需要一天时间。"

"那就明天继续训练吧。"

他闭上眼睛，休息。

这次任务，虽然遇到了一些危险，但最终成功完成。

没有人受伤，任务目标达成。

这是一次成功的经历。

而且，他也学到了很多东西。

团队合作的重要性。

保护队友的责任感。

面对强敌的冷静判断。

这些，都是宝贵的经验。

"继续努力吧。"他想。

"目标是Lv.50，影级的门槛。"

"还有很长的路要走。"

"但我会一步一步，达到目标。"

"这就是我的忍道。"

...

几天后。

李武继续训练，终于升级到了Lv.29。

【恭喜！等级提升！】
【Lv.28 → Lv.29】

【属性提升】
【力量：166 → 172】
【敏捷：144 → 149】
【体质：152 → 157】
【智力：128 → 132】

【生命值上限：6,900 → 7,100】
【念气值上限：5,240 → 5,440】

【获得属性点：5】
【获得技能点：1】

"Lv.29了。"李武满意地点头。

"距离Lv.30只差一级。"

"一个月的目标，快要达成了。"

他分配属性点和技能点，继续训练。

日子一天天过去。

李武的实力，也在稳步提升。

他知道，前方还有很多挑战等着他。

晓组织的威胁。

更强大的敌人。

更艰难的任务。

但他不会退缩。

他会继续变强。

保护木叶。

保护小李。

保护所有重要的人。

这就是他的目标。

这就是他的忍道。
