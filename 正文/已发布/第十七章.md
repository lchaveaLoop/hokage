# 第十七章：闪电之舞

大桥上，卡多的尸体倒在血泊中。

数百名雇佣兵看着这一幕，陷入了恐慌。

"老大...死了？"

"卡多死了！"

"我们的钱...怎么办？！"

雇佣兵们面面相觑，不知所措。

他们是被卡多雇佣来的，现在雇主死了，他们也失去了战斗的理由。

但就在这时，一个雇佣兵头目站了出来。

"兄弟们！"他大喊，"卡多死了，但他的财产还在！"

"只要我们杀了这些木叶忍者，卡多的所有财产就是我们的了！"

"对啊！"

"卡多的金库里有几千万两！"

"杀了他们，钱就是我们的！"

雇佣兵们眼中闪过贪婪，再次举起武器。

"冲啊！"

数百名雇佣兵蜂拥而上！

卡卡西脸色一变："糟糕，他们被金钱冲昏了头脑！"

"大家小心！"

第七班和第四班立即进入战斗状态。

但雇佣兵的数量太多了，足足有数百人！

"可恶...这么多人！"鸣人咬牙。

"我们能应付吗？"小樱担心。

就在这时，李武走到了最前方。

"你们退后。"他平静道。

"李武？"卡卡西疑惑。

"这些杂兵，交给我。"李武看着冲来的雇佣兵，眼中闪过一丝冷意。

他身上的金色念气开始沸腾，比之前任何时候都要强烈。

【技能：闪电之舞·Lv.1】
【效果：瞬间移动到多个目标身边，进行连续打击】
【锁定目标数：8】
【判定：必中】

"这个技能...还没用过。"李武心中暗道，"正好试试效果。"

他深吸一口气，念气在体内疯狂流转。

然后——

轰！

李武的身影瞬间消失！

下一秒，他出现在第一个雇佣兵身边！

砰！

一拳！

雇佣兵还没反应过来，就被轰飞出去！

但李武没有停下，他的身影再次消失——

出现在第二个雇佣兵身边！

砰！

又是一拳！

第三个！

砰！

第四个！

砰！

第五个！

砰！

李武的身影如同闪电般在雇佣兵群中穿梭，每次出现，都会有一个雇佣兵倒飞出去！

金色的残影在空中交织，形成了一道道闪电般的轨迹！

"这...这是什么？！"雇佣兵们惊恐。

"他的速度...太快了！"

"根本看不清！"

砰！砰！砰！砰！砰！砰！砰！砰！

八拳！

不到三秒，八名雇佣兵倒飞出去，撞倒了一大片同伙！

【系统提示：闪电之舞·Lv.1使用成功】
【命中目标：8/8】
【造成伤害：优秀】

李武落地，身上的念气依然在流转。

"还不够。"

他再次发动闪电之舞！

轰！

金色的闪电再次在雇佣兵群中穿梭！

砰！砰！砰！砰！砰！砰！砰！砰！

又是八个雇佣兵倒下！

"怪...怪物！"

"这家伙是怪物！"

雇佣兵们开始后退，眼中满是恐惧。

但李武没有停下。

第三次闪电之舞！

第四次！

第五次！

金色的闪电在大桥上不断闪烁，每次闪烁，都会有雇佣兵倒下！

不到一分钟，数十名雇佣兵已经失去战斗能力！

"逃...快逃！"

雇佣兵们终于崩溃了，转身就跑！

但李武没有追击。

"够了。"他平静道，"剩下的，交给你们。"

卡卡西、凯老师和两个班的学生立即冲上去，收拾残局。

有了李武的闪电之舞清场，剩下的雇佣兵已经士气全无，很快就被制服了。

...

大桥另一边。

所有人都被刚才的一幕震撼了。

"那...那是什么技能？！"鸣人瞪大眼睛。

"瞬间移动...连续攻击..."佐助难以置信，"这种速度...简直不可思议！"

"李武...你到底有多少种能力？"小樱喃喃道。

卡卡西看着李武，眼中满是震撼。

"闪电之舞...这个名字很贴切。"他低声道，"李武的战斗方式，越来越像一个完整的体系了。"

"念气感知、霸体护甲、寸拳、念气罩、闪电之舞..."

"每一个能力都有明确的用途，组合起来几乎没有弱点。"

凯老师也走了过来："李武这小鬼...真是个天才。"

"不，"卡卡西摇头，"不只是天才。他的战斗体系，是经过深思熟虑设计出来的。"

"这种系统化的思维...不像是一个十二岁的孩子能有的。"

...

半小时后。

战斗结束。

所有雇佣兵都被制服，波之国的村民们也赶来帮忙。

"太好了！"

"卡多死了！"

"我们自由了！"

村民们欢呼雀跃，眼中满是喜悦。

达兹纳站在大桥上，看着即将完工的大桥，眼眶湿润。

"终于...终于完成了..."

他转身看向李武和众人："谢谢你们！如果没有你们，这座桥永远不可能完工！"

"这是我们应该做的。"卡卡西微笑道。

"不，"达兹纳认真道，"你们不只是完成了任务，你们还拯救了整个波之国！"

"卡多的暴政结束了，我们终于可以自由地生活了！"

村民们也纷纷围了上来，向木叶忍者们表示感谢。

"谢谢你们！"

"你们是我们的恩人！"

"木叶的忍者，真是太厉害了！"

鸣人挠着头，有些不好意思："嘿嘿，没什么啦..."

佐助别过脸，但嘴角勾起一丝笑意。

小樱也笑了："能帮到大家，真是太好了。"

李武站在一旁，默默看着这一切。

他想起了白。

如果白还活着，也能看到这一幕就好了。

波之国的人民获得了自由，大桥完工了。

但白...却永远看不到了。

"对不起，白。"李武心中默念，"我没能救你。"

"但我会记住你的牺牲。"

...

第二天。

大桥正式完工。

波之国举行了盛大的庆典。

达兹纳站在大桥中央，向所有人宣布：

"这座桥，连接了波之国和外界！"

"从今天起，我们不再是孤岛！"

"而这座桥，将被命名为——"

他顿了顿，看向李武和众人：

"鸣人大桥！"

"什么？！"鸣人震惊，"用我的名字？！"

"对！"达兹纳笑道，"是你们给了我勇气，让我坚持完成了这座桥！"

"尤其是你，鸣人！你的不服输精神，激励了我们所有人！"

"所以，这座桥就叫鸣人大桥！"

村民们欢呼起来：

"鸣人大桥！"

"鸣人大桥！"

鸣人眼眶湿润，激动得说不出话来。

"我...我的名字...被刻在了桥上..."

这是他第一次被这么多人认可。

第一次感受到，自己的存在是有意义的。

"谢谢...谢谢大家..."鸣人哽咽道。

佐助看着鸣人，眼中闪过复杂的情绪。

"吊车尾...你确实成长了。"

小樱也笑了："鸣人，恭喜你！"

卡卡西拍了拍鸣人的肩膀："做得好，鸣人。"

李武也走到鸣人身边："鸣人，你值得这份荣誉。"

"李武..."鸣人看着李武，"谢谢你！如果没有你的指导，我不可能进步这么快！"

"不用谢。"李武淡淡道，"这是你自己努力的结果。"

...

庆典结束后。

木叶的忍者们准备离开波之国。

达兹纳和村民们来送行。

"真的要走了吗？"达兹纳不舍。

"嗯。"卡卡西点头，"任务已经完成了，我们该回村了。"

"那...以后还会再来吗？"伊那利问。

"当然！"鸣人大声道，"我一定会再来看鸣人大桥的！"

"好！"伊那利笑了，"我等你！"

众人告别后，踏上了回村的路。

李武走在队伍最后，回头看了一眼大桥。

"鸣人大桥..."他低声道，"希望这座桥，能给波之国带来真正的和平。"

他想起了白和再不斩。

他们也是为了和平而战斗的。

虽然方式不同，但目标是一样的。

"白，再不斩，"李武心中默念，"安息吧。"

"你们的牺牲，不会被遗忘。"

他转身，跟上了队伍。

波之国的任务，结束了。

但他的旅程，才刚刚开始。
