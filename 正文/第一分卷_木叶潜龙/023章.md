# 第二十三章：笔试开始

第二天，上午九点。

木叶忍者学校，301号考场。

李武准时到达，推开教室的门。

教室里已经坐满了人，各国的忍者们三三两两地聚在一起，气氛紧张。

李武扫了一眼，找到了自己的座位——第17号。

他走过去坐下，周围的忍者们纷纷看向他。

"是那个李武..."

"昨天一拳打穿墙壁的家伙..."

"听说他是单独参加考试的..."

窃窃私语声此起彼伏。

李武没有理会，闭目养神。

不久后，鸣人、佐助、小樱也进来了。

"李武！"鸣人兴奋地挥手。

李武睁开眼睛，点了点头。

"你也在这个考场啊。"佐助走了过来。

"嗯。"李武平静道。

"第一场考试...会是什么？"小樱有些紧张。

"很快就知道了。"李武看向讲台。

...

九点整。

教室的门突然被推开。

一群戴着黑色头巾的忍者走了进来，气势凶悍。

为首的是一个满脸伤疤的中年男子，脸上有一道恐怖的烧伤痕迹，眼神冰冷。

森乃伊比喜，木叶特别上忍，拷问部队队长。

【目标：森乃伊比喜】
【等级：Lv.35】
【威胁等级：高】
【当前生命值：紫色（BOSS级）】
【特殊能力：拷问术，心理战】

"Lv.35..."李武眼神一凝。

伊比喜的等级，比他高了15级。

而且作为拷问部队队长，他的心理战能力极强。

"安静！"伊比喜的声音如同雷鸣，震慑全场。

教室里瞬间鸦雀无声。

所有人都被他的气势压制，不敢出声。

"我是第一场考试的主考官，森乃伊比喜。"他冷冷地扫视全场，"从现在开始，你们要听从我的命令。"

"如果有人敢违抗，立即取消考试资格！"

他的眼神如同刀锋，让人不寒而栗。

"现在，听好规则！"伊比喜走到讲台前，"第一场考试，是笔试！"

"笔试？"鸣人愣住了。

"没错。"伊比喜冷笑，"考验你们的情报收集能力。"

"规则如下："

"第一，每人初始10分，答错一题扣1分。"

"第二，这是团队考试。如果队伍中有人得分为0，全队淘汰。"

"第三，考试中如果被发现作弊，每次扣2分。被抓到五次，全队淘汰。"

"第四，考试时间一小时。"

"第五，第十题将在考试结束前公布。"

听到这些规则，考场里响起一片哗然。

"作弊扣分？"

"那怎么答题？"

"这些题目肯定很难..."

伊比喜冷笑："我可没说不能作弊。我只是说，不能被抓到。"

"情报收集，本来就是忍者的基本能力。"

"如果连作弊都做不好，那就没资格当中忍！"

他一挥手："发卷！"

助考们开始分发试卷。

李武拿到试卷，扫了一眼。

第一题：请解释查克拉的五种性质变化及其相互克制关系，并举例说明在实战中的应用。

第二题：请描述三种以上的潜入敌方据点的方法，并分析各自的优缺点。

第三题：请根据下图的密码表，破译以下情报...

...

"这些题目..."李武皱眉。

每一道题都极其专业，涉及忍术理论、战术分析、密码破译等高难度内容。

对于刚毕业的下忍来说，几乎不可能全部答对。

"难怪要作弊。"李武心中暗道。

这场考试的真正目的，不是考察知识，而是考察情报收集能力。

也就是说，要在不被发现的情况下，从其他人那里获取答案。

李武看向周围。

教室里，已经有人开始行动了。

...

第一排，一个戴着墨镜的木叶忍者，悄悄推了推眼镜。

【目标：油女志乃】
【技能：寄坏虫侦查】

李武的念气感知捕捉到，无数微小的虫子从志乃身上飞出，悄无声息地飞向其他考生。

这些虫子会停在别人的试卷上，将答案传回给志乃。

"虫子作弊...不愧是油女一族。"李武心中评价。

另一边，日向宁次闭着眼睛，但额头的青筋暴起。

【技能：白眼透视】

白眼的透视能力，可以看穿前后左右的试卷。

对于日向一族来说，这种考试简直是作弊神器。

佐助也在行动。

他的写轮眼悄悄睁开，红色的瞳孔快速转动。

【技能：写轮眼·洞察】

写轮眼可以看清别人的笔迹，甚至预判下一笔的走向。

只要盯着一个会做题的人，就能完美复制答案。

"写轮眼...确实方便。"李武想。

除了这些血继限界，还有其他的作弊方法。

一个砂隐忍者，悄悄操控着一只沙眼，飞到天花板上，俯视全场。

【技能：沙眼侦查】

一个雨隐忍者，用水在桌面上形成镜面，反射其他人的试卷。

【技能：水镜术】

甚至还有人用通灵术，召唤出小动物去偷看答案。

"各种各样的作弊手段..."李武暗暗点头。

忍者的世界，果然充满了奇思妙想。

但他也注意到，伊比喜和助考们，正在密切监视着全场。

每当有人的作弊手段太过明显，就会被记录下来。

"已经有三个人被抓到了。"李武观察道。

那三个人的脸色煞白，显然很紧张。

被抓五次就会被淘汰，压力很大。

...

李武看着自己的试卷，陷入思考。

"我要作弊吗？"

以他的念气感知，完全可以感知到周围所有人的答案。

甚至比白眼和写轮眼更方便。

但...

"没必要。"李武做出了决定。

他拿起笔，在试卷的第一题下方，写下了四个大字：

**绝对实力**

然后，第二题，还是四个字：

**绝对实力**

第三题，依然是：

**绝对实力**

...

一直到第九题，李武的答案都只有四个字——绝对实力。

他放下笔，靠在椅背上，闭目养神。

周围的考生注意到了李武的举动，纷纷侧目。

"那个人...在干什么？"

"他只写了四个字？"

"绝对实力？这是什么答案？"

"他疯了吗？这样会得零分的！"

窃窃私语声再次响起。

伊比喜也注意到了李武，眼中闪过一丝兴趣。

"有意思..."他走到李武身边，看了一眼试卷。

试卷上，九道题的答案都是"绝对实力"四个字，笔迹工整，力透纸背。

"你就打算这样交卷？"伊比喜问。

"是的。"李武睁开眼睛，平静地看着伊比喜。

"你知道这样会得零分吗？"伊比喜冷笑。

"我知道。"李武点头，"但这就是我的答案。"

"哦？"伊比喜饶有兴趣，"说说看，为什么？"

李武站起身，环视全场。

所有人都在作弊，用各种各样的手段获取答案。

但在李武看来，这些都是小聪明。

"忍者的世界，充满了阴谋和诡计。"李武缓缓道，"情报、战术、配合，这些都很重要。"

"但归根结底，"他看向伊比喜，"决定胜负的，是绝对的实力。"

"如果实力足够强，就不需要情报。"

"如果实力足够强，就不需要战术。"

"如果实力足够强，就不需要配合。"

"一力降十会，这就是我的忍道。"

他的声音不大，但却清晰地传遍了整个教室。

所有人都停下了手中的动作，震惊地看着李武。

"绝对实力..."鸣人喃喃道。

"一力降十会..."佐助眼中闪过思索。

伊比喜沉默了片刻，然后突然笑了。

"有意思！非常有意思！"他大笑道，"你是第一个敢在我面前说这种话的人！"

"但是，"他的笑容收敛，眼神变得锐利，"你有这个实力吗？"

"有。"李武毫不犹豫。

两人的视线在空中碰撞，仿佛有无形的火花迸发。

几秒后，伊比喜转身离开。

"很好。"他回到讲台，"我记住你了，李武。"

"希望你能用实力证明你的话。"

李武重新坐下，闭目养神。

他知道，自己的这个举动，会引起很多人的注意。

但他不在乎。

这就是他的风格——

用绝对的实力，碾压一切。

...

考场里，气氛变得更加紧张。

李武的话，给所有人都留下了深刻的印象。

"绝对实力...他真的有那么强吗？"

"昨天他一拳打穿了墙壁..."

"而且他是单独参加考试的..."

"也许...他真的有这个实力？"

我爱罗在角落里，眼中闪过一丝兴趣。

"绝对实力...有意思。"他低声道，"我也想试试，你的实力到底有多强。"

音隐三人组也在窃窃私语。

"这个李武...太嚣张了。"托斯推了推眼镜。

"但他确实很强。"金承认道。

"大蛇丸大人让我们关注他，看来是对的。"茨姆低声道。

...

时间一分一秒过去。

四十五分钟后，伊比喜突然开口：

"好了，现在公布第十题！"

他的声音如同惊雷，让所有人都紧张起来。

"第十题，是一道特殊的题目。"伊比喜冷笑，"在公布之前，我要给你们一个选择。"

"选择放弃，还是继续？"

"如果选择放弃，你和你的队友都会被淘汰，但可以参加下次考试。"

"如果选择继续，但答错了第十题，"他的眼神变得冰冷，"你将永远失去参加中忍考试的资格！"

"什么？！"

"永远失去资格？！"

"这太过分了！"

考场里炸开了锅。

伊比喜冷笑："这就是中忍考试。如果连这点压力都承受不了，就没资格当中忍！"

"现在，做出你们的选择！"

"放弃的人，举手！"

气氛瞬间凝固。

所有人都在犹豫，在挣扎。

永远失去资格，这个代价太大了。

但如果放弃，就意味着前功尽弃。

李武看着这一幕，嘴角勾起一丝笑意。

"心理战...伊比喜果然厉害。"

他知道，第十题根本不存在。

这只是伊比喜的心理测试，考验考生的决心和勇气。

"但对我来说，"李武闭上眼睛，"根本不需要考虑。"

"因为，我有绝对的实力。"
