# 第五十三章：火影就任

第二天。

火影大楼。

纲手正式就任第五代火影的仪式开始了。

所有的上忍、中忍、下忍都聚集在火影大楼前。

李武站在人群中，看着台上的纲手。

她穿着白色的火影袍，头戴火影帽，气质高贵。

"各位木叶的忍者们。"纲手大声道，"我是千手纲手，从今天开始，我是木叶的第五代火影。"

"三代火影为了保护村子，献出了自己的生命。"

"他的牺牲，我永远不会忘记。"

"而我，作为第五代火影，会继承他的意志。"

"保护木叶，保护所有人。"

"这就是我的承诺。"

台下响起雷鸣般的掌声。

李武也鼓掌，心中感慨。

"纲手...第五代火影..."他思考。

"木叶，终于有了新的领导者。"

"接下来，村子会重建，会变得更强。"


纲手继续说道："木叶在大蛇丸的攻击中受到了重创。"

"很多建筑被摧毁，很多忍者受伤。"

"但我们不会倒下。"

"从今天开始，我们要重建木叶。"

"所有人，都要参与重建工作。"

"上忍负责指挥，中忍负责协调，下忍负责执行。"

"我们要让木叶，变得比以前更强！"

"是！"所有人齐声回应。

纲手满意地点头："很好。"

"现在，开始分配任务。"

她看向自来也："自来也，你负责情报收集。"

"晓组织的威胁还在，我们必须掌握他们的动向。"

"明白。"自来也点头。

纲手又看向其他上忍："卡卡西，你负责训练新一代忍者。"

"凯，你负责重建训练场。"

"阿斯玛，你负责协调各部门。"

"红，你负责医疗救援。"

"是！"所有上忍点头。

纲手最后看向李武和鸣人："李武，鸣人，你们也要继续修炼。"

"晓组织的威胁还在，你们必须变得更强。"

"是！"两人用力点头。

仪式结束后。

李武回到家中，开始制定训练计划。

"纲手大人说得对，我必须变得更强。"他思考。

"晓组织的威胁还在，鼬Lv.82，鬼鲛Lv.76..."

"以我现在的实力，完全不是对手。"

"必须尽快提升等级。"

他打开系统界面，查看当前状态。

【当前状态】
【等级：Lv.34】
【经验值：10,000/15,000】
【距离升级：5,000经验值】

"只差5,000经验值就能升级到Lv.35。"他计算。

"如果狩猎野兽，一只Lv.15的野兽给150经验值。"

"需要狩猎34只。"

"大概需要一天时间。"

"那就开始吧。"

他离开家，前往木叶附近的森林。

...

森林中。

李武用念气感知扫描周围。

【技能：念气感知·Lv.5】
【感知范围：1,000米】

"Lv.5的念气感知，范围达到1,000米。"他满意地点头。

"可以轻松找到猎物。"

很快，他就找到了一群野猪。

【目标：野猪】
【等级：Lv.10】
【威胁等级：低】
【当前生命值：白色（普通级）】

"Lv.10的野猪，一只给100经验值。"他思考。

"这里有15只，足够我升级了。"

他冲向野猪群。

"寸拳！"

轰！

金色的念气爆发，一拳击中野猪！

野猪瞬间倒地！

【击败野猪Lv.10】
【获得经验值：100】

【当前经验值：10,000→10,100/11,000】

"很好，继续。"李武继续狩猎。

砰砰砰砰砰！

一只接一只的野猪倒下。

【击败野猪Lv.10 x 9】
【获得经验值：900】

【当前经验值：10,100→11,000/11,000】

"经验值满了！"李武眼神一亮。

【系统提示：经验值已满】
【是否升级？】

"升级！"李武确认。

轰！

金色的光芒从他体内爆发！

【恭喜！等级提升！】
【Lv.34 → Lv.35】

【属性提升】
【力量：150 → 165】
【敏捷：130 → 140】
【体质：140 → 150】
【智力：120 → 128】

【生命值上限：6,100 → 6,500】
【念气值上限：4,600 → 5,000】

【获得属性点：5】
【获得技能点：1】

"升级了！"李武兴奋道。

"Lv.35！"

"属性全面提升，生命值和念气值上限也提升了。"

"而且获得了5属性点和1技能点。"

他查看当前状态。

【当前状态】
【等级：Lv.35】
【职业：散打(Striker)】
【生命值：6,500/6,500】
【念气值：5,000/5,000】
【经验值：0/16,000】

【属性点：5】
【技能点：1】

"从Lv.35升到Lv.36，需要16,000经验值。"他计算。

"比之前多了1,000。"

"升级所需经验值在递增。"

"但没关系，只要持续狩猎，就能升级。"

他思考如何分配属性点。

"5属性点，可以加在力量、敏捷、体质、智力上。"

"力量影响攻击力，敏捷影响速度，体质影响生命值，智力影响念气值。"

"我的战斗风格是近战，所以力量和敏捷最重要。"

"体质也很重要，可以提升生存能力。"

"智力可以提升念气值，但优先级较低。"

"那就2点力量，2点敏捷，1点体质。"

【分配属性点】
【力量：156 + 2 = 158】
【敏捷：135 + 2 = 137】
【体质：145 + 1 = 146】

【生命值上限：6,300 → 6,400】

"很好。"李武满意地点头。

"力量和敏捷提升了，攻击力和速度都会提升。"

"体质提升了，生命值上限也提升了。"

"现在，还有1技能点。"

他查看技能列表。

【当前技能点：1】

【可提升技能：】
- 霸体护甲·Lv.2 → Lv.3（需要1技能点）
- 念气罩·Lv.2 → Lv.3（需要1技能点）
- 念气波·Lv.2 → Lv.3（需要1技能点）
- 念气爆发·Lv.2 → Lv.3（需要1技能点）
- 闪电之舞·Lv.2 → Lv.3（需要1技能点）
- 疾风连拳·Lv.2 → Lv.3（需要1技能点）
- 旋风腿·Lv.2 → Lv.3（需要1技能点）
- 碎骨·Lv.2 → Lv.3（需要1技能点）
- 铁山靠·Lv.2 → Lv.3（需要1技能点）
- 影分身之术·Lv.2 → Lv.3（需要1技能点）

"只有1技能点，只能提升一个技能。"他思考。

"念气爆发是核心技能，可以提升全属性。"

"提升到Lv.3，效果应该会更强。"

"就选念气爆发吧。"

【使用技能点：1】
【念气爆发·Lv.2 → Lv.3】

【技能：念气爆发·Lv.3】
【效果：全属性提升80%，持续40秒】
【消耗：念气值1,000】

"全属性提升从60%提升到80%了！"李武眼神一亮。

"而且持续时间从30秒提升到40秒！"

"消耗也从800提升到1,000，但完全值得。"

"这样的话，我的爆发能力更强了。"

他握紧拳头，感受着体内的力量。

"Lv.35，力量165，敏捷140，体质150。"他思考。

"攻击力应该在1,800左右，防御力在1,200左右。"

"如果使用念气爆发，攻击力可以达到3,240，防御力可以达到2,160。"

"这样的实力，应该可以对抗Lv.45左右的敌人了。"

"但距离影级（Lv.70+）还差很远。"

"必须继续努力。"

他继续在森林中狩猎，积累经验值。

...

三天后。

李武回到村子，来到训练场。

小李正在那里训练。

"哥哥！"小李看到李武，兴奋地跑过来。

"你回来了！"

"嗯。"李武笑了笑，"我去森林狩猎了几天。"

"你的伤势恢复得怎么样了？"

"已经完全恢复了！"小李兴奋道，"纲手大人的医疗忍术太厉害了！"

"我的腿和手臂都完全恢复了！"

"太好了。"李武松了一口气。

他一直担心小李的伤势，现在终于可以放心了。

"哥哥，你变强了。"小李认真道，"我能感觉到。"

"你的气息，比之前更强了。"

"嗯。"李武点头，"我升级到Lv.35了。"

"Lv.35？"小李疑惑，"那是什么？"

"这是我对实力的评估方式。"李武解释道，"Lv.35，大概相当于特别上忍的实力。"

"好厉害！"小李羡慕道，"哥哥，我也想变强！"

"你会的。"李武拍了拍小李的肩膀，"只要坚持训练，你一定能变强。"

"嗯！"小李用力点头。

"对了，"李武说道，"凯老师呢？"

"凯老师在重建训练场。"小李说道，"纲手大人让他负责这个任务。"

"原来如此。"李武点头。

"那我们也去帮忙吧。"

"好！"小李兴奋道。

两人来到训练场，看到凯老师正在指挥重建工作。

"凯老师。"李武打招呼。

"李武！小李！"凯老师竖起大拇指，"你们来得正好！"

"来帮我一起重建训练场吧！"

"这就是青春的力量！"

"是！"两人笑着回应。

三人一起工作，重建训练场。

李武用念气搬运重物，小李用体术清理废墟，凯老师指挥协调。

工作进行得很顺利。

几个小时后，训练场的基础框架就搭建好了。

"很好！"凯老师满意地点头，"这就是青春的力量！"

"李武，小李，你们做得很好！"

"谢谢，凯老师。"两人笑道。

"对了，李武。"凯老师突然说道，"我听说你在寻找纲手的任务中表现很好。"

"甚至击败了兜。"

"那只是我和鸣人联手的结果。"李武谦虚道。

"不，你做得很好。"凯老师认真道，"兜Lv.65，你Lv.35，差距30级。"

"能击败他，已经很了不起了。"

"这说明，你的战斗技巧和团队配合都很出色。"

"谢谢，凯老师。"李武说道。

"继续努力吧。"凯老师拍了拍李武的肩膀，"你还有很大的成长空间。"

"总有一天，你会达到影级，甚至超越影级。"

"我相信你。"

"嗯。"李武用力点头。

他看向天空，眼神坚定。

"影级...超越影级..."他思考。

"这就是我的目标。"

"我会达到的。"

"一定会。"
