# 第五十四章：日常训练

重建工作结束后。

李武回到家中，开始制定详细的训练计划。

"现在是Lv.35，距离Lv.50还有15级。"他思考。

"每升一级需要的经验值在递增。"

"Lv.35→Lv.36需要16,000经验值。"

"如果按照这个趋势，Lv.36→Lv.37可能需要17,000。"

"从Lv.35升到Lv.50，大概需要25-30万经验值。"

"如果每天狩猎获得5,000经验值，需要60-80天。"

"也就是2-3个月。"

"但这只是理论值，实际可能更久。"

"必须制定高效的训练计划。"

他拿出纸笔，开始写训练计划。

【李武训练计划·v2.0】

**目标：**
- 短期目标：Lv.40（1个月内）
- 中期目标：Lv.45（3个月内）
- 长期目标：Lv.50（6个月内）

**训练内容：**
1. 狩猎升级（每天4小时）
   - 目标：Lv.15-20的野兽
   - 预期收益：5,000经验值/天

2. 技能训练（每天3小时）
   - 使用影分身加速训练
   - 重点提升：散打技能、念气螺旋、闪电之舞

3. 体能训练（每天2小时）
   - 力量训练：负重跑、俯卧撑、深蹲
   - 敏捷训练：闪避训练、速度训练

4. 实战训练（每周2次）
   - 与小李对练
   - 与凯老师对练
   - 参加任务

5. 冥想修炼（每天1小时）
   - 提升念气质量
   - 提升念气值上限


"很好，计划制定完成。"李武满意地点头。

"从明天开始，就按照这个计划训练。"

...

第二天。

清晨5点。

李武起床，开始晨练。

他来到训练场，开始热身。

"先跑10圈。"他思考。

训练场一圈是500米，10圈就是5,000米。

他开始奔跑，速度很快。

金色的念气在体表流转，减少空气阻力。

10分钟后，他跑完了10圈。

"呼...呼..."他喘着粗气。

"10分钟5,000米，平均速度30公里/小时。"

"比普通人快很多，但还不够。"

"必须继续提升。"

他开始做俯卧撑。

100个，200个，300个...

一直做到500个，才停下来。

"力量训练完成。"他满意地点头。

"接下来是技能训练。"

他结印："影分身之术！"

轰！

四个分身出现！

【念气值：4,800→2,800】

"四个分身，消耗2,000念气值。"他计算。

"剩余2,800念气值，足够维持训练。"

他让四个分身分别练习不同的技能。

分身A练习疾风连拳。
分身B练习旋风腿。
分身C练习念气螺旋。
分身D练习闪电之舞。

四个分身同时训练，效率极高。

李武本体则在旁边冥想，恢复念气值。

金色的念气在体内流转，缓慢恢复。

【念气值：2,800→3,000→3,200】

两个小时后。

李武解除分身。

【系统提示：分身经验反馈】
【疾风连拳熟练度+40】
【旋风腿熟练度+40】
【念气螺旋熟练度+40】
【闪电之舞熟练度+40】

"两个小时，四个技能的熟练度都提升了40点。"李武满意地点头。

"影分身训练法的效率很高。"

"如果继续这样训练，很快就能将所有技能提升到Lv.3。"

就在这时——

一个声音响起。

"李武！早啊！"

李武转身，看到鸣人跑过来。

"鸣人，早。"他打招呼。

"你也在训练啊？"

"是啊！"鸣人兴奋道，"我要继续练习螺旋丸！"

"好色仙人说，我还差最后一步就能完成了！"

"加油。"李武鼓励道。

"嗯！"鸣人用力点头。

他拿出橡皮球，开始练习。

查克拉注入橡皮球，橡皮球开始剧烈震动。

但还是没有破裂。

"可恶...还是不行..."鸣人咬牙。

"慢慢来。"李武说道，"螺旋丸是A级忍术，不是那么容易学会的。"

"你已经完成了第一阶段和第二阶段，只差最后一步了。"

"嗯...我会继续努力的！"鸣人说道。

两人一起训练，互相鼓励。

...

上午9点。

李武离开训练场，前往森林狩猎。

他用念气感知扫描周围，寻找合适的猎物。

【念气感知·Lv.5】
【感知范围：1,000米】

很快，他就找到了一群Lv.15的野狼。

【目标：野狼】
【等级：Lv.15】
【威胁等级：低】
【当前生命值：白色（普通级）】

"Lv.15的野狼，一只给150经验值。"他计算。

"这里有20只，可以获得3,000经验值。"

他冲向狼群。

"寸拳！"

轰！

金色的念气爆发，一拳击中野狼！

野狼瞬间倒地！

【击败野狼Lv.15】
【获得经验值：150】

李武继续狩猎，一只接一只击败野狼。

他的动作越来越熟练，效率越来越高。

半小时后，20只野狼全部倒下。

【击败野狼Lv.15 x 20】
【获得经验值：3,000】

【当前经验值：0→3,000/12,000】

"3,000经验值，还差9,000就能升级。"他思考。

"继续狩猎。"

他继续在森林中搜索，寻找更多猎物。

很快，他又找到了一群Lv.18的野熊。

【目标：野熊】
【等级：Lv.18】
【威胁等级：低】
【当前生命值：蓝色（精英级）】

"Lv.18的野熊，而且是蓝色精英级。"李武眼神一亮。

"应该能给更多经验值。"

他冲向熊群。

这次，野熊的实力更强，攻击力和防御力都比野狼高。

但李武的实力也提升了，Lv.26对Lv.18，有8级的优势。

"疾风连拳！"

砰砰砰砰砰！

五拳连击，全部击中野熊！

野熊被击退，但没有倒下。

"防御力很高。"李武思考。

"需要更强的攻击。"

"寸拳·蓄力！"

他蓄力3秒，金色的念气在拳头上凝聚。

轰！

一拳轰出，野熊被击飞，重重摔在地上！

【击败野熊Lv.18】
【获得经验值：250】

"250经验值！"李武眼神一亮。

"比野狼多100！"

"蓝色精英级的经验值更高。"

他继续狩猎，击败所有野熊。

一个小时后，15只野熊全部倒下。

【击败野熊Lv.18 x 15】
【获得经验值：3,750】

【当前经验值：3,000→6,750/12,000】

"6,750经验值了，还差5,250。"李武计算。

"如果继续狩猎Lv.18的野熊，需要再击败21只。"

"大概需要1.5小时。"

"那就继续吧。"

他继续在森林中搜索。

...

下午1点。

李武回到村子，完成了今天的狩猎任务。

【当前经验值：12,000/12,000】

"经验值满了！"他眼神一亮。

"可以升级到Lv.27了！"

【系统提示：经验值已满】
【是否升级？】

"升级！"李武确认。

轰！

金色的光芒从他体内爆发！

【恭喜！等级提升！】
【Lv.26 → Lv.27】

【属性提升】
【力量：158 → 164】
【敏捷：137 → 142】
【体质：146 → 151】
【智力：124 → 128】

【生命值上限：6,400 → 6,600】
【念气值上限：4,800 → 5,000】

【获得属性点：5】
【获得技能点：1】

"升级了！Lv.27！"李武兴奋道。

"一天就升了一级！"

"如果保持这个速度，一个月就能升到Lv.57了！"

"但实际不可能，因为升级所需经验值在递增。"

他查看当前状态。

【当前状态】
【等级：Lv.27】
【职业：散打(Striker)】
【生命值：6,600/6,600】
【念气值：5,000/5,000】
【经验值：0/13,000】

【属性点：5】
【技能点：1】

"从Lv.27升到Lv.28，需要13,000经验值。"他计算。

"比之前多了1,000。"

"果然在递增。"

他分配属性点。

"还是2点力量，2点敏捷，1点体质。"

【分配属性点】
【力量：164 + 2 = 166】
【敏捷：142 + 2 = 144】
【体质：151 + 1 = 152】

【生命值上限：6,600 → 6,700】

"很好。"李武满意地点头。

"现在，还有1技能点。"

"这次提升什么技能呢？"

他思考了一会儿，决定提升疾风连拳。

"疾风连拳是散打的核心技能，使用频率很高。"

"提升到Lv.3，威力和速度都会提升。"

【使用技能点：1】
【疾风连拳·Lv.2 → Lv.3】

【技能：疾风连拳·Lv.3】
【效果：连续五拳，每拳威力提升60%，攻击速度提升50%】
【消耗：念气值400】

"威力从50%提升到60%，速度也提升了50%。"李武满意地点头。

"而且消耗从500降到400。"

"效率提升了。"

他回到家中，吃午饭，休息。

...

下午3点。

李武来到训练场，继续技能训练。

他再次创造四个影分身，让分身练习技能。

两个小时后，解除分身。

【系统提示：分身经验反馈】
【疾风连拳熟练度+40】
【旋风腿熟练度+40】
【念气螺旋熟练度+40】
【闪电之舞熟练度+40】

"今天总共训练了4小时，每个技能的熟练度提升了80点。"李武计算。

"如果继续这样训练，大概5天就能将这些技能提升到Lv.3。"

"效率很高。"

...

下午5点。

李武和小李开始对练。

"哥哥，我要全力以赴了！"小李认真道。

"来吧。"李武笑道。

小李冲向李武，速度很快！

"木叶旋风！"

一脚踢向李武！

李武用霸体护甲硬抗！

砰！

小李的攻击被挡住了。

"好硬的防御！"小李震惊。

"继续。"李武说道。

小李继续攻击，使用各种体术。

李武则用散打技能反击。

两人的对练很激烈，但都控制着力道，不会真正伤到对方。

半小时后，对练结束。

"呼...呼...哥哥好强..."小李喘着粗气。

"我完全打不中你..."

"你也很强。"李武说道，"你的速度和力量都提升了很多。"

"继续努力，你会变得更强。"

"嗯！"小李用力点头。

...

晚上7点。

李武回到家中，开始冥想。

金色的念气在体内流转，越来越纯粹。

一个小时后，冥想结束。

【系统提示：念气质量提升】
【念气值上限：5,000→5,020】

"念气值上限提升了20点。"李武满意地点头。

"虽然不多，但积少成多。"

"如果每天冥想1小时，一个月就能提升600点。"

"这是一个稳定的成长方式。"

他躺在床上，回顾今天的训练。

"今天完成了所有训练计划。"他思考。

"狩猎4小时，获得12,000经验值，升级到Lv.27。"

"技能训练4小时，四个技能的熟练度各提升80点。"

"体能训练2小时，力量和敏捷都有提升。"

"实战训练0.5小时，与小李对练。"

"冥想1小时，念气值上限提升20点。"

"总共11.5小时，非常充实。"

"如果保持这个节奏，一个月后，我应该能达到Lv.30。"

"三个月后，Lv.40。"

"六个月后，Lv.50。"

"到那时，我就能达到影级的门槛了。"

他闭上眼睛，进入梦乡。

明天，又是新的一天。

又是新的训练。

他会继续变强。

一直到达到目标。
