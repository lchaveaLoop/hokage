# 第五十八章：突破Lv.38

任务结束后的第三天。

李武继续执行他的训练计划。

每天狩猎4小时，技能训练3小时，体能训练2小时，冥想1小时。

严格按照计划执行，从不懈怠。

他的经验值也在稳步增长。

【当前经验值：12,000/18,000】

"还差6,000经验值就能升级到Lv.38了。"李武思考。

"今天应该能达成。"

他来到森林，开始狩猎。

用念气感知扫描周围，寻找合适的猎物。

很快，他就找到了一群Lv.28的野猪。

【目标：野猪王】
【等级：Lv.35】
【威胁等级：中】
【当前生命值：紫色（BOSS级）】

"野猪王Lv.35，紫色BOSS级。"李武眼神一亮。

"应该能给不少经验值。"

他冲向野猪群。

野猪王看到李武，发出愤怒的吼叫。

"吼！"

它带领野猪群冲向李武！

李武没有躲避，而是启动霸体护甲！

【技能：霸体护甲·Lv.2】

金色的念气在体表形成护甲！

野猪们的冲撞击中霸体护甲，全部被弹开！

"疾风连拳！"

李武冲向野猪王，连续五拳！

砰砰砰砰砰！

野猪王被击退，但没有倒下。

"防御力很高。"李武思考。

"需要更强的攻击。"


他启动念气爆发！

【技能：念气爆发·Lv.3】

轰！

金色的念气从他体内爆发，气势暴涨！

攻击力从1,800提升到3,240！

"寸拳·蓄力！"

他蓄力3秒，金色的念气在拳头上凝聚。

轰！

一拳轰出，野猪王被击飞，重重摔在地上！

【击败野猪王Lv.35】

【Lv.37 → Lv.38】

【重要里程碑：达到Lv.30】
【解锁新功能：技能进化系统】

"什么？！"李武震惊。

"技能进化系统？"

【系统提示：技能进化系统已解锁】
【说明：当技能达到Lv.5时，可以选择进化方向】
【进化后的技能将获得新的效果和更强的威力】

"技能进化..."李武眼神一亮。

"这是一个重要的新功能。"

"当技能达到Lv.5时，可以进化。"

"这意味着，我的技能可以变得更强。"

他查看当前状态。

【当前状态】
【等级：Lv.30】
【职业：散打(Striker)】
【生命值：7,300/7,300】
【念气值：5,640/5,640】
【经验值：0/16,000】

【属性】
【力量：172 → 178】
【敏捷：149 → 154】
【体质：157 → 162】
【智力：132 → 136】

【获得属性点：5】
【获得技能点：1】

"Lv.30了！"李武兴奋道。

"一个月的目标，达成了！"

"而且，解锁了技能进化系统。"

"这是一个巨大的提升。"

他分配属性点。

"还是2点力量，2点敏捷，1点体质。"

【分配属性点】
【力量：178 + 2 = 180】
【敏捷：154 + 2 = 156】
【体质：162 + 1 = 163】

【生命值上限：7,300 → 7,400】

"力量达到180了。"李武满意地点头。

"攻击力应该在1,900左右。"

"如果使用念气爆发，可以达到3,420。"

"这样的实力，应该可以对抗Lv.45左右的敌人了。"

他查看技能列表。

【技能列表】
- 霸体护甲·Lv.2
- 念气罩·Lv.2
- 念气波·Lv.2
- 寸拳·Lv.3
- 钢筋铁骨·Lv.2
- 念气感知·Lv.5（可进化）
- 念气爆发·Lv.3
- 闪电之舞·Lv.2
- 疾风连拳·Lv.3
- 旋风腿·Lv.2
- 碎骨·Lv.2
- 铁山靠·Lv.2
- 影分身之术·Lv.2
- 念气螺旋·Lv.2

"念气感知已经Lv.5了，可以进化！"李武眼神一亮。

"看看进化选项。"

他点击念气感知。

【技能：念气感知·Lv.5】
【可选择进化方向】

【进化方向A：念气探测】
【效果：感知范围提升至2,000米，可探测Lv.100以下目标，可分析目标详细信息】
【特性：探测型，适合侦查和情报收集】

【进化方向B：念气锁定】
【效果：感知范围1,500米，可锁定3个目标，持续追踪，无视隐身和伪装】
【特性：追踪型，适合战斗和追击】

【进化方向C：念气预警】
【效果：感知范围1,000米，可提前5秒预警危险，自动触发防御】
【特性：防御型，适合保护和生存】

"三个进化方向..."李武思考。

"念气探测：范围最大，2,000米，适合侦查。"

"念气锁定：可锁定3个目标，适合战斗。"

"念气预警：可提前预警危险，适合生存。"

"该选哪个呢？"

他仔细思考。

"念气探测的范围最大，2,000米，而且可以探测Lv.100以下目标。"

"这对我来说很重要，因为晓组织的成员大多在Lv.70-90之间。"

"如果选择念气探测，我可以提前发现他们，避免遭遇。"

"而且，2,000米的范围，可以覆盖很大的区域。"

"这对侦查和情报收集很有帮助。"

"就选念气探测吧。"

他做出决定。

【选择进化方向：念气探测】

【技能进化中...】

轰！

金色的光芒从李武体内爆发！

【恭喜！技能进化成功！】
【念气感知·Lv.5 → 念气探测·Lv.1】

【技能：念气探测·Lv.1】
【效果：感知范围2,000米，可探测Lv.100以下目标，可分析目标详细信息】
【消耗：念气值150/分钟】
【特性：探测型，适合侦查和情报收集】

"成功了！"李武兴奋道。

"念气探测·Lv.1，感知范围2,000米！"

"而且可以探测Lv.100以下目标！"

"这意味着，除了极少数超影级强者，我都能探测到。"

他立刻启动念气探测。

【技能：念气探测·Lv.1】

轰！

金色的念气从他体内扩散，覆盖周围2,000米！

瞬间，他感知到了周围所有的生命体。

野兽、昆虫、鸟类...

甚至远处的木叶村，他都能感知到。

"太强了..."李武震惊。

"2,000米的范围，几乎覆盖了整个森林。"

"而且，我能清楚地感知到每个目标的位置、等级、状态。"

"这个技能，太有用了。"

他关闭念气探测，回到村子。

...

回到家中。

李武躺在床上，回顾今天的收获。

"升级到Lv.30，达成了一个月的目标。"他思考。

"而且，解锁了技能进化系统。"

"念气感知进化成念气探测，感知范围提升到2,000米。"

"这是一个巨大的提升。"

"接下来，要继续努力，冲击Lv.40。"

"按照计划，三个月内达到Lv.40。"

"现在是Lv.30，还差10级。"

"如果保持现在的速度，应该能达成。"

他打开系统界面，查看整体进度。

【李武·成长记录】
【起始等级：Lv.1】
【当前等级：Lv.30】
【成长时间：约2个月】
【平均升级速度：15级/月】

【短期目标：Lv.30（已达成）】
【中期目标：Lv.40（3个月内）】
【长期目标：Lv.50（6个月内）】

"2个月时间，从Lv.1升到Lv.30。"他计算。

"平均每月15级。"

"如果保持这个速度，3个月后应该能达到Lv.45。"

"超过了Lv.40的目标。"

"但实际上，升级所需经验值在递增。"

"Lv.30→Lv.31需要16,000经验值。"

"Lv.40→Lv.41可能需要25,000经验值。"

"所以，升级速度会逐渐变慢。"

"必须更加努力。"

他握紧拳头，眼神坚定。

"Lv.30只是一个开始。"

"我的目标是Lv.50，影级的门槛。"

"还有很长的路要走。"

"但我会一步一步，达到目标。"

...

第二天。

李武来到训练场，继续训练。

小李也在那里。

"哥哥！"小李看到李武，兴奋地跑过来。

"你变强了！"

"我能感觉到，你的气息比之前更强了！"

"嗯。"李武笑了笑，"我升级到Lv.30了。"

"Lv.30？"小李震惊，"好厉害！"

"哥哥，你真的好强！"

"你也会变强的。"李武拍了拍小李的肩膀。

"只要坚持训练，你一定能达到你的目标。"

"嗯！"小李用力点头。

"哥哥，我们一起训练吧！"

"好。"李武笑道。

两人开始对练。

李武控制着力道，不会真正伤到小李。

但小李的攻击，却越来越强。

"木叶旋风！"

"木叶烈风！"

"木叶刚力旋风！"

小李的体术越来越熟练，速度和力量都在提升。

"小李进步很快。"李武思考。

"他的天赋很好，只是不能使用忍术和幻术。"

"但体术方面，他有巨大的潜力。"

"如果他能开启八门遁甲..."

"他会变得非常强大。"

对练结束后，两人坐在地上休息。

"哥哥，"小李突然说道，"我想问你一个问题。"

"什么问题？"李武问道。

"你...为什么要变强？"小李认真道。

"你已经很强了，为什么还要继续训练？"

李武思考了一会儿，说道："因为，我想保护重要的人。"

"保护你，保护木叶，保护所有我在乎的人。"

"而且，这个世界很危险。"

"晓组织、大蛇丸、还有其他威胁。"

"如果我不够强，就无法保护任何人。"

"所以，我必须继续变强。"

"直到我强大到，可以保护所有人。"

"哥哥..."小李眼中涌出泪水。

"我也要变强！"

"我也要保护重要的人！"

"我要保护你，保护木叶，保护所有人！"

"嗯。"李武笑了笑，"我相信你。"

"我们一起努力吧。"

"嗯！"小李用力点头。

两人握拳，相互鼓励。

这就是兄弟。

互相支持，互相成长。

一起变强，一起保护重要的人。

这就是他们的羁绊。

这就是他们的忍道。
