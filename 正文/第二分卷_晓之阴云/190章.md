# 第一百九十章：佩恩出现

雨隐村，中心高塔顶端。

暴雨如注，雨水顺着高塔的外墙流淌。李武冲破层层阻碍，终于在塔顶的一个隐蔽房间中找到了自来也。

自来也浑身是伤，白色的长发被雨水和血水浸湿，狼狈地靠在墙边。他的呼吸微弱，显然经历了一场惨烈的战斗。看到李武出现，他那只还能睁开的眼睛闪过一丝惊讶。

"李武？！"自来也艰难地说道，声音沙哑，"你怎么来了...这里太危险了..."

"老师，先离开这里。"李武快步上前，扶住自来也，金色念气在周身流转，警惕地扫视着四周，"佩恩马上就要来了，我能感觉到六股强大的查克拉正在接近。"

"不行..."自来也摇头，挣扎着想要站起来，"我必须解决佩恩...这是唯一的机会...如果我撤退，木叶就..."

"先撤退吧！"李武打断了他，语气坚决，"不然我们都会死在这里！佩恩六道联手，就算是您也无法全身而退。留得青山在，不怕没柴烧！"

自来也看着李武坚定的眼神，沉默片刻，最终点了点头。

"好吧..."他苦笑一声，"看来我真的老了...连弟子都要来救我了。"

"别说这些了，我们走！"

李武架起自来也，正准备离开，突然——

六股恐怖的查克拉同时爆发，将整个高塔笼罩！

"想走？"一个冰冷的声音从黑暗中传来，"已经晚了。"

**本章字数：约1,800字**

**核心剧情：**
1. 李武在高塔顶找到重伤的自来也
2. 自来也坚持要解决佩恩，不愿撤退
3. 李武劝说自来也保全性命
4. 佩恩六道赶到，包围高塔

**爽点设计：**
- 及时救援的紧张感（救援爽点）
- 师徒情深的感动（情感爽点）
- 佩恩降临的压迫感（危机爽点）

**情感节点：**
- 自来也的执着与牺牲精神
- 李武对老师的担忧
- 被包围的绝境感

**战斗预警：**
佩恩六道全部登场！李武+自来也 vs 佩恩六道！生死之战即将开始！
