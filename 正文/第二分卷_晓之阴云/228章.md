# 第二百二十八章：紧急召集·秽土大军来袭

木叶村，紧急警报声响彻全村。

李武正在家中修炼，突然听到外面传来刺耳的警报声。他立即冲出房门，看到整个村子都陷入了紧张状态。

"紧急召集！所有上忍立即前往火影办公室！"

【系统提示：紧急任务触发】
【任务：防御木叶】
【敌人：秽土转生大军】
【危险等级：S级】

李武二话不说，化作一道金色流光冲向火影办公室。

...

火影办公室内，气氛凝重。

纲手站在地图前，面色严肃。奈良鹿久、山中亥一、秋道丁座等木叶高层全部到齐。卡卡西、凯、阿斯玛等上忍也已经到位。

"各位，"纲手开口，声音低沉，"刚刚收到前线情报。晓组织发动了大规模进攻，他们使用秽土转生复活了大量死去的强者。"

她指着地图上的几个标记："第一波攻击来自三个方向：草隐村方向有300名秽土忍者；雨隐村方向有500名；汤隐村方向有200名。总计1000名秽土转生忍者正向木叶推进。"

"1000名？！"阿斯玛倒吸一口冷气。

"而且..."纲手的眼神更加凝重，"其中不乏影级强者。根据情报，二代土影、三代雷影、四代风影都被复活了。"

会议室内一片沉默。

影级强者...每一个都拥有毁灭性的力量。而现在，他们要面对的不止一个。

"李武。"纲手看向他，"我需要你率领第一梯队迎击草隐村方向的敌人。那里有大约300名秽土忍者，带队的是..."

她顿了顿："二代土影·无。"

李武眼神一凝。二代土影，拥有尘遁血继淘汰的恐怖忍者！

"明白。"李武点头，"交给我。"

"卡卡西，你带第二梯队支援李武。"

"是。"

"凯、阿斯玛，你们负责雨隐村方向。小李、宁次、天天，你们跟随凯。"

"明白！"小李握紧拳头，眼中燃烧着战意。

...

木叶大门前。

李武站在最前方，身后是200名木叶精锐忍者。卡卡西站在他身侧，写轮眼已经开启。

"李武，"卡卡西低声说道，"二代土影的尘遁非常麻烦，一旦被击中，连防御的机会都没有。"

"我知道。"李武平静地说道，"尘遁是原子级别的分解术，常规防御无效。"

【系统分析：尘遁特性】
【类型：血继淘汰（风+土+火）】
【效果：原子级分解】
【破解方法：在术完成前打断施术者】

"所以..."李武眼中闪过一丝金光，"必须在术完成前解决施术者。"

远处的地平线上，烟尘滚滚。秽土大军正在接近。

李武能看到那些熟悉的身影——曾经的英雄、曾经的敌人，此刻都被复活成了傀儡。他们的眼睛空洞无神，皮肤呈现出不自然的灰白色。

【系统扫描】
【秽土忍者×300】
【等级范围：Lv.40-Lv.75】
【领队：二代土影·无（Lv.78）】
【特性：不死之身，查克拉无限】

"来了..."李武握紧拳头，金色的念气开始在他周身燃烧。

"所有人听令！"他大喝一声，"第一梯队，随我冲锋！"

"是——！"

李武化作一道金色流光，率先冲向敌军！

【技能：极武领域·Lv.1】
【技能：念气爆发·Lv.5】

金色的领域展开，李武的战斗力瞬间提升到极致！

"极武圣李武...在此！"
