# 第六章：日常训练

清晨，天刚蒙蒙亮。

李武已经在训练场上开始了今天的训练。

“寸拳...”

“寸拳...”

“寸拳...”

他的每一拳都蕴含着强大的念气。

虽然现在等级还低，但日积月累的训练，让他的技巧越来越熟练。

【系统提示】

【寸拳熟练度：入门 → 熟练】
【获得新技能：连环腿（入门）】

“连环腿吗？正好可以配合寸拳使用。”

就在这时，小李气喘吁吁地跑了过来。

“哥哥！”

“来了？”

“嗯！”

“凯老师说...今天要测试我的速度...”

“那就开始吧。”

“啊？”

“现在？”

“对。”

“让我看看你这几天训练的成果。”

“是！”

“来。”

“喝啊！”

他一套连拳打出。

虽然动作还有些生疏，但已经有模有样了。

“速度不错。”

“力量...差了点。”

“再来！”

他一个飞踢。

“反应也变快了。”

“呼...呼...”

“哥哥...我...”

“进步很大。”

“真的？”

“但是——”

“但是？”

“还不够。”

“...！”

“小李。”

“嗯？”

“你想不想...看看真正的力量？”

“真正的力量？”

“嗯。”

“霸体。”

“霸体？”

“看好了。”

他身上突然泛起淡淡的金光。

“这是...？”

“所谓霸体...”

他伸出拳头。

“...就是无视一切攻击的防御。”

“无视...一切攻击？”

“看好了。”

他深吸一口气，然后——

轰！！！

一拳打出。

那棵需要两人合抱的大树，竟然直接被轰断了！

“这...这...”

“这就是霸体。”

“哥哥...”

“你好强...”

“现在还不算强。”

“这只是入门级。”

“入门级就这么厉害？！”

“嗯。”

“以后...你会更强的。”

“嗯！”

“我也要学这个！”

“好。”

“从今天开始，我会教你霸体的基础。”

“太好了！”

“哟。”

一个银发蒙面男子，出现在训练场边缘。

那是旗木卡卡西——李武的指导上忍。

（按理说，李武作为特别应对班，应该由火影亲自安排。但三代火影认为，卡卡西作为村里最出色的上忍之一，又有丰富的带队经验，是最适合观察李武的人选。）

“卡卡西老师。”

“卡卡西老师好！”

“我听说你在这里训练。”

“所以过来看看。”

他的目光落在那棵断树上：

“这是...？”

“我干的。”

“哦？”

“实力见长啊。”

“老师过奖了。”

“李武。”

“嗯？”

“你有兴趣...做我的部下吗？”

“部下？”

“嗯。”

“我旗下现在只有一个名额。”

“本来是给佐助的...”

他看着李武：

“但我觉得，你可能更适合。”

“卡卡西...原著中，卡卡西应该是第七班的指导上忍。但这个世界...我被分到了他的旗下？”

“谢谢老师好意。”

“但是...”

“但是？”

“我想自己走一条路。”

“...有意思。”

“老师还有什么要说的吗？”

“下周一，有任务。”

“C级。”

“记得准备。”

“是。”

“走了。”

“C级任务...应该是波之国吧。”

“哥哥！卡卡西老师跟你说了什么？”

“没什么。”

“继续训练。”

“是！”

【等级提升：Lv.10 → Lv.12】
【获得技能：霸体（入门）】
【获得成就：第一次展示霸体】
【获得奖励：属性点+5】

“霸体...全加力量吧。”

【系统提示】

【力量+5 → 当前力量：25】

旗木卡卡西站在屋顶上，看着远处的训练场。

“那个少年...不简单。总有一天，他会成为比我更强的存在。”

他转身离去。

（本章完）
