# 第十一章：桥上初战（优化版）

清晨的海风带着咸湿的气息，吹拂着未完工的大桥。

李武站在桥头，金色的念气在体内缓缓流转。昨夜的战斗让他对念气的运用更加纯熟，但他知道，这还不够。

“Lv.17...”

他感受着体内的力量，距离Lv.50的精英上忍，还有很长的路要走。

旗木卡卡西走到李武身边，看着远处的海面。

“李武，"他低声道，"昨天那一战，再不斩只是试探。”

李武点头："我知道。他会全力以赴。"

“嗯。"卡卡西点头，"今天的战斗，会更加艰难。”

达兹纳看着正在施工的大桥，脸上满是愁容。

“卡卡西先生，"他犹豫了一下，"那个雾隐...还会再来吗？”

卡卡西看了他一眼："会。而且下次，他不会留手。"

达兹纳的脸色变得苍白："那...那我们怎么办？"

“放心。"李武平静道，"有我们在。”

突然，远处海面上，出现了那艘熟悉的小船。

浓雾，开始升起。

“昨天是我大意了。”

“今天...你们没那么好运了！”

浓雾瞬间笼罩整个大桥！

“雾隐之术！"卡卡西脸色一变，"小心！”

鸣人、佐助、小樱立刻围在达兹纳身边。

李武闭上眼睛，金色的念气扩散！

“念气感知...全开！”

在他的感知中，两个强大的查克拉波动清晰可见。

一个在正前方——再不斩。

另一个在右侧——白。

“两个敌人。再不斩在正前方，白在右侧20米。”

卡卡西点头："你继续感知，我来应付他们。"

白从浓雾中走出，双手结印。

“冰遁·千杀水翔！”

无数冰针，从雾中凝结，如同暴雨般射向李武！

“雕虫小技。”

金色的念气在他体表形成一层透明的护甲！

叮叮叮叮！

冰针击中护甲，全部被弹开！

“又是这种能力？”

“白的攻击竟然对他无效？”

李武站在原地，任由冰针攻击，寸步不退。

金色的念气护甲在阳光下熠熠生辉！

“这就是...霸体。”

“任何攻击...都无法突破。”

白的脸色变得难看。

“怎么可能？”

“我的千杀水翔...竟然连破防都做不到？”

“让我来！”

再不斩从浓雾中冲出，斩首大刀斩向李武！

铛！！！

斩首大刀击中念气护甲，发出巨大的声音！

“好硬的防御！”

李武纹丝不动。

“你...太慢了。”

白的攻击再次袭来！

“冰遁·燕雪！”

更多的冰针从四面八方射来！

叮叮叮叮！

护甲继续承受攻击，但依然稳固。

“念气护甲的强度...比我想象的更高。Lv.17的防御力，已经可以硬钢Lv.40的冰遁攻击。”

“这样下去不是办法...”

他看向卡卡西："老师，再不斩交给你了。"

卡卡西点头："小心白的秘术。"

卡卡西冲向再不斩！

“你的对手是我！”

再不斩冷笑："正合我意！"

两人战在一起！

白看到卡卡西被牵制，立刻结印。

“冰遁·魔镜冰晶！”

嗡——！！！

李武周围，突然出现了数十面晶莹的冰镜！

每一面冰镜中，都映照着白的身影！

“在这个术中...我是无敌的。”

“你逃不掉的。”

“来了...魔镜冰晶。原著中让卡卡西吃了大亏的秘术。”

李武闭上眼睛。

“念气感知...全开！”

金色的念气扩散，穿透所有冰镜！

在他的感知中，只有一面镜子里的白是真身，其他都是残影！

“找到了。”

他冲向其中一面冰镜！

“寸拳！”

砰！

冰镜破碎！

白的真身被击中，摔倒在地！

"什么？！”

“你...你是怎么找到我的？!”

“你的查克拉...我能感知到。”

“高速移动...在我的感知中就像灯塔一样明显。”

白的眼神变得复杂。

“...原来如此。”

再不斩看到白受伤，立刻后退。

“白！”

他冲到白身边，警惕地看着李武。

“又是这一招...”

“这个小鬼...完全克制我们的战术！”

“还要继续吗？”

再不斩沉默了。

他看向卡卡西，又看向李武。

“...今天就到这里。”

“但是...下次...你们就没这么好运了。”

他扶起白，消失在浓雾中。

浓雾渐渐散去。

“你没事吧？”

李武摇头："没事。"

卡卡西看着李武，眼中闪过一丝复杂。

“你的实力...已经超出了我的预期。”

“白的魔镜冰晶...竟然被这样破解了。”

“赢得很勉强...如果不是有念气感知...输的就是我了。”

“太好了！又击退了他们！”

“大桥...大桥还有希望！”

卡卡西点头："今天就先到这里。"

“大家回达兹纳家休息。”
