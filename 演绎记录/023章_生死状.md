# 第二十三章：生死状

忍校毕业考试的通知贴出来了。

体术实战。一对一淘汰赛。签生死状。阳光照在通知上，给每一个字都镀上了一层金色的光辉。

"生死状？！"

小李看着通知，脸都白了。他的嘴唇在发抖，眼睛里充满了恐惧。

"意思是……可能会死？"

"不是'可能会死'。是'死了不追究责任'。"

李武说。他的声音很平静，但带着一种不容置疑的坚定。

"考试的时候一般不会出人命——但如果真的出了，学校不负责。"

"可是……"

小李的手在发抖。阳光照在他的手上，给他的手镀上了一层金色的轮廓。

"如果对手是宁次……"

"那就认输。"

"我不要！"

小李攥紧拳头。他的眼睛里闪烁着坚定的光，像两颗星星。

"我不能认输！"

"为什么？"

"因为……因为如果我认输了……我就真的变成吊车尾了……"

他的声音越来越小。阳光照在他的脸上，给他镀上了一层金色的轮廓。

"我不要当吊车尾……"

李武看着弟弟。

他知道小李在想什么。不是"怕死"——是"怕被看不起"。对小李来说，认输比死更可怕。因为认输意味着——他承认自己确实不行。

"小李。"

"嗯？"

"你觉得'不认输'和'变强'是一回事吗？"

小李愣了一下。他的眼睛里闪烁着困惑的光。

"……不是吗？"

"不是。"

李武的声音很平静。

"'不认输'是态度。'变强'是结果。态度可以让你不放弃——但它不能让你变强。变强需要方法、时间、和正确的训练。"

"你现在不缺态度。你缺的是方法。"

小李低下头。他的眼睛里闪烁着失望的光。

"可是……我已经很努力了……"

"努力是前提，不是方法。"

李武站起来。阳光照在他的身上，给他镀上了一层金色的轮廓。

"走吧。"

"去哪？"

"训练场。离考试还有两周。我要在这两周里，让你的体术再上一个台阶。"

小李抬起头。他的眼睛里闪烁着希望的光。

"可是……两周够吗？"

"不够。"

"那——"

"但两周够做一件事——够让你知道，你这两个月的进步不是假的。"

李武转身往训练场走去。阳光照在他的身上，给他镀上了一层金色的轮廓。

小李跟在后面。他的脚步很重，每一步都踩在地上，发出沉闷的响声。

到了训练场，李武让小李站好。

"踢我。"

"啊？"

"用你最强的侧踢，踢我。"

"可是哥哥——"

"踢。"

小李犹豫了一下，然后深吸一口气。

侧踢。

腰发力。速度很快。力量也不小。

李武抬起手臂挡住。

砰！

他被踢退了一步。

"再来。"

小李又踢了一次。

"再来。"

第三次。

"再来。"

连续踢了二十次之后，小李停下了。

"哥哥……你在干什么？"

"我在测试你的极限。"

"极限？"

"你踢了二十次，前十五次的力度是一样的。第十六次开始下降，第十八次下降了百分之二十，第二十次下降了百分之三十。"

李武放下手臂。

"你的体力极限大概是十五次全力攻击。超过十五次，你的力量就会衰减。"

"那……那怎么办？"

"在十五次之内解决战斗。"

"可是如果对手比我强——"

"那就不要硬拼。"

李武走到小李面前。

"你的优势不是力量。是速度。你的速度在同龄人里是顶尖的——除了宁次，没人比你快。"

"利用速度。先手攻击。在对手还没反应过来之前，用最快的速度打出最强的一击。"

"一击制胜？"

"对。"

小李想了想。

"可是……如果一击打不中呢？"

"那就撤。不要纠缠。拉开距离，等下一次机会。"

"可是——"

"没有可是。"

李武的表情变得严肃。

"你的体力是有限的。每多打一拳，你就弱一分。如果不能在前十五击之内解决战斗——你就输了。"

"所以你必须学会：什么时候该打，什么时候该撤。"

小李点了点头。

"我懂了。"

"好。继续练。"

那天晚上，小李练到了很晚。月光从训练场的边缘洒下来，在地上投下斑驳的影子。远处传来虫鸣声，像一首轻柔的摇篮曲。

不是自残式的训练——是精准的、有针对性的训练。每一脚侧踢都带着"一击制胜"的目标，每一拳都带着"打完就撤"的意识。月光照在他的身上，给他镀上了一层银色的轮廓。

李武坐在旁边，看着他。月光照在他的脸上，给他镀上了一层银色的轮廓。

进步很快。比他预期的还快。小李的天赋确实不高——但他有一种"一旦理解了就立刻执行"的能力。别人可能需要一周才能养成的习惯，他一天就能做到。

这就是"不肯放弃"的力量。不是蛮力——是专注。李武的心里涌起一种复杂的情绪——有感动，有责任，还有一丝隐藏得很深的坚定。他知道，从这一刻开始，一切都将改变。他会用行动证明，即使没有天赋，没有血继限界，也可以变强。

（本章完）
