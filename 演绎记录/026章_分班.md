# 第二十六章：分班

毕业考试结束。

通过名单贴在训练馆门口。白纸黑字，密密麻麻的名字排成几列。李武的名字在第一位，小李的名字在最后一位——但也在上面。阳光照在名单上，给每一个名字都镀上了一层金色的光辉。

"我通过了！哥哥！我通过了！"

小李激动得跳起来，一把抱住李武。他的身体在发抖——不是害怕，是兴奋。四个月的努力，终于有了结果。他的眼睛里闪烁着泪光，但嘴角却挂着灿烂的笑容。

"嗯。"

"可是……我是最后一名……"

"通过就够了。"

李武的声音很平淡，但他的手在小李背上轻轻拍了两下。他的心里涌起一种满足感——不是因为自己考了第一名，而是因为小李通过了。这是他的胜利，也是小李的胜利。

分班结果在下午公布。

所有人聚集在走廊上，等着伊鲁卡念名单。走廊里挤满了人，空气闷热，议论声嗡嗡作响。阳光从窗户照进来，把走廊染成暖黄色。

"第七班：漩涡鸣人、春野樱、宇智波佐助。"

鸣人高兴得差点从栏杆上翻下去，被佐助一把拉住。

"第八班：日向雏田、犬冢牙、油女志乃。"

雏田低着头，脸红红的。牙在旁边大喊："太好了！和雏田一个班！"

"第十班：奈良鹿丸、山中井野、秋道丁次。"

鹿丸叹了口气："麻烦死了……"井野瞪了他一眼。

"第三班：李洛克、天天、日向宁次。"

小李的眼睛亮了。

"第三班！是凯老师的班！"

他高兴得手舞足蹈，差点撞到旁边的人。

"太好了！和宁次一个班！"

宁次站在远处，面无表情。但他的目光在小李身上停留了一秒——非常短暂的一秒，然后就移开了。

李武等着念到自己的名字。

伊鲁卡翻了一页，皱起眉头。

"第四班……"

他停顿了一下。

"只有一个人？"

全场安静下来。

"第四班：李武。"

伊鲁卡抬起头，看着李武。

"指导上忍……旗木卡卡西。"

走廊上的议论声瞬间炸了。

"只有一个人？"

"什么意思？"

"特殊班？"

"旗木卡卡西？那个传说中的……"

李武走上前，接过任务卷轴。

卷轴上写着：

```
第四班
指导上忍：旗木卡卡西
成员：李武（队长）
备注：独立行动小组，直属火影管辖
```

直属火影管辖。

他把卷轴收起来，转身离开走廊。

"哥哥！"

小李追上来。

"为什么你是一个人？"

"不知道。"

"可是……你一个人怎么完成任务？"

"慢慢来。"

李武的声音很平淡。

"你先去凯老师那里报到。"

"可是——"

"去。"

小李看着哥哥的背影，犹豫了一下，然后转身跑向第三班的方向。

李武走出学校，站在阳光下。阳光照在他的脸上，给他镀上了一层金色的轮廓。远处传来鸟鸣声，像一首欢快的交响曲。

他知道为什么自己被分到了"独立行动小组"——不是因为能力，是因为"被监视"。三代火影想近距离观察他，但又不想让他接触到其他学生。这是一个明智的选择，也是李武预料之中的选择。

直属火影。旗木卡卡西。独立行动。

这三件事拼在一起，答案很清楚：他是"嫌疑人"。李武的心里涌起一种复杂的情绪——有无奈，有理解，还有一丝隐藏得很深的坚定。他没有生气。也没有不安。因为他早就知道会有这一天。

他转身走向训练场。阳光照在他的身上，给他镀上了一层金色的轮廓。远处传来蝉鸣声，像一首永不停歇的交响曲。

该继续训练了。他知道，从这一刻开始，一切都将变得更加复杂。但他不害怕，因为他已经准备好了。

（本章完）
