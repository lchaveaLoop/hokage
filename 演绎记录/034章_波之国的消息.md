# 第三十四章：波之国的消息

卡卡西带来消息的时候，李武正在训练场练拳。

"A级任务。"

"A级？"

李武停下动作。

下忍能接的最高等级是C级。A级任务一般是中忍或上忍才能接的——除非特殊情况。

"护送一个人去波之国。"

卡卡西把任务卷轴递过来。

李武接过卷轴，展开。

```
任务等级：A级
委托人：达兹纳
任务内容：护送达兹纳安全抵达波之国，并协助完成大桥建设
备注：波之国附近有雾隐村叛忍活动，危险等级高
```

雾隐村叛忍。

他心里一跳。

他知道是谁——桃地再不斩。原著中波之国篇的boss，雾隐村的精英上忍，被称为"雾隐之刀"。

"为什么是A级？"

"因为波之国附近不太平。"

卡卡西的声音很平淡。

"委托人造桥的工程触动了某些人的利益。那些人可能会派人来破坏。"

"什么人？"

"一个叫卡多的商人。他控制了波之国的海上贸易，不想让大桥建成。"

李武沉默了一下。

原著中的剧情开始了。

"出发时间？"

"后天。"

"好。"

卡卡西转身要走，又停下来。

"李武。"

"嗯。"

"这次任务……和上次不一样。"

"怎么不一样？"

"上次的山贼是普通人。这次的敌人可能是忍者。"

卡卡西的表情变得严肃。

"如果你遇到忍者——不要硬拼。交给我。"

"好。"

卡卡西走了。

李武站在原地，看着手里的卷轴。

波之国。再不斩。白。

原著中，白是一个"被命运裹挟的人"。他没有自己的意志，只是再不斩手中的工具。最后为了保护再不斩而死。

李武闭上眼睛。

他能改变白的命运吗？

系统弹出一行碎片信息：

**"检测到……关键节点……等级不足……无法预判……"**

又是一句没用的话。

他把卷轴收起来，转身继续训练。

不管能不能改变——他都要去试试。

那天晚上，李武坐在院子里，看着夜空。

星星很亮，像一颗颗钻石撒在黑色的天鹅绒上。

他想起很多事。

想起原著中波之国的剧情。

再不斩。白。卡多。达兹纳。

白的牺牲。

再不斩的死亡。

卡多的覆灭。

达兹纳的桥。

波之国的平民。

他们都在等着他。

等着他改变这个世界。

李武攥了攥拳头。

"我会的。"

他低声说。

"不管付出什么代价。"

风吹过。

树叶沙沙作响。

像是在回应他的话。

第二天。

李武开始准备。

他检查了自己的装备——苦无、手里剑、绳索。

他检查了自己的状态——等级Lv.20，念气感知Lv.3，寸拳Lv.4。

他检查了自己的计划——如果遇到再不斩，用感知克制雾隐术；如果遇到白，用寸拳打破魔镜冰晶。

准备好了。

第三天。

出发。

李武和卡卡西站在木叶村大门前。

达兹纳还没有来。

"他会来吗？"

李武问。

"会。"

卡卡西说。

"他需要我们。"

十分钟后，达兹纳来了。

一个五十多岁的老头，头发花白，胡子拉碴，浑身酒气。

"你就是……嗝……忍者大人？"

"走吧。"

李武说。

三人离开木叶村，朝波之国的方向走去。

路上，达兹纳一直在抱怨。

"你们忍者收费也太贵了……一个A级任务要多少钱来着……"

"十万两。"

"十万两！够我喝一年的酒了！"

"你可以选择不委托。"

"那不行……那座桥必须建完……"

他的声音低了下去，像是在自言自语。

李武走在最后面，闭着眼睛。

念气感知展开。八十米范围内，所有活物的波动都清晰可见——前面的卡卡西、达兹纳、路边的鸟、树上的松鼠。

一切正常。

但他知道——这只是开始。

真正的危险，还在后面。

走了大概两个小时，他们到了一个水洼旁边。

卡卡西停下脚步。

"休息一下。"

达兹纳一屁股坐在地上，掏出酒壶开始喝。

李武没有坐。他的感知范围内出现了一个异常波动。

水洼里。

不是鱼——是人。一个人潜伏在水洼底部，气息压得很低，几乎和水融为一体。

他的感知已经能分辨出这种细微的差别了——水的波动是均匀的，而人的波动是有节奏的。心跳、呼吸、肌肉的微小张力……这些在念气感知里都像灯塔一样明显。

他没有声张。

走到水洼旁边，假装蹲下来洗手。

"卡卡西老师。"

他的声音很轻。

"水洼里有人。"

卡卡西的眼睛从书上移开，扫了一眼水洼。

"我知道。"

"怎么办？"

"等他先动。"

李武站起身，走回原位。

三十秒后，水洼里的人动了。

两道身影从水中冲出——是分身术。两个一模一样的"卡卡西"出现在水面上。

"哦？"

达兹纳吓了一跳。

"两个卡卡西先生？"

真正的卡卡西合上书，看着那两个分身。

"雾隐村的中忍？"

分身没有回答。他们同时冲向达兹纳。

卡卡西动了。

他的速度快到李武只看到一道影子——然后两个分身同时被击碎，化成两滩水。

"走吧。"

卡卡西把书收起来。

"这只是试探。真正的敌人还在后面。"

达兹纳的脸色变得苍白。

"真……真正的敌人？"

"嗯。"

李武看着卡卡西的背影，心里想：再不斩。

波之国的任务，正式开始了。

（本章完）
