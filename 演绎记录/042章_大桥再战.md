# 第四十二章：大桥再战

第三天。大桥。

浓雾再次升起。但这次，李武准备好了。雾气冰冷而潮湿，带着海水的咸腥味。能见度不到两米，仿佛整个世界都被白色的幕布笼罩了。

"达兹纳先生，躲到桥墩后面。"

"好……好……"

达兹纳哆哆嗦嗦地跑到桥墩后面蹲下。他的身体在发抖，眼睛里充满了恐惧。

李武站在桥中央，感知全开。他的心跳开始加速，血液在血管里奔涌。

两个波动。再不斩在正前方，白在右侧。他的感知穿透浓雾，锁定了两个敌人的位置。

"你来了。"

再不斩的声音从雾中传来。

"今天，我不会留手。"

斩首大刀从雾中劈出。

李武没有硬接——侧身躲开，让刀锋贴着他的肩膀滑过。然后他反手一拳，打向再不斩的手腕。

再不斩的反应很快。他收回刀，变劈为扫。

李武提前半拍后退，让刀锋从他胸前扫过。

"你的反应比昨天快了。"

再不斩的声音里带着一丝意外。

"但还不够。"

他再次冲上来。

斩首大刀连续劈出三刀——左、右、上。每一刀都带着破空声，速度快到肉眼几乎跟不上。

李武全躲开了。

不是因为他比再不斩快——是因为他的感知提前告诉了他每一刀的方向。在再不斩出手之前的零点几秒，他就已经知道刀会从哪里来。

"你怎么锁到我的位置？"

再不斩的声音里第一次出现了困惑。

"不靠看。"

李武说。

"你的呼吸和杀意，比脚步更响。"

再不斩沉默了一下。

然后他笑了。

"有意思。"

他退后一步，双手结印。

"雾隐之术·改！"

浓雾变得更浓了。能见度降到零——连自己的手都看不见。

但李武不需要眼睛。

他的感知穿透浓雾，锁定了再不斩的位置。

"找到了。"

他一步踏出，穿过雾墙，手掌直接扣住再不斩的咽喉边缘。

再不斩的身体僵住了。

"你……"

"够了。"

卡卡西的声音从雾中传来。

"再不斩。你输了。"

浓雾渐渐散去。

再不斩站在原地，斩首大刀掉在地上。李武的手扣在他的咽喉上，只要再用一点力，就能捏碎他的喉骨。

但李武没有用力。

"你为什么不杀我？"

再不斩的声音很沙哑。

"因为你不是我的敌人。"

"我是来杀达兹纳的。"

"我知道。但你不是'想'来杀他的——你是'被雇'来杀他的。"

李武松开手。

"走吧。"

再不斩看着他，眼神复杂。

"你这个小鬼……"

他捡起斩首大刀，转身消失在雾中。

白从雾里走出来，站在再不斩刚才的位置上。

他看着李武，眼神里有一种李武读不懂的东西。

"你很强。"

白说。

"但你还不够强。"

"我知道。"

"下次见面——我不会留手。"

"我也不会。"

白转身走了。

李武站在桥中央，看着他的背影。雾气渐渐散去，阳光从雾的缝隙中照进来，给一切都镀上了一层金色的光辉。

他知道白说的是真的。下次见面，他们会是敌人。但他也知道——白不想成为敌人。白的眼睛里，闪烁着一种复杂的情绪——有无奈，有痛苦，还有一丝隐藏得很深的善良。

李武的心里涌起一种复杂的情绪——有紧张，有期待，还有一丝隐藏很深的坚定。他知道，从这一刻开始，一切都将改变。他要改变白的命运，改变再不斩的命运，改变波之国的命运。

（本章完）
