# 第四十七章：命运节点

第六天。大桥。

工程接近尾声。桥面已经铺好了三分之二，再过两天就能完工。工人们在夕阳下忙碌着，汗水浸透了衣服，但脸上带着希望的笑容。这座桥对他们来说不仅仅是连接两地的通道，更是摆脱卡多压迫的象征。

但李武知道——再不斩还会来。

海风从桥的另一端吹来，带着咸腥的味道。远处的海面上，几只海鸥在盘旋，叫声凄厉而悠远。李武站在桥头，看着渐暗的天空，心里有一种不安的感觉。

不是因为他想来——是因为卡多不会放过他。卡多付了钱，再不斩就必须完成任务。如果再不斩失败，卡多会杀了他。这是一个简单的雇佣关系，但背后却隐藏着更深层的矛盾。

再不斩是忍者，卡多是商人。在忍者的世界里，实力就是一切。但在这个偏远的地方，金钱却能买到忍者的服务。这是一种扭曲的权力关系，而李武正在试图打破它。

这是一个死局。

除非——有人打破它。

那天晚上，李武坐在房间里，闭上眼睛。

房间很安静，只有窗外海浪拍打岸边的声音。月光透过窗户洒进来，在地板上投下斑驳的影子。李武盘腿坐在床上，开始每天的念气循环，但今天他总是无法集中精神。

白的脸庞不断浮现在他的脑海中。那个有着清澈眼睛的少年，为了再不斩可以付出一切。李武知道，在原著中，白会死在明天。他会为了保护再不斩而挡下卡卡西的雷切，然后像一片落叶一样飘落。

不，他不能让这件事发生。

就在他下定决心的时候，系统突然弹出了一条前所未有的信息。

不是碎片——是完整的面板。

```
【检测到关键命运节点：白的牺牲】

【节点描述：桃地再不斩将在明日的战斗中被卡卡西重创。
白将为保护再不斩而挡下雷切，死亡。】

【方案A：强制干预】
- 代价：等级-3
- 效果：白存活，再不斩存活
- 风险：剧情偏离，未知后果

【方案B：不干预】
- 代价：无
- 效果：维持原轨迹
- 风险：白死亡

【请在24小时内做出选择】
```

李武盯着面板，沉默了很久。

房间里的空气仿佛凝固了。窗外的海浪声变得遥远，只剩下系统面板散发出的幽蓝色光芒照亮了他的脸。

方案A——救白。代价是等级-3。他现在是Lv.20，降到Lv.17，会失去很多进度。这五个月的努力，每天凌晨四点起床训练，循环念气，突破感知等级……所有这些，都会因为这个选择而打乱。

方案B——不救。白会死。

他在心里权衡，手指不自觉地握紧又松开。

等级-3不是小代价。接下来的中忍考试、转职窗口、整个节奏都会被打乱。更关键的是——白和再不斩之间的关系，并不是"救下来"就能解开的。白是自愿挡在再不斩面前的。那是他的选择，是他认为自己存在的意义。

如果李武强行干预——白会感激吗？还是会怨恨？他会不会觉得李武剥夺了他为再不斩而死的权利？

他不知道。

窗外的月亮被乌云遮住了，房间变得更暗。李武闭上眼睛，深吸一口气，又缓缓吐出。

但他知道一件事：如果他不救，白会死。

而他不想让白死。

不是因为白是"好人"——是因为白不应该以那种方式死。白不应该成为再不斩的"工具"，然后为了保护"工具的主人"而牺牲。白不应该在十七岁就结束自己的生命，甚至没有机会去体验这个世界更多的可能性。

白应该有自己的人生。

"系统。"

他低声说。

"方案A。"

系统沉默了几秒。

然后面板消失了。

```
【选择确认：方案A】
【代价：等级-3】
【请在战斗中执行干预】
```

李武睁开眼睛。

窗外的月亮不知何时又从云层中露了出来，很圆，很亮。月光洒在海面上，波光粼粼，像无数颗星星在闪烁。

明天——一切都将改变。

他站起身，走到窗前。海风吹在他的脸上，带着咸腥的味道和夜晚的凉意。远处，再不斩和白居住的山洞方向，隐约可以看到一点火光。

李武看着那点火光，心里想：白，你还不知道吧？明天，你本来应该死在那里。但我会改变这一切。不是因为你是什么重要的人物，而是因为每个人都有权利选择自己的人生。

你选择了为再不斩而死。但我想给你另一个选择的机会。

至于这个选择会带来什么后果——那就让我们一起面对吧。

（本章完）
