# 第四十八章：雷切

第七天。大桥。

决战。

再不斩站在桥头，斩首大刀扛在肩上。他的脸色比昨天更差了——绷带下面渗出新的血迹。阳光照在他的脸上，给他镀上了一层金色的轮廓。他的眼睛里闪烁着一种复杂的情绪——有愤怒，有疲惫，还有一丝隐藏得很深的绝望。

白站在他身边，面具已经完全碎了。露出的脸很清秀，但眼神很疲惫。他的衣服上有几处破损，显然昨天的战斗让他消耗很大。

"今天，做个了结。"

再不斩说。他的声音很沙哑，像砂纸磨过铁皮。

卡卡西走上前。阳光照在他的银发上，闪闪发光。

"再不斩。"

"卡卡西。"

两人对视。空气仿佛凝固了，只剩下远处海浪的声音。

"你的伤还没好。"

"不需要好。"

再不斩举起刀。

"杀了你，任务就完成了。"

他冲上来。

斩首大刀劈向卡卡西——卡卡西侧身躲开，反手一击。雷光在掌心凝聚——

"雷切。"

银色的闪电直奔再不斩的胸口。

再不斩想躲——但他的伤太重了，动作慢了半拍。

就在雷切即将命中的瞬间——

白动了。

他挡在再不斩面前。

"再不斩大人——"

李武的瞳孔缩到了极限。

他一直在等这一刻。

方案A——强制干预。

他的身体在大脑反应过来之前就已经动了。

不是冲向白——是冲向卡卡西。

"卡卡西老师！收手！"

他的声音撕裂了空气。

卡卡西的雷切停在半空中——距离白的胸口不到十厘米。

"李武？！"

"不要杀他！"

李武挡在白和卡卡西之间。

"让开。"

卡卡西的声音很冷。

"他是敌人。"

"他不是敌人。"

李武转过头，看着白。

白的眼睛里全是震惊——他没想到李武会挡在自己面前。

"你……为什么……"

"因为你不想死。"

李武说。

"你挡在再不斩面前，不是因为你想死——是因为你不知道还能做什么。"

白的嘴唇在发抖。

"我……我是再不斩大人的工具……"

"你不是工具。"

李武的声音很坚定。

"你是人。你有自己的意志。你不想死——你只是被训练成'以为自己想死'。"

白沉默了。

再不斩在后面喊："白！你在干什么！杀了他！"

白转过头，看着再不斩。

他的眼神里第一次出现了犹豫。

"再不斩大人……我……"

"动手！"

"我……"

白的拳头攥紧了。

然后——松开了。

"对不起，再不斩大人。"

他的声音很轻。

"我不想死。"

再不斩的脸色变了。

"你——"

卡卡西收回雷切。

"够了。"

他的声音很平淡。

"再不斩。你的任务失败了。走吧。"

再不斩看着白，又看着李武，最后看着卡卡西。

他的表情从愤怒变成困惑，从困惑变成疲惫。

"……走。"

他扛起斩首大刀，转身消失在雾中。

白站在原地，没有动。

"你……为什么救我？"

他问李武。

"因为你值得被救。"

李武说。

白的眼眶红了。

但他没有哭。

他转身，追上再不斩的背影。

李武站在桥中央，看着他们消失的方向。阳光照在他的身上，给他镀上了一层金色的轮廓。远处传来海浪的声音，像一首轻柔的摇篮曲。

系统弹出一条信息：

```
【命运节点干预成功】
【代价：等级-3】
【当前等级：Lv.17】
【白存活。再不斩存活。剧情偏离。】
```

他感觉到一股力量从体内被抽走——很痛，像有什么东西从骨髓里被拔出来。但他不后悔。因为他救了一个人，一个不应该死的人。

白活着。再不斩活着。这就是改变。小到几乎看不见，但很真实。他的心里涌起一种满足感——不是因为自己变强了，而是因为他改变了命运。改变了白的命运，改变了再不斩的命运，改变了波之国的命运。

他知道，从这一刻开始，一切都将改变。但他已经准备好了。

（本章完）
