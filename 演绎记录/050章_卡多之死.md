# 第五十章：卡多之死

第八天。大桥完工。

清晨的阳光洒在大桥上，给一切都镀上了一层金色的光辉。海风吹过桥面，带着咸腥的味道和希望的气息。远处的海面上，几只海鸥在盘旋，叫声欢快而悠扬。

达兹纳站在桥头，看着自己亲手造出来的桥，眼眶红了。他的手颤抖着，抚摸着桥栏杆上的木纹，仿佛在抚摸一个珍贵的梦想。

"终于……终于建完了……"

工人们欢呼起来。他们的声音里充满了喜悦和希望，像一首胜利的交响曲。有人在拥抱，有人在哭泣，有人在大笑。

李武站在桥中央，看着远处的海面。他的表情平静，但心里却有一种不安的感觉。阳光照在他的脸上，给他镀上了一层金色的轮廓。

他知道——事情还没结束。

卡多不会善罢甘休。这种人，永远不会放弃。他们会用各种方式回来，直到达到目的或者被彻底消灭。

果然——

桥头出现了人影。

不是一个人。是几十个人。穿着破烂的衣服，手里拿着武器。领头的还是那个矮胖的男人。

卡多。

"达兹纳！"

他的声音尖锐得像指甲刮过黑板。

"你以为桥建完就赢了？"

他抬手一挥。

"杀了他们！所有人！"

几十个人冲向大桥。

李武没有动。

他在等。他的心跳开始加速，血液在血管里奔涌。阳光照在他的身上，给他镀上了一层金色的轮廓。

等卡多靠近。

卡多的人冲到桥中央的时候，李武动了。他的动作快如闪电，像一阵风暴席卷而过。

"闪电之舞。"

他消失在原地。

第一脚踢断前排一个人的膝盖，骨头断裂的声音清晰可闻。第二击反肘砸塌另一个人的肩膀，那人惨叫着倒在地上。第三记旋身下劈把三个人同时掀飞，他们像断线的风筝一样飞出去，重重地摔在桥面上。

十秒。

十二连击。

卡多的人成片倒下。他们甚至没看清李武的动作，就已经躺在了地上。剩下的扔刀就跑，跑得比兔子还快。

卡多的脸色煞白，跌坐在地。他的墨镜掉在地上，露出一双充满恐惧的小眼睛。

"别……别过来！"

李武停在他面前。

"我说过——如果你再来，我不会留手。"

"我……我有钱！你要多少——"

"你的账，不用钱结。"

寸拳落下。

干净。利落。

卡多仰面倒地，再没爬起来。

桥上安静了。

工人们看着这一幕，没有人说话。

李武站在卡多的尸体旁边，呼吸很重。阳光照在他的身上，但他却感到一阵寒意。海风吹过他的脸，带着咸腥的味道。

他杀了人。

第一次。

不是山贼——是一个人。一个活生生的人。刚才还活蹦乱跳、威胁着要杀死所有人的卡多，现在躺在地上，一动不动。

他没有感到愧疚。卡多该死——他压迫了一个国家，杀死了无数平民，把波之国变成了人间地狱。津奈美的丈夫，还有无数无辜的人，都是因为卡多而死。

但他也没有感到快意。

只是……空。一种深深的、无法言喻的空虚感。像心里被挖掉了一块，什么都没有剩下。

"李武。"

卡卡西走过来。他的脚步很轻，但每一步都踩得很稳。阳光照在他的银发上，闪闪发光。

"你还好吗？"

"还行。"

"第一次杀人？"

"嗯。"

"感觉怎么样？"

李武想了一下。海风吹过他的头发，把几缕发丝吹到脸上。远处的海面上，波光粼粼，像无数颗星星在闪烁。

"没什么感觉。"

卡卡西沉默了一下。他的眼睛里，闪烁着一种复杂的情绪——有理解，有担忧，还有一丝隐藏得很深的同情。

"那就好。"

他转身走了。他的背影在阳光下显得有些落寞，仿佛想起了什么往事。

李武站在桥中央，看着远处的海面。阳光照在海面上，波光粼粼，像无数颗星星在闪烁。远处的天空中，几朵白云悠闲地飘过，与桥上的宁静气氛形成和谐的画面。

波之国的海还在翻涌。但压在这片土地上的阴影——终于裂开了一道口子。阳光从那道口子里照进来，照亮了曾经黑暗的角落。

```
【等级提升：Lv.17 → Lv.20】
【命运节点：白存活。再不斩存活。卡多死亡。】
【剧情偏离度：15%】
```

他看着系统面板，心里想：15%的偏离度。不多。但够了。

白活着。再不斩活着。波之国自由了。

这就是改变。不是改变整个世界，而是改变身边的人。不是推翻整个体系，而是在这个体系中找到一丝裂缝，然后让它慢慢扩大。

就像这座桥一样，连接两个世界，让希望可以流动。

他转身走回桥面，工人们正在欢呼庆祝。达兹纳走过来，眼睛里闪烁着感激的光芒。

"谢谢你，忍者大人。"

李武点了点头。他的心里，有一种满足感——不是因为杀了卡多，而是因为改变了命运。

白不会死了。再不斩不会死了。波之国不会被卡多压迫了。

这就是他来到这个世界的意义。

（本章完）
