# 第五十四章：团藏

回到木叶的第三天晚上。

李武一个人在家。小李去凯那里训练了。

门被敲响。

他打开门——门口站着一个老人。

右眼缠着绷带，下巴上有道伤疤，表情冷得像冰。

团藏。

"李武。"

他的声音很低，像砂纸磨过铁皮。

"我能进来吗？"

李武沉默了一下，让开路。

团藏走进屋子里，扫了一眼。

"很简陋。"

"习惯了。"

团藏在沙发上坐下，从怀里掏出一份文件。

"这是你弟弟的医疗报告。"

他把文件放在茶几上。

李武的心沉了一下。

他拿起文件，展开。

```
姓名：李洛克
年龄：12岁
诊断：经络多处断裂，肌肉纤维不可逆损伤
建议：退役
```

"你弟弟的经络损伤是不可逆的。"

团藏说。

"按照现在的训练强度，不出两年，他的腿就会废掉。"

李武攥紧文件。

"你想说什么？"

"根部有资源。"

团藏的声音很平淡。

"我们可以治好你弟弟的经络。条件是——"

"不用。"

李武把文件还给他。

团藏的眉毛微微挑了一下。

"你不想救你弟弟？"

"我想。但不是用你的方法。"

"我的方法有什么问题？"

"你的方法——是要我加入根部。"

"是。"

"我不加入。"

团藏沉默了一下。

"为什么？"

"因为根部的手段，我不认同。"

"手段？"

"暗杀、监视、操控——这些是根部的手段。我不想成为那种人。"

团藏看着李武，眼神里没有任何情绪。

"你会后悔的。"

"也许。"

"你弟弟的腿——"

"我会用自己的方法治好他。"

团藏站起身。

"好自为之。"

他转身走了。

李武关上门，靠在门板上。

他看着手里的文件——"建议退役"四个字像针一样扎在眼睛里。

但他没有动摇。

因为他知道——团藏的"好意"背后，是控制。一旦他加入根部，他就不再是自己了。

他会成为团藏的工具。

而他不想成为任何人的工具。

（本章完）
