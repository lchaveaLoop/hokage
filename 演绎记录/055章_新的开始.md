# 第五十五章：新的开始

中忍考试的报名通知贴出来了。

贴在忍校门口的公告栏上，白纸黑字：

```
中忍选拔考试
时间：一个月后
报名条件：下忍，由指导上忍推荐
```

小李看到通知的时候，兴奋得差点跳起来。

"哥哥！中忍考试！我们一起参加吧！"

李武看着通知，没有说话。

中忍考试。

原著中的中忍考试是整个故事的第一个大高潮——大蛇丸入侵、木叶崩溃、三代死亡。无数的战斗、无数的牺牲、无数的改变。

而现在，他要亲自走进这个漩涡。

"哥哥？"

小李拉了拉他的袖子。

"你想参加吗？"

李武想了一下。

"想。"

"太好了！"

小李高兴地跳了一下。

"那我去告诉凯老师！"

他转身跑掉了。

李武站在公告栏前，看着那张通知。

一个月。

一个月后，中忍考试就要开始了。

他闭上眼睛，感知了一下自己的状态。

```
【姓名】：李武
【等级】：Lv.20
【职业】：无（未转职）
【技能】：
  - 念气感知 Lv.2
  - 寸拳 Lv.3
  - 念气护脉术 15%
```

Lv.20。距离一觉Lv.50还有30级。

很远。但他不急。

因为他知道——中忍考试不只是"考试"。它是一个节点。一个让所有人、所有命运交汇在一起的节点。

在那里，他会遇到大蛇丸。会遇到宁次的笼中鸟。会遇到小李vs我爱罗。会遇到三代的死亡。

每一个事件，都是一个"命运节点"。

每一个节点，他都要做出选择。

救，还是不救。

改，还是不改。

他睁开眼睛。

远处的火影岩上，四位火影的雕像在阳光下显得格外庄严。

波之国只是开始。

真正的风暴——还在后面。

他转身离开公告栏，走向训练场。

该继续训练了。

（本章完）
