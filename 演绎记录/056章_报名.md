# 第五十六章：报名

中忍考试报名处设在忍校三楼。

门口挤满了人——来自各个村子的下忍，穿着不同颜色的制服，戴着不同标志的护额。木叶的、砂隐的、音忍的、草忍的……整个走廊像一锅煮沸的粥。

李武站在人群外面，感知扫过全场。

波动很多。大部分是普通下忍的水平——淡而散，像一团雾。但有几个波动不一样。

左侧走廊尽头，三个砂隐忍者站在一起。其中一个的波动很奇怪——不是"强"，是"重"。像一块石头沉在水底，周围的水都被压得变形。

我爱罗。

右侧楼梯口，三个音忍站在一起。其中一个的波动被什么东西遮住了——像暗部的外壳，但更粗糙。

大蛇丸伪装的草忍？不对——音忍。大蛇丸应该还没到。

"李武。"

卡卡西的声音从身后传来。

"你的推荐信。"

他递过来一张纸。

"卡卡西老师的推荐？"

"三代大人亲自批的。"

李武接过推荐信，扫了一眼。

```
推荐人：旗木卡卡西
被推荐人：李武
推荐理由：独立完成A级任务，实力达到中忍水平
备注：直属火影特别关注对象
```

"直属火影特别关注对象"——这几个字让他皱了皱眉。

"走吧。"

卡卡西转身走了。

李武走进报名处。

负责登记的是一个戴眼镜的中忍，面无表情。

"名字？"

"李武。"

"指导上忍？"

"旗木卡卡西。"

中忍翻了翻名单，找到了李武的名字，在上面打了个勾。

"考试在下周。笔试。"

"笔试？"

"对。第一场考试是笔试。通过之后才能参加第二场。"

"好。"

李武转身离开报名处。

走到走廊上的时候，他看到了小李。

"哥哥！"

小李跑过来，兴奋得脸都红了。

"我也报名了！凯老师推荐的！"

"嗯。"

"宁次和天天也报名了！"

"嗯。"

"还有鸣人！第七班也报名了！"

"嗯。"

小李看着李武的表情，愣了一下。

"哥哥……你不高兴吗？"

"高兴。"

"那你为什么——"

"我在想别的事。"

李武看着走廊尽头。

我爱罗已经不在了。但他的波动还残留着——一种很沉的、"被什么东西压着"的波动。

他知道我爱罗是谁。一尾人柱力。砂隐的"武器"。原著中差点杀死小李的人。

命运的齿轮，开始转动了。

（本章完）
