# 第六十二章：咒印

李武找到小李的时候，天已经黑了。

第三班在一个山洞里扎营。小李坐在火堆旁边，看到李武回来，立刻站起来。

"哥哥！你没事吧？！"

"没事。"

"你的脖子——"

小李的目光落在李武的脖子上。

那里有一个黑色的印记。形状像三条弯曲的线，从脖子后面延伸到肩膀。

"这是什么？"

"咒印。"

"咒印？！"

小李的脸色变了。

"谁干的？"

"一个很强的人。"

李武没有说大蛇丸的名字。现在说出来只会让小李担心。

"能治好吗？"

"能。但需要时间。"

他坐到火堆旁边，闭上眼睛。

念气在经络里循环，经过脖子的时候，会触碰到咒印。每次触碰，咒印都会退缩一点——但每次退缩之后，它又会慢慢回来。

像潮水。退了又涨，涨了又退。

他需要找到一种方法，让咒印永远退下去。

念气护脉术。

如果他能把念气压缩成一层膜，覆盖在咒印的表面——是不是可以隔绝咒印的能量供给？

他试着做。

第一次，失败了。念气膜太薄，咒印的能量直接穿透了。

第二次，稍微好一点。念气膜挡住了咒印一半的能量。

第三次——

系统弹出一条信息：

```
【检测到外来能量入侵】
【类型：咒印（大蛇丸）】
【威胁等级：高】
【建议：使用念气护脉术隔离】
【当前隔离进度：35%】
```

35%。

比之前的15%高了——因为这次有了"敌人"。念气护脉术在应对实际威胁的时候，推演速度会加快。

他继续推演。

一个小时后，隔离进度达到了50%。

咒印的能量被挡住了一半。灼烧感减轻了很多，但还在。

"够了。"

他睁开眼睛。

"今晚先休息。明天继续。"

"好。"

小李在他旁边躺下。

李武看着弟弟的脸，心里想：大蛇丸的目标是佐助。但他现在也盯上了自己。

这场中忍考试——比他想象的更危险。

（本章完）
