# 第七十章：一个月

预选赛结束后，所有人有一个月的准备时间。

李武没有浪费时间。

他每天凌晨四点起床，循环念气。上午训练体术。下午推演念气护脉术。晚上研究大蛇丸的咒印。

一个月后，他的状态：

```
【姓名】：李武
【等级】：Lv.25
【职业】：无（未转职）
【技能】：
  - 念气感知 Lv.3（范围：120米）
  - 寸拳 Lv.4
  - 崩拳 Lv.2
  - 念气护脉术 45%
  - 咒印隔离 70%
```

进步很大。

念气感知的范围从八十米扩大到了一百二十米。寸拳的威力提升了百分之三十。念气护脉术从百分之十五提升到了百分之四十五。

最重要的是——咒印隔离达到了百分之七十。

大蛇丸的咒印被压制得很厉害。灼烧感几乎消失了，只在情绪激动的时候才会隐隐作痛。

但他知道——咒印没有根除。它只是被压住了，像一条冬眠的蛇，随时可能醒来。

一个月的最后一天。

李武坐在训练场上，看着远处的火影岩。

决赛就在明天。

对手可能是宁次、可能是我爱罗、可能是任何人。

但他不担心战斗。

他担心的是——大蛇丸。

大蛇丸不会只来参加中忍考试。他有更大的目的——摧毁木叶。

而李武要阻止他。

不是用武力——是用智慧。

他站起身，走向家的方向。

明天——一切都会改变。

（本章完）
