# 第七十七章：秽土转生

两具棺材从地面升起。

棺材板缓缓打开。

灰尘散落。

两个人从棺材里走出来。

初代火影——千手柱间。

二代火影——千手扉间。

两位传说中的忍者，木叶的创始人。他们的身体是灰色的——像被灰尘覆盖的雕塑。眼睛是空洞的——没有意识，只是被操控的傀儡。

"初代……二代……"

三代的声音在发抖。

"大蛇丸……你竟然亵渎死者的灵魂……"

"亵渎？"

大蛇丸笑了。

"我只是让他们'复活'了而已。"

他双手结印。

"去吧。杀了他。"

初代和二代同时动了。

初代的双手按在地上——

"木遁·树界降诞！"

无数树木从地面升起，像活物一样向三代缠绕过去。树根、树枝、树叶——整个竞技场都被树木覆盖了。

二代的双手结印——

"水遁·水龙弹之术！"

巨大的水龙从侧面撞向三代。水龙的体积比普通人大了十倍，冲击力足以撞碎一栋房子。

三代躲开了大部分攻击——但还是被树根缠住了脚。

"呃——"

他挣扎着想挣脱，但树根越缠越紧。

"老师。"

大蛇丸的声音很冷。

"你老了。"

三代没有回答。

他深吸一口气，双手结印——

"尸鬼封尽！"

一个巨大的死神虚影出现在他身后。

死神的身体是白色的，像一具骷髅。手里握着一把刀，刀刃上缠绕着黑色的雾气。

李武的瞳孔缩到了极限。

来了。

三代要和大蛇丸同归于尽了。

原著中，三代用尸鬼封尽封印了大蛇丸的双手，然后自己也死了。

但现在——

他要阻止。

"系统！"

他在心里喊。

"改命三代！代价是什么？！"

系统弹出一条信息：

```
【检测到关键命运节点：三代火影之死】
【方案A：强制干预】
- 代价：等级-5
- 效果：三代存活
- 风险：大蛇丸逃脱，后续剧情偏离

【请在60秒内做出选择】
```

等级-5。

他现在是Lv.25。降到Lv.20。

代价很大。

但三代不能死。

"方案A！"

他冲向结界。

（本章完）
