# 第七十八章：李武介入

李武冲进结界的时候，三代的尸鬼封尽已经发动了一半。

死神的虚影越来越清晰——一只巨大的手正在从三代的身体里"拉"出什么东西。

"三代大人！停手！"

李武喊。

三代转过头，看到李武，瞳孔缩了一下。

"你……你怎么进来的？！"

"不要用尸鬼封尽！"

"来不及了——"

"来得及！"

李武冲到三代面前，一只手按在他的肩膀上。

念气涌入。

暖流从李武的身体涌进三代的经络——不是攻击，是"隔断"。他用念气在三代和死神之间建了一道屏障，把尸鬼封尽的能量挡住。

"什么？！"

大蛇丸的表情变了。

"你竟然能干扰尸鬼封尽？！"

李武没有回答。

他的全部注意力都集中在那道屏障上。死神的力量太强了——他的念气屏障在不断被侵蚀，像一层薄冰在被火焰融化。

"系统！"

他在心里喊。

"给我更多力量！"

系统弹出一条信息：

```
【改命执行中……】
【代价：等级-5】
【正在转移能量……】
```

一股强大的力量从他体内涌出——不是念气，是另一种东西。更纯，更亮，像一道金线。

金线涌进三代的经络，和念气混合在一起，形成一道更厚的屏障。

死神的手被挡住了。

"呃——"

三代的身体颤抖了一下。

"你……你在做什么……"

"救你。"

李武咬紧牙关。

屏障越来越厚。死神的力量被一点点推回去。

十秒。二十秒。三十秒。

终于——

死神的虚影消失了。

尸鬼封尽被中断了。

三代瘫坐在地上，大口喘气。

"你……你竟然……"

李武也瘫坐在地上。

他的身体被抽空了——等级-5的代价，让他失去了大量的能量。

```
【改命成功】
【代价：等级-5】
【当前等级：Lv.20】
【三代火影存活】
```

大蛇丸看着这一幕，眼神复杂。

"你……你到底是什么人？"

李武抬起头，看着大蛇丸。

"我是李武。"

"你不是普通的下忍。"

"我从来没说过我是。"

大蛇丸沉默了几秒。

然后他笑了。

"有意思……太有意思了……"

他转身消失在结界外。

初代和二代的身体开始崩解——秽土转生失去了施术者的控制，自动解除了。

结界破碎。

李武躺在地上，看着天空。

他做到了。

三代活了。

（本章完）
