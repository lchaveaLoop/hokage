# 第七十九章：战后

大蛇丸撤退了。

音忍四人众被木叶的暗部追击，全部被捕。他们的尸体被带回木叶，交给情报部门分析。

三代火影被送往医院。

尸鬼封尽虽然被中断了，但已经消耗了他大量的生命力。他的头发全白了，脸上的皱纹比以前深了十倍。

"三代大人……"

卡卡西站在病床前，声音很低。

"您……"

"我没事。"

三代的声音很虚弱。

"只是……老了。"

李武站在病房外面，没有进去。

他的身体也很虚弱——等级-5的代价，让他的经络多处受损，念气循环几乎停止。但他没有去治疗——因为他在等。

等三代醒过来。

等三代告诉他——接下来该怎么办。

门开了。

卡卡西走出来，看到李武，点了点头。

"三代大人想见你。"

"好。"

李武走进病房。

三代躺在床上，脸色苍白。但眼睛依然锐利——像两把刀，能看穿一切。

"李武。"

"三代大人。"

"你救了我的命。"

"嗯。"

"你知道这意味着什么吗？"

"知道。"

"你知道——你暴露了。"

三代的声音变得严肃。

"你的能力——能干扰尸鬼封尽——这已经超出了下忍的范畴。甚至超出了上忍的范畴。"

"整个村子都在议论你。"

"有人说你是天才。有人说你是怪物。有人说你是异世界的人。"

李武沉默了一下。

"我不在乎。"

"你应该在乎。"

三代的声音变得沉重。

"木叶不是铁板一块。有各种势力、各种利益、各种阴谋。你暴露了能力——就等于把自己放到了风口浪尖上。"

"团藏会盯上你。长老团会盯上你。甚至其他国家的间谍也会盯上你。"

"你要小心。"

李武点了点头。

"我知道。"

"好。"

三代闭上眼睛。

"你先去休息吧。等我好一点——我们再谈。"

"好。"

李武转身离开病房。

走到门口的时候，他回头看了一眼。

三代躺在床上，呼吸很轻。

他活下来了。

但代价是——他的生命力被大幅消耗了。按照原著的设定，三代应该在大蛇丸入侵中死亡。但现在他活下来了——只是身体大不如前。

这不是"完美"的改命。

但这是"可以接受"的改命。

因为三代活着——木叶就不会崩溃。

（本章完）
