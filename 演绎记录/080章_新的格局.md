# 第八十章：新的格局

大蛇丸入侵之后，木叶的格局发生了巨大变化。

三代火影活下来了——但身体大不如前。他开始培养接班人，准备退休。

团藏的势力被削弱了——大蛇丸入侵的时候，根部没有任何行动，被三代质问。

卡卡西被提拔为特别上忍——因为在大蛇丸入侵中的表现。

而李武——

成了木叶的英雄。

"你救了三代火影。"

卡卡西站在病床前，声音很平淡。

"整个村子都知道了。"

"嗯。"

"但你也暴露了。"

"我知道。"

"你的能力——能干扰尸鬼封尽——这已经超出了下忍的范畴。"

"嗯。"

"三代大人想见你。"

李武点了点头。

三天后，他出院了。

第一件事，就是去火影办公室见三代。

三代坐在桌子后面，比以前瘦了很多。但眼神依然锐利。

"李武。"

"三代大人。"

"你救了我的命。"

"嗯。"

"我欠你一个人情。"

"不用。"

"你想要什么？"

李武想了一下。

"我想要自由。"

"自由？"

"我不想被任何人控制。不想被团藏利用，不想被任何人当作工具。"

三代沉默了一下。

"好。"

"我答应你。"

他站起身，走到窗边。

"从今天开始，你是木叶的特别下忍。直属火影管辖，但不受任何部门约束。"

"你可以自由行动，自由选择任务，自由使用你的能力。"

"条件是——在木叶需要你的时候，你必须出手。"

李武点了点头。

"好。"

三代转过身，看着他。

"你这个孩子……很特别。"

"谢谢。"

"不是夸你。"

三代的声音变得严肃。

"你的能力太强了。强到会引来嫉妒、恐惧、和觊觎。你要小心。"

"我知道。"

"好。去吧。"

李武转身离开火影办公室。

走到门口的时候，他回头看了一眼。

三代站在窗边，阳光照在他身上。

他活下来了。

命运改变了。

```
【改命记录更新】
1. 白（存活，代价等级-3）
2. 三代火影（存活，代价等级-5）
【当前等级：Lv.20】
【剧情偏离度：25%】
```

25%的偏离度。

不多。但已经改变了很多事情。

三代活着。木叶没有崩溃。团藏被削弱。

而他——站在了新的起点上。

（本章完）
