# 第八十一章：转职

大蛇丸入侵后的第三天。

李武坐在训练场上，闭着眼睛。

身体还没有完全恢复——等级-5的代价让他的经络多处受损，念气循环几乎停止。但他等不了了。

因为他知道——时间不等人。

晓组织在活动。大蛇丸在暗处窥视。未来的敌人只会越来越强。

他需要更强的力量。

系统弹出一条信息：

```
【当前等级：Lv.20】
【距离转职要求：已满足】
【主职业转职任务开启：散打】
【是否进入梦境位面试炼？】
```

他犹豫了一下。

转职——散打。

DNF里的散打是格斗家的转职之一，以拳系攻击为主，特点是"爆发"和"判定"。寸拳、崩拳、铁山靠——这些都是散打的招牌技能。

但现在转职，合适吗？

他的等级是Lv.20，比计划中低了五级（因为改命三代的代价）。如果现在转职，可能会错过一些转职奖励。

但如果不转职——他的实力就无法进一步提升。

而他需要实力。

大蛇丸还在外面。晓组织还在活动。未来会有更多的敌人、更多的战斗。

"转职。"

他说。

视野骤暗。

再亮时，他已经站在一座纯白擂台中央。

四周空无一人，只有风声从高处压下来。

对面，一道身影缓慢凝实。

身高、体型、呼吸节奏——全都和他一模一样。

"试炼守护者。"

系统音在耳边响起。

"击败过去的你，才能承载新的职业判定。"

守护者抬起头，咧嘴一笑。

那笑容和李武一模一样——但眼神不一样。守护者的眼神是冷的，像一面没有感情的镜子。

"来。"

守护者说。

李武深吸一口气。

他知道这场试炼的本质——不是"打败敌人"，是"超越自己"。守护者代表的是他"现在的极限"，只有突破这个极限，才能获得新的力量。

他摆好架势。

守护者也摆好架势。

两人对视。

然后——同时启动。

（本章完）
