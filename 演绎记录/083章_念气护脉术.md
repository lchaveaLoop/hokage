# 第八十三章：念气护脉术

转职之后，李武开始全力推演念气护脉术。

小李的八门遁甲训练越来越深入——凯已经开始教他"开门"了。每次训练完，小李的经络都会受到不同程度的损伤。

李武用念气给他止痛——但这只是治标，不是治本。

他需要找到彻底保护经络的方法。

念气护脉术。

原理已经清楚了：把念气压缩成一层膜，覆盖在经络表面。当八门遁甲开启时，查克拉流量暴增——念气膜会先一步吸收部分冲击，减少对经络的直接伤害。

问题是——怎么让念气膜更厚、更韧、更持久？

他开始实验。

第一次：把念气膜的厚度增加一倍。

结果：经络的灵活性下降了——像穿了一层太紧的衣服，动作受限。

第二次：把念气膜的韧性增加一倍。

结果：念气消耗太大——循环速度从七秒变成了十二秒。

第三次：把念气膜的持续时间增加一倍。

结果：成功了。

他发现——念气膜不需要"厚"，也不需要"韧"。它需要"持久"。只要能持续覆盖在经络表面，就能起到保护作用。

他调整了念气循环的方式——让一部分念气"驻留"在经络表面，而不是全部参与循环。

系统弹出信息：

```
【念气护脉术推演进度：60%】
【当前效果：八门反噬降低30%（理论值）】
【警告：尚未通过实战验证】
```

60%。

比之前的45%高了十五个百分点。

他继续推演。

一个小时后，进度达到了70%。

"够了。"

他睁开眼睛。

70%的护脉术，可以把八门遁甲的反噬降低百分之三十。虽然不能完全消除，但至少能让小李在使用八门遁甲的时候，不会受到致命伤害。

他站起身，走向小李的房间。

"小李。"

"嗯？"

"我找到保护你经络的方法了。"

小李的眼睛亮了。

"真的？！"

"真的。但需要测试。"

"怎么测试？"

"你开一次门。我用念气保护你的经络。看看效果。"

小李犹豫了一下。

"好。"

（本章完）
