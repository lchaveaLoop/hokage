# 第八十四章：测试

训练场。

小李站在空地中央，深吸一口气。

"准备好了吗？"

李武问。

"准备好了。"

"开门——开！"

小李体内的查克拉开始暴增。第一道"门"——开门，被强行冲破。

一股强大的力量从他体内涌出。

但同时——经络开始灼烧。

李武立刻出手。

念气从他的掌心涌出，覆盖在小李的经络表面。一层极薄的、但极其坚韧的膜，像一层皮肤一样贴在经络壁上。

查克拉的冲击撞在膜上——被吸收了一部分。

灼烧感减轻了。

"感觉怎么样？"

李武问。

"好……好多了！"

小李的声音里带着惊喜。

"以前开门的时候，经络会很疼……现在只有一点点！"

"继续。"

"休门——开！"

第二道门被冲破。查克拉流量再次暴增。

念气膜承受了更大的冲击——但没有破裂。

"生门——开！"

第三道门。

这是小李目前的极限。以前开到第三门的时候，他的经络会受到严重的损伤——需要休息好几天才能恢复。

但现在——

"我……我还能动！"

小李的脸上全是汗水，但眼睛亮得像两颗灯泡。

"经络没有受伤！"

李武收回念气。

系统弹出信息：

```
【念气护脉术实战测试完成】
【效果：八门反噬降低35%（实测值）】
【推演进度：70% → 75%】
```

35%的反噬降低。

比理论值高了五个百分点——因为实战中的压力让念气膜变得更坚韧了。

"成功了。"

李武说。

小李激动得差点跳起来。

"哥哥！谢谢你！"

"不用谢。"

李武看着弟弟的笑脸，心里想：这只是第一步。

35%的反噬降低，可以让小李安全地开到第三门。但如果要开到第五门、第六门、甚至第八门——还需要更强的护脉术。

他会继续研究。

总有一天，他会让小李可以安全地使用八门遁甲的所有门。

（本章完）
