# 第八十五章：新的开始

中忍考试结束一个月后。

木叶村恢复了平静。大蛇丸入侵的痕迹被慢慢抹去——倒塌的建筑被重建，受伤的人被治愈，死去的人被安葬。

但有些东西，永远无法恢复。

三代火影的身体大不如前。他开始培养接班人——不是团藏，是卡卡西。

"卡卡西。"

"三代大人。"

"你愿意接任火影吗？"

卡卡西沉默了很久。

"我……考虑一下。"

"好。"

李武站在火影楼的窗边，看着外面的街道。

中忍考试结束了。他没有成为中忍——因为他没有参加最终决赛（大蛇丸入侵中断了比赛）。但他的名声已经传遍了整个木叶。

"那个救了三代火影的下忍。"

"能干扰尸鬼封尽的怪物。"

"直属火影的特别下忍。"

各种称号贴在他身上。有敬畏的，有恐惧的，有嫉妒的。

但他不在乎。

因为他知道——中忍考试只是开始。

真正的风暴，还在后面。

晓组织。佩恩。大筒木。

这些敌人会一个接一个地出现。而他——要做好准备。

```
【姓名】：李武
【等级】：Lv.22
【职业】：散打（Lv.2）
【技能】：
  - 念气感知 Lv.3（范围：120米）
  - 寸拳 Lv.5
  - 崩拳 Lv.2
  - 念气护脉术 75%
  - 咒印隔离 70%
【改命记录】：
  - 白（存活，代价等级-3）
  - 三代火影（存活，代价等级-5）
【剧情偏离度】：25%
```

Lv.22。

距离一觉Lv.50还有28级。

很远。但他不急。

因为他知道——变强不是一朝一夕的事。需要时间，需要积累，需要在每一次战斗中成长。

他转身离开火影楼，走向训练场。

该继续训练了。

远处的火影岩上，四位火影的雕像在阳光下显得格外庄严。

新的故事——从这里开始。

（本章完）
