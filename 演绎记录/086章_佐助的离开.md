# 第八十六章：佐助的离开

中忍考试结束后两个月。

深夜。

李武的感知捕捉到了一个异常——有人在离开木叶。

不是普通的"出门"——是"逃离"。波动移动得很快，方向是村外。

佐助。

他睁开眼睛，从床上坐起来。

原著中的剧情开始了——佐助被大蛇丸的咒印吸引，深夜逃离木叶，前往音忍村。

他要不要阻止？

系统弹出一条信息：

```
【检测到关键命运节点：佐助离村】
【方案A：强制拦截】
- 代价：等级-2
- 效果：佐助留下
- 风险：咒印反噬，佐助可能失控

【方案B：不干预】
- 代价：无
- 效果：佐助离开，夺还队出发
- 风险：佐助可能被大蛇丸夺取身体

【请在30秒内做出选择】
```

李武沉默了。

方案A——拦截佐助。代价等级-2，他现在是Lv.22，降到Lv.20。但佐助会留下。

方案B——不干预。佐助会离开，但夺还队会出发。

他想了很久。

最后，他选择了方案B。

不是因为他不想救佐助——是因为他知道，佐助需要自己做出选择。

如果他强行把佐助留下，佐助不会感激——只会怨恨。因为佐助的离开，不是被大蛇丸"操控"的，是他自己的选择。

他想要力量。想要复仇。想要变强。

而大蛇丸给了他这个机会。

"让他去吧。"

李武低声说。

"等他想回来的时候——他会回来的。"

系统确认了选择。

```
【选择确认：方案B】
【佐助离村。夺还队即将出发。】
```

李武躺回床上，闭上眼睛。

他知道——夺还队会失败。原著中，夺还队被音忍四人众打得七零八落，最后只有鸣人追上了佐助。

但这一次——不一样。

因为他会参与。

（本章完）
