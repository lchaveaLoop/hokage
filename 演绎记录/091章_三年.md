# 第九十一章：三年

佐助离开后，时间开始加速。

三年。

三年的时间里，发生了很多事情。

三代火影退休了——身体大不如前，把火影的位置交给了卡卡西。交接仪式很简单——三代把火影斗笠递给卡卡西，说了一句"木叶就交给你了"，然后转身走进了夕阳里。

卡卡西成为第六代火影。

团藏的势力被彻底清算——大蛇丸入侵的时候根部没有任何行动，被三代抓住把柄，连根拔起。团藏本人被软禁在根部的地下基地里，再也没有出现在公众面前。

佐助离开了——去了音忍村，追随大蛇丸。他的目的没有变：变强，然后杀了鼬。

鸣人跟着自来也修行——去了很多地方，学了很多东西。三年后归来，实力大增，学会了仙人模式。

小李的八门遁甲训练继续——在李武的念气护脉术保护下，他已经能安全地开到第五门了。每次训练完，经络的损伤都比以前轻了很多。

宁次的笼中鸟研究继续——李武用念气感知分析了笼中鸟的结构，找到了破解的关键。三层外壳，每一层都有不同的防护机制。如果能找到三层外壳的弱点，就可以逐层破解。但还需要更多的时间来推演。

雏田的柔拳进步很大——在小李的鼓励下，她不再躲在树后面偷偷练习了。她开始在训练场上公开训练，虽然还是会脸红，但至少不再逃避。

而李武——

```
【姓名】：李武
【等级】：Lv.45
【职业】：散打（Lv.15）
【技能】：
  - 念气感知 Lv.5（范围：300米）
  - 寸拳 Lv.8
  - 崩拳 Lv.6
  - 铁山靠 Lv.4
  - 念气护脉术 90%
  - 咒印隔离 85%
【改命记录】：
  - 白（存活，代价等级-3）
  - 三代火影（存活，代价等级-5）
【剧情偏离度】：30%
```

Lv.45。

距离一觉Lv.50还有5级。

很近了。

三年的时间里，他的实力发生了质的飞跃。念气感知的范围从一百二十米扩大到了三百米——可以覆盖整个木叶村。寸拳的威力提升了三倍——一拳可以打碎一栋房子。念气护脉术从百分之七十五提升到了百分之九十——几乎可以完全消除八门遁甲的反噬。

最重要的是——他对"改命"的理解更深了。

改命不是"救人"——是"给选择"。

他救了白，是因为白"想活"。他救了三代，是因为三代"不该死"。

每一次改命，都是在尊重对方的选择。

而不是强加自己的意志。

这就是他的"道"。

（本章完）
