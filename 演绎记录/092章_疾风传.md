# 第九十二章：疾风传

三年后。

木叶村。

李武站在训练场上，看着远处的火影岩。

六代火影——旗木卡卡西的雕像已经刻上去了。和前五代火影并排站着，在阳光下显得格外庄严。

三年的时间，改变了很多事情。

三代火影退休了——身体大不如前，把火影的位置交给了卡卡西。

团藏的势力被彻底清算——大蛇丸入侵的时候根部没有任何行动，被三代抓住把柄，连根拔起。

佐助离开了——去了音忍村，追随大蛇丸。

鸣人跟着自来也修行——三年后归来，实力大增，学会了仙人模式。

小李的八门遁甲训练继续——在李武的念气护脉术保护下，他已经能安全地开到第五门了。

宁次的笼中鸟研究继续——李武用念气感知分析了笼中鸟的结构，找到了破解的关键，但还需要更多的时间来推演。

而李武——

```
【姓名】：李武
【等级】：Lv.45
【职业】：散打（Lv.15）
【技能】：
  - 念气感知 Lv.5（范围：300米）
  - 寸拳 Lv.8
  - 崩拳 Lv.6
  - 铁山靠 Lv.4
  - 念气护脉术 90%
  - 咒印隔离 85%
【改命记录】：
  - 白（存活，代价等级-3）
  - 三代火影（存活，代价等级-5）
【剧情偏离度】：30%
```

Lv.45。

距离一觉Lv.50还有5级。

很近了。

"哥哥！"

小李跑过来。

三年的时间，小李长高了很多。肌肉更结实了，眼神也更坚定了。不再是那个"只会用腿硬踢"的吊车尾——而是一个真正的忍者。

"凯老师说——今天有任务！"

"什么任务？"

"晓组织的情报！"

李武的心跳加速了一拍。

晓组织。

三年的调查，他已经收集了很多情报——晓组织有十个成员，每个都是S级叛忍。他们的目标是收集尾兽，实现某种"月之眼计划"。

但具体的细节，他还没有搞清楚。

"走吧。"

他说。

"去火影办公室。"

"好！"

两人赶到火影办公室。

卡卡西坐在桌子后面，表情严肃——比平时严肃得多。

"晓组织开始行动了。"

他说。

"他们的第一个目标——是一尾人柱力。"

"我爱罗。"

李武说。

"嗯。"

卡卡西点头。

"砂隐村已经发来求援。我们需要派人去支援。"

"派谁？"

"你。"

卡卡西看着李武。

"你的感知能力是我们最强的追踪工具。我需要你去砂隐，找到晓组织的位置。"

"好。"

李武接过任务卷轴。

他知道——晓组织的成员是迪达拉和蝎。两个S级叛忍，实力远超普通上忍。

但他不害怕。

因为他已经准备好了。

（本章完）
