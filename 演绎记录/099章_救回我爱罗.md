# 第九十九章：救回我爱罗

蝎死后，四人带着我爱罗回到了砂隐村。

砂隐的医疗班立刻开始治疗。

一尾的查克拉被抽取了大半——我爱罗的生命垂危。但如果及时治疗，还有救。

"需要输血。"

医疗忍者说。

"他的查克拉流失太多了，需要补充。"

"用我的。"

李武说。

"你的？"

"我的查克拉……不对，我的能量和普通人不同。但应该可以补充他的流失。"

医疗忍者犹豫了一下。

"好。"

李武伸出手，掌心贴在我爱罗的胸口。

念气涌入。

暖流从李武的身体涌进我爱罗的经络——填补了流失的查克拉空缺。

十分钟后，我爱罗的脸色恢复了红润。

"他……他活过来了！"

医疗忍者震惊地看着李武。

"你……你用的是什么力量？"

"一种特殊的力量。"

李武收回手。

系统弹出信息：

```
【我爱罗存活确认】
【剧情偏离度：35%】
```

35%的偏离度。

比之前高了十个百分点——因为他救了我爱罗。

原著中，我爱罗在这次事件中死亡（后来被千代用转生术复活）。但现在——他活下来了。

"谢谢你。"

千代走到李武面前。

"你救了我爱罗。"

"不用谢。"

"你……你用的不是查克拉吧？"

李武沉默了一下。

"不是。"

"那是什么？"

"念气。"

千代的表情变了一下。

"念气……我听说过这个词。传说中，有一种比查克拉更古老的力量——来自阿拉德大陆。"

"你……你是从那里来的？"

李武没有回答。

千代也没有追问。

"不管你是从哪里来的——你救了砂隐的风影。砂隐欠你一个人情。"

"好。"

李武转身走出医疗室。

小李跟在后面。

"哥哥……你没事吧？"

"没事。"

"可是你刚才……给了很多能量……"

"休息几天就好了。"

小李点了点头。

两人走在砂隐村的街道上。

夕阳在他们身后，把影子拉得很长。

（本章完）
