# 第一百章：第一觉

回到木叶后，李武开始准备第一次觉醒。

等级已经到了Lv.48。距离Lv.50还差2级。

他需要再升2级，才能触发一觉任务。

系统弹出信息：

```
【当前等级：Lv.48】
【距离一觉：Lv.50】
【建议：完成高难度任务加速升级】
```

高难度任务。

他去找卡卡西。

"有什么高难度的任务吗？"

"有。"

卡卡西递过来一张任务卷轴。

"S级任务。调查晓组织的据点。"

"S级？"

"嗯。晓组织最近在雨隐村附近活动频繁。我需要你去调查他们的目的。"

"好。"

李武接过卷轴。

他知道——雨隐村是长门和小南的地盘。晓组织的真正总部就在那里。

但他不准备现在去总部。他只需要调查外围的情报——确认晓组织的下一步行动。

三天后，他出发了。

一个人。

因为S级任务不需要队友——队友只会成为累赘。

他用念气感知追踪晓组织的痕迹——查克拉残留、脚印、还有……某种特殊的能量波动。

那种波动很奇怪——不是查克拉，也不是念气。是另一种东西。更古老，更原始。

轮回眼。

长门的轮回眼。

他追踪着波动，来到了雨隐村的边界。

然后——他感知到了一个异常。

前方一公里处，有两个强大的波动。一个很"重"——像一座山。另一个很"轻"——像一片羽毛。

长门和小南。

他没有继续前进。

因为他知道——现在的他，还打不过长门。

但他做了一件事——用念气感知"记录"了长门的波动特征。

以后——他会回来的。

```
【等级提升：Lv.48 → Lv.50】
【一觉任务激活】
【请进入梦境位面完成觉醒试炼】
```

他闭上眼睛。

梦境位面展开。

（本章完）
