# 火影忍者时间线（木叶历·官方设定版）

> 根据官方设定书《阵之书》《临之书》整理
> 第四次忍界大战结束时距离木叶村建立约 67 年
> 鸣人出生定为**木叶 51 年**，四战大致在**木叶 68 年**前后

---

## 纪元说明

| 纪年 | 说明 |
|------|------|
| 木叶历前 | 大筒木降临至战国时代 |
| 木叶元年 | 木叶隐村建立 |
| 木叶 51 年 | 鸣人出生（主线基准） |
| 木叶 68 年 | 第四次忍界大战 |
| 木叶 70 年 + | 博人传时代 |

## 附录 D：木叶历换算说明

| 纪年方式 | 换算公式 |
|---------|---------|
| 木叶历 → 鸣人出生前 | 鸣人前 = 54 - 木叶历 |
| 鸣人出生前 → 木叶历 | 木叶历 = 54 - 鸣人前 |

**关键年份对照：**
- 木叶元年 = 鸣人出生前 53 年（建村）
- 木叶 51 年 = 鸣人出生年（九尾之乱）
- 木叶 54 年 = 鸣人 0 岁
- 木叶 62 年 = 主线开始（鸣人 12 岁毕业）
- 木叶 68 年 = 四战结束（鸣人 17 岁）
- 木叶 88 年 = 鸣人任七代火影
