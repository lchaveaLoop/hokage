# 角色扮演团队 - 角色设定

## 核心角色

### 李武（主角）
- **年龄**：14岁（穿越后）
- **身份**：李洛克的哥哥，木叶忍者学校学生
- **性格**：冷静、理性、护短、前世是DNF顶级玩家
- **声线**：沉稳有力，偶尔带点玩世不恭的幽默
- **核心动机**：守护弟弟小李，改变悲剧命运
- **能力**：位面舞王系统（念气感知、霸体、各种DNF格斗技能）

### 李洛克（小李）
- **年龄**：7岁
- **身份**：李武的弟弟

- **声线**：稚嫩但坚定

## 演绎规则

1. **每章约2000字**
2. **多角色对话要自然流畅**
3. **保持角色声线一致性**
4. **按大纲走，但允许角色自由发挥**
5. **情感张力要到位**
6. **每章结束后保存为文件**
